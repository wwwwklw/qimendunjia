# 官司诉讼预测

## 定义与基本属性

官司诉讼预测是通过奇门遁甲判断诉讼双方的强弱、案件走势、判决结果以及律师水平等问题。以值符为原告、天乙（值符下临地盘之星）为被告，开门为法官，惊门为律师，景门为诉状——形成了一个完整的诉讼角色体系。

> **核心原则**：值符为原告主动方，天乙为被告被动方。开门为审判者，惊门为辩论，景门为文书。各方生克关系决定诉讼走向。

## 核心用神

### 诉讼双方
| 用神 | 含义 | 特征 |
|------|------|------|
| [[值符（八神）|值符]] | **原告** | 起诉方、主动方、进攻方 |
| [[天乙]] | **被告** | 被诉方、被动方、防守方 |

> **天乙说明**：天乙 = 值符落宫地盘之星。值符所临之宫的地盘星就是天乙。

### 辅助用神

## 取用神方法

### 第一步：确定双方用神
```
值符落宫 = 原告方
天乙落宫 = 被告方（值符下临地盘之星）
```

### 第二步：判断力量对比
1. **[[开门]]生[[值符（八神）|值符]]**：法官倾向原告（▲原告有利）
2. **[[开门]]生[[天乙]]**：法官倾向被告（▲被告有利）
3. **[[值符（八神）|值符]]克[[天乙]]**：原告占上风，可能胜诉
4. **[[天乙]]克[[值符（八神）|值符]]**：被告占上风，可能反诉成功

### 第三步：看律师能力
1. **[[惊门]]生何方**：律师对谁有利
2. **[[惊门]]旺相且临吉星**：律师能力强
3. **[[惊门]]休囚且临凶星**：律师能力差

### 取用流程
```
排盘 → 定位值符 → 定位天乙 → 对比双方生克 
     → 看开门偏向 → 看惊门律师 → 看景门文书 
     → 综合判断胜负应期
```

## 判断要点

### 胜负判断
1. **[[值符（八神）|值符]]旺相，[[天乙]]休囚**：原告赢面大
2. **[[天乙]]旺相，[[值符（八神）|值符]]休囚**：被告赢面大
3. **双方比和**：和解可能性高
4. **[[开门]]生何方**：法官偏袒何方

### 律师判断
1. **[[惊门]]生值符**：原告律师得力（▲原告有利）
2. **[[惊门]]生天乙**：被告律师得力（▲被告有利）
3. **[[惊门]]克何方**：律师对何方不利
4. **[[惊门]]临[[死门]]**：律师无能，帮倒忙

### 诉状/文书判断
1. **[[景门]]生值符**：原告诉状有力，证据充分
2. **[[景门]]生天乙**：被告答辩有利
3. **[[景门]]临[[杜门]]**：证据不足，文书有漏洞
4. **[[景门]]旺相**：证据链条完整

### 诉讼走向判断
1. **[[值符（八神）|值符]]临[[开门]]**：原告公开起诉，案件受理顺利
2. **[[天乙]]临[[开门]]**：被告主动应诉
3. **双方临[[伤门]]**：诉讼伴随强制措施（查封扣押）
4. **[[六合]]生合双方**：调解和解可能
5. **双方临[[死门]]**：案件漫长僵持

### 判决应期
1. **[[开门]]旺相之日**：开庭/判决日
2. **值符/天乙被克之日**：败诉方失利日
3. **[[庚格定应期|庚格]]出现**：诉讼转折时间点

## 实占要领

1. **值符为原告**：打官司必先定位值符（原告）和天乙（被告），这是双方对立的两个核心。
2. **开门看法官**：开庭前的研判——看开门偏向哪一方。
3. **惊门看律师**：律师好坏直接影响诉讼结果，不可忽视。
4. **景门看证据**：诉状和证据是诉讼的基础，景门不利则证据不足。
5. **生克判胜负**：谁旺相谁有优势，谁被克谁可能败诉——简单直白。
6. **六合看和解**：六合（合同、和解）生合双方则有可能庭外和解。
7. **注意角色**：如果是自己当原告，值符就是自己；如果是被告，天乙才是自己——求测人的立场要清楚。

## 开悟之门补充

以下案例引自《开悟之门》，原案例来自《神奇之门》书中实例，张志春老师用以讲解官司诉讼/劳动纠纷中主客策略的运用。

### 开悟之门补充：劳动纠纷案例——主客策略的经典运用

《神奇之门》实例六杜新会测劳动纠纷：李师傅首先主动上告了工厂领导（为**客**，主动进攻方）。经过预测判断**原告必胜**，被告虽然当时气势汹汹，但必然会泄气撤火。

基于此判断，为李师傅制定的策略却并非继续进攻，而是让他转为**被动策略**——等待对方来找自己，且要**见好就收、适可而止**。结果工厂不仅没有处分李师傅，反而**答应了他的全部条件**，连过去遗留长期没有解决的问题也给解决了。

> **核心决策智慧**：预测准确之后的决策更加关键。虽然预测原告必胜，但策略上选择"转客为主"、以被动应对主动，让对方自己退让——这种"以退为进"的策略比一味进攻效果更好。"该进则进，该退则退"，预测与决策配合才能取得最佳效果。

**奇门中主客判断的九条原则**（《开悟之门》总结）：
1. 动者为客，静者为主
2. 先动为客，后动为主
3. 积极主动为客，消极被动为主
4. 进攻为客，防守为主
5. 远者为客，近者为主
6. 外者为客，内者为主
7. 天盘三奇六仪为客，地盘为主
8. 甲、乙、丙、丁、戊五阳时利客；己、庚、辛、壬、癸五阴时利主
9. 星门反吟局利客，星门伏吟局利主

## 开悟之门补充

以下案例引自《开悟之门》，原案例来自《神奇之门》实例，张志春老师以此讲解诉讼预测中主客策略的决策方法。

### 开悟之门案例：测劳动纠纷（实例六——杜新会测李师傅）

- **时间干支**：见《神奇之门》第359-362页
- **奇门格局**：主客格局；李师傅首先主动上告了工厂领导
- **判断分析**：经过预测，原告必胜，被告虽然当时气势汹汹，但必然会泄气撤火；于是为李师傅决策，让他采取被动策略——等待对方来找自己，而且要见好就收，适可而止；奇门分主客原则：动者为客、静者为主，先动者为客、后动者为主
- **实际结果**：工厂不仅没有处分李师傅，反而答应了他的全部条件，连过去遗留长期没有解决的问题也给解决了

**要点**：预测后需要决策——"急则从神缓从门，动静先后分主客"；原告必胜时采取被动策略反而更能获全胜；见好就收、适可而止是诉讼决策的重要原则

## 金函玉镜补充

### 占词讼吉凶（卷八）

其法以惊、景二门主之。凡有词讼，看惊门乘旺相之气者，讼不息，景门亦然。若二门入墓，落空亡之宫者，主讼不成。

### 占问罪轻重（卷八）

以甲午辛为罪人，开门为问官。如开门落宫生甲午辛之落宫，问官怜悯不加罪。相比者，罪必轻。相冲克及凶格相乘，罪必重。若甲午辛落空亡，得奇门吉格相救者，罪必赦宥。

### 占囚禁（卷八）

以甲午为罪人，辛为天狱，壬为地牢，癸为天网。看日干之落宫下临甲午辛者，主凶禁。甲午辛落宫，下临地盘壬癸者，为误入天牢，待冲破之日，必出。若天盘壬癸之干下临地盘甲午辛者，为天网蒙头，主囚禁。又如天上星仪，落地盘墓库及壬癸者，终不出狱，竟作狱底之鬼。若落空亡之宫者，为空狱，不囚禁。

### 占官司牵连（卷八）

以甲辰壬为螣蛇推之，值符落宫下临甲辰壬、螣蛇，主原告牵连多人。乙奇落宫下临甲辰壬与螣蛇相并者，主被告牵连多人。开门落宫下临甲辰壬、螣蛇者，主问官牵连多人。甲辰壬、螣蛇落空亡之官者，不牵连。若见庚癸大格，纵牵连，不妨。

### 占被审责否（卷八）

以庚格决之。庚金所乘之宫来克值符宫者，主原告被责。克乙奇落宫者，被告受责。克六合宫者，干证受责。若庚金入墓及落空亡之宫，官怒不责。

### 占状词（卷八）

以景门、朱雀为状词。开门为官长，如景、惊二门乘旺相气，再有三奇吉格，主情词恳切。不被开门冲克者，必准。惊、景二门不得奇仪吉格，再与开门冲克者，主官嗔怒，不准；即准，反招罪责。若惊、景二门入墓，主情词不明，不准。惊、景二门落空亡者，主平空捏词，不准。开门入墓及落空亡者，官不动情，不准。

### 占官嘱托（卷八）

以开门为问官，值符为原告，乙奇为被告。值符落宫生开门宫者，主原告有托，乙奇亦然。二宫若被开门冲克，纵嘱托，官必不准。

### 占审官司得理否（卷八）

以开门所落之宫为问官，值符落宫为原告，乙奇落宫为被告，六合落宫为干证。开门落宫生值符者，官向原告，生乙亦然。生六合主听干证之言归结。相生者，得理。相克者，不得理。开门入墓，问官胡涂，审不明白。开门落宫亡，不审。反吟，须换官审。

### 占词讼（卷八）

以值符为原告，值符所落地盘所乘之星为被告。天盘星克地盘星，原告胜。地盘星克天盘星，被告胜。
