# 刑事案件预测

## 定义与基本属性

刑事案件预测是奇门遁甲中用于判断犯罪类型、案犯特征、侦破进展及破案应期的应用。以天蓬星（抢劫杀人）和玄武（小偷）为案犯分类用神，白虎和伤门为公安机关，庚格为破案应期的关键判断依据。

> **核心原则**：天蓬为凶犯（强盗、杀人），玄武为小偷（盗窃）。白虎伤门为公安武装力量。逢庚必变，庚格出现即有破案时机。

## 取用神方法

### 第一步：判断犯罪类型
```
看天蓬星状态 → 旺相且克日干/时干 → 暴力犯罪
看玄武状态   → 临日干/时干     → 盗窃/诈骗
```

### 第二步：判断案犯特征
1. **[[天蓬星]]所临宫位**：案犯所在方位
2. **[[玄武]]所临宫位**：小偷/诈骗犯所在方位
3. **案犯所临八神**：判断案犯身份特征
4. **案犯所临八门**：判断作案手段

### 第三步：判断案件进展
1. **[[白虎]]/[[伤门]]生[[时干]]**：公安在行动，案件有进展
2. **[[白虎]]/[[伤门]]克[[天蓬星]]/[[玄武]]**：公安克制案犯，即将破案
3. **[[天蓬星]]/[[玄武]]生[[白虎]]/[[伤门]]**：案犯自投罗网，或已落网
4. **案犯空亡**：案犯已逃离

### 取用流程
```
排盘 → 看天蓬星/玄武位置 → 判断犯罪类型 
     → 看白虎/伤门位置 → 看公安与案犯生克 
     → 看庚格 → 判断破案应期
```

## 判断要点

### 犯罪性质判断
1. **[[天蓬星]]旺相且临凶门**：重大暴力刑事案件（抢劫杀人）
2. **[[玄武]]旺相且临[[杜门]]**：技术型盗窃（如网络犯罪）
3. **[[天蓬星]]+[[玄武]]同宫**：团伙作案，有组织犯罪
4. **[[天蓬星]]临死门**：案件有人员死亡
5. **[[玄武]]临开门**：案件已公开，受害人多

### 案犯方位判断
1. 天蓬星/玄武落宫方位 = 案犯所在/逃往方向
2. 天蓬星/玄武临[[九天]]：案犯逃往远处
3. 天蓬星/玄武临[[九地]]：案犯潜伏在本地区
4. 天蓬星/玄武空亡：案犯已脱离追查范围

### 侦破进展判断
1. **[[白虎]]克[[天蓬星]]/[[玄武]]**：公安占上风，能破案（▲吉）
2. **[[天蓬星]]/[[玄武]]克[[白虎]]**：案犯嚣张，破案困难（▼凶）
3. **[[伤门]]生日干**：公安主动出击，对求测人有利
4. **[[开门]]临白虎**：案件已公开侦办
5. **[[开门]]临伤门**：公开抓捕行动

### 破案应期判断（庚格）
1. **[[庚格定应期|庚格]]出现**：破案的最强时间信号
2. **白虎克天蓬/玄武之日**：抓捕成功日
3. **天蓬/玄武填实空亡之时**：案犯现身日
4. **马星到白虎宫**：抓捕行动日

## 实占要领

1. **天蓬武玄分清楚**：天蓬星是暴力犯罪（杀人抢劫），玄武是盗窃诈骗——定准性质后再展开判断。
2. **白虎伤门为公安**：公安的行动力量看白虎（威慑力）和伤门（行动力）。
3. **生克定胜负**：白虎能克案犯则能破案，案犯克白虎则破案困难。
4. **庚格看破案**：[[庚格定应期|庚格]]是刑事案件预测的"金钥匙"——逢庚必破。
5. **开门看公开程度**：案件是秘密侦办还是已对公众公开，看开门临何处。
6. **空亡看逃脱**：案犯空亡 = 暂时脱逃，填实之日是抓捕时机。
7. **八神看身份**：案犯所临八神可辅助判断其身份（如太阴为隐秘、螣蛇为虚诈）。

## 开悟之门补充

以下案例引自《开悟之门》，原案例来自《神奇之门》实例，张志春老师以此讲解刑事案件预测中内外盘断远近、用神旺衰、庚格定应期等核心方法。

### 开悟之门案例：测北京抢银行大案（实例四）

- **时间干支**：1996年6月3日上午8点多发案，丙子年癸巳月辛未日壬辰时，阳3局，甲申旬（午未空）
- **奇门格局**：年干丙在8宫，月干癸在3宫，日干辛在1宫，时干壬在6宫；天蓬星落8宫（案犯）；白虎和值使死门为公安，分别落在3、4宫；兑7宫庚+壬时格；坤2宫庚上之干为丁；马星寅在艮8宫
- **判断分析**：内盘、外盘和日干、时干一内一外，确定不会很快破案，慢断年月，但因有庚+壬时格又主快，综合断月；天蓬星落8宫为土在巳月为相地（案犯尚强）；白虎在3宫、死门在4宫，月令巳火，3/4宫木处休地，但死门在4宫对应辰巳月当令，公安仍能克案犯；值使死门落巽4宫含3、4、5、8四个数，断至少三个月后破案；按辛未日为阴日，"阴日看庚上之干"，坤2宫庚上之干为丁，应断当年丁月——丁酉月（农历八月）
- **实际结果**：6月3日发案，9月9日（刚入丁酉月）破了案，三个月多一点

**要点**：内外盘定快慢——内盘主快外盘主慢；庚格出现提示可破但需参考其他条件；"阳日看庚下之干，阴日看庚上之干"定破案应期；值使门落宫数判断大致期限

## 金函玉镜补充

### 占捕捉（卷七）

以六合为逃人，以伤门为捕人（全以八门为主，以八门为人盘，主人事也）。如六合宫克伤门宫，不可得，反此易得。伤门宫生六合宫，捕人不实力，必受其赂。伤门、六合同宫，两相通同。值年月日时格，则可获，天网低亦然。

### 占失物（卷九）

以日干落宫为失主，时干落宫为失物，各以类推之。看时干落宫乘旺相气来生日干落宫者，仍得。反吟者，亦得。落空亡、墓、绝之宫，不得。

各宫所主之物：干为金锁、宝物、铜铁、圆圈之物，又为帽缨、为马。坎为水晶、珍珠、笔墨、毛发细软之物，又为猪。艮为山、玉石、器皿、蹬靴之物，又为牛、犬、猫。震为车、船、木器、碧色衣服之物，又为驴、骡。巽为丝紬、缎布细软之物，又为彩色细长成队之物。离为文明图书、手卷、字画、印信、文劵、彩禽、暖衣之物，又为马。坤为铜铁、鼓磬、釜中空有声、象牙之物，又为牛羊。兑为金银、首饰、有口对衿之物，又为羊鸡、飞禽。看其有气，便为活生之物，无气即为死囚废旧之物。

### 占何人盗（卷九）

以蓬元为盗主，神乘旺相气，又得奇门吉格者，乃是贵人为盗。不乘旺相气，不得奇门吉格者，乃是小人为盗。仍配卦以决之：干为老阳，震为长、壮，坎为中男、始壮，艮为少男、童年，坤为老妇，巽为长女，离中女，兑少女。在内为亲近之人，在外为他人。

### 占捕盗贼（卷九）

刦人财物者，谓之盗；杀人取财者，谓之贼。以天蓬为大贼，玄武为小贼，勾陈为捕盗之人，杜门为捕获之方，皆以天盘主之。勾陈落宫克蓬元之落宫者，捕之必获。若蓬元克勾陈之宫，主贼旺，捕人不敢获。蓬元与勾陈宫比和者，必捕人通同为盗。勾陈、蓬、元同宫，必捕人为盗。蓬、元宫生勾陈宫者，捕人受贿，纵之不捕。总以庚格主之，年格年获，月格月获，日格日获，时格时获，不格不获，杜门有格必获，否亦不获。
