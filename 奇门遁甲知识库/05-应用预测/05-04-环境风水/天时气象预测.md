# 天时气象预测

## 定义与基本属性

天时气象预测是通过奇门遁甲判断当前和未来的天气状况，包括晴雨、风向、温度、云雾雷电等。奇门遁甲用不同的九星来代表不同的气象现象——天辅为风、天英为火晴、天柱与天蓬为雨。这是奇门遁甲在农事、出行、军事等活动中的重要辅助应用。

> **核心原则**：晴天看天英星（火），雨天看天蓬星和天柱星（水）。旺相者气盛，所主天气明显；休囚者气衰，所主天气不显。

## 取用神方法

### 晴天判断
1. **看[[天英星]]**：天英星旺相 -> 晴天、热天
2. **看[[九天]]**：临九天 -> 晴空万里
3. **看[[景门]]**：旺相 -> 天气晴好、热

### 雨天判断
1. **看[[天蓬星]]和[[天柱星]]**：旺相 -> 有雨
2. **看[[休门]]**：旺相 -> 多雨
3. **天蓬+天柱同旺** -> 大雨

### 取用流程
```
排盘 → 看天英星旺衰（晴） → 看天蓬星/天柱星旺衰（雨） 
     → 九天/九地看高度 → 八门辅助判断 
     → 综合判断当前和近期天气
```

## 判断要点

### 晴天判断要点
1. **[[天英星]]旺相**：天气晴朗，阳光强烈
2. **天英星临[[景门]]**：晴热天气，高温
3. **天英星临[[九天]]**：万里无云
4. **天英星休囚且被水星克制**：晴转多云或阴

### 雨天判断要点
1. **[[天蓬星]]旺相**：有持续降雨，阴雨连绵
2. **[[天柱星]]旺相**：有大暴雨，降水强度大
3. **天蓬星+天柱星同旺**：暴雨倾盆
4. **天蓬星临[[休门]]**：阴雨天气，适合居家休息
5. **天蓬星休囚且被火星克制**：雨云消散，即将停雨

### 风天判断
1. **[[天辅星]]旺相**：有风，旺相越强风力越大
2. **天辅星临[[伤门]]**：狂风，破坏性强
3. **天辅星临[[白虎]]**：暴风，有灾害可能
4. **天辅星休囚**：微风或无风

### 极端天气判断
1. **[[白虎]]+天柱/天蓬旺相**：暴风雨预警
2. **[[天冲星]]旺相**：雷电天气
3. **[[天芮星]]旺相且临九地**：大雾天气
4. **多星同旺**：天气变化剧烈

### 天气变化时间
- 看旺相之星转入休囚的时间（降雨/晴天转）
- [[反吟]]格局：天气变化快
- [[伏吟]]格局：天气持续不变

## 实占要领

1. **晴看天英**：天气好不好，先看天英星——天英旺则晴，天英囚则阴。
2. **雨看天蓬天柱**：有没有雨，看天蓬星和天柱星——两个水星旺相就有雨。
3. **旺衰定程度**：旺相 = 所主天气明显且强烈；休囚 = 所主天气不明显甚至不存在。
4. **九天九地看高度**：九天主高空晴、九地主低空阴——补充判断云量。
5. **五行的生克关系**：火旺克水则无雨，水旺克火则无晴。
6. **反吟速变**：[[反吟]]格局下天气变化极快，说变就变。
7. **伏吟不变**：[[伏吟]]格局下天气持续不变，预报准确度高。
8. **星宫同看**：单看九星不够，要结合所临宫位的五行生克综合判断。

### 晴雨占——九宫诀

> 占验雨晴论天乙，须究五行起元例，得令失令与刑格，生尅制伏阴晴别，太阴元武司阴雨，六合白虎雷风生，朱雀蛇辖雷雹虹，勾陈阴雾曰霾取，日时月格刑悖逢，阴阳星宫雨旸踪，九天九地奇仪到，阳开阴阖究玄宗，雨占奇仪化合神，方位时刻可详料，白虎到巽六合干，旺相风生休废烟，朱雀坎位武九，主胜客气雨必然，惟有太阴并九地，或晴或雨论宫缠。

### 晴雨占——值符诀

> 水局主雨金局寒，门金符水教出潭，若是金符门到水，只疑飞龙入九渊，土局阴霾未有涯，时或四野起黄沙，冬深符土门金冻，门土符金飞六花，符门之间火木局，天地晴和寔可夸，木局风生火局晴，春夏逢之断得灵，惟惧囚墓废时日，反致风停细雨零。

### 晴雨占——值使诀

> 值使各逐本宫求，旺气逢生从类搜，刑制隔害吟反伏，阴晴变易有来由，休囚死废无定据，纵有风雨霎时收。

### 晴雨占——干支诀

> 日干为天时为地，纳音五行阴阳帝，天尅地兮风生林，地尅天兮空雾气，天地生合论化神（如甲己化土之类），金水甘露天未霁，细雨纳音生尅情，五行时令消息意，假如合水纳是金，管教刻下风雨至。

### 晴雨占总诀

**占晴**须论符、天、朱、蛇、勾、白之神，与八门生、景、死、杜、辅、禽、英、芮，并起元于火木局，三奇得位，戊己当权，阳星开必主晴。

**占雨**须看地、阴、元、白之神，与八门之休、惊、开、伤，九星之蓬、柱、心、冲，并起元于金水局，三奇虚墓，庚辛壬癸当权，阴星阖必阴雨也。
