# 地理环境预测

## 定义与基本属性

地理环境预测（风水预测）是奇门遁甲中判断住宅（阳宅）和墓地（阴宅）吉凶的应用。以日干为人、时干为宅为核心，生门为阳宅房屋、死门为地皮（兼阴宅用神）。值符代表新宅，值使代表旧宅。阴宅预测则以死门为核心，天地盘分死者与生者。

> **核心原则**：人宅须相合——日干（人）与时干（宅）相生为吉，相克为凶。生门看房屋吉凶，死门看地皮吉凶（阳宅）或阴宅吉凶（阴宅）。

## 取用神方法

### 阳宅预测取用
```
排盘 → 定位日干落宫（人） → 定位时干落宫（宅） 
     → 看人宫与宅宫生克 → 看生门落宫（房屋） 
     → 看值符（新宅）/值使（旧宅） → 综合判断
```

### 阴宅预测取用
```
排盘 → 定位死门落宫 → 看死门旺衰 
     → 天盘干=生者 → 地盘干=死者 
     → 看天地盘生克 → 综合判断
```

## 判断要点

### 阳宅吉凶判断

#### 人宅关系（核心）
1. **[[时干]]（宅）生[[日干]]（人）**：住宅利人，居住者好运（★★★★★ 最佳）
2. **[[日干]]（人）生[[时干]]（宅）**：居住者付出，但可居住
3. **时干与日干比和**：人与住宅协调，居住安稳（▲吉）
4. **[[日干]]克[[时干]]**：人克宅，居住者不服环境
5. **[[时干]]克[[日干]]**：宅克人，住宅对居住者不利（▼凶）

#### 房屋质量判断
1. **[[生门]]旺相**：房屋质量好，风水佳
2. **[[生门]]休囚**：房屋一般或老旧
3. **[[生门]]空亡**：房屋虚空，不适合居住
4. **[[生门]]临[[死门]]**：房屋不吉，阴气重
5. **[[生门]]临[[天芮星]]**：房屋有病气，可能影响健康
6. **[[生门]]临[[白虎]]**：有煞气冲射

#### 新宅旧宅
1. **[[值符（八神）|值符]]旺相生合日干**：新房适合居住（▲吉）
2. **[[值符与值使|值使]]旺相生合日干**：旧房适合居住（▲吉）
3. **值符新宅临[[九天]]**：新宅位置高、阳气足
4. **值使旧宅临[[九地]]**：旧宅位置低、有阴气
5. **值符克日干**：新房不宜购买/入住

#### 地皮判断
1. **[[死门]]为地皮**（注意：死门在阳宅预测中只代表地皮，不代表死亡）
2. **死门旺相且临吉门**：地皮好，适合建房
3. **死门临凶门**：地段不好，不宜开发
4. **死门临[[生门]]**：土地有利润（兼看求财）

### 阴宅吉凶判断

1. **天盘干（生者）生地盘干（死者）**：生者孝顺，阴宅福泽后人（▲吉）
2. **地盘干（死者）生天盘干（生者）**：祖宗荫庇，后人受益（▲吉）
3. **天盘克地盘**：生者遗忘先人，后世不利
4. **地盘克天盘**：坟墓不吉，可能对后人不利（▼凶）
5. **[[死门]]旺相且临吉门**：阴宅风水好
6. **[[死门]]休囚且临凶门**：阴宅风水差，需迁坟
7. **死门临[[九天]]**：坟地在高处
8. **死门临[[九地]]**：坟地在低处

### 选房/选址
1. **[[开门]]所在方位**：适合作为大门朝向
2. **[[生门]]所在方位**：适合作为主要活动区域
3. **[[休门]]所在方位**：适合作为卧室
4. **[[白虎]]所在方位**：有煞气，不适合做大门

## 实占要领

1. **人宅关系是根本**：时干（宅）与日干（人）的生克关系是第一判断基准——宅生人则吉，宅克人则凶。
2. **阳宅看生门**：生门是阳宅房屋的核心用神，旺衰决定房屋质量好坏。
3. **地皮看死门**：死门在阳宅预测中专看地皮（宅基地），不要误解为死亡。
4. **新旧看值符值使**：值符新宅、值使旧宅——买房看值符，旧居看值使。
5. **生克方向不可反**：宅生人为吉（住宅给人添福），人克宅为掌控（但费心费力）。
6. **煞气看白虎**：任何用神临白虎都要考虑是否有形煞冲射。
7. **阴宅看死门**：阴宅预测的核心用神是死门，天地盘的生克决定祖坟对后人的影响。
8. **天地盘分生死**：天盘为生者（活在世上的人），地盘为死者（已故祖先）。

#### 十干诀

> 十干查分所主，莫教鬼尅入关拦，尤嫌坐临忌煞位，此地当年拟废间，甲乙兴隆左脉威，制主须教林木芟，庚辛眠外无惊险，昂首冲空右脉残，丙丁得令跳峯秀，一有瑕疵口舌关，壬癸得元来脉厚，枯涩休囚后裔难，戊己球毡胎息地，须要生煞入其间，细查何干孤尅缺，堪断其方缺废残，子父财官区以别，富贵儿孙一局颁。

#### 九宫诀

> 九宫起元论山祖，元局不孤纔可许，若有一局坠旬空，化气不真非为美，即令全局总无伤，再将本局细参取，起从何宫飞向何，穿田过峡须寻迤，更看归局历几辰，详考变胎可脱体，倘得节次入生乡，此是佳城天付与，上局得令左偏宜，中元依中下右取，应从节次别元情，得令失令因时举。

#### 门户诀

> 门户休囚易得征，只看阴阳贵人星，阳吉星临门发早，阴吉星临户后兴，门吉户凶门渐替，户吉门凶先拂情，若见门户全凶吉，终始如类说前程。
