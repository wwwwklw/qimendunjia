# 晴雨占法

## 定义与基本属性

晴雨占法是通过奇门遁甲判断天气变化、降雨降雪、雷电风暴等天时现象的应用。以天禽（司命）、天柱（雨师）、天冲（雷神）、天辅（风伯）、天蓬（水神）及壬癸水干为核心用神。

> **核心原则**：天禽临旺相，天柱带壬癸游于一三七宫，则大雨。诸星落空亡墓库则阴雾不雨。冬月看天心星及天任带壬癸则雪。

## 金函玉镜补充

### 占雪（卷七）

以干、兑二宫为主。天心宫即干，天心星乘壬癸二干到兑，或天柱、兑宫乘壬癸二干到干，皆主雪。如以落宫所得之干，地盘宫中日干以定其期。

### 占河水消涨（卷九）

以天蓬、休门决之。休门乘旺相气而得三奇者，水虽涨不至泛滥。休门乘旺相气而值庚格者，主河水壅塞，泛滥涌涨。再看有甲辰壬带螣蛇，主孽龙舞水为害。休门落于二五八宫，水被土克，主立消或无水。休门不乘庚格与甲辰壬者，水虽旺发，亦不出岸。

### 占雨（卷十）

以天禽星为司命之主，天柱为雨师，天冲为雷神，天辅为风伯，螣蛇为雷光，天蓬为水神，甲辰壬为龙神。

天禽临旺相之宫，天柱、甲辰壬游于一、三、七宫，或下临甲辰壬，主大雨。带甲寅癸下临甲寅癸于一、三、七宫，主小雨。甲辰壬加临震宫为龙登雷门，主雷雨。丁临地盘螣蛇，主闪电。乘凶门临地盘螣蛇为龙雷发蛟，震怪。

落空亡、墓库宫，主阴雾不雨。

冬月看天心星及天任土宿，带壬癸，下临壬癸，或乘玄武、白虎，主水土凝结而为雪。空亡、入墓，主阴寒。

更看白虎猖狂，主大风。青龙逃走，主风起云散，无雨。白入荧主冰雹。荧入白主晴。

若天柱游于一、三、七宫下带壬癸，或下临壬癸之宫，不见壬癸，而见螣蛇者，主现虹霓不雨。如落宫下临朱雀，或落空亡者，主亢旱不雨，强求反有火灾。

## 奇门宝鉴补充

> 出处：《御定奇门遁甲·奇门宝鉴》（金志文点校）卷五·元机赋上「占风雨法」 — 参见 [[御定奇门遁甲奇门宝鉴（金志文）]]

### 占风雨总纲

占天时之风雨，以**六甲直符宫**之生克为本方之晴雨。各方晴雨，看各方之生克。

> 风雨初寻六甲，各方得气生死。次将师伯细分，凌犯交冲起止。

**占断次第**：先以六甲直符所到之宫审其得地不得地，推风雨之有无；次以五符加本日干禄上顺行，看风伯、雷公、雨师、风云所到之宫，与天盘直符时干两宫或凌犯或交冲者，定风雨起止。

### 十二神将占风雨

查本日日干禄位上起五符，顺行天曹、地府于十二宫，视风伯所到之方主有风，雨师所到之方主有雨。

|------|------|
| 五符 | 宜出兵大胜，耀武扬威；商贾竖造迁改婚嫁大利 |
| 天曹 | 宜上章奏书；从此方来传报者多为奸计，当防之 |
| 地府 | 宜埋藏固守，设伏暗计；不宜出兵谋为 |
| 风伯 | 出入忧惶；若此方有风对面而来，出兵大败 |

| 雨师 | 主有阴雨，宜穿井挖河取水；出兵恐遭暴雨涨泛之险 |

### 凌犯交冲之法

其法于**地盘日干禄位**起五符顺行，看其凌犯交冲，以定风雨。

- **同宫曰「凌犯」**：凌轻犯重
- **对宫曰「交冲」**：逢冲即散，晴变为雨，雨变为晴（即[[反吟]]也）
- **同宫相克曰「战」**：如甲子加甲午、甲午加甲申、甲申加甲寅、甲寅加甲戌、甲戌加甲子

> 风雨看甲，次看师伯。凌犯战冲，成气重重。**三四主雨，一二行风**。

十干禄位：甲禄寅、乙禄卯、丙戊禄己、丁巳禄午、庚禄申、辛禄酉、壬禄亥、癸禄子。

### 水入龙宫占雨

> 子癸临巽，水入龙宫。符水加辰，旺水临龙。龙临大海，在于一宫。速速雨霁，无甲亦同。子癸临申，用之多功。

- 辰属巽宫，甲子戊、甲寅癸到巽宫为「水入龙宫」，又为水入库中，**无雨泽**
- 直符子癸加于甲辰之上，为「旺水入龙宫」
- 甲辰直符加于坎一宫，为「龙归大海」，**主无雨，天色立时晴朗**
- 子癸临坤宫申地及甲申上，为水到长生之地，**主风雨**

### 风雨变动占君臣民

凡风雨变动，当查审日期，以**甲己入元日**为始。

| 六甲加日 | 所占对象 |
|---------|---------|
| 子午卯酉日有变 | 占在君 |
| 辰戌丑未日有变 | 占在臣 |
| 寅申巳亥日有变 | 占在民 |

- 有狂风：主怨怒
- 有暴雨：主悲泣
- 风雨凄惨：主阴谋叛逆
- 暴雷冰雹、拔屋折木：主暴兵四起，关梁阻塞，饥馑流亡
