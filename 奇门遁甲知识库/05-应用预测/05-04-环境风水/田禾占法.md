# 田禾占法

## 定义与基本属性

田禾占法是通过奇门遁甲判断农作物种植、收成丰歉的应用。以九星配八门代表不同作物，乘旺相得三奇吉格则丰收，休囚空亡则歉收。

> **核心原则**：九星各有所主之谷，乘旺相气得三奇吉门者大收，不得吉格者薄收，空亡入墓者歉收。

## 金函玉镜补充

### 占布种五谷（卷九）

以九星所乘之门以类推之。乘旺相气而得三奇吉门者：

| 星门 | 作物 |
|------|------|
| 开门、天心 | 麦田 |
| 伤门、天冲 | 稻谷 |
| 杜门、天辅 | 麦稷，又为棉花 |
| 景门、天英 | 高梁 |
| 休门、天蓬 | 豆田 |
| 死门、天芮 | 荞麦，又为蒜 |

各门各星乘旺相气带三奇吉格者，大收。乘旺相气不得三奇吉格者，薄收。九星内有被克，或值空亡、入墓，而得凶格者，歉收，不必种。

### 占走失六畜（卷九）

驴、骡、车、船以伤门、天冲主之。牛、羊以死门主之。看落天盘何宫，即往何方寻之。马以干、兑宫所得之星主之，若得天蓬星，当于近水处寻之，如乘玄武为人盗去，再看得之之日必逢庚格。

## 奇门法窍补充

> **source:** [[奇门法窍（锡孟父子）]]
>
> 本节出自《奇门法窍》卷四「占禾稼」「占种植」「占蚕」「占蝗蝻」「占岁中丰歉」「占果木结实」「占牛马怀驹牝牡」等章节，系（飞盘体系）飞盘奇门对田禾占法的独特论述。

### 占禾稼（飞盘体系）

以**天任**落艮震宫主丰，不落者为歉。以**贵神月将**分旱涝——**伤门**为麦，**生门**为谷；蛇雀巳午为旱，传送天后为涝。

### 占种植（飞盘体系）

以**天冲**为花木，以**天辅**为树木，各看落于旺相之宫为生，休囚为死；相生者活，相克者死。

### 占蚕（飞盘体系）

以**三奇三吉门吉星**落于金木之宫为吉，下克上为良；不到金木之宫而克门者不利。

### 占蝗蝻（飞盘体系）

以**死伤杜惊**临于**日干**之宫，主有蝗蝻。

### 占岁中丰歉（飞盘体系）

以**立春时刻日建**起看天地二盘以决之，看奇吉落于何宫，以九宫分野定之。

### 占果木结实（飞盘体系）

以**伤门**落宫为树木，**生门**落宫为果实。乘虎为风，乘元为雨，又以旺相生克决之。

### 占牛马怀驹牝牡（飞盘体系）

各以其宫所乘之星推之：

| 六畜 | 宫位 |
|------|------|
| 马 | 离 |
| 牛 | 坤 |
| 骡驴 | 震 |

阳星为牝，阴星为牡。
