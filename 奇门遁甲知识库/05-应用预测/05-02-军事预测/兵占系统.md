# 兵占系统

## 概述

兵占系统是金函玉镜独有的军事预测体系，将奇门遁甲的十干、九星、九宫、八门、八神全部赋予军事角色和职能，形成一套完整的军事决策工具。从排兵布阵到攻守进退、从将领配置到后勤保障，均有对应的符号体系。

核心包括：千金诀、三甲开阖图、禹罡图、兵占十论（值符、值使、十干、九星、九宫、门户、三甲、干支、吉凶）。

## 千金诀

### 人专煞贡起例

四孟月（寅、申、巳、亥）甲子日起妖星
四仲月（子、午、卯、酉）甲子日起惑星
四季月（辰、戌、丑、未）甲子日起禾刀

逐日轮算，周而复始。

九大煞星对应：妖星——元武，惑星——朱雀，禾刀——白虎，煞贡——青龙，直星——金匮，卜木——道符，角己——太阴，人专——金堂，立旱——勾陈。

## 三甲开阖图

三甲即孟甲（甲寅）、仲甲（甲子甲午）、季甲（甲辰甲戌），各分阴阳开阖，决定主客利弊。

### 孟甲
- **内开（甲寅）**：合阳星，阳气在内，利主不利客，利守
- **外阖（甲申）**：合阴星，阴气在外，利主不利客，利守

### 仲甲
- **半开（甲子）**：合阳星，阳气，利主利后应
- **半阖（甲午）**：合阴星，为格，利客不利守

### 季甲
- **俱阖**：甲辰合阳星阳气在外，利客利伤兵；甲戌合阴星阴气在内，利主利迎敌

## 上元经补充

### 飞盘+转盘双视角兵占体系

《奇门遁甲上元经》兵战章的核心突破在于区分了**飞盘奇门**与**转盘奇门**在军事预测中的不同应用场景：

| 体系 | 优势场景 | 劣势 |
|------|---------|------|
| **飞盘奇门** | 短时间内发生/结束的单场战役，能细致刻画地面战斗详情 | 不善于长时预测（排斥时间圆转因素） |
| **转盘奇门** | 历时长久的战争整体走向、海陆空一体化现代战争 | 刻画战役细节不如飞盘 |

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]兵战章

### 敌我定位的系统化方法

#### 一、符庚对阵（阵地战/对攻战）

值符与庚的对立适用于**双方没有明显主客攻守性质的阵地战**。

**庚不永为敌的三种情况**：

| 情况 | 判断 | 示例 |
|------|------|------|
| 印比关系（庚宫生助值符宫） | 庚甲矛盾暂缓，庚不代表直接敌军 | 清朝借英法洋枪队剿太平天国 |
| 明克暗合（三合/五合/六合） | 转克为生，庚失去直接对手资格 | 杨辅清暗助清军打击石达开 |
| 甲申庚为值符 | 值符方内部存在巨大矛盾，外部敌人转为冲制值符宫位者 | — |

#### 二、日时之辩（局部战况）

- **日干** = 我方（正确）
- **时干** ≠ 敌方（错误！时干地位低于日干，战争双方对等无隶属关系）
- **时干** = 当下战局概况（客观描述战争进程，不偏袒任何一方）

日时生克判断：
- 时干生助日干 → 战争进程于我有利
- 时干克制日干 → 我军暂时处于不利阶段

> **注意**：日时关系仅代表战争全局中的一个阶段，其生克不能决定整个战争走向。如曾国藩祁门被围（日被时克）但安庆会战大胜（战略全局胜）。

#### 三、进攻方用神（值符+开门）

- 进攻方为动态 → 用**值符**表征
- 攻城关键在于入城门路 → 观察**开门**

开门与值符关系判断：
- 开门生助值符 → 攻破城池可能性大
- 开门内出现庚加丙（贼必来）→ 破城指日可待
- 开门内戊加庚、乙加辛 → 不利，宜鸣金收兵

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]兵战章

### 飞盘中门战术

飞盘奇门独有的**中门**在兵战中有特殊价值：

**中门的军事意义**：
- 地盘中门 → 紫禁城大门，直通中宫帝王居所
- 天盘中门 → 飞布外八宫，临于哪个宫哪个宫就有直达中宫的通路

**中门出奇制胜的条件**（如邓艾偷渡阴平）：
1. 外八盘中门落宫与中宫有符号联通（两宫皆有庚字等连接）
2. 中门落宫格局与中宫格局必定全部呈现凶象

**中门无涉中宫时**的三种状态：

| 状态 | 含义 |
|------|------|
| 中门落宫被敌军冲制 | 皇命不达，传令官被截 |
| 中门克值使 | 皇帝对当下防守作战不满 |
| 中门克值符 | 将在外见疑于大内 |
| 中门同值符 | 君臣无隙，同心同德 |
| 中门同值使 | 上谕达于将士 |

> **历史案例**：袁崇焕案——中门克值符（崇祯对袁本人成见），虽指挥防守战役胜利（值使），仍不免被害。

### 景惊二门破阵法

破阵时首选景门和惊门作为进攻方向，理由有二：

1. **辨识通路**：景门属火（视觉感知），惊门属金（听觉感知）→ 通过视觉和听觉信息认清阵式机关方位
2. **同声相应**：景门为血光之象（战争流血），惊门为金锣之声（战争喧惊），与战争意象吻合

**各门在破阵中的角色**：

| 门 | 破阵角色 |
|----|----------|
| 景门 | 主攻，首选方向之一 |
| 惊门 | 主攻，首选方向之一 |
| 开门 | 退兵通道 |
| 休门 | 退兵通道（较少激烈战斗）|
| 生门 | 退兵通道 |
| 死门 | 大忌，此路不通 |
| 杜门 | 大忌，此路不通 |
| 伤门 | 辩证：旺相可多杀伤敌军，休囚有伤己危险 |

> **注意**：景惊二门只是门盘系统中的一层，整体胜败还需参看星、神、干、宫等多重因素。

### 玄武、天蓬的军事应用

在军事预测中，传统负能量符号有其独特价值：

**玄武**（谍报活动）：
- 玄武与己方用神关联 → 间谍为己方服务
- 玄武与敌方用神关联 → 间谍刺探我方军情
- 玄武与值符同宫（格局良好）→ 可出奇兵破敌
- 地盘玄武与敌方关联 + 天盘玄武与值符同宫 → 敌方密探已深入我营

**天蓬**（敢死/偷袭部队）：
- 天蓬 = 亡命徒代称，适宜作为担负偷袭、劫营的机动部队
- 敢于冒险、让敌人摸不透战略意图的统兵大将也可用天蓬表征

### 热兵器时代的奇门军事应用

面对洋枪洋炮等热兵器，上元经给出了取类比象的对应方法：
- 天冲星 → 枪炮之利（古为弓矢）
- 丙、丁 → 火器弹药
- 午、戌等支 → 对应炮弹发射

**现代战争的对应用神**：

| 现代战争元素 | 奇门对应 |
|-------------|---------|
| 卫星侦察系统 | 星盘（九星显象之能）|
| 信息战干扰 | 螣蛇、玄武、朱雀（神盘）|
| 空对地导弹攻击 | 天盘干（阳动）v.s. 地盘干（阴静）|
| 飞机轰炸被拦截 | 壬加己、己加壬、戊加辛、辛加戊（暗支相冲格）|
| 导弹袭城被摧毁 | 丙加壬、丁加癸（火水对冲格）|

> **注意**：飞盘奇门在传统地面战刻画上更细腻（因天地同飞模式更贴近地面战斗），而转盘奇门的天盘圆转布局模式更适用于描述现代立体化多维战争类型。

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]兵战章
