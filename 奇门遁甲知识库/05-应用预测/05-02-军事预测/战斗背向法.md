# 战斗背向法

## 定义与基本属性

**战斗背向法**是奇门遁甲中战斗时选择背向方位的法则，共有七法，依次为背太岁大将军、背月建大将、背孤击虚、背游都击鲁都、背天雄击地雌、背生神击死神、背干游击干鲁。

> 出处：《御定奇门遁甲·奇门宝鉴》（金志文点校）卷一·释义四十四则 — 参见 [[御定奇门遁甲奇门宝鉴（金志文点校）]]

## 战斗背向的原则

1. **背**：背向凶方，避开其锋
2. **击**：击其冲方，攻其虚处
3. 凡背凶击吉，百战百胜
4. 七法可单独使用，亦可合并使用
