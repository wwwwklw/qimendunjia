# 军事占法

## 定义与基本属性

军事占法是奇门遁甲的原始应用场景之一。金函玉镜作为军事奇门经典，以天冲（武士）、值符（主帅）、六庚（攻者/敌军）、天禽（守者）为核心用神，通过主客生克、旺相休囚判断军事情势。

> **核心原则**：奇门主客之道，以值符为主帅、六庚为敌军。值符克庚则主胜，庚克值符则客胜。旺相为胜，休囚为负。

## 金函玉镜补充

### 占投军

天冲为武士，值符为主帅。值符宫生天冲宫，及天冲宫生值符宫，一见即投合，如彼此相克，定不收录。天冲若坐值符，一去即为部长，后必大用，伏吟空回，反吟反复不准。

### 占攻城

以六庚为攻者，以天禽为守者（庚者兵众，禽者中心者攻之，故受攻）。如六庚乘旺相，得开门加中五宫，城必破。又看地盘天禽所乘之宫，得旺相及吉门者，其守将不可擒也，反此必死。

### 占守城

以天禽为守者，天蓬、六庚为攻者。天禽宫得休、生、开、景，又旺相有六丙，其城不破。如无旺相及吉门，再犯天蓬、六庚，又中宫，不能守断。

### 占敌兵（盗贼）

敌兵来去先分界限，冬至以后以坎艮震巽为内，离坤兑干四宫为不至。六庚落宫被克者，为安营不稳，自惊而退。六庚克所落之宫，又逢玄武、天蓬、白虎之神，乘必大战。庚得九天，则大张声势，鸣鼓而进。庚得九地，则掩旗息鼓而来。如贼已入境，占其何时去？看六庚在内四宫为不去，在外四宫为去。总以六庚地盘干支年月日时为去来之期。如太白入荧，虽贼来，若在外界（外四宫），亦主不来；荧入太白，虽主贼去，若在内界，亦不去。

### 占贼临境城可守否

以时干宫为客，时支宫为主（即符使之宫），看其生克何如。如时支宫地盘受值符所落之宫克制，而时支又乘六庚、玄武，此城当弃。或值使宫自被下克，此城亦不可居，速宜退避。如值符宫与值使宫相生、相比，或值符宫自受刑克，敌人三来，不能取胜，守之无妨。

### 占胜败

凡战阵以景、惊二门主之。经曰：景门宜破阵。又治乱之法，要视惊门，当以值符所落之宫为主，六庚所落之宫为客。值符宫克六庚之宫，主胜；六庚宫克值符之宫，客胜。又论旺相为胜，休囚为负。如主得惊、景二门，或二门宫与客宫相生则客胜。如主客宫相生，来必讲和；如主客所乘皆旺相，俱得二门，不相刑克，其力相等，则两相恐惧，不战而退。如六庚为值符，是主客同宫，二家不分胜负。又日干加庚主胜，庚加日干客胜。

### 占远信（军情）

以景门为信。歌曰："景上投书并破阵"。景门临外界来迟，临内界为速，上带吉格，信吉；凶格，信凶。门迫、投江水，无信。

### 占武举

以值符为主考，时干为举子，甲申庚为箭，甲午辛为红心，景门为策论。专看甲申庚落宫若克甲午辛之宫，或冲其宫皆为箭中红心，再看景门得旺相，又与值符宫相生者，主中。缺一不中。

### 占船主善恶

以震三宫为船主，以天盘上所得之星，为船主之善恶。震宫上得辅、心、禽三星为上吉，冲、任为中吉，其船家是好人。得英、芮、柱、蓬为大凶，船不可登。

### 行兵杂摘（卷三）

统宗卷三「行兵杂摘」集兵占断验之要，涵盖格局吉凶、主客进退、安营劫营、置阵之法等：

#### 格局吉凶与主客

| 格局 | 主客吉凶 |
|------|---------|
| **天乙飞宫** | 不宜进兵，后应则吉 |
| **白入荧**（庚加丙）| 利主，贼来偷营，即伏兵于北方迎战 |
| **荧入白**（丙加庚）| 利客，遇贼兵即时发兵，少迟则让贼为客 |
| **龙逃走**（乙加辛）| 不利客，宜叛应；贼来偷劫败，自败而返 |
| **雀投江**（丁加癸）| 不利客，宜叛应 |
| **蛇妖矫**（辛加乙）| 不利主，宜先举；不可对敌，宜移帐到戊己方 |
| **虎猖狂**（辛加乙）| 不利主，宜先举；不会合惊开二门，不甚为客 |
| **大格、刑格** | 必败 |
| **六仪击刑** | 最凶，虽值符亦不可用 |
| **三奇入墓** | 百无一成 |
| **格勃**（值符是庚，宜避丙丁之宫）| 非回首也 |
| **回首、跌穴、三诈、九遁** | 最忌下克上 |

#### 反吟伏吟与阴阳时

| 局势 | 用法 |
|------|------|
| **反吟有吉格** | 亦宜进兵，当乘乱破杀 |
| **伏吟** | 宜藏兵暗地，使敌入我伏中，择九天开门值符下出兵吉 |
| **阳时** | 要天盘强 |
| **阴时** | 要地盘强 |
| **时干克日干** | 主下犯上，主胜客 |

#### 天乙飞伏宫

| 格局 | 应对 |
|------|------|
| **天乙飞宫** | 切勿进兵；将军当随天上值符而飞 |
| **天乙伏宫** | 须移幔丙丁方避之；将军当随值符而伏 |

#### 安营劫营

- **奇仪相合**：安营最利
- **立寨**：须看六庚与玄武所临之宫，主有劫营之贼，所遇无非\*细
- **阳时阴神**：遇两重玄武，贼必来偷劫
- **乙加辛**：贼来偷劫败，自败而返；若乙得旺气，此一路必有伏兵
- **丙加壬**：亦有文书奇缠遗失

#### 八门战守

| 八门 | 战守宜忌 |
|------|---------|
| **伤门** | 交战最利 |
| **景门** | 胜而不久 |
| **休门** | 止宜坚守 |
| **六合临主** | 敌有人降，事必成 |

#### 置阵之法

### 统宗版特点

- 统宗卷三「行兵杂摘」是兵占实战断验的集大成，涵盖格局、主客、安营、劫营、八门、置阵六大类
- 格局吉凶论主客进退：白入荧利主、荧入白利客、龙逃走不利客等，与金函玉镜体系一致
- **天乙飞伏宫**为统宗独有：飞宫将军随天上值符而飞，伏宫将军随值符而伏
- **置阵之法**以日干五行定阵形（直锐方圆曲），与元机赋「金圆木直而土方，水曲丙丁而火焰」一致
- **反吟伏吟用兵**：反吟乘乱破杀，伏吟藏兵暗地——统宗独有之兵法
- 「龙回首主兵大利客亦不凶」「虎猖狂不会合惊开二门不甚为客」——统宗对格局力量的精微辨析
