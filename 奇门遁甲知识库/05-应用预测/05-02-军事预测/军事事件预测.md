# 军事事件预测

## 定义与基本属性

军事事件预测是奇门遁甲最原始的用途。上古时期黄帝与蚩尤之战中，九天玄女授奇门遁甲于黄帝，最初就是为了军事战争。因此，军事预测是奇门遁甲的"本行"，所有应用预测皆从此衍生。庚为攻方（兵器）、值符为主方（统帅），九天九地定攻守之势。

> **核心原则**：九天之上好扬兵（居高临下利攻击），九地之下可伏藏（隐匿埋伏利防守）。此乃军事预测的最高准则。

## 取用神方法

### 第一步：分清攻守
```
判断是谁先发动战争 → 先发动者为攻方（庚）
我方 = 值符（主方/统帅方）
敌方 = 庚（进攻方/来犯之敌）
```

### 第二步：看攻守态势
```
[[九天]]临何宫 → 该宫方适合采取攻势
[[九地]]临何宫 → 该宫方适合采取守势
```

### 第三步：判断胜负
```
值符旺相克庚 → 主方胜
庚旺相克值符 → 攻方胜
双方比和 → 相持不下
```

### 取用流程
```
排盘 → 定位值符（主方/我方） → 定位庚（攻方/敌方） 
     → 看九天九地落宫 → 定攻守策略 
     → 看双方生克旺衰 → 判断胜负 
     → 看八门八神 → 综合判断战术细节
```

## 判断要点

### 攻守策略判断（九天九地）
| 用神临 | 策略 | 古文 |
|--------|------|------|
| [[九天]] | **采取攻势**，主动出击 | "九天之上好扬兵" |
| [[九地]] | **采取守势**，埋伏防守 | "九地之下可伏藏" |

> **注意**：九天和九地不是按主客方分的，而是按落宫判断——九天在哪个方位，哪个方位适合主动进攻。

### 胜负判断
1. **[[值符（八神）|值符]]旺相，[[庚（阻隔之神）|庚]]休囚**：主方占优，能取胜
2. **[[庚（阻隔之神）|庚]]旺相，[[值符（八神）|值符]]休囚**：敌攻方势大，主方危险
3. **[[值符（八神）|值符]]克[[庚（阻隔之神）|庚]]**：主方能克制敌方（▲主方胜）
4. **[[庚（阻隔之神）|庚]]克[[值符（八神）|值符]]**：敌方克制主方（▼主方败）
5. **[[庚（阻隔之神）|庚]]临空亡**：敌军虚张声势，实力不强
6. **[[值符（八神）|值符]]临空亡**：主方准备不足，不宜应战

### 战术细节判断
1. **[[伤门]]临白虎且克敌方**：打歼灭战的好时机
2. **[[杜门]]旺相**：防御工事坚固
3. **[[太阴]]临九天**：适合夜袭
4. **[[六合]]生合己方**：有盟军支援
5. **[[腾蛇|螣蛇]]临庚**：敌军使诈，不要轻信
6. **[[景门]]临九天**：情报准确，可行动
7. **[[景门]]临螣蛇**：假情报，不要上当
8. **[[死门]]临己方**：有重大牺牲，需谨慎
9. **[[休门]]临己方**：需要休整，不宜再战

### 应期判断
- 值符旺相克庚之日（开战吉日）
- [[庚格定应期|庚格]]出现之日（战争转折）
- [[白虎]]克敌方之日（决战之日）
- [[马星]]发动之日（总攻信号）

## 实占要领

1. **九天九地为第一法则**：军事预测的根基是九天九地——"九天之上好扬兵，九地之下可伏藏"。这是不可违背的军事原则。
2. **分清主客**：值符是己方、庚是敌方——自己站哪一边要清楚。
3. **旺衰定实力**：双方落宫旺衰直接反映军事实力对比。
4. **生克定胜负**：谁克谁谁就有可能获胜。
5. **九地看防守**：九地下可伏藏——防守方要善用地势，攻方要谨防敌方埋伏。
6. **九天看进攻**：九天之上好扬兵——居高临下、势如破竹。
7. **八神看战术**：太阴（隐秘）、六合（联军）、螣蛇（诡计）——不同八神提示不同战术手法。
8. **死门最凶**：死门临任何一方的用神，都意味着该方将付出惨重代价。

### 占迷路法

> 「三路迷失时，天罡随于孟仲，两歧感应日辰。」

- **三叉路**：天罡加孟（寅申巳亥）左道通，加季（辰戌丑未）右道通，加仲（子午卯酉）中道通
- **两歧路**：日干吉者左道通，辰支吉者右道通

### 密访与情报

> 「欲得贼消息，天目听之吉（天目：大吉、小吉方）。欲访机密事，地耳不须疑（地耳：太冲、从魁方）。」

### 渡河

> 「天有三河壬子癸，地有三井卯酉辰。月将加时看三河，一河临井必覆舟。」

太冲加日辰主风雨；神后加季，急渡则吉。

### 刦粮觅水

> 「我军欲刦粮，大小吉下访。我军欲求水，小吉太冲访。」

小吉、太冲为井泉之神，地下见之可得。

### 敌使虚实

> 「敌有使来看日辰，日上为客辰主人。日干上神辰克辰上，其言虚谬不可闻。若是辰上克日上，彼必畏我言词真。」

- 朱雀、天空、太阴乘日上（日上）：主彼诈
- 朱雀、天空、太阴乘辰上：主我诈

### 突围法

> 「或被兵围不要忙，加时出路是天罡。若天罡方无路可出，当寻三官时（亥子丑）。」

- **绛官**：亥临子午卯酉，向天盘申酉下出
- **明堂**：子临子午卯酉，向天盘卯方出
- **玉堂**：丑临子午卯酉，向天盘戌方出
- **八极张**：逢豪强之贼势不可当，向天罡可出

### 察贼所在

> 「闻贼未知贼所在，加时春乙夏居丁，秋辛冬癸名天目，贼当在下伏其形。」

天目方位：**春辰、夏未、秋戌、冬丑**。以月将加时视天目之方，贼在其下。

### 抽军避寇

> 「贼势平凌我未强，抽军回避看天罡。」

- 天罡加孟（寅申巳亥）：从右道避之
- 天罡加仲季（子午卯酉、辰戌丑未）：向左藏之

避藏吉方：从魁(酉)、小吉(未)、太冲(卯)为天三门。大吉(丑)为紫房，神后(子)为华盖，能藏万物。
