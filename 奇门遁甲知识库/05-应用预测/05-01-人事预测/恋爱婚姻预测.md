# 恋爱婚姻预测

## 定义与基本属性

恋爱婚姻预测是通过奇门遁甲中乙庚合、六合等符号，判断男女双方感情发展、婚姻状况、离婚诉讼等问题的应用。其核心原理是"乙庚合为夫妇之象"，以双方落宫生克定吉凶。

> **核心原则**：乙为阴木（妻），庚为阳金（夫），乙庚合金是天地自然之合。恋爱婚姻预测的一切判断都从此展开。

## 核心用神

### 男女双方
| 用神 | 含义 | 五行 |
|------|------|------|
| [[乙奇]] | **女方**（未婚女友/已婚妻子） | 阴木 |
| [[庚（阻隔之神）|庚]] | **男方**（未婚男友/已婚丈夫） | 阳金 |
| [[天干化合|乙庚合]] | 男女结合之象，成婚的征兆 | 金（合化） |

### 辅助用神
| 用神 | 含义 |
|------|------|
| [[日干]] | 求测人（本人） |
| [[时干]] | 所测之事（婚恋之事体） |
| [[六合]] | 媒人、介绍人、婚姻中介 |

## 取用神方法

1. **测自己**：日干为自己，对比乙/庚落宫与自身日干的关系。
2. **测他人**：直接看乙落宫（女方）与庚落宫（男方）的生克。
3. **测婚姻状态**：乙庚两宫相生为和，相克为不和。
4. **测成婚**：看[[天干化合|乙庚合]]是否成立，六合是否生合乙庚。

### 性别切换注意
- 求测人是**男性**：日干为自己（庚方），看[[乙奇]]落宫与己宫生克
- 求测人是**女性**：日干为自己（乙方），看[[庚（阻隔之神）|庚]]落宫与己宫生克
- 测**孩子婚姻**：直接取乙（女）庚（男）两宫判断

## 判断要点

### 婚姻吉凶判断
1. **[[乙奇]]落宫生[[庚（阻隔之神）|庚]]落宫**：女方爱男方，婚姻和美（▲吉）
2. **[[庚（阻隔之神）|庚]]落宫生[[乙奇]]落宫**：男方爱女方，婚姻和美（▲吉）
3. **[[乙奇]]与[[庚（阻隔之神）|庚]]落宫比和**：男女平等，互相尊重（▲吉）
4. **[[庚（阻隔之神）|庚]]落宫克[[乙奇]]落宫**：男方克女方，婚姻不利女方
5. **[[乙奇]]落宫克[[庚（阻隔之神）|庚]]落宫**：女方克男方，婚姻不利男方
6. **两宫相冲**：性格不合，容易争吵
7. **两宫相害**：有心结，难以沟通

### 成婚判断
- **[[天干化合|乙庚合]]成立**：天地盘相合，成婚之象明显
- **[[六合]]临乙庚两宫**：有媒人撮合，婚事易成
- **乙庚两宫逢[[开门]]**：婚姻公开，明媒正娶
- **乙庚两宫逢[[休门]]**：感情休憩，关系松弛
- **乙庚两宫逢[[死门]]**：婚姻死局，难成或无感情

### 特殊格局（重要）
| 格局 | 含义 |
|------|------|
| [[青龙逃走]] | 女方主动离开男方（乙+辛，女主逃走） |
| [[白虎猖狂]] | 男方主动提出离婚（辛+乙，男主疯狂） |
| [[朱雀投江]] | 口头争吵导致离婚，诉讼离婚 |
| [[腾蛇夭矫|螣蛇夭矫]] | 婚姻中有口舌是非，纠缠不断 |
| [[太白入荧]] | 男方出轨 |
| [[荧入太白]] | 女方出轨 |

> **记忆窍门**：青龙=乙（女），逃走=辛（革动）。白虎=庚（男）或辛，猖狂=辛+乙。记住"乙辛相反"即可掌握两种格局。

### 婚姻应期
- **[[天干化合|乙庚合]]成立之日**
- **两宫相生之日**
- **马星发动、填实空亡之时**

## 实占要领

1. **首定性别**：先明确求测人是男是女，否则用神取错全盘皆错。
2. **看乙庚两宫**：不看别的，先看乙奇与庚落宫的生克关系，这是婚姻预测的基石。
3. **看生克方向**：谁生谁、谁克谁，代表感情的流向与家庭地位的高低。
4. **乙庚合为补**：如果生克不够理想，看[[天干化合|乙庚合]]是否有望弥补。
5. **六合为辅**：六合代表外部助力（媒人/长辈），旺相且生合乙庚则婚事有贵人相助。
6. **格局定情形**：特殊格局非常关键——[[青龙逃走]]、[[白虎猖狂]]直接定性婚姻走向。
7. **离婚诉讼看朱雀**：[[朱雀投江]]格局下，离婚靠诉讼解决的可能性极高。

## 开悟之门补充

以下案例引自《开悟之门》，原案例来自《神奇之门》书中实例，张志春老师用以讲解恋爱婚姻预测中空亡填实的应期判断方法。

### 开悟之门补充：空亡填实——结婚应期的关键方法

在测婚姻应期时，**空亡填实**是关键方法之一。丈夫所在的宫位如果逢空亡，填实之年自然很可能是结婚之年。

《神奇之门》实例五测外籍医生四柱，判断她大约1967年或1968年结婚。判断逻辑：1967年丁未、1968年戊申，地支未、申所在**坤2宫临庚**（庚为丈夫），但此宫逢**空亡**。空亡表示丈夫尚未实至——填实之年（未年或申年）丈夫由虚变实，自然成就婚姻。

> **核心方法**：配偶用神（男方看庚宫、女方看乙宫）逢空亡时，以空亡填实之年/月为结婚应期。此法简洁直接，关键是先找准配偶所在的宫位。

出处：《开悟之门》

## 开悟之门补充

《开悟之门》中未提供此预测类型的独立完整案例。但张志春老师在书中多处提及婚姻相关判断要点，可参考以下原则：

- **婚期判断**：地支空亡填实之年为婚期——若坤2宫逢空亡（未、申），则丁未年或戊申年为可能的结婚之年（见人生机遇预测中的外籍医生案例）
- **乙庚关系**：乙奇（妻）+ 庚（夫）为婚姻核心用神，乙+庚或庚+乙为刑格，主夫妻不和；乙庚合则为良缘
- **日干相合之干**：也可作为配偶用神的参考，与对方年命落宫综合对比
- **六合吉凶**：六合为媒妁之神，临吉门吉格则婚易成，临凶门凶格则婚难成

### 占婚姻（卷七）

以六乙为女，六庚为男，取甲以乙妹娶庚之义也。如乙庚二干落宫两相生合则成，两相克则不成。又以天上六合为媒人，如六合宫生六乙宫媒向女家，生六庚宫媒向男家。六庚宫克六乙宫，女家畏男而不嫁。六乙宫克六庚宫，男家嫌女而不娶。六乙宫带击刑，主女性凶恶；带德合，女性坚贞。六庚宫带凶神，主男性暴烈，带德合，主温厚。

### 占婚姻（卷十）

以庚金为男家，乙奇为女家，六合为媒妁。庚之落宫生乙之落宫，主男家爱女家；乙之落宫生庚之落宫，主女家爱男家。得奇门吉格者，婚姻必成。比和不相克制者，亦成。若庚宫克乙宫，主男家嫌女家，不成。乙宫克庚宫，主女家嫌男家，不成，强成之后必有刑克。庚金入墓而乘凶格者，刑夫。乙奇入墓而乘凶格者，克妻。

### 占纳宠（卷十）

以乙奇为妻，丁奇为妾，太白庚金为夫。若乙、丁奇之落宫生庚之落宫，其女必肯嫁。乙丁落宫克庚之落宫者，不肯嫁。乙丁宫相比和者，主妻妾和谐。如乙克丁宫，主妻不能容妾。或丁克乙宫，主妾欲欺妻。若乙丁入空陷库绝之宫，主不能成，成亦不利。如庚金宫生丁奇之宫，主徒劳无成。

## 上元经补充

### 用神的历时动态变化

《奇门遁甲上元经》婚姻章的最大突破是指出婚恋占测用神随阶段而变化，不能机械地一上来就看乙庚：

| 阶段 | 用神体系 | 依据 |
|------|---------|------|
| **少年情愫（舞勺/破瓜之年）** | 日时 + 桃花之位 | 懵懂初恋非真正婚姻，只是桃花 |
| **谈婚论嫁意向（男女未确立关系）** | 日时生克法 | 一方有意一方尚未表态，不能启用乙庚 |
| **确立恋爱关系/订婚** | 乙庚之合 | 已有和合事实基础（古代：父母之命；现代：身体接触） |
| **决定结婚走向** | 值符落宫（核心） | 乙庚相合的目的是合庚卫甲，值符决定整体走向 |

> **核心原则**：乙庚合的使用是有一定条件的，不能一上来就看乙庚二宫的生克比和。
> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]婚姻章

### 少年情窦初开占法

- 日干临桃花 → 已遇中意之人，该人情况看日干五合之落宫
- 日干下临之干临桃花 → 正在盼望心上人
- 遇壬加己、己加辛、戊加庚、乙加辛等冲击格局 → 桃花为过眼烟云，镜花水月
- 遇吉格吉局 → 心上佳人不久会出现
- 日干临值符 + 临桃花 → 孩子王，仰慕者众多（需观六仪、甲帅明暗两种情况）

### 谈婚论嫁阶段（未确立关系前）

#### 男方求测追求女方

日干 = 男方，时干 = 女方（妻财从属于日干）：

| 关系 | 判断 |
|------|------|
| 时干生助/印比日干 | 女方同意追求 |
| 时干明克暗合日干 | 女方有意，但存在障碍（家庭地位差/索要彩礼） |
| 日干克制时干 | 看是否有宫位暗联→有则可成，无则难度大 |

**特殊：男方追求年长有财富地位的女性**

此时女方具有类似值符的地位，转向值符宫 → 天盘六仪代表女方，地盘干代表男方，女上男下，观天地干生克。庚加丙格局利客不利主 → 虽能成但对男方发展无益。

#### 女方求测能否与男方发展

女方不适合以日干表征 → 以地盘时干代表自己，天盘时干代表男方，观天地盘生克。

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]婚姻章

### 乙庚合成立后的判断

#### 两宫各自反映的只是个人情况

- 九星 → 先天品性
- 天干 + 八门 → 当下穷通祸福
- 落宫旺衰 → 个人运势

> **纠偏**：乙庚落宫符号不吉利或直接冲克 ≠ 双方无法成婚。残疾人对照也组家庭、私德都有问题的鱼找鱼虾找虾。宫位对冲反映性格处事差异，不直接决定婚姻结果。

#### 值符是决定结婚的关键

**为什么不是六合决定婚姻？**

六合有双重意象：
- 古代婚前 → 媒妁（中介人）
- 经历性行为后 → 饮食男女之事

现代社会婚前性行为普遍 → 六合反映的是性生活情况，不表示一定领证结婚。**值符落宫的情况才是观测双方是否能结为秦晋之好的关键所在。**

**为什么不是休门决定婚姻？**

休门仅反映休闲娱乐活动（家庭聚餐、电影、旅游等），精英人群工作繁忙无暇休闲 → 休门不佳 ≠ 感情不好。

#### 日时关系在确立关系后的角色

确立恋爱关系后，乙庚+值符已成为主要判断体系，日时关系反映的是**起局时间附近对日干婚姻造成影响的事件**，不再代表对方。

### 值使门在婚姻中的真实作用

> **纠偏**：值使门 ≠ 父母之命

值使门是甲帅在人盘的具体活动记录，代表婚姻中的**人间琐事**（柴米油盐的日常家庭生活）。符使一体三面：

| 用神 | 婚姻侧面 |
|------|---------|
| 六合 | 夫妻性生活 |
| 休门 | 休闲娱乐活动 |
| 值使门 | 繁琐的日常家庭生活 |
| 值符 | 婚姻长远走向和全局趋势 |

**古代结婚择日看值使门**：对女方而言"过门"的门就是值使门。

**现代民政登记看开门**：开门是政府机构的象征。开门门迫/击刑/入墓 → 婚姻登记存在手续瑕疵；开门空亡 → 未在民政机构领证，只是办了婚礼仪式。

### 婚姻中介人的认定

六合充当中介人的条件：
1. 自己从未谈过恋爱 → 六合为中间人
2. 已分手，处于空窗期 → 天盘六合 = 中间人，地盘六合 = 前男女友生活情况
3. 已有恋爱对象 → 六合表征现恋爱实景，中介人转向时干宫

中介人出现时 → 乙庚不能指代男女双方（双方还不认识），婚姻认定回到日时关系。

### 第三者/纳妾占测体系

**丁奇**：女性第三者符号。庚加丁 → 庚明知丁会带来麻烦但喜火炼 → 出于性爱角度的眷恋。

**丙奇**：男性第三者符号。丙加庚（贼必去）→ 男方分手概率大。

**多角恋**：单一丁不能覆盖时，最先接触者取丁，其余按年命落宫分别表征。

> **注意**：年命在一般婚姻预测中不重要（乙庚关系 + 值符 + 值使 + 休门 + 六合 + 日时已完整），仅在多角恋预测中有价值。

### 妻妾关系与家庭财产占测

#### 符使定妻妾存亡

**值符宫凶恶 + 值使宫凶恶**：
- 天地两干同时入墓 → 婚姻走到尽头
- 乙加辛/丙加庚出现 → 离婚必然
- 明冲暗冲格局 → 双方裂痕巨大但不必然离婚

**值符宫凶 + 值使宫吉**：
- 丈夫出轨妻子不知情（丈夫演技高超）
- 或双方都在演戏维持表面家庭
- 需结合六合/休门验证（六合空亡 + 休门空亡 = 表面功夫）

**值符宫吉 + 值使宫出现乙加辛/丙加庚**：
- 不能轻断离婚（值符大趋势不坏，值使离去只是暂时性）

**符使均为好坏参半**：
- 观察值符与丁火落宫关系
  - 值符生助丁 → 第三者在竞争中占优
  - 丁加乙 → 侍妾地位高过原配（现代社会：第三者受宠程度高于原配）
  - 乙加丁 → 原配维持表面主导地位
  - 值符、乙、丁三合成局 → 妻妾和谐相处

#### 家资财产争夺判断

- 生门为主，戊土为辅（流动资金）
- 生门生助哪方 → 哪方控制财产主要部分
- 明克暗合（如生门克乙但天干五合暗合）→ 乙表面受损但实控大部分财产
- 生门与乙、丁三合成局 → 双方在财产问题上达成一致

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]婚姻章

### 非正常婚恋行为占测

飞盘中门在婚恋预测中可指称同性恋及变性者。

断定同性恋的三个必要条件（缺一不可）：
1. 天盘乙落宫五行与天盘庚落宫五行相同
2. 中门位于两宫中的任何一宫
3. 六合宫中出现了加己/己加丁/己加癸/癸加己等非正常性行为符号

## 奇门法窍补充

> **source:** [[奇门法窍（锡孟父子）]]
>
> 本节出自《奇门法窍》卷四「占婚姻」「占招赘」「占纳宠」「占娶女容貌性格」「占婚后有益」「占婚后子息」「占是否有妊」等章节，系（飞盘体系）飞盘奇门对婚姻预测的独特论述。

### 占婚姻（飞盘体系）

以**乙**为女家，以**庚**为男家，以两家落宫相生合决其成否；又以天盘**六合**为媒人，生合男女决之。

### 占招赘（飞盘体系）

以天庚生地乙宫，庚上得吉者良；男求女须以地乙生天庚为可成。

### 占纳宠（飞盘体系）

以**乙奇**为妻，**丁奇**为妾，**庚**为夫，以生克空陷决之。

### 占娶女容貌性格（飞盘体系）

以**乙奇**所乘之星推之，以九星决其性情美恶。

### 占婚后有益（飞盘体系）

以**庚乙**之宫推之，以奇吉生克决之。

### 占婚后子息旺否（飞盘体系）

以**得使**之宫为子息，如值阳吉星多男，阴吉星多女；仍以乘旺相相生断之。

### 占是否有妊（飞盘体系）

以**杜门**落宫为月信，以**日干**落宫为占人，各以生克定之。

### 占产育（飞盘体系·九星论男女）

占胎产男女，以**时干**为母，**直符**为父，**时干加地盘九星**为子，端看主位相生则吉，相克则凶。以九星阴阳论男女，九星加奇仪为之生期。
