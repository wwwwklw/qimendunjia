# 出行出国预测

## 定义与基本属性

出行出国预测是通过奇门遁甲判断出行是否顺利、交通工具安全程度、归期以及出行方向的吉凶。按照交通工具分为陆路、水路、空路三类，各自有不同的用神体系。

> **核心原则**：日干为出行人，所往方向与日干生克关系为主线。不同交通工具各有专属用神——伤门为车，休门为船，九天为航线，开门为飞机。

## 取用神方法

### 出行取用流程
```
排盘 → 定日干落宫 → 定目的地所在方位之宫 → 看生克 
     → 定交通工具用神 → 看交通工具旺衰与日干关系 
     → 看八门八神 → 综合判断吉凶
```

### 归期取用流程
```
排盘 → 看庚格位置 → 看马星是否发动 
     → 看日干是否临开门 → 看空亡填实 
     → 综合判断归期
```

## 判断要点

### 出行吉凶判断
1. **所往方向之宫生日干**：出行顺利，目的地有利（▲吉）
2. **日干生所往方向之宫**：出行辛苦但最终到达（△中）
3. **所往方向之宫克日干**：出行不利，有危险（▼凶）
4. **日干克所往方向之宫**：可控但艰辛
5. **两宫相冲**：途中必有事端

### 交通工具安全判断

#### 陆路（车 + 公路）
1. **[[伤门]]（车）生日干**：车辆安全，驾驶顺利（▲吉）
2. **日干克[[伤门]]**：有驾驶风险，但人可控制
3. **[[伤门]]克日干**：车有问题，可能出事故（▼凶）
4. **[[景门]]（道路）生日干**：路况好，畅通
5. **[[景门]]克日干**：路况差，拥堵或封路
6. **伤门临[[死门]]**：严重交通事故的警示
7. **伤门临[[白虎]]**：车祸前兆，需特别注意

#### 水路（船 + 水路）
1. **[[休门]]（水路）生日干**：水路畅通，航行安全（▲吉）
2. **[[休门]]克日干**：水上风险，可能晕船
3. **休门临[[惊门]]**：水路有风波
4. **休门临[[天蓬星]]**：水上事故的警示

#### 空路（飞机 + 航线）
1. **[[九天]]（航线）生日干**：飞行顺利，天气好（▲吉）
2. **[[开门]]（飞机）生日干**：航班正常，起降安全
3. **[[九天]]克日干**：航线有湍流，飞行不适
4. **开门临[[死门]]**：航班可能取消或延误
5. **开门临[[白虎]]**：航空事故的先兆预警

### 出发日期判断
1. 看[[开门]]下临地盘之干对应的天干日（如甲子日、乙丑日等）。
2. [[开门]]旺相之日为最佳出发日。
3. [[马星]]发动之日为出发信号。

### 归期判断
1. **[[庚格定应期|庚格]]出现**：归期将至的有力信号。
2. **马星临日干宫**：归心已动，即将返程。
3. **空亡填实**：如日干空亡，填实之时为归期。
4. **日干临[[反吟]]**：去而复返，很快返回。
5. **日干临[[伏吟]]**：迟迟不归，归期延后。

## 实占要领

1. **先查安危**：出行预测的第一要务是判断安全，看目的地宫与日干是否相克、看交通工具用神是否克日干。
2. **工具分类**：先确定出行方式——陆路看伤门+景门，水路看休门，空路看九天+开门。
3. **方向定吉凶**：所去方向之宫是第一判断基准。生我则去，克我则慎。
4. **白虎为凶兆**：任何交通工具用神临[[白虎]]，都是事故的前兆，需要特别警示。
5. **庚格看归期**：『逢庚必变』——庚和庚格是判断归期的关键参考。
6. **马星看动身**：动身与否看马星，马星不动，计划再多也走不了。
7. **死门看取消**：所选用神临死门，行程很可能取消。

## 开悟之门补充

《开悟之门》中未提供此预测类型的独立完整案例。但张志春老师在书中论及出行预测的核心要点：

- **开休生三吉门**：出行宜选此三门方向，吉门生宫为"和"，出行顺利
- **死惊伤三凶门**：出行忌此三门方向，凶门克宫事更凶
- **内外盘定远近**：内盘主近、主本地；外盘主远、主外地——出行距离的参考
- **反吟伏吟**：反吟主快去快回，伏吟主迟滞不行或停留
- **天辅星方向**：求学出行宜向天辅星所临方位

## 金函玉镜补充

### 占出外预定归期（卷八）

其法以四维长生决之，看出门之日干系何干？即看四维是何干长生之宫？以本宫地盘之干，即回家之日也。

### 占出外不知家安否（卷八）

其法专以四维决之。四维者，乃四长生之方也。倘甲乙日看干宫，此四大长生之根也。若得门、得奇、吉格，则吉；凶格，俱凶。

### 占出外之人安否（卷七）

先定方向，看其方上下二盘得三奇吉门天地二门俱有奇门及诸吉格者，安；反此不安。

### 占出行水陆吉凶（卷八）

以休、景二门所乘之宫分水旱二路。休门落空，合天地二盘有三奇相乘者，水路吉；景门有三奇者，旱路吉，行船忌。青龙逃走、白虎猖狂凶格，主风暴。螣蛇妖矫，主凶灾。朱雀投江，主沉溺。

又以伤门为船，伤加休上为浮、为顺；休加伤上，为舟行水底，主沉溺。景门为旱路，又以伤门为马、为车，忌太白入荧，主盗。荧入太白主火惊，玄武、天蓬主失盗。水路忌惊门来克，恐伤船，艮无水，利旱路。休加景主泥泞难行。若二门入墓，主关梁阻隔，行动艰难。

### 占出行（卷八）

以日干落宫为出行谋干之人，所往何方，有吉门、吉格来生日干之宫者，往必大利。或不得奇门吉格来生，而与日干比和者，亦利，反此不利。再被凶门、凶格来冲克日干者，大凶。若临空亡、入墓之方，或日干年命刑、墓、空亡方位者，不利。

## 金函玉镜补充

以下内容引自《金函玉镜奇门遁甲秘笈全书》卷十三"出行占"，系统阐述了值符、值使、十干在出行中的判断体系。

### 出行占——干支诀

> 日元为人时元地，人元尅地去向利，时元如若尅人元，所往之方多不济，时纳所藏是中途，不伤人元堪策骥，若得再来生日元，未到地头先称意，最忌截空与刑害，进退趦趄方另议。

### 出行占——值符诀

> 出行道行宜生旺，六合所主同伴当，勾陈白虎依程途，朱蛇文信口舌项，元武所主盗与奸，前后俱忌相利创，地阴尅制路多岐，九天生合宜去向。

### 出行占——值使诀

> 出行直使最为重，加临无伤始堪用，本门起自何卦中，飞临生合人趋奉，其间用空与落空，反伏迫格及刑冲，起程不得去无向，劝君另自卜门从。

### 出行占——十干诀

> 奇仪所主六亲神，旅舍风尘亦同因，图谋宜入财宫位，投托须临父舍恩，忌入官乡与刑格，再据地方车折轮。
