# 经营求财预测

## 定义与基本属性

经营求财预测是奇门遁甲中应用最广泛的预测类别之一，涵盖交易求财、投资求财、买卖货物、房地产、开店、合伙、借贷、讨债等多种与金钱相关的场景。以甲子戊（资本）、生门（利润）为核心，日干（求财人）与各用神之间的生克关系为判断主线。

> **核心原则**：甲子戊为资本之本，生门为利润之源。**生门生甲子戊则获利，克之则赔本**——此乃求财预测的第一铁律。

## 分类预测详解

### 一、交易求财

**判断要点**：
1. **[[生门]]生[[甲（遁甲）|甲子戊]]**：有利润，可做（▲吉）
2. **[[甲（遁甲）|甲子戊]]生[[生门]]**：需追加投资才有利润（△中）
3. **[[生门]]克[[甲（遁甲）|甲子戊]]**：赔本买卖，不可做（▼凶）
4. **[[甲（遁甲）|甲子戊]]克[[生门]]**：投资可控，但利润微薄

**实占要领**：先看生门与甲子戊的生克方向，再看日干与生门的关系。如果生门生甲子戊，同时生门也生日干，则生意一定赚钱。

### 二、投资求财

**判断要点**：
1. **[[日干]]与[[生门]]相生**：投资有利可图
2. **[[生门]]旺相**：投资回报快
3. **[[生门]]休囚**：投资回报慢或没有
4. **[[值符（八神）|值符]]（投资方）生日干**：投资环境对己有利
5. **临[[空亡]]**：投资有风险，可能血本无归

**实占要领**：投资看生门旺衰比交易更重要。生门旺相是第一条件，旺相才有讨论利润的余地。

### 三、买货

**判断要点**：
1. **[[时干]]（货物）生日干**：货物对买家有利，买进有利（▲吉）
2. **日干生[[时干]]**：买家付出多，可能买贵了
3. **[[值符（八神）|值符]]（货主）生日干**：货主诚信，买卖顺利
4. **[[时干]]临[[死门]]**：货物质量差，不要买

**实占要领**：买货看时干与日干的生克。时干生日干说明货物有增值空间，值得购买。

### 四、卖货

**判断要点**：
1. **日干生[[时干]]**：卖出货物顺利
2. **[[时干]]生日干**：货物滞销，不易出手
3. **[[值符与值使|值使]]（购货人）生日干**：有买家上门
4. **[[开门]]旺相生日干**：市场活络，好卖

**实占要领**：卖货方向与买货相反。日干生时干货物才出得去（买家带走货物）。

### 五、房地产

**判断要点**：
1. **[[生门]]为阳宅房屋**、**[[死门]]为地皮**
2. **生门生日干**：住宅对居住者有利（▲吉）
3. **日干克生门**：需付出代价购房
4. **死门临吉格**：地段好，值得投资
5. **死门临凶格**：地段差，不宜投资
6. **[[值符（八神）|值符]]生合日干**：卖方诚信，交易顺利

**实占要领**：买房看生门与日干关系；买地看死门与日干关系。

### 六、开店

**判断要点**：
1. **[[开门]]（店铺）生日干**：店址好，对经营者有利（▲吉）
2. **日干生[[开门]]**：经营轻松，但可能不赚钱
3. **[[开门]]临[[生门]]**：开店利润好
4. **[[开门]]临[[死门]]**：店面无生意
5. **[[六合]]临开门**：适合合伙开

**实占要领**：开店看开门，门庭若市还是门可罗雀，全在开门一宫。

### 七、合伙

**判断要点**：
1. **[[六合]]（合作关系）生日干**：合伙对己有利
2. **[[六合]]临吉门吉星**：合作伙伴可靠
3. **[[六合]]临[[玄武]]**：合作伙伴有欺诈
4. **双方年命相生**：适合长期合伙

**实占要领**：合伙最关键看六合。六合凶则合作必败，再好的项目不能合伙。

### 八、贷款借钱

**判断要点**：
1. **[[值符（八神）|值符]]（银行/出借方）生日干**：借贷可得（▲吉）
2. **[[日干]]克[[值符（八神）|值符]]**：需费周折但可能借到
3. **[[值符（八神）|值符]]克[[日干]]**：借贷无望
4. **[[休门]]临值符**：贷款审批中

**实占要领**：借钱看值符，值符代表放贷方，生我则借得到。

### 九、放贷

**判断要点**：
1. **[[值符与值使|值使]]（借贷人）生日干**：放贷安全，能收回（▲吉）
2. **日干克[[值符与值使|值使]]**：可控制借贷人
3. **[[值符与值使|值使]]克日干**：借贷人可能赖账
4. **值使临[[玄武]]**：借贷人信用差，慎放

**实占要领**：放贷看值使，值使代表借款人，克我者慎之又慎。

### 十、讨债

**判断要点**：
1. **日干克[[值符与值使|值使]]**：能讨回债务（▲吉）
2. **[[伤门]]/[[惊门]]生日干**：讨债过程不顺利，但可能讨回
3. **[[值符与值使|值使]]临空亡**：钱已不在借款人处
4. **[[庚格定应期|庚格]]出现**：讨债有希望

|--------|----|----|

## 实占要领

1. **求财看生门**：无论何种求财类型，生门是第一核心用神。
2. **甲子戊看本金**：本金安全与否看甲子戊是否旺相、是否空亡。
3. **两大铁律**：生门生甲子戊则利、生门克甲子戊则赔。生门生日干则赚钱对我有利，克日干则赚不到我手里。
4. **分类取用**：卖货日干生时干、买货时干生日干——方向不能搞反。
5. **六合看合伙**：测合作必看六合，六合凶则合作不可行。
6. **值符值使分角色**：值符=货主/银行，值使=购货人/借款人——角色定位要准确。
7. **玄武警惕**：任何求财活动中出现[[玄武]]，都要警惕欺诈和财务损失。

> 日元为人时元财，两支中间债物船，纳音之中定市价，孤虚旺相着裁量。

## 开悟之门补充

以下案例引自《开悟之门》，原案例来自《神奇之门》实例，张志春老师以此讲解经营求财预测中四大要素分析、用神旺衰、定应期等核心方法。

### 开悟之门案例：测李月珍引进外资建商城（实例二）

- **时间干支**：己卯月（1995年）
- **奇门格局**：日干庚（代表李月珍）落坤2宫为冠带、临官之地得地利；临天英星（从落坤2宫看为火生土，天时旺相）；临死门伏吟（人和不佳，正应当时有人告状、与原来项目老板闹矛盾）；上乘六合吉神（有合作伙伴相助）；庚+丙为太白入荧凶格（但为客进利）；庚+壬为上格（移荡格，主动撤出有利）；日干2宫逢空亡；月干己落3宫为木，克2宫之土；时干癸落4宫为木，克2宫之土；年干乙落坎1宫，受日干庚落宫所克；年命壬落兑7宫属金
- **判断分析**：日干庚落坤2宫属土，到辰月土旺，她重新注册公司得以成功；土能生金，到开门所在的戌亥之月（10月18日丙戌月壬午日）太和电子城破土动工；经过一年多建设，于1996年底丑月（生门所在艮8宫当令之月）土旺之月，太和电子城竣工；随即于1997年丁丑年元月8日（辛丑月庚戌日）正式开业运营
- **实际结果**：太和电子城顺利建成并投入运营

**要点**：竖看四大要素——天时（天英星火生土旺）、地利（庚落坤2宫冠带临官）、人和（死门不佳但主动撤出反吉）、神助（六合吉神）；空亡主变——日干逢空亡主动抽身；月令为纲——土旺之月事事有成；年命生克定人际关系——年命壬金克月干、时干木，说明同事小人克不住她

### 开悟之门案例：测失钱（实例五）

- **时间干支**：3月30日丙寅日前
- **奇门格局**：时干为失钱（用神）；时干壬临天蓬阳星；巽4宫庚下之干为丙
- **判断分析**：按时干临阳星看"庚下之干"定应期；巽4宫庚下之干为丙，应于丙日；3月30日恰为丙寅日
- **实际结果**：3月30日把自己挪动了收藏之地的6800元人民币如数找到

**要点**：失物找回应期——时干临阳星看庚下之干；庚格法第三种：时干为用神时以九星阴阳判断庚格方向

## 金函玉镜补充

### 占求财（卷七）

当分休囚，以生门所落之宫，为休生开上所乘之星为用，用生休吉，休生咎，则不吉。囚旺休衰，休虽克囚，不为大吉。休旺衰囚，虽克休亦无大害，大抵看门所落宫分，再看上下三盘，格局吉凶何如？吉信吉星，所求如意，一有不吉，所求仅半，休囚不吉，所求全无。

### 占借贷（卷七）

以值符为物主，天乙（即值使）为往借之人，各以所落宫分生克论之。值符宫生天乙宫，天乙宫克值符宫，借贷必遂；值符宫克天乙宫，天乙宫生值符宫，借贷不遂。

### 占借贷（卷十）

其占法以所往之方天盘所得之星为求借之人，又以天盘之星所落地盘之星为借贷之家。地盘所乘之星生天盘星者，必得借。比和者，虽借主迟疑。相克主不借，反惹羞辱。若天盘所得之干入地盘墓库者，彼寔吝啬，不肯借入。临空亡宫者，彼寔无，不必去借。

### 占买货（卷十）

以日干落宫为收买货物之人，以时干落宫为货物。时干落宫乘旺相气，再得奇门、吉格者，为美货。乘休囚之气，不得奇门、吉格者，为废货。时干落宫来生日干落宫者，其货不拘美恶皆有。利息来克日干落宫，或乘墓绝及空亡者，货皆无利，不必买。

### 占脱货求财（卷十）

以值符甲子戊为我，以天乙为脱货之人，以六合为经纪牙人。若天乙落宫乘奇门吉格来生值符之宫者，其货可脱。比和者，脱亦无碍，反此必不可脱，必有失。又看六合落宫生天乙落宫，主经纪以脱货之人相生，吉。值符与甲子戊之宫者，主经纪与货主同心。倘六合入墓或落空亡之宫，必有奸诈、欺骗之事，断不可脱。

以日干落宫为货主，时干落宫为货物，以甲子戊落宫为资本，生门落宫为利息。看日干之落宫生时干之落宫，主人恋货，不肯脱。若时干之落宫生日干之落宫，主货恋人，不能脱。再看时干之落宫生甲子戊与生门之落宫者，有利息，反此无利息。倘时干落宫乘凶神、凶格来冲克日干宫者，主折耗。惟日干落宫克时干落宫者，虽欲急脱，而货之甚迟。时干落宫克日干之落宫者，售之必速。

### 占开店肆（卷十）

以开门主之，开门乘旺相气，带奇门吉格来生日干之落宫者，大吉。相比和不克制者，次吉。若开门入墓或返吟，及落空亡者，不利。开门落宫乘凶神、凶格来冲克日干宫者，更不利。

### 占合伙求财（卷十）

以日干落宫为我，时干落宫为伙计。时干乘奇门吉格来生日干者，则合伙有益于我。日干乘奇门吉格而生时干者，则有益于他。二宫相比和者，主合伙公平，各无猜忌。若时干所落之宫乘凶神、凶格来克日干宫者，不利。再以生门之生我，不生我详之，百无一失。

### 占贸易（卷十）

以甲子戊为资本，生门为利息。生门所落之宫得奇门吉格来生甲子戊之落宫者，必获倍利。二宫比和者，亦得中利。生门来克甲子戊之落宫，再乘凶神、凶格者，必折本。甲子戊之宫生生门之宫，主加添资本，仍得利息。生门若临墓绝之地，再有凶神、凶格相乘者，必耗尽资本，尤凶。

### 占求财（卷十）

以甲子戊为财神，生门为财方。看二宫天盘落于何方？以二方生克、比和验其得失。天盘甲子戊与生门落于坎、艮、震、巽四宫，阳遁为内为近，又为速。再得奇门吉格者，得之必多。若得门不得奇，得奇不得门，得之不多。门奇俱不得，不落空亡、墓绝，不受地盘克制，得之必少。或甲子戊与生门落于一内一外，得之必迟。二宫俱在外，必求千里之财。二宫落空亡、返吟、墓绝，再以凶神、凶格相并者，必不得财，反遭是非。

### 占交易（卷十）

以日干落宫为我，时干落宫为他，六合为经纪。日干生时干，买主受置。时干生日干，业主愿意。日干克时干，买主不要。时干克日干，业主不卖。六合生日干，经纪人向买主。生时干者，向卖主。二干落宫相比和者，两家公平交易，主成。二干有一落空亡者，交易不成。

### 占买房屋（卷十）

以值符为买主，生门为住宅，死门为土地。生死二门乘三奇吉格来生值符宫者，主买后发达。二门不得吉格来生值符宫者，中吉。比和者，平安。二门乘休废之气，再有凶神、凶格来克值符宫者，主买后破败家财。值符生此二门者，主因宅产赢败不利。

### 占讨债（卷十）

以天乙所落之宫为欠债之人，以勾陈落宫为使讨之人。勾陈落宫克天乙落宫者，其人实心去讨。天乙落宫克勾陈宫者，所使之人畏彼，不敢讨。勾陈与天乙相生和者，彼此通同，不讨。勾陈被天乙生者，主嘱托贿赂，所使之人不肯讨。天乙落宫得旺相气来生值符者，必还。若落空亡来生值符者，有心无力。天蓬、天乙落宫克制值符宫者，安心不还，必不能讨。

### 占放债（卷十）

以六甲值符为放债之人，值符下临之星为天乙，即借债之人。若地盘之星乘旺相之气生天盘之星者，借债必还。乘凶格克天盘之星者，必不还。落空亡者，主死亡负债。乘墓绝之宫者，主混赖不还。

### 占打抽丰（卷十）

以日干落宫为打抽丰之人，所往之方为抽丰之处。得奇门吉格来生日干者，所往必利，即不得奇门吉格而来生者，亦利。比和者，平平。克制日干之宫者，去必有损。临空亡、墓绝之宫，主不见，徒劳无益，不必去。

## 上元经补充

### 生门与戊土作为求财用神的易学依据

《奇门遁甲上元经》财资章纠正了旧说"因五行属土=土地资源=财富"的肤浅解释：

**真正依据**：
1. 奇门核心在甲木，甲木克者为土 → 生门/戊土本质上是**甲木之财**（"我克者为财"）
2. 生门居艮宫 = 甲木临官禄地 → 禄财并行，事业高峰处财运自旺
3. 死门居坤宫 = 甲木墓绝之地 → 甲衰弱至极，无力创造财富

**偏财 vs 正财**：
- 戊/生门属阳 → 甲阳克阳土 = **偏财**（商业活动/投资等量大不稳定之财）
- 己/死门属阴 → **正财**（正常劳动收入，稳定但较少）
- 奇门所关注的主要是偏财 → 与古代生产力低下、普通劳动仅够果腹的社会结构一致

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]财资章

### 戊土与生门的阴阳区分

| 用神 | 阴阳 | 属性 | 含义 |
|------|------|------|------|
| 戊土 | 干（阳，动态） | 投入流通的资本/资金 | 资金数额大小，不表示最终收益 |
| 生门 | 门（阴，静态） | 商业活动后的实际收益 | 最终是否赚钱及利润大小 |

> **关键**：戊土生旺只表示投入资金量大，不等于最终盈利。生门宫内符号搭配决定利润多寡。

### 生门利润的判断细则

**门迫唯一情形**：生门落坎宫 → 门迫（坎宫=休门之所在，冬季休息与生门生发之义冲突），其余八宫不论。

**九星与生门的关系**：星辰不能直接与生门产生生克，但星的吉凶衰旺可**显象**——反映出财富数量与财富性质来源。

**天干格局判断**（重阳抑阴、喜合惧冲）：
- 三奇为基点的阳性格局 + 无墓/刑 → 正面，多重收益
- 五合格局 → 尤其吉利
- 单一负能量（如击刑）→ 不能完全抵消阳性格局正能量，财富量度有所减小

**宫位月令旺衰** → 区分利润层次与产业规模（同样戊加丙临生门，寅月旺相 = 大型公司收益；酉月衰弱 = 小店级收入）。

### 求财性质与生门的阴阳主客

- 玄武/天蓬落日干宫 → 求测者从事捞偏门（非正路）
- 玄武/天蓬落生门宫 → 求财过程不顺利，利润遭受不明损失

### 长久运势判断：符生命

| 组合 | 判断 |
|------|------|
| 生门旺 + 值符旺 | 长期盈利，长久不衰 |
| 生门旺 + 值符凶 | 当下收入不错，过了风口逆转（行业非长久之计） |
| 日时吉利相生 + 值符凶 | 短期可赚钱，但从长远看收益不会好 |

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]财资章

### 适合行业的笼统判断

- 各宫（日时、年命、符使之宫）与生门生助比合 → 适宜求财方向
- 直接冲克生门的宫位 → 投资禁忌之首
- 年命/日时/符使直接临生门且搭配良好 → 财来找我，求财异常顺畅

**行业选择以门盘为首选用神**：开/休/生三吉门占优；死/惊/伤三凶门需满足（不门迫 + 阳性格局+ 无墓刑冲 + 吉星吉神加持）才能产生正面意象。

### 买货占测（区分自用与求利）

#### 买货自用

- 时干宫 → 产品质量（宫内格局） + 市场认可度（宫位旺衰）
- 日时关系 → 购买是否对己有利（非产品质量判断）
- 紧要之物 → 观值符宫（合同要约、长远影响）

#### 买货求财（投资占测）

- 时干 → 购买这件事（不再指代货品质量）
- 时干与生门关系 → 利润判断
- 日时关系 → 对自身影响的好坏（如时干生助生门但冲克日干 = 虽赚钱但对自身安全造成威胁）
- 戊土不需观察（购物瞬间完成，资本没有经历周期运转）

### 卖货占测

> **纠偏**：旧说"日克时主恋货、时克日货恋人"是错误的。

**正确判断**：
- 时干宫 → 卖货这件事是否顺利（吉门吉星三奇 = 顺利；旺相 = 产品被市场认可）
- 时干与生门 → 利润多寡
- 日时关系 → 卖货对自身的影响好坏

> 日时关系和时干生门关系可能不同步：亏本出售对日主有利（日时相生），但时干克生门（无利润）——"鱼和熊掌不可兼得"。

### 以货易货

- 值符（阳动）= 出货过程
- 值使（阴静）= 收货过程
- 时干 = 交易总体描述
- 符使任何一宫空亡 → 一方未如期交货（需分真空假空）

### 股票/股权投资求财（奇门合柱法）

清末洋务运动发行中国第一只股票后，上元经提出全新判断体系：

> 单纯时家奇门局的信息量不能满足股票类偏财占测需要。股票/博彩类偏财求财需同时满足先天命势+后天时运两个条件，缺一不可。

**常规用神的局限性**：
- 时干吉凶 → 与收益无直接关系（劣质股可被炒作到高价）
- 戊土落宫 → 只反映买入时的涨跌，不展现后续价格变化
- 景门 → 火性炎上=单向性，不足以表征双向涨跌行情

**奇门合柱法概要**（详见《奇门遁甲中元经》）：
1. 将八字四柱 + 大运流年 + 流月流日八柱分入四正四维宫
2. 四正宫尤为重要：震宫=顺势上升，离宫=高位运行，兑宫=逐渐下跌，坎宫=跌落谷底
3. 结合清微派"通气决"综合研判气机运化点
4. 符使在此法中具有重要地位

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]财资章

### 开门与求财风水

固定经营场所 → 必须看开门。开门不在判断盈利，而在判断**经营场所的风水**：

**内部格局**（尊阳抑阴、喜合惧冲）：
- 三奇加临 → 阳气充足，鬼魅退避
- 五合加临 → 生机勃发
- 阴性格局逢旺相 → 危害最甚

**落宫月令旺衰** → 行业时运（战争时开门休囚；和平繁盛时开门旺相）

**开门冲击的他宫影响**：
- 冲克值符 → 最大忌讳（直接对主帅进攻）
- 冲克太岁 → 年内遭遇厄运
- 冲克生门 → 求财不顺利，非无利可图
- 开门旺相+阴性格局 → 最强负能量冲击（有压制力 + 心怀恶意）

### 房产占测：生门死门的阴阳本质

> **纠偏**：生门=房屋、死门=地皮的说法是偏颇的。生门=人栖息生活的生阳之气；死门=人死后非人物质力量的归魂状态。

**生门死门在转盘与飞盘中的区别**：
- 转盘：生门死门始终相对 → 阴阳相隔、人鬼殊途
- 飞盘：生死二门位置灵活，可生克比合 → 表征更细腻（"得鬼神庇佑"= 飞盘状态下死门印比生门）

**飞盘死门与生门的三种关系**：

| 关系 | 吉凶 | 含义 |
|------|------|------|
| 死门克生门 | 最凶 | 以阴伐阳，阴阳不和，不宜居住 |
| 死门印比生门 | 大吉 | 以阴助阳，阴阳和谐 |
| 死门合会生门 | 吉凶参半 | 阴阳和谐但容易梦魇见鬼神 |

> 飞盘奇门在房产等地气占测上优于转盘。

### 借贷预测（以银行/票号方为主体）

**批判值符天乙对待法**（值符宫天地二星分别指代借款方与贷款方）：
1. 用神类别选择错误（贷款属人事行为，应优先用人盘符号→天地二干而非星辰）
2. 阴阳主客关系错误（天盘对应主方/贷款方反成客方，不能自圆其说）

**正确方法**：
- 值符与生门的关系 → 盈利判断（符印比/合会生门 = 出贷有利）
- 值使 → 贷款执行过程
- 日干（经办人）+ 时干（贷款方）→ 只有日干是普通办事人员时才需关注日时关系

> **纠偏**：不能仅凭时干宫玄武/螣蛇就断对方信用造假（单一凶神不决定整体）。必须凶星+门迫+阴性天干组合并临才考虑资质问题。最终判断必须上升到值符高度。

### 借贷预测（以借贷方为主体）

- 日时相生 → 贷款有希望，但不能决定是否能拿到钱
- 值符印比/生合生门 → 最终可能拿到贷款的关键
- 值使宫克制日干/生门 → 分管部门或办事人员设置障碍

### 以官求财模式（封建社会特殊现象）

> 清末官员敛财远超商人阶层，奇门由此发展出独特的"求财看官"模式。

**官员认定**：
- 日干/年命临值符或太岁
- 落宫于乾/震/离
- 日干落宫处于临官状态（临官 = 领官职）

**官旺自生财的判断**：
- 开门（职位）+ 生门（财富）+ 日干/年命/值符构成三合/三会 → 能量极大扩充，官旺自生财
- 普通人生门印比日干 ≠ 以官求财（必须三合三会）

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]财资章
