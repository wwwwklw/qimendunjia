# 杂项预测

## 定义与基本属性

杂项预测是奇门遁甲中不属于前面十六类常规预测范畴的各种零散应用的综合。奇门遁甲"万物皆可测"，其余法则是灵活变通——在基本取用神体系（日干、时干、九星、八门、八神、天地盘）的基础上，根据所测之事的性质，灵活选取最匹配的用神进行判断。

> **核心原则**：奇门遁甲取用神的核心逻辑是"以类附之"——什么事取什么类神的符号。杂项预测考验的是对符号含义的灵活运用能力。

## 各类杂项详解

### 一、测开会人数

#### 用神
| 用神 | 含义 |
|------|------|
| [[开门]] | **大会**、会议、集会场所 |
| [[时干]] | **参会人**、与会人员 |

#### 判断方法
1. **[[时干]]生[[开门]]**：参会人愿意来，人数多
2. **[[开门]]生[[时干]]**：会议对参会人有利，出席率高
3. **[[时干]]克[[开门]]**：参会人不愿来，人数少
4. **[[开门]]克[[时干]]**：会议对参会人不友好，可能缺席
5. **[[时干]]旺相**：参会人数众多
6. **[[时干]]休囚**：参会人数稀少
7. **[[六合]]临开门或时干**：多人参会，团体出席

#### 实占要领
- 时干旺衰决定人数多少——旺则人多，囚则人少。
- 开门与时干的生克决定出席意愿——生则愿来，克则不想来。

### 二、测爆炸事件

#### 用神
| 用神 | 含义 |
|------|------|
| [[天冲星]] | **爆炸**、冲击、震动 |
| [[景门]] | 火光、爆炸的视觉效果 |
| [[伤门]] | 爆炸的破坏 |

#### 判断方法
1. **[[天冲星]]旺相且临[[景门]]**：爆炸事件严重
2. **[[天冲星]]临[[伤门]]**：有人员伤亡的爆炸
3. **[[天冲星]]临[[死门]]**：有人员死亡的爆炸
4. **[[天冲星]]临[[九天]]**：爆炸剧烈，范围大
5. **[[天冲星]]休囚**：爆炸威力小，或没有爆炸

#### 实占要领
- 天冲星是爆炸的专用符号，其旺衰决定爆炸的严重程度。
- 天冲星临什么门，爆炸造成什么后果。

### 三、测抽奖/摸奖

#### 用神
| 用神 | 含义 |
|------|------|
| [[时干]] | **摸奖人**（参与抽奖的人） |
| [[开门]] | **主办方**（组织抽奖的单位） |
| [[生门]] | 奖品、利益 |
| [[丁奇]] | 中奖信息 |

#### 判断方法
1. **[[开门]]（主办方）生[[时干]]（摸奖人）**：主办方对摸奖人有利，可能中奖
2. **[[时干]]克[[开门]]**：摸奖技术好，自控力强
3. **[[生门]]生[[时干]]**：奖品送到手，能中奖（▲吉）
4. **[[时干]]生[[生门]]**：需要付出努力，中奖概率低
5. **[[生门]]临[[时干]]宫**：中奖机会极大（★★★ 最佳）
6. **[[丁奇]]生日干**：有好消息，中奖信息到达
7. **时干空亡**：没有参与资格或中奖无望

#### 实占要领
- 生门是奖品/利益的用神，生我则中，克我则不中。
- 生门与时干同宫是中奖的最好信号。

### 四、测失物招领

#### 用神
| 用神 | 含义 |
|------|------|
| [[时干]] | 拾到物品的人（或失物所在） |
| [[日干]] | 失主 |
| [[六合]] | 转交/归还的纽带 |

#### 判断方法
1. **[[时干]]生[[日干]]**：拾物人愿意归还
2. **[[时干]]克[[日干]]**：拾物人想据为己有
3. **[[六合]]生合双方**：通过中间人归还
4. **时干临[[开门]]**：失物被公开捡到，归还概率大
5. **时干临[[玄武]]**：拾物人偷偷拿走，不归还

#### 实占要领
- 与钱物丢失预测区别：钱物丢失预测是"谁偷了我的东西"，这里是"谁捡了我的东西"。
- 时干的角色由"失物"变成"拾物人"，注意角色转换。

| [[景门]] | 见面地点 |

#### 判断方法
1. **[[时干]]生[[日干]]**：对方愿意见面，态度好（▲吉）
2. **[[日干]]生[[时干]]**：我主动，对方可能被动
3. **[[时干]]克[[日干]]**：对方可能爽约或态度不好（▼凶）
4. **[[六合]]临吉门**：见面顺利，愉快
5. **[[六合]]临[[死门]]**：见面可能取消或很不愉快

#### 实占要领
- 见面看六合（和合之事），态度看时干（对方）。
- 时干生我 = 对方有意，时干克我 = 对方无心。

## 取用神通用原则

奇门遁甲杂项预测的取用神有一个通用原则：

1. **定事类**：想清楚你所测之事属于哪一类。
2. **找对应**：在九星、八门、八神、天干中找到与事类最匹配的符号。
3. **看生克**：看该符号与日干（求测人）的生克关系。
4. **断吉凶**：生我则吉，克我则凶。

> **灵活应用口诀**：万物皆可以类附之，以事变通之。《神奇之门》第十七章总结：取用灵活，不可拘泥一格。

## 实占要领

1. **以类附之第一原则**：杂项预测没有固定用神体系，需要根据"什么事"类"什么神"。
2. **日干永远是自己**：无论测什么杂项，日干代表求测人这一原则不变。
3. **生克永恒**：无论取什么用神，生我者吉、克我者凶的法则不变。
4. **举一反三**：本章列出的九类杂项只是举例，实际应用中可以无限扩C——关键在于掌握取用神的底层逻辑。
5. **宁简勿繁**：杂项预测不要取太多用神，一两个核心用神+生克判断即可。
6. **参考常规预测**：如果杂项与前面十六类预测有相似之处，可以参考对应预测的用神体系。

## 开悟之门补充

以下案例引自《开悟之门》，原案例来自《神奇之门》实例，张志春老师以此讲解失物寻找、庚格定应期等方法。

### 开悟之门案例：测找发票（实例一）

- **时间干支**：12月22日左右
- **奇门格局**：日干己、时干壬均在内盘主快；八门反吟也主快；值使休门落9宫和时干落3宫均主3天左右；时干为丢失之物为用神；时干壬临天禽阳星；6宫庚下之干为癸
- **判断分析**：按时干临星阴阳找庚格——时干壬临天禽阳星，"时干临阳星看庚下之干"，6宫庚下之干为癸，判定当在癸日找到，即12月23日癸未
- **实际结果**：12月22日壬午日就找到了，比预测提前了一天

**要点**：庚格法第三种——时干为用神时以所临九星阴阳判断庚格方向；反吟主快；值使门落宫数大致定天数

### 开悟之门案例：测找发票（补充）——八门反吟定快慢

- **时间干支**：同上
- **判断分析**：日干、时干均在内盘主快；八门反吟也主快（反吟在时间上主快主速，在空间上主远）；值使休门落9宫含3天，时干落3宫也含3天左右
- **实际结果**：3天左右找到

**要点**：反吟局利客，主动寻找可见效；值使门落宫之数往往就是事物成败的期限

## 金函玉镜补充

### 占迁移吉凶（卷七）

以九星及九宫分方向定可否，如此方上有三奇、吉门，再时得天禽，四季皆吉；得天辅，春、夏大吉；得天心，秋、冬大利，余星俱不利。各以来时占，看何星为天乙定之，观此则以其人来之时发占，不似六壬之筮时也。

### 占雀噪（卷七）

以朱雀为主，看朱雀所临何奇、何门？以决其事。如开门得奇，主有亲朋至，或行人远归，或主酒食（此皆以开门断之，人盘以断人事，此举一隅也）。休门得奇，主有喜事（休即喜也）、喜信及婚姻之事。生门得奇，主得田产、财物、猪畜之事（俱切生门断法）。若不得三门及门迫、奇墓，俱主无所关系。要看景门临吉格，则有喜信，凶格则有忧信，或小烦恼。

### 占干谒（卷七）

干谒专看休门宫分（如天盘休加震为本宫），为所见之人，时干宫（天上时干加地盘坎方为水之类）为所往之人，加休门宫生时干宫，而时干宫又有三奇临之，遂意。如相克制又无三奇，不得见，或不喜悦，所求不遂。又要彼此二宫旺相，有一处休咎，即不吉。若所见之方得休门临之，亦相见也。

### 占访友（卷七）

大凡奇门用事，专择方向与时耳。如我欲访友寻人，以所往之方地盘为主，天盘为客，主客要相生合，又得吉门、吉星必相迁。若门凶，上下两盘又相克制，则不迁也。庚为年月日时格，值此亦不迁也。

### 占梦吉凶（卷八）

其法专责螣蛇所乘天盘门仪下临地盘是何门仪。合吉门、吉格者，吉；凶格，则凶。若落地盘空亡、墓库者，则无吉凶。

### 占禽鸟怪鸣（卷八）

其法专责天禽星落宫下临地盘之宫是何干以决之。若天禽落地盘得奇门吉格者吉，凶格者凶。各以八门配之，吉凶自验。

### 占何怪（卷八）

专责螣蛇及所乘星门以决之。若螣蛇落宫在坎，为水怪、神怪。落艮，石怪、山精。落震，木怪、狐狸。落巽，花妖、龙蛇。落离，火怪、鸟怪、龟蛇。落坤，老妇、牛羊及房屋、金灶怪。落兑，飞禽、羊怪、金银，埋久作怪。落干，神愿猪羊犬首及铜铁器皿作怪。若不得奇门吉格，而乘凶格者，必至死亡、孝服、官司之凶。落空亡者，无碍。

### 占避难（卷八）

专以杜门决之，看其落于何方？即以其方避之。再看所乘之干是何干？见戊为贵人潜避。若杜门乘三奇则去无阻隔，大吉。若见庚，须抱木行，方免凶灾。见辛为天狱，壬为地牢，必不能逃。若杜门落地盘癸为天网之格，在坎艮二宫，可用三、四尺之木压之而逃；若干兑二宫，可匍匐而过之。逃方在艮离二宫，或高八、九尺，可挺身而行。若杜门同癸、震、巽二宫，天网盈门，不能逃脱。若杜门临干兑二宫，木被金克，虽然逃去，后亦拿获。若有三奇吉格，落日干之宫者，有救。

### 占入山访道（卷八）

以天芮为访道之人，天辅为传道之人。若天辅得奇门、吉格来生天芮者，得高人传授。相比和者，空见人，不传道。相克者，不能见人。惟推阳日得僧道传道，阴日得羽士传道。

### 占买人（卷八）

以值符为买奴之人，以天芮之落宫为奴婢。若天芮落宫生值符者，主奴益主。相比和者，不逊。克值符者，反逆。乘玄武、天蓬者，偷盗。乘青龙逃走者，失。螣蛇妖矫者，病亡。乘白入荧、虎猖狂克值符落宫者，弒主，大凶。乘玉女守门，淫乱污主。荧入太白，懒惰庸愚。

### 占书馆（卷八）

以天辅为师长，天芮为弟子。若天芮落宫生天辅宫者，主弟子聘师长。天辅落宫生天芮宫者，主师长就弟子，或旧徒。天辅宫克天芮宫者，师长嫌弟子。天芮克天辅，弟子嫌师长。若天芮乘旺相气，又得奇门吉格者，必是厚馆。如天芮不得奇门吉格，不乘旺相，其馆必菲。辅、芮返吟者，主半途而废。天芮落空亡、墓库者，终不得馆。

### 占访人（卷八）

以天盘所往之方为我，地盘之星为他，天地二盘相比和者，必见。再得奇门、吉格及天地二盘所得之干相合者，并有酒食。若相克制，主猜忌，不见。如临星干入墓，彼在家，不肯见。如地盘之星落空亡者，彼实不在家。

### 占人来访（卷八）

以来人所来之方天盘之星为来人、为客，地盘所乘之星为主、为我，天盘星得奇门来生地盘之星者，为贵客，有益于我，可见。若天盘之星克地盘星，再有凶门、凶格相乘者，必损我之人，不可见。

### 占信息虚实（卷八）

以景门、朱雀信息决之。若景门乘旺相气，又得三奇者，的信。景门休囚，不得三奇，而乘朱雀者，为诈信，不可听。景门落空亡、入墓，而带朱雀者，乃道路之言，或忤悖架勾之词，更不可信。

### 占请客（卷九）

以值符为客，天乙为主。天乙所乘之宫来生值符之宫者，必来。时干生日干者，亦来。天盘星生地盘星者，亦来。反此不来。

### 占选妃（卷九）

以值符为差官，岁干为天子，柱星为室女。室女年命日干为值符所生者，主差官看中，必选。天柱为值符所生者，亦选。若室女年命日干所乘之宫，或天柱之落宫为值符所乘之宫来克者，不选。

### 占选后如何（卷九）

以岁干所乘之宫与室女年命日干之宫，来合天柱之落宫，看岁干之宫来生室女年命日干并天柱之宫者，主上喜隆宠。若甲子戊之宫，依生年命日干及天柱之宫者，主后有王妃贵人。克年命、天柱宫者，主上不喜，仅充不陈。

### 朝觐引见（卷九）

以休门、岁干、年命日干之落宫决之。岁干为天子，日干为引见之人，年命日干得岁干来生，再有三奇相扶者，大吉。又有休门落宫得奇仪相乘者，主引见，天子必喜，而得隆宠。如岁干不生命日干之落宫，亦不相克制，而休门得奇来生年命日干之落宫者，亦吉。若岁干来克年命日干，再入空亡、墓废，并休门不得奇仪吉格者，不利。

### 占走失六畜（卷九）

驴、骡、车、船以伤门、天冲主之。牛、羊以死门主之。看落天盘何宫，即往何方寻之。马以干、兑宫所得之星主之，若得天蓬星，当于近水处寻之，如乘玄武为人盗去，再看得之之日必逢庚格。

### 占人寿夭吉凶（卷十）

法以冲、柱二星合死门远近以定之。男以天冲星起顺布奇仪，女以天柱星起逆布奇仪，看至死门相隔几宫，一宫十年，起算致死门几宫，作寿几十年。余几十年外，再以宫一宫为一年，看零几年是几岁，零数以四正四宫，一宫只作一宫，惟四维四宫，一宫便作二宫，因为四维乃八支也。

### 占人年命吉凶（卷十）

以来人所乘之方，合天地正时与其年命所坐之方，合而推之。来方得奇门吉格者，其年命必吉；年命落宫合奇门吉格者，亦吉。年命坐方俱合奇门吉格，再乘旺相之气，必有奇遇横发。入如地盘墓库者，主昏晦。落空亡者，百事不成。入死门者，死。伤门者，病。惊门者，口舌词讼。景门者，血光、火灾。杜门者，忧疑。开门者，见贵。休门者，进谒。生门者，发财得喜。大抵得奇门者，吉；乘凶门相克年命者，凶；不克者，小凶。

### 占谒贵人（卷十）

现任官以天盘开门落宫主之，未任致仕官以天盘甲子戊主之，以日干为求谒之人。开门甲子戊乘旺相气，再得奇门吉格来生日干落宫者，去谒必见，见且有益，所求如意。开门甲子戊与日干比和者，但求谒主少迟。开门甲子戊同宫者，主贵人有客，宜稍待。若开门甲子戊返吟者，主贵人他在，不得见。开门甲子戊入库，不能见。开门甲子戊受制，贵人有忧事，不见，见亦无益。开门甲子戊落宫冲克日干宫者，贵人不喜，枉惹羞辱。若谒武官，看杜门，亦仿此。

### 占开挖水道（卷九）

以开门主之。开门乘旺相气而得三奇者，主开挖有益，顺利无害。若开门返吟、入墓，不宜挑挖，倘值太白凶格及甲辰壬者，后必有大害。

### 占立窑（卷十）

其占法以景门主之。景门乘旺相气，得三奇吉格来生我年命日干者，吉，可立。若带凶神、凶格克我年命日干者，凶。乘朱雀投江、螣蛇夭矫者，大凶，断断不可立。
