# 工作就业预测

## 定义与基本属性

工作就业预测是通过奇门遁甲判断求职成败、岗位适合度、工作环境优劣、升迁调动等问题。以开门（事业之门）为工作的核心用神，开门与日干的生克关系是判断一切就业问题的主线。

> **核心原则**：开门为工作单位/事业之门，日干为求职者。开门生合日干则工作对求测人有利，反之则不利。

## 核心用神

### 主要用神
| 用神 | 含义 |
|------|------|
| [[开门]] | **工作单位**、事业、岗位、职位 |
| [[日干]] | 求职人/求测人本身 |

### 辅助用神
| 用神 | 含义 |
|------|------|
| [[时干]] | 所测之事（求职/升迁这件事） |
| [[值符（八神）|值符]] | 上级领导、老板、总经理 |
| [[年命|太岁]] / [[年命|年干]] | 最上级领导、单位的主管部门 |
| [[值符与值使|值使]] | 人事部门、直属上级 |
| [[用神|月干]] | 同事、同级别员工 |
| [[时干]] | 下属、基层员工（兼看） |

### 不同情况
| 情况 | 用神选择 |
|------|----------|
| 求职面试 | 日干 + 开门 + 值符 |
| 在职升迁 | 日干 + 开门 + 太岁/年干 |
| 工作调动 | 日干 + 开门 + 马星 |
| 辞职能否 | 日干 + 开门 + 时干 |
| 与同事关系 | 日干 + 月干 |

## 取用神方法

1. **测求职**：看[[开门]]与日干的生克关系。
2. **测面试**：看[[值符（八神）|值符]]（面试官）与日干的生克。
3. **测升迁**：看[[年命|太岁]]/[[年命|年干]]（上级领导）与日干的生克。
4. **测调动**：看[[马星]]是否在日干或开门宫发动。

### 取用流程
```
排盘 → 定位日干落宫 → 定位开门落宫 → 看生克关系 
     → 看值符/年干（领导） → 看八门八神旺衰 
     → 综合判断吉凶应期
```

## 判断要点

### 求职成败判断
1. **[[开门]]生[[日干]]**：工作单位欢迎求职者，求职成功（★★★★★ 最佳）
2. **[[日干]]生[[开门]]**：求职者向往单位，能否进去不确定（△中）
3. **[[开门]]与[[日干]]比和**：求职顺利，人与单位匹配（▲吉）
4. **[[日干]]克[[开门]]**：求职者可掌控局面，但可能不满意
5. **[[开门]]克[[日干]]**：单位排斥求职者，求职失败（▼凶）

### 工作环境判断
1. **开门临[[休门]]**：工作清闲，但可能没发展
2. **开门临[[生门]]**：工作有收获，效益好
3. **开门临[[伤门]]**：工作辛苦劳累，易有工伤
4. **开门临[[杜门]]**：工作技术性强，但可能闭塞
5. **开门临[[景门]]**：公开透明，适合文职策划
6. **开门临[[死门]]**：单位死气沉沉，没有前途
7. **开门临[[惊门]]**：单位竞争激烈，压力大

### 领导关系判断
1. **[[值符（八神）|值符]]/[[年命|年干]]生日干**：领导赏识，有提携之象（▲吉）
2. **日干生[[值符（八神）|值符]]/[[年命|年干]]**：迎合领导，但未必有效
3. **日干克[[值符（八神）|值符]]/[[年命|年干]]**：不服领导，可能导致矛盾
4. **[[值符（八神）|值符]]/[[年命|年干]]克日干**：领导打压，职场不利（▼凶）

### 升迁调动判断
1. **日日相生，马星到日干宫**：工作变动在即
2. **马星临[[开门]]**：单位有变动（重组/搬迁）
3. **日干临[[九天]]**：有升迁之象
4. **日干临[[九地]]**：稳定不动，难以升迁
5. **日干临[[开门]]+[[九天]]+马星**：升迁调动的最佳组合

### 应期判断
- [[开门]]旺相临日干之日（入职上班日）
- 马星发动之日（人事变动日）
- [[值符（八神）|值符]]生日干之日（面试通过日）
- 空亡填实之日（等待期结束日）

## 实占要领

1. **先定位开门**：开门是工作预测的第一用神，先看它在哪一宫、旺衰如何。
2. **生克定吉凶**：开门与日干的相生相克是判断一切就业问题的主线。
3. **生门看收入**：如果关注工资待遇，兼看[[生门]]（利益）与日干的关系。
4. **八门看环境**：开门所临八门，直接反映工作单位的氛围和性质。
5. **值符看领导**：面试/晋升必看值符（主考官/直属上级），年干/太岁看更高层领导。
6. **马星看变动**：马星是关键——没有马星参与，调动升迁都不会发生。
7. **八神看趋势**：九天主上升、九地主稳定、值符主贵人、螣蛇主变数。

## 上元经补充

### 科举/应试取仕的核心判断体系

《奇门遁甲上元经》职占章指出，科举取士不是单纯的考试——最终决定权在上级官员或皇帝手中。因此**丁火、景门只反映了试卷答题情况，不能决定是否被录取**。

#### 决定录取的第一因素：符使

| 组合 | 结果 |
|------|------|
| 符使俱吉 | 金榜题名 |
| 符凶使吉 | 名不位前（主要官员不认可，名次居末尾） |
| 符吉使凶 | 好事成空（考前上峰有意录取，具体操作环节横生事端） |

> **注意**：符使在科举试占中有先后顺序——符为天象在先，使为人盘在后，最终结果看值使落宫。

#### 决定录取的第二因素：太岁

- 一年之内，太岁影响力大于值符
- 但跨越年度（如笔试→次年面试）时太岁更替，旧太岁不再发挥作用
- 太岁生助日主 → 需同时与值符形成符号关联才能发挥推荐效能（太岁不会越过值符直接跟值使商量）

> **案例**：年轻精英竞聘，董事长口头说"我很欣赏你"，但没有交代人力部门在行动上支持 → 晋升成镜花水月。太岁单方面生助日主 ≠ 实质帮助。

#### 符使与日主的关系：不排斥克制

古代官场讲究上克下——上级对下级具有克制才是权力运行的基础。符使克制日主合乎常理。唯独符使与太岁构成三合/三会来冲克日主 → 所有决定录取的因素都站在了对立面 → 录取可能性极低。

### 科举中符使与日主的关联判断

1. **宫内干支符号相通**：如考生宫壬加乙 + 值符宫乙加戊 → 通过乙/戊符号连接
2. **宫位合会关系**：三合力最强，可抵消个别宫位凶象。符使与日主三合成局 → 即使符使某宫出现凶象，也被三合抵消。

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]职占章

### 与考试/文章相关符号的精细区分

| 符号 | 层次 | 角色 | 特殊性 |
|------|------|------|--------|
| **天辅星** | 星盘（天） | 文曲星，代表先天学习潜质/顶层高端文化 | 必须与考生同宫才算直接助力（自宫星力最强，外宫星力量有限） |
| **朱雀**（飞盘独有） | 神盘 | 笔锋锐利、观点鲜明 | 双面：优秀文章 vs. 因言获罪 |
| **六合** | 神盘 | 文章旁征博引、融合多观点 | 中庸平和，衰弱则东拼西凑类抄袭 |
| **丁奇** | 人盘（干——阳动） | 考前→书写之象（试卷"还没写"用丁） | 上元经首次明确区分丁与景的时序差异 |
| **景门** | 人盘（门——阴静） | 考后→阅检之门（试卷"已写完"用景） | 也可表示负责收卷阅卷的机构 |

**丁与景的组合判断（主客观差异）**：
- 丁吉景吉 → 主观感受与阅卷人感受一致，双方满意
- 丁吉景劣 → 自我感觉良好但分数不理想
- 丁劣景良 → 有贵人帮忙（房师力荐），文章不好但被推荐

> **重申**：丁、景不能决定最终录取结果（决定权在符使、太岁）。

### 科举监察——开门在考试中的特殊角色

开门在科举考试中类似现代监察机构，负责检验考生资格及维护考场秩序。开门不能用来断是否被录取：
- 开门逢空 ≠ 没被录取
- 开门克制日主需满足两个条件：(1)直接冲克不能明克暗合 (2)开门与值符或太岁通过合会构成内在联系

### 孩童前途代占

古代孩子出生后家长代占前途：

| 用神 | 反映 |
|------|------|
| 年命宫星辰 | 先天品性与基因 |
| 年命宫所临之门 | 以后人生道路/社会人际关系 |
| 天盘天干 | 能力大小与能达到的高度（壬加戊 = 小蛇化龙上升过程；壬加辛 = 由阴入阴，结局不好）|
| 落宫月令旺衰 | 是否有施展抱负的舞台 |

**符使在终身预测中的作用**：
- 值符 → 事业拼搏中能达到的顶尖高度
- 值使 → 最终结局/人生归宿

> 年命、符使必须同看：符为高点，使为结局，两者俱佳才是圆满人生。

**职业方向选择**：分别观察八门落宫的吉凶旺衰，选择搭配最好的宫位为"出路"。不一定是开/生/休三吉门最佳——死门丁加戊落离宫临天英 = 名医之象。如多门吉凶程度相似，以和合年命落宫者优先。

### 进士及第后所授官职的判断

#### 以开门判断工作单位

开门落宫内部搭配 → 单位好坏（如壬加乙落乾宫 = 单位效益扭亏为盈）：

| 开门与日主关系 | 辩证判断 |
|---------------|---------|
| 开门冲克日主 | 不一定是职位不利 —— 格局吉祥时 = 工作压力大而非被革职（类似子平七杀） |
| 开门生合日主 | 不一定是利 —— 开门壬加己且天干五合合于日主 = 负能量意象合到自己身上 |
| 开门印比日主 | 不一定是帮助 —— 开门格局破败 = 即将破产的公司 |

#### "文职看开，武职看杜"之批判

该说是主客不分、阴阳未分的错误：
- 开门可指代工作机构——无论多封闭的单位，工作的人必须通过大门进入 → 开门指代工作具有普遍性
- 杜门是封闭/阻塞 → 切断涉密人员与外界联系 → 落脚点在于人员本身
- **杜门临于日干或年命且宫内格局适当** → 才可断从事保密性质的工作

#### 官职升降任免（核心）

**两大决定因素**：(1)上级态度（太岁、值符）(2)自身能力

太岁与值符力量合与不合：
- 岁月同宫 → 力量一致，吉凶易辩
- 岁月相左 → 观值使（值使倾向于命主 → 提拔手续和流程正在进行）
- 二克之下 → 仍有擢升可能（唯见于不世之才落于旺相之宫 + 月令之助——极端条件）

> 现代文明社会时干宫最重要（时干=人民群众），封建社会时干宫作用不大。

#### 职位调动的判断

**调动的前提**：主客双方通过马星引动形成合会关系。

| 条件 | 说明 |
|------|------|
| 仅单宫临马星 | ≠ 调动（只表示策划变动或因事情来回奔波） |
| 日干/年命（主）+ 开门/值使/时干（客）+ 马星引动 | 主客共同引动才能表达调动之象 |

**必动格局**（单个宫位的格局意象）：
- 戊加庚（飞宫）、乙加辛（龙逃走）、丙加庚（必去）、庚加丙（贼必来）、庚加壬（移荡）
- 庚加戊 ≠ 必动（"伏宫"——想动也动不了，处处受打压）
- 壬加庚 ≠ 必动（太白擒蛇，无移动象意）

**大象阴阳 + 格局不能简单对应升降**：
- 阳性格局（戊加丁、戊加丙等）→ 工作得心应手，施展才能 ≠ 一定升迁
- 阴性格局（己加辛、壬加辛等）→ 工作不顺利、阻碍 ≠ 一定辞职开除

**开门逢冲击与变动**：
- 开门辛加戊落坎宫（子午两重暗冲）≠ 调动（"困龙被伤"中"困"字导致移动困难）
- 开门九天 ≠ 调动（需得格局支持或马星引动合会）
- 开门击刑 ≠ 辞职（是外在伤害/人际关系倾轧，需格局决定是否构成移动）
- 开门天干入墓 → 真墓（死气沉沉）和假墓（马星引动/合会出墓→先抑后扬）

**开门落震巽二宫门迫的差异**：
- 震宫力强 → 波澜较多、棘手的麻烦事 → 遇到多重负能量时工作干不长
- 巽宫力柔和 → 进退维谷、面临多个选择（大学生找工作）、反复不定（反吟非马上变动）

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]职占章

### 新岗位与旧岗位的二选一判断

**前提**：必须确认新工作是否为真（不是上级空口许诺）：
- 太岁、值符是否对日主有实质性支持
- 日主或时干是否空亡（空 = 镜花水月）

**新旧工作的用神定位**：
- 开门 → 现有工作（占测时已在原单位，开门反映与原工作密切相关的信息）
- 时干 → 新工作（时干既有即时性表述，又有新近事物意味）

> **纠偏**：不是开门地盘干转宫后看新工作（开门作为没有生命力的物质系统，不能主动随时间变化）。

### 犯事入狱的占断体系

#### 己测（自由状态下自测）

- 不以辛金为主要用神（辛为锁链→当事人尚为自由身）；只有被控制后代占，辛才能指代当事人
- 首观日干/年命宫内搭配旺衰 → 吉旺则无事；负能量符号（己加庚、庾加己、癸加辛、壬加辛 + 击刑/入墓/伤死杜门）→ 必受惩罚

#### 代占（已失去自由后）

辛金为用神：
- 天盘辛 → 案件处于收集证据阶段，尚未正式刑罚，有翻盘可能
- 地盘辛 → 已判刑，需观察是否还有上诉机会

**阴干牢狱意象区分**：
| 符号 | 意象 |
|------|------|
| 辛 | 细锁链，身带刑具之人 |
| 庚 | 肉刑/刑罚，庚下之干 = 刑罚应期 |
| 壬（阳水） | 有一定活动空间的劳役场所 |
| 癸（阴水） | 完全限制自由的狭窄囚室 |
| 癸加辛 | 阴上加阴，看不到光明希望，难出头 |

**辛逢冲的判断**：
- 暗支刑冲 → 出问题的原因或与其他囚犯矛盾，不是出狱
- 辛加戊落坎宫（辛暗支午冲坎宫子）→ 有出去的可能，但需年命/值符等其他宫助力才能正常出狱（否则为越狱逃出）
- 癸落坤宫（三杀并临：墓/刑/冲）→ 更凶恶，非冲网而出
- 壬落乾宫（旺而逢冲）→ 更换关押地点
- 庚辛同宫 → 暴力打斗意象（屈打成招）
