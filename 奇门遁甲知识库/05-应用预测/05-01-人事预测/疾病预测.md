# 疾病预测

## 定义与基本属性

疾病预测是奇门遁甲中通过天芮星（病星）及其相关符号组合，推断人体疾病的部位、性质、轻重及治疗效果的应用。疾病预测遵循"星为病、门为症、奇为医"的核心逻辑，多符号联动综合判断。

> **核心原则**：天芮星为主病之源，八门定病之症状，三奇六仪判病之深浅变化，天心星与乙奇代表医治力量。

## 取用神方法

1. **确定病神位置**：查看[[天芮星]]落在哪一宫，该宫对应的身体部位即为病位。
2. **确定病症性质**：查看[[天芮星]]所临的八门、天地盘干，判断病症类型（寒热虚实）。
3. **确定医治能力**：查看[[天心星]]和[[乙奇]]是否旺相，与病宫的生克关系。
4. **代测转换**：代他人测病时，用相应六亲转换为用神，但病星始终看[[天芮星]]。

### 实战取用流程
```
排盘 → 定位天芮星落宫 → 看宫位身体部位 → 看八门症状 
     → 看病宫天地盘干 → 看天心星/乙奇与病宫生克 
     → 综合判断吉凶与应期
```

## 判断要点

### 病灶判断
1. **天芮星旺相**：病情严重、病势在发展
2. **天芮星休囚**：病情减轻、病势在消退
3. **天芮星临死门**：病情危急，有生命危险
4. **天芮星逢开门**：病有出路，可治愈

### 治疗判断
1. **天心星/乙奇生合病宫**：医生对症，治疗有效
2. **天心星/乙奇克病宫**：药物有效，但病人可能承受痛苦
3. **病宫生天心星**：病人配合，治疗效果慢但安全
4. **病宫克天心星/乙奇**：医生无能，药不对症

### 好转与恶化
- **临开、休、生三吉门**：病情好转，可达康复
- **临死、伤、杜三门凶**：病情加重，治疗困难
- **临九天**：病情外散，向外排解（好）
- **临九地**：病情内陷，深入脏腑（坏）

### 应期判断
- 天芮星逢空亡填实之时
- 天心星/乙奇旺相之日
- 病宫被克最重之时（物极必反）

## 实占要领

1. **先看天芮**：无论测自己还是他人，[[天芮星]]始终是疾病预测的第一用神。
2. **次看宫位**：天芮星落何宫，病在何部——这是定位病灶的快捷通道。
3. **再看八门**：八门决定病症表现形态，比天地盘干更直观。
4. **最后看乙奇**：[[乙奇]]为药，天心为医，二者旺相且与病宫吉则为治，凶则难医。
5. **注意空亡**：病星空亡，看似无病但隐患潜伏；医星空亡，有医无药或药不对症。
6. **综合判断**：不能单看一个用神，需天时（星）、地利（宫）、人和（门）、神助（八神）四合一。

## 开悟之门补充

以下案例引自《开悟之门》，原案例来自《神奇之门》书中实例，张志春老师以此讲解疾病预测的方法与应期判断。

### 开悟之门案例：女测父病（实例二）

- **时间干支**：1996年丙子年
- **奇门格局**：年干丙为用神（其父1921年生，时年75岁），落离9宫为帝旺之地，但逢空亡（"老怕帝旺"）；年命甲午辛在巽4宫形成午午自刑（其父为医生，自知药石无效，拒绝服药）；星门均伏吟
- **判断分析**：以用神死、墓、绝法断其父将死于酉月或戌月；又以年命午午自刑，刑以合为应期（午与未合），故断死于酉月或戌月的未日；日干、时干一内一外和星门伏吟都主迟，但年干丙和病神天芮均在内盘又主近，故按月、日断
- **实际结果**：死于当年阳历9月7日（丁酉年丁未日）

**要点**：老怕帝旺——年老者临帝旺反为凶；自刑以合为应期——午午自刑应于未日；病星空亡主凶；伏吟主慢

### 开悟之门补充：自刑提前死亡应期——年命午午自刑的特殊作用

在测父病案例中，按常规以用神入墓之月（戌月）断死亡即可，但由于**年命甲午辛**在巽4宫形成**午午自刑**，产生了提前效应。其父本人是位医生，知道吃药也没用，干脆什么药也不吃了，自己等死（自刑的心理表现——自我放弃），所以死亡提前到**一进酉月**就去世了，而非等到戌月。

> **关键逻辑**：自刑（午午自刑）在疾病预测中会**加速死亡进程**，使应期提前于常规判断的入墓之月。自刑反映了当事人主观上的自我放弃（"自己给自己造成伤害"），叠加在命理层面产生加速效应。实际死于酉月丁未日（刑以合为应期，午与未合）。

出处：《开悟之门》

### 开悟之门案例：判断何时出院（实例三）

- **时间干支**：甲午旬，求测日为5月9日丙午日
- **奇门格局**：时干用神丙临太岁、临吉门、落宫旺相；年命乙临吉星、吉神、落宫旺相；病星天芮受制，凶不起来；日干、时干均在内盘主快；时干和值使休门均落3宫，3宫含3、4、8三个数字
- **判断分析**：判断病能愈；因昨天刚做手术，拆线至少需一周，故不断3天，断不超过13天；生门落宫逢空亡（辰、巳空），空亡填实之日即为出院应期；10天后5月19日丙辰日、20日丁巳日恰是填实之日
- **实际结果**：5月20日上午痊愈出院

**要点**：空亡藏玄机——填实之日即为应期；值使门落宫之数定大致期限；生逢生日，死逢死期——病愈看生门填实

### 开悟之门补充：生门填实冲死门——出院应期的深层机理

在测病愈出院时，有一个深层判断机制：**生门填实之日冲开死门**即可出院。《神奇之门》实例三中，甲午旬辰巳空，空亡之地支辰、巳恰在巽4宫临生门。生门为生机之门，因空亡而"虚"，等到辰日、巳日填实之时，生门由虚变实，其生机之力冲开对宫的死门，病人即可痊愈出院。一查万年历，近期5月19日丙辰日、20日丁巳日恰是填实之日，故断这二天出院。实际5月20日上午痊愈出院。

> **关键逻辑**：非简单看生门旺相——而是利用空亡填实的"由虚转实"之力，以生门之实冲死门之虚，此为"生冲死"的独特应期机制。

## 金函玉镜补充

### 占病何日愈（卷七）

以病人日干为主（八字内之日），以天芮为病神，以生死二门为生死。如日干落在生门，生主不死（皆天盘），得死门者，难愈。更遇日干休囚，得凶星、凶格，必死；其余六门主缠绵，以天芮废没之日为愈期。占儿女病，时干入墓必死。

### 占病愈（卷八）

以天芮落宫为病症，克天芮落宫之干支者为愈期。如甲乙木克戊己土之类。

### 占请医（卷八）

以天心星为医者，又以天乙奇为医者，所落之宫乘奇门吉格为良医，二神乘旺相之宫。不逢奇门吉格为时医，不得旺相气及奇门吉格为庸医。不论良医、庸医，但能克天芮病神之宫者，医必有功。若病神落宫克二神落宫者，虽良医亦不能治也。

### 占何病症（卷八）

以天芮所落之宫决之，后以戴九履一之法论是何病：

- **离宫**：头、眼目，在内为心、为火
- **坤宫**：腹，在内为胃，在病为蛊账，在外为肌肤，又为右肩、右耳，在病为疮
- **兑宫**：咽喉、胸膈、为肺，在内为咳嗽、噎疾、喘急、喑哑，在外为口齿、额角、右肋，在病为痞
- **干宫**：腿足，亦为头，在内为大肠，在病为膀胱、便闭、壅结，外则腿足筋骨疼痛，又为疮
- **坎宫**：在内为小肠、肾气、丹田，在病为寒疾、遗精、泄泻或淋漓、便闭，或茶酒久宿腹痛，在外为肾，在病为阴虚、疮痒、疝气
- **艮宫**：在内为脾，内病主虚胀，在外为腿足脚气，在病为麻木风湿，又为疮
- **震宫**：在内为肝胆，在外为左肋，内病则为血虚、为痨症、吐血、寤寐、惊悖、狂言，外病则目盲、耳聋及肤疯、癞疮痛、左股肋眷之疾
- **巽宫**：在内为胃、口、膏盲，又为胆病，内则中风不语、肝肺相伤、三焦虚炎、感伤风热、喘急，在外为左耳、左肩、左胘，又为筋，外病则手足浮热、四肢无力，为火为疾，又为瘫疾、狂悖、筋舒之疾

再以天芮落宫所带之干，验其寒热虚实，详审节气时令，方为的验。

### 占病吉凶（卷八）

以天芮之落宫为病，以生死二门推之。天芮得生门者，生；得死门者，死。又看天芮落干、兑二宫为旺，不能治。落离宫、中五，其病缠绵。落震、巽二宫，病神受克，主不药而愈。落坎宫为休囚，病虽缠绵，犹可医治。新病落空亡者，生；久病落空亡者，死。又看日干带死囚之气，带凶神、凶格，或不得奇门者，亦死。若天芮落宫乘凶神、凶格，日干虽得旺相气，而被天芮来冲克年命者，亦死。又看病人年命日干（即生日八字中之日干）入墓者，亦死。衢仙曰："死门加生门，占病，死者复生"。

## 上元经补充

### 年命在疾病占测中的优先地位

《奇门遁甲上元经》疾病章的核心突破之一是指出**年命落宫在病占中的作用高于值符**。

不同年龄的人即使局象完全一致，也不能作相同吉凶论断：

| 情况 | 年命宫旺相 | 年命宫入墓且空亡 |
|------|-----------|---------------|
| 天芮旺相、庚加丙、符使破败 | 疾病严重但不危及生命 | 疾病对生命构成严重威胁 |

> **核心原则**：疾病预测最关心的是"病是否会对生命造成影响"，而年命从总体上反映生命体征。类似高血压、冠心病等不易根治的慢性病 → 值符宫破败但年命宫无大问题 → 该病一段时间内不会对生命造成致命威胁。
> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]疾病章

### 不同年龄阶段的疾病占测区别

#### 幼婴孩童（代占）

- 问事纯粹，只用看健康，不用顾及财官
- 符使和时干集中反映健康相关问题
- 在年命落宫基础上结合符使、时干综合判断

#### 成年人

- 表面问疾病，实质是问"病是否会影响求财/求官/求名"（**疾病预测的二重性**）
- 符使落宫反映的不一定是疾病本身发展趋势，而是病对工作事业的影响
- 需要采用**满盘占**的论断思路，不放弃每个宫内符号含义及其与他宫联系

> **案例**：运动员求测伤病——天芮旺相冲克日干，符使破败且空亡。不能断"年命不保"，实际运动员关心的是"能否参加比赛"。

### 脏腑宫位的精细区分

上元经在传统九宫对应基础上，提出了几个关键宫组的精细区分：

#### 震巽二宫（肝胆）

| 宫 | 主 | 辅 | 依据 |
|-----|------|------|------|
| 震宫（四正宫） | 肝（脏，主） | 胆（腑，表） | 乙木禄地在卯，居震宫为乙木正地 |
| 巽宫（四维宫） | 胆（腑，辅） | — | 四维主外，肝部问题很少显于巽宫 |

#### 坎坤兑三宫（生殖疾病）

- 坤宫 → 内生殖器官（子宫等）病变
- 兑宫 → 女性生殖疾病 + 肾（兑为少女）; 丁加己/己加丁落兑宫 → 非正常性行为引起的生殖疾病

- 兑宫：肺部内热引起的症状

#### 艮坤二宫（脾胃）

| 宫 | 特点 | 病机 |
|-----|------|------|
| 艮宫 | 左腹部，木临官、火长生之地 | 阳气运化不畅 |
| 坤宫 | 右腹部，水长生、金临官之地 | 阴气运化不畅 |

### 九星在疾病中的完整角色

上元经指出，九星并非只有天芮才有疾病诊断价值，八星各有特定角色：

| 九星 | 疾病角色 | 特征 |
|------|---------|------|
| **天芮** | 主要疾病/疾病的主要方面 | 病占首观之 |
| **天蓬** | 阴症、寒症、性病 | 现代因贪恋空调诱发关节炎也可表征 |
| **天柱** | 肺部/喉咙呼吸器官 | 也可指脊柱（辛加戊落坎宫逢伤门 = 脊柱下侧增生） |
| **天冲** | 向上生发之气的阳症 | 肝胆内热上冲→头昏目赤耳鸣耳聋 |
| **天英** | 炎燥特性（无上冲力） | 肝胆单纯阳燥→补阴为主（六味地黄丸）而非泻肝 |
| **天辅** | 风邪（外感） | 外感+内毒=表里皆实（防风通圣丸类） |
| **天任** | 中气匮乏 | 脾虚胃热造成的虚火→健脾养胃 （天任与天芮对立：天任在先→良性肿瘤；天芮在后→恶性肿瘤） |
| **天心** | 心脏疾病 + 脑疾病 | 双重特性（思维+实体器官） |
| **天禽**（飞盘独有）| 至中至正的化育之机、先天体质 | 不指代具体脏器，指人身中气所在 |

**星辰治病角色**：
- 天心星 = 病患心理特征和认知思维（心 = 精神力量），而非西药象征
- 天辅星 = 气功修炼的疗效，吐故纳新
- 天任星 = 患者体质强弱，能食尚饮、步伐矫健 → 病虽在身无大碍

### 天干病象深化

#### 甲乙之辩
- 甲隐不现 → 乙承担实际功用
- 乙弯曲之象：血管、神经、经络
- 乙加戊落离宫（戊阻塞被暗支冲动 + 惊门 + 螣蛇）→ 惊吓造成的血管栓塞
- 乙单独落坎宫（不搭配丁）→ 女性阴道与输卵管，非男性生殖器

#### 丙丁之辩
- 丙火（盛大阳刚外显）：炎症、烫伤、流血、伤口等显性症状
- 丁火（阴火）：心脏、眼睛（自身不发光热，由丁表征）
- 地盘丁 + 吉星吉门 → 多干遇之皆主祥瑞，可转危为安

#### 戊己之辩
| 天干 | 特征 | 对应器官 | 肿胀性质 |
|------|------|---------|---------|
| 戊（阳） | 隆起、肿大、阻塞、干燥 | 鼻、乳等突起器官 | 栓塞性、增生性 |
| 己（阴） | 下陷、非突起、阴湿 | 口腔、肛门 | 囊肿（含液体成分），刑/穿/墓下可恶化 |

#### 庚辛之辩
- 庚 = 病情加重象征（天芮逢庚 = 严重疾病，但不一定是绝症）
- 庚代表最严重的不治之症，是病占首要解决的焦点
- 辛 = 小的/细的/坚硬的颗粒（如乙加辛落震宫 = 胆结石）
- 庚力量大于辛：庚落艮宫（墓/刑/冲并临）仍可产生较大危害

#### 壬癸之辩
- 壬胫癸足（体外之象）
- 壬癸落巽宫 → 腿脚毛病 + 车祸概率高
- 水主隐私 → 生殖器/性行为
- 壬（阳水）= 主动一方的性行为；癸（阴水）= 女性一方或隐蔽性行为
- 天壬地癸 = "幼女奸淫，家有丑声"——天壬起传播作用

### 救神体系的多元结构

#### 人盘救神

- **丁奇**（手术/针灸） → 以克制庚（丁克庚），刚性医疗

- **开门**（手术通路）→ 与丁奇共同构成手术全过程（丁在前=手术刀切开，开在后=切口形成）

> 乙庚之合（合制）比丁克庚（克制）更具优势——反映中西医文化认知差异。

#### 星盘救神

- 天心星 → 病患心理素质（非西药）、精神力量
- 天辅星 → 气功修炼疗效
- 天任星 → 患者体质强弱

#### 救神与病神的宫位生克复杂关系

| 关系 | 判断 |

| 三合中病神为中神 | 病情加重，医疗效果失当 |
| 救神宫印比病神宫（无三合） | 救治思路不对，反使病情加重 |

### 病星空亡的四种状态

| 空亡类型 | 含义 |
|----------|------|
| **真空** | 根本没有病（多是父母代占装病的孩子） |
| **假空** | 暂时没病但未来某时段会患病——**病占空亡判断的核心**（转盘奇门更擅长此判断） |
| **动空** | 疾病已经康复，空亡宫符号搭配表示曾经的疾病 |
| **半空** | 可能患有某种疾病的可能，但缺乏断定为具体疾病的客观标准（如甲流病毒携带者，无症状） |

> 真空与半空的区别：年命、值符落宫表象一致，但时干宫在半空状态下会出现墓/迫/刑/冲等负能量符号，且病星落宫符号与时干宫及其他负能量宫位形成三合/五合/六合内在联系。
