# 怀孕分娩预测

## 定义与基本属性

怀孕分娩预测是通过奇门遁甲排盘判断是否怀孕、胎儿情况、分娩顺逆及胎儿性别等问题的应用。以坤二宫为母体，天芮星为孕育之神，综合九宫八门判断生育全貌。

> **核心原则**：坤为母为腹，天芮星为孕育之神。坤宫与天芮星的旺衰直接反映母子状态。

## 核心用神

| 用神 | 含义 |
|------|------|
| 坤二宫 | **母体**、子宫、腹部、孕育之地 |
| [[天芮星]] | **孕妇**、孕育之神（此处置为正面角色） |
| [[时干]] | 事体，判断分娩是否顺利 |
| [[日干]] | 求测人 |

> **注意**：在疾病预测中天芮星是病神，但在怀孕预测中天芮星是孕育之神，为吉。同一符号在不同应用中含义不同，初学者要灵活理解。

## 取用神方法

1. **测是否怀孕**：看坤二宫与[[天芮星]]的旺衰、空亡情况。
2. **测胎儿情况**：看[[天芮星]]所在宫的天盘干与地盘干。
3. **测分娩顺逆**：看[[时干]]与坤二宫、[[天芮星]]的生克。
4. **测性别**：看[[天芮星]]所临的星门与天地盘干的阴阳属性。

### 取用流程
```
排盘 → 看坤二宫状态 → 看天芮星落宫 → 看时干与坤宫/天芮生克
     → 看八门吉凶 → 看八神辅助 → 综合判断
```

## 判断要点

### 是否怀孕判断
1. **坤二宫旺相且不空亡**：有孕之象
2. **天芮星旺相且不空亡**：胎儿存在且健康
3. **坤二宫临[[开门]]或[[休门]]**：怀孕过程顺利
4. **坤二宫空亡或死门**：可能未怀孕或有流产风险

### 胎儿情况判断
1. **天芮星临吉门（开休生）**：胎儿健康安好
2. **天芮星临凶门（死伤惊）**：胎儿有险，需注意保胎
3. **天芮星临[[杜门]]**：胎位可能不正，或有堵塞
4. **天芮星逢[[九天]]**：胎儿发育快，或胎位偏上
5. **天芮星逢[[九地]]**：胎儿偏小，发育偏慢

### 分娩顺逆判断
1. **时干生坤二宫**：分娩顺利，母子平安
2. **坤二宫克时干**：分娩有困难，可能需要剖腹
3. **时干与坤宫比和**：顺产可能性大
4. **时干克坤宫**：分娩有险，需格外注意
5. **临[[反吟]]格局**：分娩过程有反复，可能漫长
6. **临[[伏吟]]格局**：分娩迟缓，可能需要催产

### 胎儿性别判断
传统奇门判断性别的方法（仅供参考）：
1. **天芮星临阳星阳门**：多主男孩
2. **天芮星临阴星阴门**：多主女孩
3. **天地盘干阳多阴少**：偏向男孩
4. **天地盘干阴多阳少**：偏向女孩

> **提示**：性别判断仅为古法参考，不可作为现代医学的依据。

## 实占要领

1. **先定坤宫**：坤二宫是母体之宫，第一步必须看它的旺衰空亡。
2. **次看天芮**：天芮星在此为孕神，不是病神，要转换思维。
3. **时干为关键**：时干代表分娩这件事，与坤宫天芮的生克直接决定分娩顺逆。
4. **八门定程度**：三吉门（开休生）在分娩预测中尤为重要。
5. **格局定特征**：[[反吟]]反复，[[伏吟]]迟缓——这两个格局在分娩预测中特别重要。
6. **八神辅助**：临[[太阴]]有隐情（如未知的并发症），临[[腾蛇|螣蛇]]有疑虑。
7. **不可单一判断**：坤宫、天芮、时干三者必须综合，不能只凭一宫下结论。

## 上元经补充

### 孕产五阶段时序用神体系

《奇门遁甲上元经》孕产章的核心突破是提出孕产不是单一事件，而是一个时间上的线性链条，每个阶段用神不同。这是避免"单一方法、单一用神"错误的关键。

| 阶段 | 用神 | 判断内容 | 五行位置 |
|------|------|---------|---------|
| **备孕阶段** | 乙庚落宫 + 年命 + 天芮星 | 双方身体/工作/生活条件是否适合备孕 | 气化 |
| **性生活** | 六合落宫 | 夫妻生活和谐度、精子卵子融合的前期条件 | 气→液化 |
| **胚胎形成前** | 坎宫 | 精子在女性生殖器内部的运化过程 | 液化 |
| **胚胎成形** | 坤宫（主战场）| 母体腹部胚胎发育成形全过程 | 成形 |
| **婴儿降生** | 震宫 | 新生儿脱离母腹时的状态 | 出生 |

> **核心原则**：移步换景的用神取用方法——根据孕产周期进度依次观察六合→坎宫→坤宫→震宫，步步为营。
> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]孕产章

### 阶段一：备孕条件判断

首先看双方是否具有孕产生子的条件（"合化之基"）：

1. 观察乙、庚、丁等符号落宫及各自年命落宫
2. 注意是否存在墓、刑、迫等问题
3. 区分负能量来自工作生活还是身体
4. 身体方面 → 看与天芮星落宫是否有内在符号联系

> **注意**：年命落宫在孕产预测中具有重要价值（揭示先天穷通祸福与身体状况），这在婚姻预测中反而不重要。因为婚姻重乙庚关系，孕产重先天体质。

**特殊情形**：男性求测但备孕对象不是正妻乙（妾室或"小三"）→ 观察丁火落宫而非乙，并询问丁火年命。

### 阶段二：六合宫——新生命开始的第一阶段

六合宫反映夫妻性生活质量：

| 六合宫内搭配 | 含义 |
|-------------|------|
| 六合空亡 | 夫妻最近无性生活 |
| 六合空亡逢马星 | 有正常夫妻生活但备孕未成功 |
| 天干入墓/击刑 | 天盘干问题 → 男方（勃起障碍、早泄、精子质量差）；地盘干问题 → 女方 |
| 戊加丙/戊加丁（阳性充足） | 未来宝宝先天体质上佳 |
| 五合 | 夫妻生活圆满，备孕顺利 |
| 明冲暗冲 | 夫妻生活不协调 |
| 丁加癸（备孕阶段） | 丁火被冲，备孕成功概率不大 |

> **法则**：六合宫重阳抑阴、喜合惧冲。吉星+吉门+吉宫可抵消暗冲格的负面因素。

### 阶段三：坎宫——承上启下

坎宫描述胚胎形成之前精液在女性生殖器内部的运化过程：
- 重阳抑阴、喜合惧冲（与六合宫基本一致）
- 坎宫为子地，一阳初生，丙丁乙戊等阳性干支落坎宫均处于衰弱状态——阳尚微小，经不起冲击
- 需注意离宫（对宫）是否被引动冲击坎宫（离宫旺相冲击 → 威胁生命初生体）

> **纠偏**：精子不能用辛金表示（辛为阴金，有毁折/死亡特性，与精子阳性生发性质不符）。精子应以丁火表征（阳性生命初始元素）。

### 阶段四：坤宫——孕产预测的主战场

#### 九星在坤宫的作用

- 地盘天芮星 → 孕母之象（承载、容纳、静态）
- 天盘星辰 → 胚胎发育初期情况（星为天为先，时序最靠前）
- 天盘吉星旺相 → 胚胎发育顺利；天盘凶星旺相 → 发育有麻烦
- 星辰吉凶 → 反映孩子先天品性（性在命先）

> **纠偏一**：天盘天芮与地盘天芮性质完全不同，天盘天芮是病星（反映疾病部位），千万不能混淆。
> **纠偏二**：天盘星辰与地盘星辰不能产生直接的五行生克（天上星辰永驻夜空，不像地面生物一样经历生老病死）。
> **纠偏三**：九星不能用来判定胎儿性别。九星统辖怀胎初期（4~5周才能区分性别），展现的是先天品质属性（"性命"中的"性"），而非性别。

#### 天盘干在坤宫的落宫状态

天盘干象征胎儿，地盘干象征母亲。母子是血亲联系，与普通主客阴阳不同（不能简单用上下干五行生克论）。

| 天盘干状态 | 吉凶 | 说明 |
|-----------|------|------|
| 长生、沐浴、冠带 | 吉 | 由小到大的增长阶段，与胎儿成长一致 |
| 临官、帝旺 | 凶 | 胎儿最弱小阶段反遇旺气——往往有先天发育缺陷 |
| 衰 | 小凶 | 先天体质较弱（如出生时体重较轻） |
| 病 | 较凶 | 需区分先天性疾病还是生产时问题（结合震宫判断） |
| 死、墓、绝 | 最凶 | 须有三奇吉门相助，不能出第二个不吉符号 |
| 胎、养 | 吉 | 与怀胎之象相符 |

**壬加乙、壬加戊**：胎产专用吉格。壬为妊娠，坤宫为壬长生之地，恰好与婴儿初生象意吻合。壬加戊中戊被穿（子未穿）→ 往往是剖腹产之象（不如顺产舒服但不致命）。

#### 天地格局在坤宫的判断

> **八字总诀**：重阳抑阴、喜合惧冲

- 全阴格局 → 不吉利（如庚加壬移荡格，庚处临官，肃杀最强 → 滑胎/流产）
- 五合格局（不遇入墓击刑）→ 和谐，表示母子相处融洽
  - 丁壬合最佳（丁壬落宫状态俱佳）
  - 丙辛合次之
  - 庚乙合（天盘庚临官）、甲己合（己击刑）、戊癸合（癸击刑入墓，能量散失最大）→ 凶性渐增
- 明冲暗冲 → 百无一吉
  - 明冲主速、阳战，祸更烈（如戊庚、乙辛、丙壬）
  - 暗冲为迟（如戊辛、壬己，穿刑贯穿其中，双重打击）

> 三奇落坤宫各自特点：丁临沐浴冠带（最吉）、乙临胎养（吉）、丙临衰病——但丙火力量弱反而不是坏事（强旺反如"婴儿暴晒"）。

#### 门盘在坤宫（孕产末期）

门反映临产时胎儿脱离母体的通路：
- 开门 → 最吉，联通内外，胎儿降生通路大开
- 休门 → 安然，不会遭遇过多痛苦
- 生门 → 返吟，产程反复不顺利
- 伤门门迫 → 母子生产时受伤（需分干衰旺墓刑穿判断伤在谁）
- 杜门门迫 → 阻塞之象，大概率难产
- 死门 → 八门伏吟，脱体困难
- 景门 → 可能出血
- 飞盘中门 → 中庸，无吉无凶，趋于普通

> **门迫优先级**：门迫出现时，伤门/杜门的危害大于死门。

**剖腹产的判断**：仅临一个开门不能断剖腹产，必须有对应地盘干出现穿制符号（如丙加戊落坤宫，戊逢子未穿 + 开门）。

#### 坤宫月令旺衰的先天体质含义

坤宫月令旺衰表达的是孕母对怀孕先天承载力的强弱。无论预测时间是哪个月，只看该月坤宫旺衰 → 断定孕母先天体质。囚死二态时即使宫内搭配良好，也要多加小心，需结合符箓调节。

### 阶段五：震宫——婴儿降生

震为动，"帝出乎震"的阳气动象对所有初生婴儿（无论男女）都有统辖作用。震宫门迫/入墓/击刑等 → 婴儿降生时可能面临疾病或突发事件（如黄疸）。

### 时干与符使在孕产中的正确用法

**时干**：在婴儿出生前，时干不能直接指代胎儿（胎儿不具有独立自主个体地位），只指代与胎产相关的**事件**。仅在临产一刻，时干才可能有代表胎儿的含义。

**符使**：胎产是婚姻的延伸（乙庚合化的产物），值符管整个家庭全局，不直接反映孕产各阶段母子吉凶。值使门代表孩子降生后繁杂琐碎的家庭生活。

> 产母直符子六合，刑因旺相吉凶诀，蛇白催六是生期，地阴合六当出月，推求时日两不伤，子母团圆不须说，专责符元起何亲，轮飞之宫可生合，吉凶男女细参详，阴阳奇耦官仪拨。

### 胎产占——值使诀

> 值使推详罴与熊，门吉阳宫产豹龙，阴门阴宫瑶池女，刑格空亡最忌逢，宫制门兮多不育，门制宫兮畧有惊，若卜胎产全吉利，宫马相生天马腾。

### 胎产占——九星诀

> 九星阴阳判胎孕，旺相休囚时日论，胎逢死废孤与虚，起飞宫次五行门。入墓逢生辨速迟，当今孕寔产将近，就中形体可参详，入鬼入财与入印，化空逢生产物客，化死坐墓产忧门。

### 胎产占——九宫诀

> 九宫起元分节气，节气分时犹阻滞，胎临马兮当转移，长生得马产育易，金火逢空将临盆，水木逢空拟不济，废土空陷必堕胎，酌取年命断来意。

### 胎产占——门户三甲诀

> 占断胎产有妙诀，门户阴阳甲开合，阳开问产尽亨通，阴阖胎安不须忌。

## 金函玉镜补充

### 占生男女长命（卷十）

以所生时干天盘所落之宫，看是何星？何神？以天蓬为天贼，玄武为偷生，犯此二神，再乘休废无气之时，主不能养。若有奇门吉格、吉神，再乘旺相之气，主长命富贵。

### 占孕何日生（卷十）

以坤宫为产室，天芮为产母，天盘所得之星为小儿。天芮克天盘之星者，主产速。天盘星生地盘星天芮者，子恋母腹，产迟。天盘星克地盘星者，主母凶。地盘星克天盘星者，主子亡。若得旺相气及奇门吉格者，方吉。如天盘星落地盘库，子死母腹内。天地二盘乘凶门、凶格者，子母俱凶。以天地二盘乘死绝之气也。

### 占胎息男女（卷十）

其占法以坤宫、天芮星为母，以天盘星临坤之宫为胎息，阳星为男胎，阴星为女胎。惟天禽临之为双生，阳干是男，阴干女也。
