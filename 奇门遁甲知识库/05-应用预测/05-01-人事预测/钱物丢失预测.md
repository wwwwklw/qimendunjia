# 钱物丢失预测

## 定义与基本属性

钱物丢失预测用于判断失物去向、能否找回、是否被盗以及盗贼特征等问题。以日干为失主、时干为失物为核心，玄武为盗贼，天蓬星为抢劫犯，二者在丢失案件中的凶恶程度不同，要注意区分。

> **核心原则**：时干（失物）与日干（失主）的生克关系决定失物能否找回——生我则能回，克我则难寻。

## 核心用神

### 失物主与失物
| 用神 | 含义 |
|------|------|
| [[日干]] | 失主（丢东西的人） |
| [[时干]] | 所失之物（丢失的物品） |

### 盗贼用神
| 用神 | 含义 | 区别 |
|------|------|------|
| [[玄武]] | **小偷**、盗窃、暗中偷取 | 暗偷 |
| [[天蓬星]] | **抢劫犯**、公然抢夺 | 明抢 |

### 辅助用神
| 用神 | 含义 |
|------|------|
| [[杜门]] | 隐藏位置（失物可能在此） |
| [[伤门]] | 被破坏或损坏的失物 |
| [[六合]] | 转手方向（失物被转卖的可能） |

## 取用神方法

### 第一步：判断是丢还是偷
1. **[[玄武]]出现在盘中且与时干有关**：被偷
2. **[[天蓬星]]旺相且克时干**：被抢
3. **无玄武无天蓬**：自己遗失，非被盗窃

### 第二步：判断失物去向
1. **时干所在方**：失物目前所在方位
2. **[[杜门]]所在方**：失物被隐藏的方位
3. **时干与日干同宫**：失物没有被偷，在自己身边
4. **时干生日干**：失物能找回（▲吉）
5. **时干克日干**：找回困难（▼凶）

### 取用流程
```
排盘 → 看有无玄武/天蓬 → 判断丢失or被盗 
     → 看时干落宫 → 看时干与日干生克 → 看杜门位置 
     → 判断能否找回 → 综合判断
```

## 判断要点

### 丢失类型判断
1. **[[玄武]]临[[时干]]**：物品被小偷偷走
2. **[[天蓬星]]临[[时干]]**：物品被公然抢夺
3. **[[玄武]]+[[天蓬星]]同宫**：暴力盗窃/入室抢劫
4. **无玄武天蓬**：自己不小心遗失，非人为盗窃

### 能否找回判断
1. **[[时干]]生[[日干]]**：失物会回到失主手中（★★★ 最佳）
2. **[[日干]]生[[时干]]**：失物还在但需努力寻找
3. **时干与日干比和**：失物没走远，在附近
4. **[[时干]]克[[日干]]**：失物难找回，可能已毁坏（▼凶）
5. **时干空亡**：失物已不在，找回无望
6. **时干入墓**：失物被藏匿，不易找到

### 盗贼特征判断
1. **[[玄武]]临[[白虎]]**：盗贼凶狠，可能携带凶器
2. **[[玄武]]临[[伤门]]**：盗窃过程中有暴力行为
3. **[[玄武]]临[[杜门]]**：盗贼技术型（如黑客、技术开锁）
4. **[[玄武]]临[[生门]]**：熟人作案（有利可图才下手）
5. **[[玄武]]所临宫位判断盗贼方位**

### 追回应期
1. **[[庚格定应期|庚格]]出现**：能追回的时间信号
2. **时干填实空亡**：失物出现之时
3. [[玄武]]落宫被冲之日：盗贼落网之时

## 实占要领

1. **先判性质**：第一件事不是找失物，而是判断是丢失还是被盗。看玄武、天蓬星的有无。
2. **时干为失物**：失物所在方位即时干落宫。
3. **生克定归否**：时干生日干则能回，时干克日干则难寻——这是失物预测的铁律。
4. **区分盗贼**：玄武是小偷（暗），天蓬星是劫犯（明）——两者性质完全不同，判断推理也不同。
5. **杜门看隐藏**：失物被藏在何处，看杜门落宫方位。
6. **空亡最凶**：时干空亡几乎等于找不回来了。
7. **熟人作案看生门**：玄武临生门，很可能是熟人作案（因为有利可图才会被下手）。

## 开悟之门补充

以下案例引自《开悟之门》，原案例来自《神奇之门》书中实例，张志春老师用以讲解钱物丢失预测中的速度快慢和庚格定应期方法。

### 开悟之门补充：找发票实例——反吟主快

《神奇之门》实例一测找发票：日干己、时干壬均在内盘主快，**八门反吟**也主快。值使休门落9宫和时干落3宫均主3天左右。时干为丢失之物用神，按时干临星阴阳找庚格——时干壬临天禽阳星，"时干临阳星看庚下之干"，6宫庚下之干为癸，判定癸日可找到（12月23日癸未）。实际12月22日（壬午日）即找到，比断期提前一天。

> **要点**：内盘+反吟双重快象叠加，应期极近；庚格法精确到日，误差仅一天。

出处：《开悟之门》

### 开悟之门补充：找钱实例——庚下之干定应期

《神奇之门》实例五测失钱：时干为失钱用神，时干壬临**天蓬阳星**，按"时干临阳星，看庚下之干"的方法——巽4宫庚下之干为丙，3月30日为丙寅日，故断3月30日可找到。实际在3月30日将自己挪动了收藏之地的**6800元**人民币如数找到。

> **核心**：钱物丢失预测中，寻找应期的关键步骤：
> 1. 确定快慢（内盘/外盘、反吟/伏吟）
> 2. 卡定大致天数（值使门/时干落宫之数）
> 3. 用庚格法精确锁定日干支（时干临阳星看庚下之干，临阴星看庚上之干）

出处：《开悟之门》
