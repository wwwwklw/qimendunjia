# 行人走失预测

## 定义与基本属性

行人走失预测用于判断走失人员的方向、状态、安危以及归期。根据求测人与走失人的关系，取不同天干为用神——父母测子女用[[时干]]，测同辈用[[用神|月干]]，测长辈用[[年命|年干]]。这是六亲关系在走失预测中的典型应用。

> **核心原则**：以六亲定用神（年/月/时干按辈分），以六合定方向，以杜门定躲藏方位。吉门多则不思归，凶门多则易自回。

| 测同辈/配偶 | [[用神|月干]] | 月干为兄弟宫 |
| 测长辈/父母 | [[年命|年干]] | 年干为父母宫 |

## 取用神方法

### 第一步：确定走失人用神
```
求测人是谁 → 确定与走失人关系 → 取对应天干为用神
父母测子女 → [[时干]]
测同辈     → [[用神|月干]]  
测长辈     → [[年命|年干]]
补充       → [[年命]]（走失人出生年天干）
```

### 第二步：确定方位
```
[[六合]]落宫方位 → 走失方向
[[杜门]]落宫方位 → 躲藏/隐蔽方向
```

### 第三步：判断状态
```
用神临吉门（开休生）→ 安全
用神临凶门（死伤惊）→ 有危险
用神临[[杜门]]→ 自行躲藏不出
用神临空亡 → 可能不在寻找范围内
```

### 取用流程
```
排盘 → 六亲定位用神 → 看用神落宫吉凶 
     → 看六合落宫（方向） → 看杜门落宫（躲藏） 
     → 看八门吉凶（回否） → 看庚格/马星/空亡（归期）
```

## 判断要点

### 去向判断
1. **[[六合]]落宫方位** = 走失人前往的方向
2. **[[杜门]]落宫方位** = 走失人躲藏/隐蔽的方向
3. **六合与杜门同宫**：走失人在较为封闭的地方

### 归否判断（八门定归心）
| 用神所临 | 判断 | 原因 |
|----------|------|------|
| [[开门]]、[[休门]]、[[生门]]、[[杜门]] | **不思归** | 安全无事/躲藏不出，不会主动回来 |
| [[伤门]]、[[景门]]、[[死门]]、[[惊门]] | **易自回** | 有困难危险，反而容易主动回家 |

> **记忆法**：吉门+杜门（四门）外出舒服，不思归；凶门四门外受困苦，愿回。

### 安危判断
1. **用神临[[九天]]**：走失人去了远处/高处
2. **用神临[[九地]]**：走失人去了近处/地下室
3. **用神临[[死门]]**：走失人有生命危险
4. **用神临[[伤门]]**：走失人可能受伤
5. **用神临[[惊门]]**：走失人受到惊吓

### 归期判断
1. **[[庚格定应期|庚格]]出现**：归期的最强信号——逢庚即回
2. **[[马星]]到用神宫**：走失人在移动中，快被找到
3. **用神空亡填实**：空亡期结束即归
4. **用神临[[反吟]]**：去而复返，反复走动
5. **用神临[[伏吟]]**：停留在某处不动

## 实占要领

1. **六亲定位第一**：取错用神全盘皆错。父母测子女→时干，测同辈→月干，测长辈→年干。
2. **六合定方向**：六合主出行走动，其落宫就是走失方向。
3. **杜门定躲藏**：如果是离家出走/故意躲藏，重点看杜门落宫。
4. **八门定归心**：四门不思归（开休生杜）、四门易自回（伤景死惊）——此是走失预测的独特规律。
5. **吉凶看用神**：走失人用神的宫位吉凶直接反映走失人的处境。
6. **归期看三信号**：庚格（最强）、马星（移动）、空亡填实（时间节点）。
7. **年命为补充**：如果知道走失人的出生年命，作为用神的补充参考。

## 开悟之门补充

以下案例引自《开悟之门》，原案例来自《神奇之门》实例，张志春老师用以讲解行人走失预测中八门吉凶判断、用神旺衰、庚格定应期等核心方法。

### 开悟之门案例：测女儿离家出走（实例五）

- **时间干支**：卯月（1997年3月7日离家）
- **奇门格局**：时干壬落巽4宫临死门；死门落4宫受木制，且卯月木旺土死；日干、时干一内一外主慢；值使开门落2宫；时干壬临天禽阳星；坎1宫庚下之干为戊
- **判断分析**：开始一见死门按凶断就错了；实际上死门落4宫受木制，卯月木旺土死，凶门根本凶不起来；用神临死、惊、伤、景四门反而容易回来；值使门落宫数大致卡在10天或12天内；按时干临阳星看"庚下之干"法找见戊日，查万年历3月17日为戊午日即为应期
- **实际结果**：女儿在男朋友家住了十天，因患重感冒，于3月17日自己回到家中

**要点**：凶门在失令受克时凶不起来；行人走失用神临死惊伤景四门反而容易回来（物极必反）；庚格法定应期——时干临阳星看庚下之干

### 开悟之门案例：测女孩失踪被害（实例七）

- **时间干支**：未月
- **奇门格局**：时干乙落兑宫临死门；死门为土，生兑宫之金，凶门生宫；未月死门旺相；上乘白虎凶神；乙落兑为绝地；乙+壬为日奇入地凶格
- **判断分析**：凶门生宫，事更凶；未月死门旺相，凶上加凶；白虎凶神加临，主血光之灾；乙落绝地，日奇入地，均主绝命入土之象，故断失踪女孩被人害死埋到地下
- **实际结果**：确已被害身亡

**要点**：凶门生宫事更凶；死门旺相（月令）决定凶的程度；多凶符叠加（白虎+绝地+日奇入地+死门旺相）判断必死

### 开悟之门案例：测外甥女失踪（实例九）

- **时间干支**：辰月（4月5日清明节后，刚入辰月，头九天乙木当令）
- **奇门格局**：用神时干辛落巽4宫；临死门、天柱凶星；甲午辛+甲子戊为困龙被伤格，但子午冲，冲则不为入墓；辰月土生金，时干辛为入辰库（非墓）
- **判断分析**：开始未细分墓、库区别，一律按入墓断，加上临死门、天柱凶星，错判"凶多吉少，恐已被害"；实则辰月土生金，辛金旺相，入辰库不为入墓；子午冲解了困龙被伤格；刚入辰月头九天乙木当令，木旺金囚土死，天柱星、死门均凶不起来；用神临死门反而容易找回
- **实际结果**：此女孩被安全找回

**要点**：入库与入墓必须严格区分（旺相入为库，休囚入为墓）；月令五行旺衰决定星门吉凶程度；用神受冲不为入墓；行人走失用神临死门有反归之机

### 开悟之门补充：用神临死门反易回——凶门反断法详解

《神奇之门》实例五测女儿离家出走是对"死门逢凶必死"惯性思维的**经典纠错**。时干壬落巽4宫临**死门**，初学者一见死门就按凶断——实际断错。纠正逻辑：

1. **五行生克决定凶力**：死门（土）落巽4宫（木），**木克土**，死门被宫所制（宫克门为"制"），凶不起来。
2. **月令决定旺衰**：卯月**木旺土死**，死门在月令上处于死地，更加无力。
3. **凶门在走失预测的反断规则**：行人走失中，用神临**死、惊、伤、景**四门反而容易回来——走投无路时会自己回家。

> **核心教训**：不能一见死门就断凶。必须综合分析：门被宫制则凶力大减；月令决定门的实际旺衰；特定预测类别（行人走失）中凶门有反断规则。

实际结果：女儿在男朋友家住了十天，因患重感冒，于3月17日自己回到家中。

## 金函玉镜补充

### 占走失（卷七）

以时干为失主（与六壬之用干同理），六合为逃走之物，俱以落宫论之。又以六合宫与时干落宫，看在内外以分远近。如日干、六合俱在内，易寻；俱在外，难寻。六合在外，时干在内，难寻。又以六合所在之宫为方向，如得旺相星，又乘开休生杜四门，不可得，反此可得。乘九地、太阴，有人潜藏；乘九天远走，乘玄武被人盗去，乘螣蛇有人盘诘羁縻，乘朱雀有信，勾陈有勾引而去。又看六庚，年格则年获，月格则月获，日格则日获，时格则时获。

### 占走失奴婢（卷十）

男仆专责天盘天蓬之宫，女婢专责天盘天芮之宫，寻之必获。阳日责地盘蓬、芮之宫决之，逢格必获，不格不获。若与六合相并者，必被拐去。二神入库，有人隐藏难寻。落空亡者，必不能获。

### 占走失何方（卷十）

失去小儿，阳遁于天盘六合所在之方，阴遁于地盘六合所在之宫决之。若失女婢，阳遁于天盘太阴所在之方，阴遁于地盘太阴所在之宫定之，看此二神，即知其人所在之方矣。阳遁在坎艮震巽四宫为内，为近；落离坤兑干为外，为远，阴遁反此。若二神落日干之墓库者，远近难寻。落空亡者，其人又往他方去，之后再以干合神之宫寻之，必获。

### 占行人在外（卷十）

法以行人年命日干决之。年命日干落在坎艮震巽，阳遁为内，为近；离坤兑干为外，为远，合而决之。年命落于四维甚远，落坤艮者，为极远。阴遁以离坤兑干为内，为近，坎艮震巽为外，为远。年命日干落于休废，其人在外必不如意，非困即病。若年命日干落于墓库、空亡之宫，再乘死绝之气，其人必不在。如年命日干乘旺相之气，再得奇门吉格，行人在外发财如意。

### 占行人归期（卷十）

法以行人年命专责庚格，阳日以庚下临之干主之，阴日以庚上乘之干定之。年格年来，月格月来，日格日来，时格时来，不格不来。何谓不格？乙庚为合不为格。庚金入墓或临空亡，不为格。
