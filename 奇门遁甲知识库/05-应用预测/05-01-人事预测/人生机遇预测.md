# 人生机遇预测

## 定义与基本属性

人生机遇预测是奇门遁甲中综合性最强的应用，用于判断一个人的一生命运、先天禀赋、事业成就、社会关系等。涉及六亲（年干/月干/时干）、九星（天时）、八门（人事）、九宫（地理）、八神（自然力量）的全面联动，是奇门遁甲最高层次的综合运用之一。

> **核心原则**：日干为本命，年干为父母上级，月干为兄弟同事，时干为子女下级。九星为天时运程，八门为人事际遇，九宫为地利环境，八神为神秘力量。人生预测需要兼看[[年命]]（出生年天干）。

## 取用神方法

### 综合取用流程
```
排盘 → 定位日干落宫（本人） → 兼看年命落宫（先天命数）
     → 看九星（天时运程） → 看八门（人事际遇） 
     → 看九宫（地利环境） → 看八神（神秘力量）
     → 看年干（父母上级） → 月干（同辈） → 时干（子女下属）
     → 综合分析一生运势
```

### 测先天素质
1. **出生时间的奇门格局**：可测一个人的先天禀赋和天赋倾向。
2. **[[年命]]落宫的旺衰**：先天根基的强弱。
3. **[[九星概述|九星]]在天盘与年命宫关系**：天赋特长的方向。

### 测当前运势
1. **日干落宫旺衰**：当前运势的基本判断。
2. **[[年命|太岁]]与日干生克**：当年运势好坏。
3. **[[值符（八神）|值符]]与日干生克**：贵人运的情况。

## 判断要点

### 本人运势（日干）
1. **日干旺相且临吉门吉星**：运势好，万事顺遂
2. **日干休囚且临凶门凶星**：运势差，困难多
3. **日干临[[值符（八神）|值符]]**：有贵人相助，社会地位高（▲大吉）
4. **日干临[[九天]]**：上升期，事业上升，地位提高
5. **日干临[[九地]]**：稳定期，但上升空间有限
6. **日干空亡**：运势虚空，没有方向和目标
7. **日干入墓**：运势受到压制，被困住

### 先天素质（年命）
1. **年命与日干同宫**：先天根基好，与当前生命统一
2. **年命生合日干**：先天禀赋能支持后天发展
3. **年命与日干相克**：先天根基与后天发展有矛盾
4. **年命临[[天辅星]]**：文运好，学业天赋
5. **年命临[[天英星]]**：知名度高，社交天赋
6. **年命临[[天心星]]**：技术型天赋，医学/机械
7. **年命临[[天蓬星]]**：胆大好冒险，军警天赋

### 父母/上级（年干）
1. **年干生日干**：父母爱护，上司提拔（▲吉）
2. **日干生年干**：孝敬父母，尊从领导
3. **年干克日干**：父母压制，上级打压（▼凶）
4. **年干临[[死门]]**：父母身体不好或已故

### 兄弟姐妹/同事（月干）
1. **月干生日干**：兄弟姐妹关爱，同事合作（▲吉）
2. **日干克月干**：控制同辈关系
3. **月干克日干**：同辈竞争打单（▼凶）
4. **月干与日干比和**：同辈关系和谐

### 子女/下属（时干）
1. **日干生时干**：为子女/下属付出，责任大
2. **时干生日干**：子女孝顺，下级服从（▲吉）
3. **日干克时干**：管教严厉，掌控有力
4. **时干克日干**：子女叛逆，下属反抗（▼凶）

### 四维综合判断
1. **天时（九星）生合日干**：天时有利，抓得住机遇
2. **人事（八门）生合日干**：人和际遇好，关系网牢固
3. **地理（九宫）生合日干**：地利佳，居所有利
4. **神助（八神）生合日干**：有神秘力量庇护

### 应期
- 日干由囚转旺之时（运势转好之始）
- [[马星]]到日干宫（人生变动的信号）
- 填实空亡之时（机遇来临）

## 实占要领

1. **日干是核心**：不管看什么，日干始终是人生预测的最核心用神——它代表求测人本身。
2. **年命为根**：出生年份天干（年命）看先天素质——这是定命的基础，不可不查。
3. **四维体系要完整**：九星（天时）+ 八门（人事）+ 九宫（地理）+ 八神（神助）——体系完整缺一不可。
4. **六亲看关系圈**：年干父母上级、月干兄弟姐妹同事、时干子女下级——用六亲断人生关系网。
5. **生克定好坏**：谁生我谁对我有利，谁克我谁对我不利——这是最简单也最重要的判断法则。
6. **旺衰看起伏**：旺相时运势上升，休囚时运势低迷，入墓空亡时运势最低。
7. **出生时间看先天**：如果有出生时间，排当时的奇门格局可测先天禀赋、天赋方向——这是人生预测的最高境界。
8. **不要只看一组**：人生预测是综合性最强的预测，切忌只抓一个用神就下结论。

## 开悟之门补充

以下案例引自《开悟之门》，原案例来自《神奇之门》实例，张志春老师以此讲解人生预测中地支填实判断、流年应期的原理。

### 开悟之门案例：测外籍医生四柱（实例五）

- **时间干支**：见《神奇之门》第416-420页
- **奇门格局**：该女1967年丁未、1968年戊申年结婚；地支未、申所在坤2宫临庚（为其丈夫），但逢空亡
- **判断分析**：地支未、申所在坤2宫逢空亡，表示婚姻之事尚未落实；空亡填实之年就是结婚之年——1967年丁未年填实未，1968年戊申年填实申；故断大约是1967年或1968年结婚
- **实际结果**：准确断出其结婚年份

**要点**：地支填实之年为应期——坤2宫未、申空亡，应于丁未年和戊申年；用神逢空亡，填实之年自然就可能是该事发生之年；这是从流年地支角度判断应期的典型方法

### 开悟之门案例补充：地支与流年应期判断

张志春在书中进一步讲解：奇门判断流年、流月、流日、流时，从地支角度可以找到对应宫位来判断。如判断1996年（干支丙子）的事情，首先观察子年对应的坎宫情况；测某人农历八月情况，看兑宫；测辰日情况，看巽宫；测戌时发生的事，看乾宫。这是属静、属阴的特性——十二地支与奇门九宫格的对应关系固定不变。
