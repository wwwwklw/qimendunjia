# 求学考试预测

## 定义与基本属性

求学考试预测是通过奇门遁甲判断考生学业运势、考试成绩、录取可能以及适合的学校和专业方向的应用。以天辅星（文昌星）为核心，配合丁奇（文书）、景门（试卷），形成完整的学业判断体系。

> **核心原则**：天辅星为文昌星，主学业文运；丁奇为文书、信息；景门为试卷成绩。三方联动，判学业之成败。

## 核心用神

### 三大核心
| 用神 | 含义 | 作用 |
|------|------|------|
| [[天辅星]] | **老师**、**学校**、文昌星，主文运学业 | 教育力量 |
| [[丁奇]] | **文书**、考卷、学历证书、录取通知书 | 考试内容 |
| [[景门]] | **试卷**、成绩、考试过程 | 考试结果 |

### 辅助用神
| 用神 | 含义 |
|------|------|
| [[日干]] | 考生本人 |
| [[时干]] | 考试这件事 |
| [[值符（八神）|值符]] | 主考官、招生办 |
| [[年命|年干]] | 父母（代测子女时） |

## 取用神方法

1. **测自己**：日干为考生，看日干与天辅星、丁奇、景门的关系。
2. **测孩子**：年干为父母测子女，时干为孩子（子女宫）。
3. **测录取**：看景门与日干的生克关系。
4. **测考试难度**：看[[天辅星]]旺衰（旺则题易，衰则题难反常规）。

### 取用流程
```
排盘 → 定位日干落宫 → 看天辅星落宫（学校/老师） 
     → 看丁奇落宫（文书/考卷） → 看景门落宫（成绩） 
     → 看三者与日干的生克 → 综合判断
```

## 判断要点

### 学业能力判断
1. **日干生[[天辅星]]**：学生好学上进，喜欢老师（▲吉）
2. **[[天辅星]]生日干**：老师赏识学生，学校有利（▲吉）
3. **日干克[[天辅星]]**：学生不尊重老师，自学为主
4. **[[天辅星]]克日干**：学校/老师对学生不利，学业压力大

### 考试成绩判断
1. **[[丁奇]]/[[景门]]生日干**：考试对考生有利，成绩理想（▲吉）
2. **日干生[[丁奇]]/[[景门]]**：考生适应考试，但需努力（△中）
3. **[[丁奇]]/[[景门]]克日干**：考试对考生不利，成绩不理想
4. **[[景门]]旺相且临吉门**：考试成绩优异
5. **[[景门]]休囚且临凶门**：考试失利

### 录取判断
1. **[[值符（八神）|值符]]（考官）生日干**：考官欣赏，录取有望
2. **[[开门]]生日干或与日干比和**：公开录取机会大
3. **[[丁奇]]临开门**：通知书到达之象
4. **日干临[[死门]]**：录取无望

### 应期判断
- [[丁奇]]旺相日（通知日）
- [[景门]]旺相日（成绩公布日）
- [[开门]]临日干之日（开学/报到日）

## 实占要领

1. **分清角色**：日干是考生、天辅是老师学校、丁奇是文书、景门是考卷——四个角色不能混淆。
2. **考试类看景门**：景门是考试预测的第一参考，旺衰吉凶直接反映成绩。
3. **学业类看天辅**：测长期学业运势，天辅星比景门更重要。
4. **重点盯生克**：景门、丁奇、天辅与日干的生克关系是判断的主线。
5. **旺衰辅助**：各用神旺相时力量大，休囚时力量小——同一生克关系旺衰不同，结论可相差很大。
6. **八神看环境**：临[[值符（八神）|值符]]有贵人相助（如保送），临[[太阴]]暗中有助力（如特殊政策）。
7. **代测注意**：测孩子学业时，取时干为孩子（辅以年命），再按上述方法判断。

## 开悟之门补充

以下案例引自《开悟之门》，原案例来自《神奇之门》实例，张志春老师以此讲解考试预测中定应期的方法。

### 开悟之门案例：测在美国考行医执照（实例五）

- **时间干支**：1997年
- **奇门格局**：杜新会判断叶教授女儿最终能被录取；八门伏吟主迟；日干丁落外盘3宫、时干癸落内盘9宫，日时一内一外也主迟；迟断年月，断年不合事理故定断月；丁奇为通知书，天盘丁奇在3宫，地盘丁奇在4宫；巽4宫辰、巳逢空亡
- **判断分析**：空亡藏有玄机——通知书什么时候来，看巽4宫辰巳空亡何时填实；戌冲辰，故断阴历九月（戌月）；按1997年阳历10月8日交寒露节进入戌月，故断10月7日以后才能收到录取通知书
- **实际结果**：10月9日收到通知书（比预测晚1天）

**要点**：空亡填实/冲实为应期——辰巳空，戌冲辰即应期（戌月填实）；以节气定月份比以阴历定月份更准确（若按阴历九月断10月2日以后，与实际差7天；按节气戌月断10月8日以后，与实际仅差1天）

### 开悟之门案例补充：定应期以节气为纲

- **判断分析**：奇门判断应期时，应以节气为主、以阳历时间为主；二十四节气与八宫等分后，与阳历时间的对应更为稳定（最多相差一两天）；杜新会在本例中断10月7日以后收通知书，正是基于交寒露节气进入戌月这一判断
- **实际结果**：10月9日收到通知书，判断基本准确
