# 体育竞赛预测

## 定义与基本属性

体育竞赛预测是奇门遁甲的应用之一，脱胎于军事占测——以主客场或攻守双方划分主客，以时令长短分胜负。其核心在于主客判断：反吟利客（进攻方），伏吟利主（防守方/主场方）。

> **核心原则**：竞赛如战场，主场为守、客场为攻。反吟格局下客方获胜概率大，伏吟格局下主方获胜概率大。

## 核心用神

### 竞赛双方
| 用神 | 含义 |
|------|------|
| 日干落宫 / 主场方 | **主方**（主场队伍、防守方） |
| 时干落宫 / 客场方 | **客方**（客场队伍、进攻方） |

### 赛事用神
| 用神 | 含义 |
|------|------|
| [[庚（阻隔之神）|庚]] | **比赛器械**（球类比赛的球、赛车比赛的车辆、兵器比赛的武器） |
| [[景门]] | 比赛成绩、得分情况 |
| [[开门]] | 比赛场馆 |
| [[时干]] | 比赛这件事体 |

## 取用神方法

### 第一步：划分主客
1. **主场队伍 = 主方 = 日干落宫**
2. **客场队伍 = 客方 = 时干落宫**
3. 如果没有主场之分：日干为我方（关注方），时干为对方

### 第二步：看格局定倾向
1. **[[反吟]]格局**：利客（进攻方/客场有利）
2. **[[伏吟]]格局**：利主（防守方/主场有利）

### 第三步：判断胜负
1. **谁旺相谁赢**：旺相方占优
2. **谁克对方谁赢**：克对方方占优
3. **格局加持**：反吟+客方旺相=客赢概率极高

### 取用流程
```
排盘 → 判断是否是反吟/伏吟 → 划分主客 
     → 对比双方旺衰 → 看主客生克 → 看庚（器械） 
     → 看景门（成绩） → 综合判断胜负
```

## 判断要点

### 格局判断（最重要）
两个特殊格局直接影响竞赛结果：

| 格局 | 含义 | 对竞赛的影响 |
|------|------|-------------|
| [[反吟]] | 反复变化，动象强烈 | **利客方**，客队胜率高 |
| [[伏吟]] | 静止不动，平稳保守 | **利主方**，主队胜率高 |

### 力量对比判断
1. **日干（主方）旺相，时干（客方）休囚**：主方占优
2. **时干（客方）旺相，日干（主方）休囚**：客方占优
3. **日干落宫克时干落宫**：主方能克制客方（▲主方有利）
4. **时干落宫克日干落宫**：客方能克制主方（▲客方有利）
5. **双方比和**：势均力敌，可能平局

### 器械判断（庚）
1. **[[庚（阻隔之神）|庚]]生日干（主方）**：器材对主方有利
2. **[[庚（阻隔之神）|庚]]生时干（客方）**：器材对客方有利
3. **[[庚（阻隔之神）|庚]]临[[伤门]]**：器材可能出问题或有人受伤
4. **[[庚（阻隔之神）|庚]]临吉星**：器材状态好，比赛质量高

### 成绩判断（景门）
1. **[[景门]]生日干**：成绩偏向主方
2. **[[景门]]生时干**：成绩偏向客方
3. **[[景门]]临[[九天]]**：高分比赛
4. **[[景门]]临[[九地]]**：低分比赛

## 实占要领

1. **反吟伏吟第一看**：这是竞赛预测最重要的格局长——反吟利客，伏吟利主。
2. **主客分清**：主=我关注的方，客=对方/对方队——立场要明确。
3. **旺衰定实力**：双方落宫旺衰直接反映实力对比，这个比生克更直观。
4. **主客生克为辅**：主克客（主方克制客方），客克主（客方克制主方）。
5. **庚看器械**：球类比赛重点看庚——庚是球、是比赛器械，旺相则比赛精彩。
6. **景门看成绩**：比赛结果与景门的关系密切。
7. **综合判断不偏废**：反吟伏吟只是倾向，还要结合双方旺衰综合判断。
