# 人生命运预测古文详解

## 定义与基本属性

**人生命运预测古文详解**是《开悟之门》第五章收录的一篇奇门遁甲古文，系统阐述了如何用奇门遁甲预测一个人的终身命运，包括父母、兄弟、妻妾、子女、财运、官运、职业、吉凶等各方面。

- **古文体例**：共12段，每段先列原文，后附白话解说
- **核心主题**：从奇门格局看人生命运的富贵贫贱、吉凶祸福
- **起局方法**：采用奇门四柱起局法，以年、月、日、时四柱纳入九宫

> **通俗理解**：这篇古文是古人用奇门遁甲"算命"的总纲——相当于用奇门盘给人"画一幅人生全息地图"。从父母的寿夭到子女的贤愚，从财运大小到职业方向，都有一套完整的方法。

## 古文第一段：起局与人体对应

### 原文

> 其法先明戴九履一，细寻接气超神，八门配合九宫，九宫要带三奇。年月日时要旺相，八门九宫须推移，离宫头面，乾艮两足；肩耳在于坤巽，震兑左右胸襟，坎为阴肾，中为心腹。

### 解说

第一段主要讲奇门的[[节气与定局|定局]]和[[起局步骤|起局]]，以及九宫格与人体的对应关系：

- **"其法先明戴九履一"**：要掌握奇门测人生的方法，首先要弄明白奇门九宫格的原理
- **"细寻接气超神"**：要弄明白一年[[节气与定局|二十四节气]]、[[六十甲子]]干支系统与奇门用局的关系（此处讲[[超神接气置闰|置闰]]法；[[神奇之门（张志春）|开悟之门]]提倡用[[超神接气置闰|拆补法]]，不存在超神接气问题）
- **"八门配合九宫"**：按奇门起局方法，把[[九星概述|九星]]、[[八门概述|八门]]、[[八神概述|八神]]都配到九宫中
- **"九宫要带三奇"**：提纲挈领地讲九宫形成的主客状况，最好带有乙、[[丙奇|丙]]、丁三奇才为吉
- **"年月日时要旺相"**：年、月、日、时是纲领性的主要[[用神]]，用神旺相才为吉
- **"八门九宫须推移"**：以门为主来看人生的吉凶祸福及其推移变化

#### 九宫与人体部位对应

| 宫位 | 人体部位 |
|:---|:---|
| **离9宫** | 头部、面部 |
| **乾6宫** | 右腿、右足 |
| **艮8宫** | 左腿、左足 |
| **坤2宫** | 右耳、右肩 |
| **巽4宫** | 左耳、左肩 |
| **震3宫** | 左胸襟 |
| **兑7宫** | 右胸襟 |
| **坎1宫** | 阴部、肾、膀胱系统 |
| **中5宫** | 心脏、腹部 |

## 古文第二段：取用神的纲领

### 原文

> 年干父母，月干兄弟，日干本身，时干子女。推及妻妾，但看乙丁之落宫；若问夫君面首，庚丙二干定不够。要详细其强弱，即能决其荣枯。旺相得奇，且富且贵；死囚墓绝，极贱极贫。

### 解说

这段主要讲如何取[[用神]]，以及对用神的基本看法：

#### 六亲用神体系

| 六亲 | 用神 | 说明 |
|:---|:---|:---|
| 父母 | 年干 | 并可兼看[[八卦概述|乾宫]]（父）、[[八卦概述|坤宫]]（母） |
| 兄弟姐妹 | 月干 | — |
| 本人 | [[日干]] | — |
| 子女 | [[时干]] | — |
| 妻子 | [[乙奇]] | 男人问妻 |
| 妾/情人 | [[丁奇]] | 男问妾（现代社会可理解为情人） |
| 丈夫 | 庚 | 女人问夫 |
| 面首/男情人 | [[丙奇]] | 女问面首 |

#### 社会关系用神

- **年干**：上级领导
- **值符**：顶头上司
- **值使**：顶头上司的副职
- **月干**：同事、同行、朋友、竞争者
- **时干**：下级、群众、小人、宠物

#### 基本原则

- **"旺相得奇，且富且贵"**：用神旺相，又临三奇、吉星、吉门、吉神、吉格，自然是富贵之命
- **"死囚墓绝，极贱极贫"**：用神死囚墓绝，又临凶星、凶门、凶神、凶格，自然就是极贱极贫的命

## 古文第三段：格局与父母兄弟

### 原文

> 格局要分轻重，命运验其得失，未布局先着孤虚旺相，即开口必辩天地乘时。太白入乾坤，父母早年沦没；若临兄弟宫位，手足竟是仇敌。日奇被刑，庄生嗟叹；时干逢之，邓通无几。

### 解说

#### 分析格局的基本原则

1. **"未布局先看孤虚旺相"**：在尚未看用神具体格局以前，首先要看用神的**空亡（孤）**和旺衰
2. **"即开口必辩天地乘时"**：预测师从一开口就必须分辨用神得不得天时、地利、人和与神助的基本状况

#### 父母论断

- **"太白入乾坤，父母早年沦没"**：**太白金星**即庚。庚落[[乾六宫]]，则父亲早年去世；庚落[[坤二宫]]，则母亲英年早逝（[[地盘天盘人盘神盘|天盘]]、[[地盘天盘人盘神盘|地盘]]均算）

#### 兄弟论断

- **"若临兄弟宫位，手足竟是仇敌"**：庚落入月干所在的宫位，兄弟姐妹之间互相疏远、隔阂，甚至矛盾激化

#### 妻室论断

- **"日奇被刑，庄生嗟叹"**：[[乙奇|日奇]]即乙奇，被刑即遇上庚——[[天干化合|乙+庚]]、[[奇格|庚+乙]]均为[[刑格]]。本来乙庚相合为夫妻，但落入同一宫反而相互刑克，对妻子不利，轻则妻室经常闹病，重则妻子早亡

#### 子女论断

- **"时干逢之，邓通无几"**：时干加庚或[[庚格定应期|庚加时干]]，因时干代表子女，即使像汉代的富翁邓通一样广有财富，也往往没有后代，特别是难有儿子继承产业

## 古文第四段：财运

### 原文

> 生门产业，要得三奇；成败分于内外，生克决其得失。生门太白逢冲陷，背祖离乡之客；被冲克，售尽祖父旧园。生在外而身在内，必迁居而发富；身在外而生在内，纵有祖业亦难留。身生二宫俱在内，安享仆马之福；身生俱在外宫，远方创业之人。

### 解说

这段主要讲如何分析判断一个人的**财运**。财为养命之源，是命运的基础。

#### 财运基本论断表

| 格局 | 含义 |
|:---|:---|
| 生门临庚又冲克日干 | 背祖离乡，到外地发展 |
| 日干冲克生门与庚 | 变卖祖产，败家子 |
| 生门在外盘 + 日干在内盘 | 必须迁居外地/外国才能发达 |
| 日干在外盘 + 生门在内盘 | 即使有祖业也留不住他 |
| 日干与生门都在内盘 | 继承祖业，安享富贵 |
| 日干与生门都在外盘 | 远方创业之人 |

#### 核心要点

- **"生门产业，要得三奇"**：[[生门]]代表一个人一生的财运和产业，落宫要旺相，得三奇、吉星、吉格、吉神
- **"成败分于内外"**：看[[内盘外盘|内盘]]、[[内盘外盘|外盘]]
- **"生克决其得失"**：看与日干的生克关系

## 古文第五段：孤虚与命局

### 原文

> 日值孤，时值虚，少年无依；时落孤，日落虚，老后鳏独。命被冲克，乞食道路；日临墓绝，难释愁眉。

### 解说

古人在四柱命理学中将人的一生比作树木花草——**年柱为根，月柱为干，日柱为花，时柱为果**。

| 格局 | 含义 |
|:---|:---|
| 日干落空亡 + 时干落空亡对冲宫 | 少年时代命运不好，无依无靠 |
| 时干落空亡 + 日干落空亡对冲宫 | 老年鳏寡孤独，无子女赡养 |
| 年干被生门/日干/时干冲克 | 四处漂泊，乞食道路 |
| 日干落墓绝之地 | 一生不顺利，愁眉不展 |

## 古文第六段：九星定官职

### 原文

> 财官生旺，盖因飞鸟跌穴；鳌头独立，必得青龙返首。天辅旺相得奇门，文明翰苑；天冲旺相得奇仪，威镇边疆；天禽位镇中宫，得奇门，百官元首；英星右弼合吉格，鼎鼐元勋。天柱合式直言谏议。天心入垣医药最良。任为左辅，司农之职。天芮不可得地，逢奇门曹董之流。蓬星位镇北垣，得奇门叛君之贼。

### 解说

从九星角度讲**好命**的格局：

| 九星 | 格局条件 | 职业/命运 |
|:---|:---|:---|
| 日干临**飞鸟跌穴**（丙+戊）/青龙返首（戊+丙） | — | 财官两旺，金榜题名 |
| [[天辅星]] | 旺相 + 三奇/吉门/吉格 | 文化科技教育领域有成就 |
| [[天冲星]] | 旺相 + 三奇/吉门/吉格 | 威镇边疆的武官将领 |
| [[天禽星]] | 旺相 + 三奇/吉门/吉格 | 百官元首（总统、主席等一把手） |
| [[天英星]] | 旺相 + 三奇/吉门/吉格 | 开国元勋或栋梁大臣 |
| [[天柱星]] | 旺相 + 三奇/吉门/吉格 | 纪检、监察、人大、政协官员 |
| [[天心星]] | 旺相 + 三奇/吉门/吉格 | 领导干部、著名医生 |
| [[天任星]] | 旺相 + 三奇/吉门/吉格 | 农林水利官员 |
| [[天芮星]] | 旺相得地 + 三奇/吉门/吉格 | 曹操、董卓类奸雄（地位高但贪婪） |
| [[天蓬星]] | 旺相得地 + 三奇/吉门/吉格 | 叛君之贼（胆大妄为的盗贼首领） |

> **关键**：天芮星和天蓬星被认为是凶星，但旺相得地照样可以地位很高——只是人品倾向贪婪狠毒或胆大妄为。

## 古文第七段：八门定职业

### 原文

> 开门有奇，应是文职。伤门得令，定系虎臣。生门得奇，石崇富足。景门合局，丹天文明。惊门入式，好辩之人。休门合奇，礼宾首长。杜门得地，保密军警。死门得奇，职在司刑。

### 解说

从[[八门概述|八门]]角度讲，**八门与人所从事的职业关系最密切**：

| 八门 | 格局条件 | 职业 |
|:---|:---|:---|
| [[开门]] | 得三奇 | 文职官吏 |
| [[伤门]] | 得令/得三奇 | 武官 |
| [[生门]] | 得奇/旺相/吉格 | 像石崇一样的大富豪 |
| [[景门]] | 旺相 + 三奇/吉格 | 作家、画家、科学家、教育家 |
| [[惊门]] | 旺相得奇 | 外交家、教师、律师、歌唱家 |
| [[休门]] | 旺相 + 三奇 | 礼宾司长/秘书长/办公厅主任 |
| [[杜门]] | 得地 + 三奇 | 保密机关/军警部门 |
| [[死门]] | 旺相 + 三奇 | 法院/检察院/司法/监狱 |

## 古文第八段：失局者的命运

### 原文

> 分布八门，星神奇仪。合局者，豪富豪贵；失局者，下等平人。开门、心星皆是金，稍有气，医卜星相；若值死囚，手艺辛勤。休门、天蓬同是水，稍有气，兵弁役吏；值死囚，军卒贼人。生门、天任是土宿，得地者，田地富足；若失时，庸工耕耘。伤门、天冲皆为木，稍有气，兵役之首；值死休，马后相奔。杜门、天辅亦是木，稍有气，乃是寒儒；值墓绝，僧道山林。景门、天英为火宿，稍有气，行行威烈之表；全失局，冶炼劳碌之人。死门、天芮俱是土，稍有气，赵魏家长；值墓空，孤穷之人。惊门、天柱同属金，稍有气，幕宾教授；若失令，说唱巫祝之人。

### 解说

这段讲**失局失令**的人——中、下等平民百姓的命运。旺相得令、得奇得吉格合局的毕竟是少数。

#### 失局命运对照表

| 星/门 | 五行 | 稍有气（休/衰/病/胎/养） | 完全失局（死/囚/墓/绝） |
|:---|:---|:---|:---|
| [[开门]]/[[天心星]] | 金 | 医生、相士、占卜者 | 手艺劳动者 |
| [[休门]]/[[天蓬星]] | 水 | 排长/班长/科长/公务员 | 士兵/小偷/盗贼 |
| [[生门]]/[[天任星]] | 土 | 有一定田地，自给自足 | 打工、出卖劳动力 |
| [[伤门]]/[[天冲星]] | 木 | 部队小头头/领班 | 普通士兵/跟班差役 |
| [[杜门]]/[[天辅星]] | 木 | 清贫知识分子（寒儒） | 和尚/道士（归隐山林） |
| [[景门]]/[[天英星]] | 火 | 文化科技事业有一定成绩 | 冶炼等体力劳动者 |
| [[死门]]/[[天芮星]] | 土 | 大姓人家家长（有田产） | 孤单穷困百姓 |
| [[惊门]]/[[天柱星]] | 金 | 幕僚宾客/教书先生 | 说唱卖艺/巫婆神汉 |

## 古文第九段：九星再论职业高低

### 原文

> 八门辨别旺相，再论九星。天蓬乘时，边疆之将；失地利，军卒贼人。天任得地，田土仆马之福；受克，农圃辛苦之人。天冲有气者武贵，背时者车船江湖之流。天辅为文昌，得气者文雅仕官；失地利，僧道画工。天英火司南离，乘权者必主文明；背时者，贪暴昏庸。天芮本是黑星，得天时，性恶霸道；失天时，牧隶庸工。七赤天柱位镇西垣，有气者，舌辩当世；被冲克，优伎乐工。天心原是六白，得地者，才华国柱；若空墓，九流相寻。

### 解说

进一步从九星角度，按**紫白飞星说**讲一个人所临九星不同、旺相状态不同，在行业中的社会地位也不同：

| 九星 | 紫白 | 得时得地（合局+旺相） | 失时失地（囚/废/空/墓） |
|:---|:---|:---|:---|
| [[天蓬星]] | — | 边疆将领 | 军卒/贼人 |
| [[天任星]] | — | 广有田庄钱财 | 农田体力劳动者 |
| [[天冲星]] | — | 军中武装部门有权势者 | 车夫/司机/艄公 |
| [[天辅星]] | — | 风流儒雅文职官员 | 和尚/道士/绘画卖艺 |
| [[天英星]] | — | 文化科技有成就、闻名于世 | 贪暴昏庸 |
| [[天芮星]] | 二黑 | 性恶霸道（曹董之流） | 放牛牧羊/打工卖力 |
| [[天柱星]] | 七赤 | 外交家/演说家/律师/歌唱家 | 街头说书唱戏之人 |
| [[天心星]] | 六白 | 才华横溢的国家栋梁 | 走江湖卖艺的九流人物 |

## 古文第十段：格局定命运

### 原文

> 青龙返首，鳌头独立。飞鸟跌穴，富贵成名。青龙转光，邑长县令。戊加丁奇，富贵荣耀。白虎猖狂，凶顽之辈。青龙逃走，懦弱之人，更主因妻成败，而且驼背驼身。朱雀投江，刀笔书吏；腾蛇夭矫，毒心小人；失时者目盲耳聩，乘气者火焚伤身。太白入荧者，进，主先贫后富；荧入白者，退，主家业萧条。庚加戊，戊加庚，多成多败，此地不比他地。飞干伏干，此人不比他人。太白同宫，兄弟雷攻。太白逢星，妻室常病。星逢太白，惟薄丑声。大格者，萍迹四海；小格者，暂时清贫。年月日时四格，即六亲而推去，看是何宫测验，而为悖戾无礼相从，推及父母兄弟妻子贤否，可以详明。

### 解说

从奇门格局角度讲人生命运：

#### 吉格定命

| 格局 | 组合 | 命运 |
|:---|:---|:---|
| [[青龙返首]] | 戊+丙 | 金榜题名，出人头地 |
| [[飞鸟跌穴]] | 丙+戊 | 富贵有名 |
| [[青龙转光]] | 丁+戊 | 镇长/乡长/县长级官吏 |
| [[青龙耀明]] | 戊+丁 | 富贵荣耀 |

#### 凶格定命

| 格局 | 组合 | 命运 |
|:---|:---|:---|
| [[白虎猖狂]] | 辛+乙 | 凶狠顽固 |
| [[青龙逃走]] | 乙+辛 | 懦弱之人，因妻成败，驼背 |
| [[朱雀投江]] | 丁+癸 | 刀笔书吏（法官/调解员）；失时则眼疾 |

## 古文第十一段：特殊格局

### 原文

> 辛为天狱，壬为地牢，主低下之品，更主抑郁难伸。癸为天网，须看高低，高者化为华盖，可推贵格；低者天网缠身，寂寂孤贫。且忌玉女守门，妻随人行。乙丁妻妾，看是谁亲谁疏。乙奇入墓，妻不生子。丁乙同宫，另续相亲。乙奇在上，妻妾和谐；丁在乙上，妾幸夫心。

### 解说

#### 特殊符号含义

| 符号 | 含义 | 论断 |
|:---|:---|:---|
| [[辛（罪错之神）|甲午辛]] | 天狱 | 犯错误/犯罪/牢狱之灾，抑郁难伸 |
| [[壬（流动之神）|甲辰壬]] | 地牢 | 同上 |
| [[癸（天网之神）|甲寅癸]] | 天网 | 落6/7/8/9宫为网高→华盖→可为贵人；落1/2/3/4宫为网低→孤独贫穷 |
| [[玉女守门]] | 值使门遇地盘丁奇 | 在终身局中出现，易有红杏出墙、私通之事 |

#### 妻妾论断

| 格局 | 含义 |
|:---|:---|
| 乙奇落乾6宫（入戌墓） | 妻不生子 |
| 丁与乙同宫 | 必然会再续娶 |
| 乙奇在天盘 + 丁奇在地盘 | 妻主事，妾服从，妻妾和谐 |
| 丁奇在天盘 + 乙奇在地盘 | 小妾受宠，在家中说了算 |

## 古文第十二段：总结

### 原文

> 诸格忌落空亡，吉者减昌而苦者更苦。若临墓绝之地，无告穷民。八门推来最怕反吟，伏吟亦是凶神。五行配合斯理，取用在乎一心。余不尽言，惟应用者悟之。

### 解说

全文总结：

| 要点 | 含义 |
|:---|:---|
| **空亡** | 吉格逢空亡→吉的程度减少；凶格逢空亡→苦者更苦 |
| **墓绝** | 日干逢凶格 + 空亡 + 墓绝 → 无人帮助的穷苦百姓 |
| **[[反吟]]** | 八门反吟→一生反常、起伏大、不平静 |
| **[[伏吟]]** | 一生坎坷沉沦，难有大发展 |

> **"五行配合斯理，取用在乎一心"**：要用阴阳五行生克制化的古典哲学来理解、灵活运用以上道理。如何取用神、如何分析判断，全在于自己对道理的融会贯通。

## 应用宜忌

- **宜**：结合日干落宫的星、门、神、格局综合分析终身命运
- **宜**：用多重用神从多角度交叉验证，不同星、门角度应一致
- **忌**：片面依赖某一个格局或某一个星的论断
- **忌**：将古文论断生搬硬套，必须结合现代社会变通使用

## 奇门法窍补充

> **source:** [[奇门法窍（锡孟父子）]]
>
> 本节出自《奇门法窍》卷四「占寿」「占人生贵贱」「占人年命吉凶」「占终身」等章节，系（飞盘体系）飞盘奇门对人生命运预测的独特论述。

### 占寿（飞盘体系）

以九十为率，每宫十年。**天冲**落宫与**死门**远近以定其数——带旺一生无患，无吉者看其休囚之气断之。

### 占人生贵贱（飞盘体系）

以**年月日时**配合**八门九星九宫**，带吉奇旺相之宫为吉者、为贵，仍按十二宫推之。

### 占人年命吉凶（飞盘体系）

以来人所来之方合天地正时，与其生命所坐之方，合而推之——其人来方合奇者为吉。

### 占终身（飞盘体系·三盘分断）

占终身当分**天地人三盘**而断：
- **主星**管少年之造化
- **主门**管中年
- **主宫**管末年

然以**主星**为重。

> 问终身以得奇门诈为吉，其飞流亦须得宫门为重。
