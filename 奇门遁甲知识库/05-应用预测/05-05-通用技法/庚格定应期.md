# 庚格定应期

## 定义与基本属性

**庚格定应期**是奇门遁甲中最核心的定应期方法。庚在奇门中有特殊的阻隔、阻碍之意——事情被"格"住了，什么时候这个"格"被冲破，就是应期。庚格的本质就是判断庚（阻碍）与各种天干的组合关系，从而推断事情发生的具体时间。

通俗理解：庚就像是路上的一块石头，挡住了你的去路。如果要出门办事，你首先得过掉这块石头（庚）。看看这块石头上面写的是年干、月干、日干还是时干，就知道是年内过、月内过、日内过还是时辰内过。

## 四种庚格

### 一、年格

**庚 + 年干**

- 含义：事情在**年内**应验
- 判断：当年内会发生

### 二、月格

**庚 + 月干**

- 含义：事情在**月内**应验
- 判断：当月内会发生

### 三、日格

**庚 + 日干**

- 含义：事情在**日内**应验
- 判断：当天或近期会发生

### 四、时格

**庚 + 时干**

- 含义：事情在**时辰内**应验
- 判断：本时辰或几小时内会发生

## 庚格的判断方法

### 核心判断规则

判断庚格时，需要看庚与哪一层天干形成组合：

| 阴日/阳日 | 看什么地方 | 说明 |
|----------|----------|------|
| **阴日** | 看庚上之干（天盘干） | 即天盘落宫中的干 |
| **阳日** | 看庚下之干（地盘干） | 即地盘落宫中的干 |

> 阴日看天盘、阳日看地盘——这是判断庚格时的关键细节。

### 什么是阴日、阳日

根据起局当天的天干：
- **甲丙戊庚壬**为阳干之日 → 阳日
- **乙丁己辛癸**为阴干之日 → 阴日

## 庚不格的情况

并非所有庚的组合都算"格"，以下情况**不算庚格**：

| 情况 | 说明 |
|------|------|
| **乙庚合** | 乙庚相合，庚被合住，形不成阻隔，不为格 |
| **庚入墓** | 庚落衰死之地（入墓），无力量阻挡，不为格 |
| **庚空亡** | 庚所在宫位旬空，名存实亡，不为格 |

> 这三条是判断庚格时的"特殊豁免"——即使庚与某个天干在一起，只要满足以上任一条件，就不能按格来判断。

## 庚格的主要应用场景

庚格定应期主要在以下场景中应用：

### 一、破案（刑事案件）

判断罪犯何时落网、案件何时侦破：
- 看庚格在年格、月格还是日格
- 年格 → 年内破案；日格 → 近期破案

> 详见 [[刑事案件预测]]

### 二、行人走失

判断走失的人何时归来：
- 看庚格确定回归的时间范围
- 结合内盘外盘判断远近

> 详见 [[行人走失预测]]

### 三、其他需要定应期的场景

任何需要判断时间的具体预测，都可以运用庚格原理。

## 综合定应期的方法

庚格不应孤立使用，完整的定应期需要结合：

1. **庚格**（核心）：看庚在哪一格
2. **内盘外盘**：看用神落内盘还是外盘 → 近还是远
3. **干支组合**：看具体干支 → 锁定具体时间
4. **格局吉凶**：看整体格局 → 事情能否成

> 详见 [[内盘外盘]]、[[值符与值使]]

## 记忆口诀

> **庚加年干年内应，庚加月干当月行。**
> **庚加日干是日内，庚加时干时辰定。**
> **阴日看天阳看地，乙合入墓不格论。**

## 应用宜忌

- **宜**：定应期时优先看庚格，再结合内外盘和其他因素
- **宜**：牢记"阴日看天阳日看地"的判断规则
- **宜**：特别注意乙庚合、庚入墓、庚空亡三种不格的情况
- **忌**：不分阴阳日，一律用同一种方法看庚格
- **忌**：只看庚格，不考虑内盘外盘等辅助信息
- **忌**：以为庚格可以精确到具体哪一天——庚格给的是时间范围（年/月/日/时）

## 开悟之门补充：成功类、失败类、破获类定应期方法细化

《开悟之门》第四章第三节将定应期的方法系统地划分为三大类，并对每一类给出了详细的操作方法列表。

### 一、成功类事物定应期的方法（共8条）

前提：已判定事物**可以成功**（如婚姻可成、考试可过、职务可升等）。

| 序号 | 方法 | 说明 |
|:---|:---|:---|
| 1 | 用神旺相当令之时 | 用神最旺的时间点 |
| 2 | 以能生之时 | 用神需要被生时，以能生它的时间为应期 |
| 3 | 以克住之时 | 用神需要克制某物时，以能克住的时间为应期 |
| 4 | 填实或冲实 | 用神旬空，以填实（该地支出现时）或冲实（对冲地支出现时）为应期 |
| 5 | 冲库之时 | 用神入库者，以冲开墓库的时间为应期 |
| 6 | 以合时为应期 | 用神逢刑/冲者，以合住的时间为应期 |
| 7 | 马星当令或被冲 | 驿马星发动或被冲的时间 |
| 8 | 值使门/时干落宫之数 | 该宫所含数字往往是事物成败的期限（年/月/日/时）；临年之干有时也是应期 |

### 二、失败类事物定应期的方法（共6条）

前提：已判定事物**不能成功、必然败亡**（如恋爱不成、考试落榜、升职无望等）。

| 序号 | 方法 | 说明 |
|:---|:---|:---|
| 1 | 用神休囚死墓绝之时 | 用神最弱的时间点 |
| 2 | 忌神旺相、用神受克时 | 忌神当令最旺，克制用神之时 |
| 3 | 填实之时为失败之期 | 用神旬空，以填实之时应验失败 |
| 4 | 刑/冲以合时为应期；合以刑/冲时为应期 | 被合住的用神，等到冲开时失败 |
| 5 | 马星当令或被冲 | 驿马星发动之时 |
| 6 | 值使门/时干落宫之数 | 该宫所含数字往往是事物败亡的期限 |

### 三、破获/找寻/阻止类事物定应期的方法（庚格法）

适用于：刑侦破案、抓捕逃犯、找寻走失行人、寻找丢失财物等。

#### 庚格法的哲理基础

在奇门中**甲与庚是一对主要矛盾**，庚被称为**阻隔之神**：
- 好事受阻→不吉（血管受阻→中风，事业受阻→难升职）
- 坏事需要拦截阻挡→遇到庚反而是好事

这里有三类庚格用法：

#### 庚格方法一：直接庚格

| 类型 | 条件 | 应期 |
|:---|:---|:---|
| **年格** | 天盘庚 + 地盘年干 | 年内应验 |
| **月格** | 天盘庚 + 地盘月干 | 月内应验 |
| **日格** | 天盘庚 + 地盘点干 | 日内应验 |
| **时格** | 天盘庚 + 地盘时干 | 时辰内应验 |

> **注意**：空亡不为格、乙庚合不为格。

#### 庚格方法二：阴阳日看庚上下

如果没有方法一中的直接庚格出现：

| 日干 | 看什么 | 含义 |
|:---|:---|:---|
| **阳日**（甲丙戊庚壬） | 看庚下之干（地盘点） | 该天干往往就是应期之年干/月干/日干/时干 |
| **阴日**（乙丁己辛癸） | 看庚上之干（天盘干） | 该天干往往就是应期之年干/月干/日干/时干 |

#### 庚格方法三：时干临星法

如果没有方法一中的直接庚格出现，且用神是时干：

| 时干临星 | 看什么 | 
|:---|:---|
| **阳星** | 看庚下之干 |
| **阴星** | 看庚上之干 |

> **补充**：定应期时应尽量以阳历年、月、日、时为准，以适应当代社会习惯。

### 开悟之门补充：应期以阳历节气为准——避免阴历造成的误差

《开悟之门》特别强调：奇门遁甲判断应期时，**应以节气为主、以阳历时间为主**，这样准确率才比较高。因为二十四节气与公历日期对应稳定（一般仅差一两天），而阴历因有闰月、大小月，与节气常常错位。

以杜新会测在美国考行医执照为例：巽4宫辰巳空亡，戌冲辰，故断戌月。1997年阳历10月2日是阴历九月初一，但10月8日才交寒露节（入戌月）。杜新会断10月7日以后才能收到通知书，实际10月9日收到。**如果按阴历九月来断，则应断阳历10月2日以后，与实际相差7天，误差显著放大。**

> **核心原则**：定月应期时要查节气交节日期，以交节时刻为月的分界，不按阴历初一。例如戌月不是阴历九月初一，而是寒露节交节之时。

### 开悟之门补充：成功类定应期实例——久病逢冲则死

《神奇之门》实例二女测父病（完整应期判断示范）：已断定其父病重不治，求测人问"何时去世？"

**多层次应期判断**：
| 步骤 | 方法 | 判断结果 |
|:---|:---|:---|
| 1. 远近 | 日干/时干一内一外、星门均伏吟→主迟；年干丙/天芮均在内盘→主近 | 按月、日断 |
| 2. 死墓绝 | 用神死、墓、绝法 | 酉月或戌月 |
| 3. 自刑合期 | 年命甲午辛逢午午自刑，刑以**合**为应期 | 午与未合→未日 |
| 4. 综合 | 酉月+戌月+未日 | 死于酉月或戌月未日 |

实际结果：死于当年阳历9月7日（丁酉月丁未日）。

> **方法链总结**：失败类事物定应期的完整链条——① 远近卡定年/月/日级别 → ② 死墓绝法定节气范围 → ③ 刑冲合害法/庚格法锁定干支 → ④ 查万年历确认具体日期。

出处：《开悟之门》

### 开悟之门补充：破获类定应期实例——时干临阳星看庚下之干

《神奇之门》实例五测女儿离家出走，在奇门局中**找不到**年格、月格、日格、时格时，使用备用庚格法：

1. **卡定大致期限**：日干/时干一内一外主慢，值使开门落2宫→大致10天或12天内
2. **用时干临星法**：用神为时干壬，在巽4宫临**天禽阳星**→按时干临阳星，看"庚下之干"
3. **锁定天干**：坎1宫庚下之干为**戊**→断戊日可回
4. **查万年历定具体日**：10天左右，3月17日为戊午日→即应期

实际3月7日离家，3月17日（戊午日）自己回来了。

> **庚格法操作要点**：找不到直接庚格（年/月/日/时格）时，才用"时干临星法"。时干临阳星（天蓬/天任/天冲/天辅/天禽）→看庚下之干；时干临阴星（天英/天芮/天柱/天心）→看庚上之干。"庚下之干"指天盘庚所落宫的地盘干。

出处：《开悟之门》

## 上元经补充

### 应期的多层次概念

《奇门遁甲上元经》指出，应期不是一个简单的时间点，而是一个综合性时间信息，包含多种可能：

| 情况 | 应期含义 |
|------|----------|
| 事未起（事情未发生）| 可能是事物发生的时间、结束的时间、或经历的时间段 |
| 事已起（事情正在发生）| 同理三种情况，需分门别类解析 |
| 事有多头（多阶段事件）| 包含多个时间节点（如多期工程项目），需步步为营，仔细推详 |

> **批判**：市面上一些书籍把应期看成简单时间点，明显违反易理。
> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]应期章

### 符使定应期的层级法则

#### 符使与类占用神的优先级

- **事关全局** → 视符使之用神（值符为先）
- **事体微弱** → 观类占之用神
- 即使事体微小不足以牵动符使，观测符使也有助于从整体上把握事件发展脉络

#### 值符优先、值使备用的判断流程

```
观察值符落宫 → 值符宫旺相吉利、格局搭配得当 → 由值符定应期
              → 值符宫破败不堪、不得时令、力量衰微 → 转求值使定应期
```

特殊情况：
1. **瞬间发生的事情**（如下雨、出行具体时间）→ 以值使门定应期，但需满足前提：事体微小无关值符 + 值使门落宫与类占用神落宫产生联系（三合、五合、六合等）
2. **紧迫之事无关大局** → 专察值使亦可

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]应期章

### 锡孟樨《奇门法窍》应期法批判

#### 锡氏应期法的核心内容

锡孟樨应期法的基本思路：
1. 先定地支，后配天干
2. 以值符为核心，值符宫天盘六仪藏支与宫位藏支构成刑冲关系 → 寻找与天盘六仪藏支相合地支 → 确定应期
3. 不刑不冲 → 逢空填实
4. 不冲不刑不空 → 看天盘六仪（值使宫），逢冲看合、逢合看冲
5. 均不成立 → 以星门生克定应期（生逢生日，克逢克日）

#### 上元经对锡氏应期法的五点批判

**一、星门奇仪衰旺混谈**

锡氏说"从星门奇仪衰旺而断"，但三者衰旺性质完全不同：
- 门无衰旺，只观迫制
- 星的衰旺不涉月令，看太岁与五运六气的生化
- 唯有奇仪存在衰旺

**二、转盘与飞盘的混淆**

锡氏《奇门法窍》试图综合飞盘与转盘，创造出"空中宫以象转盘、飞布以象飞盘、中宫寄四维"的新模式，但没有真正理解奇门背后易学原理。值符冲刑寻应期的方法本质上采用飞盘思路，运用在转盘奇门中可信度打折。

**三、值符宫衰落不可定应期**

值符藏支与宫位藏支构成刑冲时（如甲寅癸值符落坤宫，癸入墓且受刑冲），除非落宫旺相且格局非大凶，否则值符宫不能定应期，必须转向值使。

天盘干支自身冲刑（如辛加戊、戊加庚），能量损失较大，除非天盘干生旺且落宫月令旺相，才可维持在值符宫找应期。

示例：甲子戊加庚落离宫（戊帝旺、庚败地，离宫旺相）→ 可定应期；甲子戊加庚临坤宫（戊弱庚强）→ 不能定应期，转向值使宫。

**四、"填实"不可一概而论**

值符逢空不止是填实那么简单：
- 事情已结束 → 空表示完结，不存在应期
- 值符宫逢马星空动 → 事情可能在以前已结束
- 事情未真正展开，只在策划层面 → 不存在应期

**五、阴比阳比的参照点必须是时干**

锡氏说空亡填实后"阳与阳比，阴与阴比"，但参照点必须是时干的阴阳。如甲辰旬首落乾宫被填实，甲属阳 → 对应戌支；丁未时丁属阴落乾宫填实 → 对应亥支，而非以日干阴阳为参照。

**六、"星门生克"不符合易理**

门只看门迫，星显象而不直接对门与宫产生生克。"生逢生日，克逢克日"成为无稽之谈。

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]应期章

### 类占用神定应期的核心法则

#### 类占用神充当应期的三种情况

1. 占测具体事务无关全局 → 直接用类占用神定应期
2. 类占用神落宫与符使落宫构成三合/五合/六合 → 全局气运积聚到类占用神落宫
3. 符使用神落宫破败不堪，羸弱无力 → 必须转向类占用神

#### 关键条件：类占用神必须旺相

- 用神落宫旺相 → 事情有成功可能，有资格定应期
- 用神处于囚、死、墓、绝 → 好事成空，无法定应期

#### 填实与冲实的区别

| 类型 | 条件 | 判断 |
|------|------|------|
| 填实 | 具体落宫旺相，宫内格局搭配得当 | 填实时可发挥效用，由此定应期 |
| 冲实 | 对宫冲击，冲击宫不能旺相且非忌神落宫 | 可定应期 |
| 冲实 | 忌神落宫状态旺相，冲击力量大 | 用神宫能量散失大，不能定应期 |

示例：夏天起局占异地调动，开门落乾宫（休囚空亡），但寅宫年命 + 午宫值符 + 乾宫构成寅午戌三合局，中神在戌不在午，乾宫得三合旺气转衰为旺 → 戌月乾宫填实时完成迁徙。

#### 刑冲状态下用神宫的判断

自身强弱决定是否承受住刑冲：
- 自身强旺 → 能承受刑冲，可继续定应期
- 自身羸弱 → 无法承受，不能定应期

> **比喻**：如同武侠小说中中毒的武功高手——若自身内力强，能把毒逼出来，依旧能纵横驰骋；若无法疏解毒性，则无法充分展现出来。

### 生死应期的特殊规则

一般的求测事项：
- 事情成 → 可找成功应期
- 事情不成 → 无应期概念

**生死大事的特殊性**：
- 抢救成功 → 可找脱离危险应期
- 抢救失败（死亡）→ 同样可推测死亡时间

> 人命大于天，事关全局的预测必然牵动符使。不可能像某些书籍所说光观察日干、天芮、生死二门就行，必须上升到符使与年命的高度。

**反向应期法则**：
- 一般事件：值符宫破败 → 无充当应期的资格
- 涉及死亡的反向预测：羸弱的值符反而为预测应期提供方便

示例：癌症病人化疗后恢复不错，但值符落宫破败且与天芮、死门构成申子辰三合，值符宫为合局中气，起局是酉月 → 子月时会出大问题。

### 庚定应期的正确方法（纠正旧说）

#### 庚充当应期的前提条件

庚一般不具备充当定应期的功能（庚是甲的仇敌），只有在以下条件下才能用庚定应期：
- 庚充当甲申庚值符
- 庚落宫与值符落宫构成五合/三合/六合等内在关联

> 切忌一见破案、拦截案例就找庚定应期。必须先看庚是否与甲帅相关——庚必须是"以甲为元帅队列中的一员"，否则不能为甲所用。

#### 庚落宫旺衰的重要性

庚落宫休囚且不得月令之助 → 庚自身羸弱 → 没有抓捕能力 → 定应期必须通过符使与类占用神。

#### 批判"阴日看庚上之干，阳日看庚下之干"

该观点错误的核心原因：
1. 时家奇门以时干为排列基准，庚是旬内运动序列的一份子，定应期与日干无关
2. 运动的庚只能由天盘干表征，相对静止的地盘干庚无法承担此职责
3. "看庚上之干"是无稽之谈

#### 庚定应期的正确操作方法

1. **庚格法**：庚下临年/月/日/时干 → 年格应一年内、月格月内、日格日内、时格时辰内
2. **地支优先**：定应期"支大于干"的法则是第一位的，首先要注重庚所在宫位的地支
3. **符使为总纲**：大局意识什么时候也不能丢
