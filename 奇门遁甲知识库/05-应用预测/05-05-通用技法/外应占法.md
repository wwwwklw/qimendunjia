# 外应占法

## 定义与基本属性

外应是指起局时突发的、与起局者内心产生感应的外部现象。外应是相对于内部用神（盘内星、神、门、干等符号）而言的——盘内用神为解局之主，外应为断言之辅。

> **核心原则**：局内用神为主（内因），外应为辅（外因）。外应不能喧宾夺主，只有在特定情况下才能联通内因对事件产生决定性影响。

## 外应的三个必要条件

外应取用必须同时满足三个条件：

1. **起局时的外在感应**：必须是在起局时意念与外在感应同步发生。没有起局的想法，即使外面天塌地陷，也不能算作外应。
2. **入脑入心的心灵感应**：该感应必须能够与起局者产生心灵共鸣。见怪不怪的现象不能成为外应。
3. **非人力蓄意寻求**：外应只能是可遇而不可求的。蓄意寻找的所谓外应反而具有人工造作痕迹，与简易思想违背。

> **出处**：《奇门遁甲上元经》外应章

## 外应与四盘的对应体系

外应的确定与区分应从起局者角度出发，根据奇门天、地、人、神四盘体系寻找对等外应：

| 感应类型 | 对应盘面 | 用神 |
|---------|---------|------|
| 视觉光线（天象）| 星盘 | [[天英星]] |
| 佩戴珠玉反光 | 门盘（人盘）| [[景门]] |
| 眼前幻觉光芒 | 神盘 | [[腾蛇]] |
| 听觉声响 | 星盘/门盘 | [[天柱星]] / [[惊门]] |
| 山川地势 | 九宫 | 对应宫位（如山→艮宫）|

> **要点**：根据五行意象分类寻找对应的外在事物，注重天、地、人、神不同体系的对应关系。

## 外应的动静之别

### 静应

观测到的外应是静止的（如突然看到形状怪异的山峰）：

- **作用**：不改变原宫位动静状态，但能提升该宫位层次、凸显预测重要性
- **判断**：若原宫位不与符使、日时、具体用神相关，外应的加入可使该宫成为解局时不可忽视的方面
- **操作**：认真观察该宫是否与符使、日时、具体用神落宫存在潜在的符号关联

### 动应

观测到的外应是动态的（如突见空中海市蜃楼、窗外鸟儿飞过发出叫声）：

**情况一**：外应落宫本身为静宫 → 动应将该宫由静转动，该宫可能主动触发与其他宫位的联系（五合、三合、六合）。需注意与对面宫位对冲后造成的结果。

**情况二**：外应落宫本身为动宫 → 动上加动，对宫冲击更加巨大。旺宫冲衰宫，类似于"旺者冲衰衰者拔"，可形成有利局面（如职位擢升中冲击竞争对手年命落宫）。

> **案例**：被测人进门时窗外一只鸟儿飞过并发出刺耳叫声，选天柱星为外应。天柱星落宫冲击日主落宫，落宫空亡且宫内出现庚加丙意象——可断前不久发生车祸，发出巨大响声，所幸人员未受大伤害。

## 外应的远近之说

动应是移动的，以起局人为参照点，有远近两种情况：

| 方向 | 含义 |
|------|------|
| **由远及近** | 外应力量增强，影响递增 |
| **由近及远** | 外应力量减弱，影响递减 |

> **案例**：起局时适逢元宵佳节燃放烟花，烟花升空由近及远，以天冲星代表动应。天冲落宫与求测人年命落宫一致，外应由近及远表明能量逐步减弱，结合宫内不良符号可断不如意情况。

## 外应的多寡之说

外应的多寡影响其重要程度：
- 单一外应：指向性明确，针对性强
- 多重外应：需分辨主次，以最入心者为重

## 实占要领

1. **勿夸大外应**：不能像某些书籍所说过分强调外应作用，把外应当成解局关键。内因是事物发展的决定性因素。
2. **内应外应联动**：外应必须联通内用神才能显其功用，单纯外应不足以定断。
3. **以我为主取应**：外应的确定从起局者出发，而非从外应物质特征分类。
4. **四盘对等取应**：根据天、地、人、神不同体系寻找对等的外应用神。
5. **动静分判作用**：静应提升层次，动应改变宫位动静状态。
6. **远近定影响力**：由远及近力量增强，由近及远力量减弱。
