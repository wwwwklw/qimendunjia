# 金锁玉环出行诀

## 金函玉镜补充

### 定义与基本属性

**金锁玉环遁甲出行要诀**是斐晋公（裴度）创制的数字课择法，以天干地支的固定数字值相加，根据总数判断出行吉凶及应验事象。此法简便易用，不需排盘，只需知道当日的干支和时辰即可。

通俗理解：这是一个"出行吉凶速查表"——把日子（干支）和时辰（地支）分别对应一个数字，加起来看总数对应的"歌诀"就知道出门是吉是凶。

### 数字赋值

#### 天干数

| 天干 | 数字 |
|------|------|
| 甲、己 | 九 |
| 乙、庚 | 八 |
| 丙、辛 | 七 |
| 丁、壬 | 六 |
| 戊、癸 | 五 |

#### 地支数

| 地支 | 数字 |
|------|------|
| 子、午 | 九 |
| 丑、未 | 八 |
| 寅、申 | 七 |
| 卯、酉 | 六 |
| 辰、戌 | 五 |
| 巳、亥 | 四 |

### 计算方法

1. 取日天干数 + 日地支数 + 时地支数 = 总数
2. 注意：日论干支，**时只论地支不论天干**

**举例**：
- 甲子日出行：甲(9) + 子(9) = 18（日数），若卯时(6)，共 24 数 → **课吉**
- 甲子日出行：甲(9) + 子(9) = 18，若午时(9)，共 27 数 → **课凶**
