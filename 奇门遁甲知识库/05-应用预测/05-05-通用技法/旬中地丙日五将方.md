# 旬中地丙日五将方

> 出处：《奇门遁甲统宗》卷之三·旬中地丙日、五将方 — 参见 [[奇门遁甲统宗（诸葛亮）]]

## 定义

**旬中地丙日**与**五将方**是《奇门遁甲统宗》卷三所载的两种兵占避忌之法。旬中地丙日为按旬首推算的凶日；五将方为按日支三合局推算的凶方。

## 一、旬中地丙日

### 定义

**旬中地丙日**是按旬首推算的凶日。将兵者不可用，犯之上将死于阵。

### 旬首推旬中地丙日表

| 旬首 | 旬中地丙日 |
|------|-----------|
| 甲子旬 | 寅日 |
| 甲戌旬 | 子日 |
| 甲申旬 | 戌日 |
| 甲午旬 | 申日 |
| 甲辰旬 | 午日 |
| 甲寅旬 | 辰日 |

### 推算规律

旬中地丙日为各旬中按一定规律逆向递减两位的地支：

- 甲子旬（子丑寅卯巳午未申酉亥）→ 寅日
- 甲戌旬（戌亥子丑寅卯巳午未申）→ 子日
- 甲申旬（申酉亥子丑寅卯巳午未）→ 戌日

每旬的地丙日为前一旬地丙日递减两位。

### 应用

> 将兵者不可用，犯之上将死于阵。

- 旬中地丙日**不可将兵**
- 犯之**上将死于阵**

## 二、五将方

### 定义

**五将方**是按**日支三合局**推算的凶方。凡遇敌必审五将方所在避之，犯之必败。

### 日支推五将方表

按日支三合局分组：

| 日支 | 五将方 |
|------|--------|
| 寅、午、戌日 | 东方 |
| 亥、卯、未日 | 南方 |
| 申、子、辰日 | 西方 |
| 巳、酉、丑日 | 北方 |

### 推算规律

五将方为日支三合局**所克之方**：

- 寅午戌（火局）→ 火克金，西方本应为凶，但取东方（木生火反受其生）
- 亥卯未（木局）→ 木克土，取南方（火泄木反受其生）
- 申子辰（水局）→ 水克火，取西方（金生水反受其生）
- 巳酉丑（金局）→ 金克木，取北方（水泄金反受其生）

### 应用法则

> 凡遇敌必审五将方所在，避之大吉，犯之必败。敌若从此方来，当军择利，使击之则胜。

- **避五将方**：避之大吉
- **犯五将方**：犯之必败
- **敌从五将方来**：当军择利使击之则胜

## 综合应用

旬中地丙日与五将方皆为兵占凶日凶方，常与以下避忌并用：

- 太岁
- 月建
- 大将军
- 太阴
- 河魁
- 天营
- 四神
- 游都
- 鲁都

## 应用宜忌

### 宜

- 将兵前查算旬中地丙日与五将方
- 避开旬中地丙日将兵
- 避开五将方出兵

### 忌

- 旬中地丙日将兵（上将死于阵）
- 犯五将方（必败）
