# 值符（八神）

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **五行** | 土（中央土） |
| **阴阳** | — |
| **别名** | 天乙之神 |
| **地位** | 八神元首、九星领袖 |
| **吉凶** | 四吉神之一 |

## 核心含义

值符是八神中的首领，居于领导地位。**禀中央土气**，为天乙之神，是八神的元首，也是九星的领袖。

> **《神奇之门》原文**：值符为天乙之神，禀中央土，是八神之首、九星领袖，所到之处，百恶消散。

## 详细象意

### 人事
- **领导、上司、老板**：位居高位之人
- **权威人士**：有话语权、有决定权的人
- **核心人物**：团队中的灵魂人物
- **法官、裁判**：有最终裁定权的人
- **高档贵重之人或物**：品质上乘、价值高昂

### 性格
- 正直、威严、大气
- 有担当、有决策力
- 光明磊落

### 地理/环境
- 核心区域、中枢地带
- 高档场所：五星酒店、高级会所
- 政府机关、权力中心

### 身体
- 心脏（身体的核心）
- 头部

## 小值符与大值符的区别

| | **小值符** | **大值符** |
|------|-----------|-----------|
| **所指** | 八神中的值符 | 九星中的天禽星（值符） |
| **属性** | 八神之一 | 九星之一 |
| **作用** | 管八神之吉凶 | 管九星之运行 |

小值符：随天盘值符星而转，在八神盘中居于核心位置。
大值符：指九星中的值符星座，有天乙贵人之意。

**关系口诀**：大值符为星，小值符为神，星动神随。

## 应用宜忌

### 宜
- 宜求见贵人、拜见领导
- 宜诉讼、谈判（有贵人相助）
- 宜婚嫁、出行（百恶消散）
- 预测得值符，事情可能有贵人插手

### 忌
- 值符落空亡，贵人无力
- 值符逢[[庚（阻隔之神）]]/[[白虎]]，主领导有麻烦

## 旺相休囚 / 落宫状态

- **旺于土宫**（坤二宫、艮八宫、中五宫）
- **相于金宫**（兑七宫、乾六宫）
- 落宫有力则贵人得力，落宫衰败则靠山不牢

## 与其他概念的关系

- [[太阴]]：值符为明处的力量，太阴为暗处的谋划
- [[九天]]：值符居中统管，九天好动外向
- [[九地]]：值符居中统管，九地好静内向
- [[天禽星]]：大值符即天禽星
- [[甲（遁甲）]]：值符是甲木元帅在八神层面的体现
- [[三奇（乙丙丁）]]：值符统领全局，三奇是值符（甲）的辅佐官

## 金函玉镜补充

> 出自《金函玉镜奇门遁甲秘笈全书》卷六——九神主论章。

### 金函玉镜版值符论述

> 值符者，诸神之元首，九星之领袖。因名值符，**其体属火**，其神所到之处，百恶消散，诸凶寂灭，至吉之神。所畏者，太白金星，忌入墓，吉处不吉，凶处更凶。天始于甲，地始于子，固谓万汇之尊者，举甲子而六甲在其中矣，故名之曰「值符」。

**差异**：《神奇之门》以值符属土（中央土），金函玉镜以值符属火。
