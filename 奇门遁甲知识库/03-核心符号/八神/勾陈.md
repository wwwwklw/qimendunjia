# 勾陈

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **五行** | 土（中央阳土） |
| **阴阳** | 阳 |
| **别名** | 顽钝之神 |
| **吉凶** | 凶神 |
| **出处** | 《金函玉镜奇门遁甲秘笈全书》卷六 |

> **注**：勾陈在《神奇之门》的八神系统中被隐藏于白虎之下，但《金函玉镜》将其作为独立的神煞列入九神系统。

## 金函玉镜论述

> 勾陈者，中央之阳土也。其性顽，专司田土、词讼之事，自甲至戊其数五，自子至辰其数亦五，艮纳丙，坎纳戊，配于东南。经云：知三避五，三五反复，凶顽之气，不可趋位。镇于艮，故名之曰「勾陈」。

### 核心含义

- **田土官司**：土地纠纷、房地产争议
- **词讼**：诉讼、打官司
- **反复无常**："三五反复"之象
- **顽钝**：固执、难缠

### 命名来源

- 甲至戊 = 五数
- 子至辰 = 五数
- 三五反复，凶顽不可趋

## 详细象意

### 人事
- 土地纠纷、房产官司
- 长期诉讼、拖而不决
- 顽固之人、难缠之事
- 反复无常的局面

### 性格
- 顽固、冥顽不灵
- 好争斗诉讼
- 不轻易让步

### 地理/环境
- 田土、土地
- 法院、官司场所
- 东南方位（配于东南、镇于艮）

## 应用宜忌

- **忌**：遇到勾陈主官司缠身、田土之争
- **忌**：宜避开此类方位
- 勾陈临宫，慎防土地纠纷和诉讼之事

## 与其他概念的关系

- [[白虎]]：在《神奇之门》系统中白虎下隐勾陈，二者皆凶
- [[六合]]：六合主和合，勾陈主争斗，形成鲜明对比
- [[惊门]]：惊门+勾陈，官司口舌加剧
