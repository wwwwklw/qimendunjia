# 朱雀

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **五行** | 火（南方火神） |
| **阴阳** | 阳 |
| **别名** | 文明之神、口舌之神 |
| **吉凶** | 吉凶参半 |
| **出处** | 《金函玉镜奇门遁甲秘笈全书》卷六 |

> **注**：朱雀在《神奇之门》的八神系统中被隐藏于玄武之下，但《金函玉镜》将其作为独立的神煞列入九神系统。

## 金函玉镜论述

> 朱雀者，南方之火神，统辖周天之野，专司文明之权，掌奏口舌、文书之职。得地则文书印信有喜，失时则有是非口舌挠乱之凶。位在丙，丙纳艮，土旺相在离，在天为赤鸟之神，属丙火，举丙寅而六丙在其中矣。故名之曰「朱雀」。

### 核心含义

- **文书**：公文、信件、证书、文章
- **口舌**：是非争论、言语纠纷
- **文明**：文化、学术、考试
- **信息**：消息传递

### 吉凶双重性

- **得地**（旺相）：文书印信有喜，考试文章有利
- **失时**（衰弱）：是非口舌挠乱，官讼缠身

## 详细象意

### 人事
- 文书、考试、文章
- 口舌是非、争吵
- 信息传递、消息
- 学术文化
- 官司状纸

### 性格
- 善言辞、口才好
- 喜欢议论是非（失时）
- 文采斐然（得地）

### 地理/环境
- 学校、考场
- 法院（文书层面）
- 传媒、报社

### 身体
- 心脏、血液
- 口舌、咽喉

## 应用宜忌

- **宜**（得地）：文书、考试、上书、献策
- **忌**（失时）：口舌是非、争吵、打官司
- 朱雀+景门：文书有利，考试大吉
- 朱雀+惊门：口舌官司，是非不断

## 与其他概念的关系

- [[玄武]]：在《神奇之门》系统中玄武下隐朱雀
- [[景门]]：朱雀为文书，景门亦主文书，二者叠加文书事显著
- [[丁奇]]：丁为星奇/文书，与朱雀的文书属性相通
- [[丙奇]]：金函玉镜指出朱雀"位在丙""属丙火"，与丙奇关系密切
