# 太常

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **五行** | 五行化气（随所遇而变化） |
| **阴阳** | — |
| **别名** | 歌饮之神、宴享之神 |
| **吉凶** | 吉凶参半（与门并吉则吉，与门并凶则凶） |
| **出处** | 《金函玉镜奇门遁甲秘笈全书》卷六 |

> **注**：太常在《神奇之门》的八神系统中不出现，是《金函玉镜》九神系统中的独有神煞。

## 金函玉镜论述

> 太常者，五行之化气也。其神好歌饮，专司宴享、祭祀、衣帛、羔雁、酒食之事。此神随天禽遍游诸方，遇火则从火，遇金则从金，遇水则从水，遇木则从木，遇土则从土，与五体相合，其性不常，故名之曰「太常」。与吉门并则吉，与凶门并则凶，主衣服、彩衣、孝服之变更也。

### 核心含义

- **宴享**：宴会、聚餐、庆祝活动
- **祭祀**：祭祀仪式、典礼
- **衣帛**：衣物、丝绸、服饰
- **酒食**：饮食之事、招待
- **变化无常**："其性不常"

### 独特性

太常最具特色的性质是**五行随所遇而变**：
- 遇火则从火
- 遇金则从金
- 遇水则从水
- 遇木则从木
- 遇土则从土

这与其他神煞有固定五行属性完全不同。

## 详细象意

### 人事
- 宴会、聚餐、招待
- 祭祀典礼、宗教活动
- 服饰、礼仪
- 酒食餐饮之事
- 彩衣/孝服变更（喜事丧事之变）

### 吉凶判断

- 与吉门同宫 → 吉祥之事
- 与凶门同宫 → 凶兆之事
- 太常本身不决定吉凶，吉凶由同宫之门决定

### 地理/环境
- 宴会场所、餐厅
- 祭祀场所、庙宇
- 服装店、丝绸店

## 应用宜忌

- **宜**：与吉门配合时宜宴会、祭祀、社交
- **忌**：与凶门配合时可能有丧事、变更之象
- 临太常主"衣"变——可能是喜服（婚庆）也可能是孝服（丧事）
- 随天禽星遍游诸方，流动性强

## 与其他概念的关系

- [[天禽星]]：太常随天禽遍游诸方
- [[六合]]：六合主婚姻交易（喜事），太常主宴享（也涉喜事）
- [[死门]]：死门+太常，可能主孝服丧事

## 出处

《金函玉镜奇门遁甲秘笈全书》卷六——九神主论章/太常。
