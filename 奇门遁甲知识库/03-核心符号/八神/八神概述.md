# 八神概述

八神是奇门**神盘**上的八种神秘能量符号，代表天地间的灵性能量场。

> 八神亦分阴阳，阳遁顺排、阴遁逆排。

## 上元经补充

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]八神九神章

### 神盘的地位辨析

张大春先生明确指出：**神盘的势能远不及星，也不如门、干**，绝不能在普通的预测中过分夸大它的作用。部分学者"神助大于一切"的观点违背易理。

> 奇门遁甲的命名"奇、门、遁甲"中就没有体现出神盘。神盘系统与人一样受到地表运行规律制约，作为万物之灵的人可以认识、把握神盘系统，也可以利用神盘体系来为自己服务。

### 转盘八神 vs 飞盘九神

| 特征 | 转盘八神 | 飞盘九神 |
|------|----------|----------|
| **数量** | 8神 | 9神（多太常、勾陈、朱雀） |
| **运行方式** | 受阴阳二遁顺逆排列制约 | 阳遁顺行飞涧跳宫，阴遁逆行飞涧跳宫 |
| **观察视角** | 依时旋转，不假后天人力 | 站在地面上的人作为太极点 |
| **运行原理** | 八神圆转，中宫虚位以待，木火金水行于天地之间 | 阳遁由勾陈而至朱雀（阳升），阴遁由白虎而至玄武（阴盛） |

### 飞盘九神排列

- **阳遁**：值符 → 螣蛇 → 太阴 → 六合 → 勾陈 → 太常 → 朱雀 → 九地 → 九天
- **阴遁**：值符 → 螣蛇 → 太阴 → 六合 → 白虎 → 太常 → 玄武 → 九地 → 九天

### 太常的作用（飞盘独有）

>（飞盘体系）太常是飞盘九神中特有的一神，起到**气机运化**的作用：
> - **阳遁**：勾陈经太常运化而终成朱雀阳极之气
> - **阴遁**：白虎经太常运化而成玄武至阴之气
>
> 阳遁太常为吉神（运化至阳），阴遁太常一般不能视为吉神（运化至玄武阴极，助纣为虐）。

### 诸神阴阳属性

- **转盘**：值符、螣蛇、六合、九天为阳；太阴、白虎、玄武、九地为阴
- **五行**：值符、六合属木，太阴、白虎属金，螣蛇为火，玄武为水；九天、九地象征乾坤二卦

### 神的旺衰判断

张大春先生否定依据月令判断神旺衰的说法。神盘是类似暗物质的特殊场能，不可能随着月令推移出现旺相休囚变化。**八神/九神的能量旺衰只体现在不同的落宫中**：
- 值符五行属木：长生乾宫亥，帝旺震宫卯，墓库坤宫未
- 诸神均依此推详
