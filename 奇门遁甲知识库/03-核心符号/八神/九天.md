# 九天

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **五行** | 金（乾金之象） |
| **阴阳** | 阳 |
| **别名** | 威悍之神、万物之父 |
| **吉凶** | 四吉神之一 |

## 核心含义

九天为**乾金之象**，万物之父，为威悍之神。**性刚好动**。

> **《神奇之门》原文**：九天为乾金之象，万物之父，威悍之神，性刚好动。九天之上可以陈兵。

## 详细象意

### 人事
- **高大**：身材高大或事情格局大
- **远处**：遥远的地方
- **飞机、高空**：与天空有关的事
- **远行、出差**：到很远的地方去
- **志向高远**：有远大理想
- **主动出击**：积极行动、不等待
- **公开的强势**：明面上的力量
- **快速推进**：进展快、不拖沓

### 性格
- 刚强、好动
- 志向远大
- 行动力强
- 自信、有魄力

### 地理/环境
- 高处（山顶、高楼顶层）
- 远方、国外
- 机场（起飞降落）
- 开阔地带

### 身体
- 头部
- 肺（乾金主肺）

## "九天之上可以陈兵"

这是奇门中关于九天的经典论述。意思是：**在九天所在的方向或宫位，可以陈兵布阵、主动出击**。类似于"天道"的作用——居高临下，势如破竹。

**应用**：
- 遇事临九天，宜攻不宜守
- 出行时朝九天方向有利
- 事业宜扩张、开拓
- 适合做大事、打大仗

## 应用宜忌

### 宜
- 宜出击、扩张、主动进攻
- 宜远行、出国
- 宜登高、扬名
- 宜公开大事

### 忌
- 忌防守、潜伏（与九天相反）
- 忌谨慎保守之事
- 落宫受克，好高骛远不切实际
- 九天+凶格，兵败如山倒

## 旺相休囚 / 落宫状态

- **旺于金宫**（兑七宫、乾六宫）
- 旺则声势浩大，如日中天
- 衰则空有理想，力不从心

## 开悟之门补充

### 现代象意：飞机航线

《开悟之门》为九天赋予了现代交通含义：

> 九天之方可以排兵布阵，行军打仗。用神上乘九天吉神，可以主动出击，大展宏图。如果是出行，自然可以坐飞机出门或出国远行了。所以在今天，**九天又可以作为飞机航线的代表符号**。

这意味着在预测出行、交通、物流等现代事务时：
- 九天临用神 → 可能需要乘坐飞机
- 九天方向出行 → 宜选择航空方式
- 九天+开门 → 国际航班、跨国远行
- 九天代表高空、快速、远距离移动

## 金函玉镜补充

> 出自《金函玉镜奇门遁甲秘笈全书》卷六——九神主论章。

### 金函玉镜版九天论述

> 九天者，乾金也。其体属金，乾纳甲壬，性刚而好动。所主者，名正言顺之事，值其令而无阻，至吉之神。若得门、得奇，万福咸集，即不得奇，亦不为凶，畏入墓而力屈。天始于甲，自甲至壬其数九，故名之曰「九天」。

**差异**：金函玉镜从乾纳甲壬的角度解释了九天的命名（甲至壬数九），强调九天"即不得奇，亦不为凶"——即使没有三奇配合也不会凶，这是其作为至吉之神的特殊之处。另外指出九天畏入墓。
