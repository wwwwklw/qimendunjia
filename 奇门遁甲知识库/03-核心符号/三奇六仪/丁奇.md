# 丁奇

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **五行** | 阴火 |
| **阴阳** | 阴 |
| **别名** | 星奇、玉女 |
| **角色** | 军需官 |
| **十二长生** | 长生在酉，帝旺在巳，墓在丑 |
| **出处** | 《神奇之门》第2章 |

## 核心含义

丁为**星奇**，又名**玉女**，五行属阴火。在奇门遁甲中担任**军需官**的角色，是[[甲（遁甲）]]之女，**以巧克庚护甲**。

> 丁火如同星星之火、烛火，不像[[丙奇]]火那样猛烈，而是阴柔精巧。丁为甲之女，女儿护父，以巧制敌。

## 丁火长生路线

```
长生在酉（兑七宫） → 帝旺在巳（巽四宫） → 墓在丑（艮八宫）
```

- **长生在酉**：酉金是丁火的"长生"之地（阴火生于酉金）
- **帝旺在巳**：巳火是丁火的帝旺之地
- **墓在丑**：丑为金库（丑又是艮八宫）

> **入墓**：丁奇落艮八宫即为入墓，玉女被困

## 玉女守门

**玉女守门**是奇门中的一个特殊格局，指丁奇加临值使门所在之宫。

> 玉女守门，如同一位美丽的女子守在门口，事情需要先从"门口"处突破。

**含义**：
- 测婚姻：有女性缘分
- 测事业：文书证件是关键
- 测诉讼：证据很重要
