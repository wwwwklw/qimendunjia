# 甲（遁甲）

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **五行** | 阳木 |
| **地位** | 元帅、主将 |
| **特点** | 隐遁在阵中，不直接露面 |
| **旬首** | 每旬的甲日是该旬之首 |
| **出处** | 《神奇之门》第2章第1节 |

## 核心含义

甲是奇门遁甲中的**元帅主将**，是整个阵法的核心。但甲木元帅从不直接出现，而是**隐遁在阵中**——这就是"遁甲"的由来。

> **《神奇之门》原文**：甲为元帅主将，隐遁在阵中，常隐藏在六仪之下。

## 六甲体系

甲不是只有一个，而是有**六个甲**——一正五副：

| 六甲 | 身份 | 隐于 | 说明 |
|------|------|------|------|
| **甲子** | 元帅 | [[六仪（戊己庚辛壬癸）|戊]] | 最高统帅 |
| **甲戌** | 大将 | [[六仪（戊己庚辛壬癸）|己]] | 五员大将之一 |
| **甲申** | 大将 | [[庚（阻隔之神）]] | 五员大将之一 |
| **甲午** | 大将 | [[辛（罪错之神）]] | 五员大将之一 |
| **甲辰** | 大将 | [[壬（流动之神）]] | 五员大将之一 |
| **甲寅** | 大将 | [[癸（天网之神）]] | 五员大将之一 |

这六甲合称"**六甲**"，每一甲都是旬首（每十天一旬的首日）。

## 甲最怕庚金克杀

**甲木最怕庚金**，这是奇门遁甲中最核心的一对矛盾关系：

- 甲为**阳木**（参天大树）
- 庚为**阳金**（利斧大刀）
- 阳金克阳木 = **七杀**（最凶的克）
- 甲一旦被庚克到，元帅就有危险

这是为什么奇门要"遁甲"的根本原因——**把甲藏起来，不让庚找到它**。

## 三奇护甲

为了保护甲元帅，奇门设置了**三奇护甲**策略：

| 三奇 | 护甲方法 | 比喻 |
|------|---------|------|
| **[[乙奇]]** | 乙合庚（乙庚合金） | **美人计**——用和亲化解 |
| **[[丙奇]]** | 丙克庚（火克金） | **儿子护父**——正面克制 |
| **[[丁奇]]** | 丁克庚（火克金） | **女儿护父**——暗中克制 |

> 口诀：乙合庚、丙克庚、丁克庚

三种方式有分工：
- **乙**用柔（合）：以柔克刚，和亲化解
- **丙**用刚（克）：以强对强，正面打击
- **丁**用巧（克）：以巧对强，暗中周旋

## 甲的寻找方法

在实际排盘和预测中，**甲不看天干，而看六仪和值符**：

1. **看六仪**：每个六仪之下都藏着对应的甲
2. **看值符**：[[值符（八神）]]所在的宫位，就是甲所在之处
3. **看旬首**：甲子戊、甲戌己……六甲就是六个旬首

> 实战口诀："值符所在即甲所在"

## 上元经补充

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]十千体性章

### 遁甲的深层原因

张大春先生指出，"遁甲"的原因不仅是甲木畏庚金克害，更深层次的原因是：**甲为统帅，是一个队伍的核心，显得特别骄贵**。如同国家首脑出访总有大量安保人员环绕，越是重要的人物越不可能直接暴露在不可预知的风险前。

> 四柱中用神与甲木同理——都是需要保护的娇贵之物。用神显露时若无保护，用神流年反而更加倒霉。

### 六仪顺排、三奇逆排的易学原理

戊、己、庚、辛、壬、癸、丁、丙、乙的固定排列源于"**数往者顺，知来者逆**"：
- 前六仪土生金、金生水依次**顺行**——"往者顺"
- 后三奇丁丙乙**逆行**——"来者逆"
- 癸、丁隔甲，甲前顺数，甲后逆行，故三奇不可顺数

### 戊土为九干之首的理据

先天一水、二火、三木、四金→冲气成五数土。**五即戊土**，是先天之气向后天之质转化的基点。甲子作为六甲之首隐藏其下，亦显肇始之机。
