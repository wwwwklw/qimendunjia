# 马星

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **含义** | 主走动、变动、快速 |
| **别名** | 驿马 |
| **作用** | 推动事物发展变化的"催化剂" |
| **出处** | 《神奇之门》第4章 |

## 核心含义

马星是奇门预测中判断**变动、出行、速度**的关键参数。马星所落之宫，必有**变化和移动**。

> 见马星即见动——这是奇门预测中一条重要的经验法则。

## 马星口诀

```
寅午戌马在申
申子辰马在寅
巳酉丑马在亥
亥卯未马在巳
```

**详细对照表**：

| 时辰地支 | 马星所在 | 对应宫位 |
|----------|---------|---------|
| 寅、午、戌 | 申 | 兑七宫 |
| 申、子、辰 | 寅 | 艮八宫 |
| 巳、酉、丑 | 亥 | 乾六宫 |
| 亥、卯、未 | 巳 | 巽四宫 |

### 推导逻辑

马星的推算基于**三合局**：
- 三合局的三个地支，马星在对冲之位
- 例如：寅午戌三合火局，水克火——马星在水之长生位申（申子辰水局长生在申）

## 马星在预测中的应用

### 1. 判断是否动
- 任何用神临马星：必有变动
- [[日干]]临马星：本人要动
- [[时干]]临马星：事情在变
- 用神宫见马星：该事物在变化中

### 2. 马星动为应期
- **马星所落之宫**的干支时间，往往就是事情发生的时间
- 例如马星落兑七宫（申位），申日/申月是应期

### 3. 出行判断
- 出行方向看马星落宫方位
- 出行时间看马星对应的地支

### 4. 事物状态
| 状态 | 含义 |
|------|------|
| **马星+吉神吉星** | 好事将动、变动有利 |
| **马星+凶神凶星** | 坏事将动、变动不利 |
| **马星+[[空亡]]** | 想动动不了、有心无力 |
| **马星+[[九天]]** | 远行、出国 |
| **马星+开门** | 工作调动 |

## 与其他概念的关系

- [[日干]]：日干临马星，本人即将行动
- [[时干]]：时干临马星，事情很快有变化
- [[空亡]]：马星+[[空亡]] = 动不起来
- [[九天]]：马星+[[九天]] = 远行吉利
- [[白虎]]：马星+[[白虎]] = 出行有风险
- [[壬（流动之神）]]：马星+壬 = 快速流动

## 上元经补充

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]神煞章

### 马星的双重引动功能

张大春先生将马星的作用分化为两个层次：

**宫位间的引动**：
- **静宫由马牵引而为动**：原本安静的宫位因临马星而变为动态
- **动宫遇马，动上加动**：本身发动状态的宫再遇马星，速度加倍
- **三合六合逢马而起**：原本需要宫位联动条件的三合、六合，逢马星即被触发，诸宫合力势不可挡
- 马星处于四维宫长生/临官状态的地支，本身就是最具动性的位置

**天盘干的引动**：
马星还可以引动宫内的天盘天干，这时需要注意天盘天干的旺衰关系：
- 壬居亥宫临马星 → 壬临官状态被放大加速，若再临开门吉星 → 升官提职征兆
- 乙临巽宫巳支（桃花位），巳为马星再配合相关符号 → 桃花运来了

### 神煞体系的取舍原则

《奇门遁甲上元经》在神煞章**只介绍了空亡、马星、太岁三个神煞**，其他神煞一概隐而不谈。这种取舍是张大春先生经过深思熟虑做出的——大量明目繁杂的神煞在奇门占测中并没有实际价值。
