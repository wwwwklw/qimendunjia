# 时干

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **含义** | 代表所测之事（类似六爻的世爻） |
| **地位** | 奇门预测的第二用神 |
| **来源** | 用事当天的时辰天干（时柱天干） |
| **出处** | 《神奇之门》第3章、第4章 |

## 核心含义

**时干代表所测之事**，在奇门中型地位仅次于[[日干]]。如果说日干是"人"，时干就是"事"。

> **类比六爻**：时干在奇门中的作用类似于六爻中的"**世爻**"——代表当前所问之事本身。

## 时干在奇门中的纲领性作用

### 1. 定事态
时干所在的宫位，代表**事物本身的状态**：
- 时干落宫旺相：事情本身条件是好的
- 时干落宫衰弱：事情本身条件不足
- 时干入墓：事情被卡住、有阻力
- 时干空亡：事情不落实、信息不准

### 2. 判断时机
时干代表当前时辰的"气"，结合时干可以判断：
- **是否得时**：事物是否有"天时"
- **是否逢时**：时机是否成熟

### 3. 日干与时干的生克关系

| 状态 | 含义 |
|------|------|
| 时干生[[日干]] | 事情来找你，顺其自然（吉） |
| [[日干]]生时干 | 你需要主动去追求（需努力） |
| [[日干]]克时干 | 你能掌控事情（能办成） |
| 时干克[[日干]] | 事情压着你（困难、不吉） |
| 日时比和 | 人和事和谐（平稳） |

### 4. 断应期
时干所在的宫位也是判断事情**应在何时**的重要参考。

## 常见用法

```
日干宫 = 求测者（主体）
时干宫 = 所测之事（客体）

判断逻辑：
日干旺 + 时干生/比和 → 事成
日干弱 + 时干克 → 事败
时干入墓 → 事情有阻
时干逢空 → 事情尚未发生/不落实
```

## 与其他概念的关系

- [[日干]]：日干为"人"，时干为"事"——二者是奇门预测的核心主客关系
- [[值符（八神）]]：时干临值符，事情有贵人
- [[年命]]：时干为当前之事，年命为长久之运
- [[空亡]]：时干落空亡，事情不实
- [[马星]]：时干临马星，事情变动快
