# 壬（流动之神）

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **五行** | 阳水 |
| **阴阳** | 阳 |
| **别名** | 天牢、流动之神 |
| **甲名** | 甲辰壬 |
| **代表** | 流动、变化、智慧、水 |
| **出处** | 《神奇之门》第2章 |

## 核心含义

壬为**阳水**，是奇门中的"流动之神"。在六仪中为**正黑旗**，藏甲辰。

> 壬水如同江河湖海之水，流动不息，主变化、动态之事。它既可以代表智慧的流动，也可以代表波折的变化。

## 详细象意

### 人事
- **流动、变迁**：搬家、调动、换工作
- **智慧**：水生智慧
- **财运**：水主财，壬代表大财
- **出差、旅行**：流动性活动
- **变化多端**：事情有变数
- **水性杨花**（情感中）

### 性格
- 聪明、机智
- 善变、灵活
- 圆滑

### 地理/环境
- 江河湖海
- 道路（水的流动性）
- 出差地

### 身体
- 肾
- 膀胱
- 血液

## 壬与丙的冲克

壬水和[[丙奇]]火之间存在**水火相冲**的关系，这在奇门中形成重要格局：

- 壬+[[丙奇]]：水克火，水火相冲，主动荡争执
- [[丙奇]]+壬：火被水克，权威受冲

## 与其他概念的关系

- [[庚（阻隔之神）]]：庚+壬=[[小格（上格）|小格]]，水中阻隔
- [[丙奇]]：丙壬相冲，水火不容
- [[癸（天网之神）]]：壬为阳水主流动，癸为阴水主陷溺
- [[玄武]]：壬水+[[玄武]]，水贼之象
- [[开门]]：开门+壬，调动、出行顺利
- [[天蓬星]]：天蓬属水，与壬水信息重叠
