# 庚（阻隔之神）

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **五行** | 阳金 |
| **阴阳** | 阳 |
| **别名** | 太白、阻隔之神 |
| **地位** | 甲木的头号克星 |
| **甲名** | 甲申庚 |
| **出处** | 《神奇之门》第2章 |

## 核心含义

庚是奇门遁甲中**最凶的干**。五行属阳金，为太白金星，专门克制[[甲（遁甲）]]（阳金克阳木为**七杀**）。

> **《神奇之门》原文**：庚为甲木之克星，奇门中最凶之干，代表阻隔、障碍。

庚本为六仪之一（正白旗，藏甲申），但在一切预测中，庚都是以"**凶神**"的面目出现。

## 详细象意

### 人事
- **阻隔、障碍**：任何事情碰到庚都有坎
- **道路**：道路不通、交通堵塞
- **车祸**：交通事故
- **官司、诉讼**：法律纠纷
- **丈夫**（女子测丈夫丈夫看庚）
- **仇人**：与自己对立的人
- **对手**：竞争中的对手方
- **金属物品**：刀、剑、汽车

### 性格
- 刚硬、固执
- 好斗、不妥协
- 强势、霸气

### 地理/环境
- 道路、高速公路
- 法院、派出所
- 金属加工厂
- 事故现场

### 身体
- 大肠
- 骨骼
- 肺部

## 庚与其他天干的关系

- **庚+[[乙奇]]**：乙庚合，美人计化解
- **庚+[[丙奇]]**：丙克庚，儿子护父（正面冲突）
- **庚+[[丁奇]]**：丁克庚，女儿护父（以巧制敌）
- **庚+[[六仪（戊己庚辛壬癸）|戊]]**：甲子藏于戊中，庚克戊即克甲（最凶）
- **庚+[[辛（罪错之神）]]**：二金相遇，冲克升级
- **庚+[[壬（流动之神）]]**：[[小格（上格）|小格]]，水中阻隔
- **庚+[[癸（天网之神）]]**：大格（严重阻隔）

## 与其他概念的关系

- [[甲（遁甲）]]：生死仇敌
- [[三奇（乙丙丁）]]：三奇制庚的三条路径
- [[白虎]]：庚为金，白虎也为金——双方属性相通
- [[值符（八神）]]：值符（甲）最怕见庚
- [[开门]]：开门+庚，道路不通
