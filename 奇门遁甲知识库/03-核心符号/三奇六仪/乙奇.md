# 乙奇

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **五行** | 阴木 |
| **阴阳** | 阴 |
| **别名** | 日奇 |
| **角色** | 文官谋士 |
| **十二长生** | 长生于午，帝旺在寅，墓在戌 |
| **出处** | 《神奇之门》第2章 |

## 核心含义

乙为**日奇**，五行属阴木。在奇门遁甲中担任**文官谋士**的角色，辅佐元帅[[甲（遁甲）]]。

> 乙木的特点是柔韧、弯曲，如藤蔓花草。它不像[[甲（遁甲）]]木那样参天挺拔，而是能够**以柔克刚**，通过合化的方式解决问题。

## 详细象意

### 人事
- **文官、谋士**：出谋划策之人
- **医生、医药**：治病救人之象
- **女子**：妻子、女友、女性
- **文书**：合同、证件、信件
- **柔中带刚**：表面柔弱，内有计谋

### 性格
- 温柔、细腻、善解人意
- 足智多谋
- 柔韧、能屈能伸

### 地理/环境
- 花园、草地
- 医院、诊所
- 文化场所

### 身体
- 肝、胆（木主肝胆）
- 毛发

## 乙木长生路线

```
长生于午（离九宫） → 帝旺在寅（艮八宫） → 墓在戌（乾六宫）
```

- **长生在午**：午火越旺，乙木越有生机（木得火而荣）
- **帝旺在寅**：寅是乙木的临官禄地，木气最旺
- **墓在戌**：戌土为火库，乙木入墓则困

> **入墓**：乙奇落乾六宫（戌）即为入墓，有力使不出

## 乙庚合——美人计

乙奇护甲的核心手段是**与庚相合**：

- **乙（阴木）+ 庚（阳金）= 合化金**
- 乙木以柔克刚，用"联姻"的方式化解庚金的威胁
- 如同**美人计**——以和亲化解敌意

> 乙庚合是奇门中最有名的合局之一，广泛用于婚姻预测。
