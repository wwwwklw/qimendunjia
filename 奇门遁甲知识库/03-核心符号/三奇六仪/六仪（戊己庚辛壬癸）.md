# 六仪（戊己庚辛壬癸）

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **组成** | 戊、己、庚、辛、壬、癸 |
| **别名** | 六支仪仗队 |
| **地位** | 元帅[[甲（遁甲）]]的外围仪仗 |
| **出处** | 《神奇之门》第2章第1节 |

## 核心含义

六仪是**六支仪仗队**，相当于元帅甲的外围卫队。每一支仪仗队的旗帜颜色不同：

| 天干 | 旗帜 | 颜色 | 隐藏的六甲 |
|------|------|------|-----------|
| **戊** | 正黄旗 | 正黄色 | 甲子 |
| **己** | 镶黄旗 | 镶黄色 | 甲戌 |
| **庚** | 正白旗 | 正白色 | 甲申 |
| **辛** | 镶白旗 | 镶白色 | 甲午 |
| **壬** | 正黑旗 | 正黑色 | 甲辰 |
| **癸** | 镶黑旗 | 镶黑色 | 甲寅 |

> **《神奇之门》原文**：六仪就是六支仪仗队：戊为正黄旗、己为镶黄旗、庚为正白旗、辛为镶白旗、壬为正黑旗、癸为镶黑旗。

## 六仪永远不变的队形顺序

六仪的顺序是**固定不变的**，这在排盘时是硬规则：

```
戊 → 己 → 庚 → 辛 → 壬 → 癸 → 丁 → 丙 → 乙
|______________________|  |_________|
        六仪                 三奇
```

从上可知，六仪排列在前，三奇紧随其后。这个顺序由六甲隐伏的位置决定，**不论阴阳遁，这个相对顺序永远不变**。

## 六甲隐于六仪

六仪最重要的一层含义：**六甲分别隐藏在六仪之中**。

| 六甲 | 隐于 | 六仪 | 含义 |
|------|------|------|------|
| 甲子 | 隐于 | 戊 | 甲子遁在戊，戊即甲子 |
| 甲戌 | 隐于 | 己 | 甲戌遁在己，己即甲戌 |
| 甲申 | 隐于 | 庚 | 甲申遁在庚 |
| 甲午 | 隐于 | 辛 | 甲午遁在辛 |
| 甲辰 | 隐于 | 壬 | 甲辰遁在壬 |
| 甲寅 | 隐于 | 癸 | 甲寅遁在癸 |

这就是"**遁甲**"二字的由来——甲元帅隐藏在六仪之中，不直接露面。

> 口诀：甲子戊、甲戌己、甲申庚、甲午辛、甲辰壬、甲寅癸

## 六仪的本质

表面上看起来是六支仪仗队，实际上**每支仪仗队的主帅都是甲木的将领**。这是奇门遁甲最巧妙的一层设计——将甲木藏起来，用六仪掩护。

## 与其他概念的关系

- [[甲（遁甲）]]：六仪是甲的藏身之所
- [[三奇（乙丙丁）]]：三奇跟随六仪之后，共同保护甲
- [[庚（阻隔之神）]]：庚既是六仪之一（正白旗），又是甲的大敌
- [[辛（罪错之神）]]：辛是镶白旗，藏着甲午
- [[壬（流动之神）]]：壬是正黑旗，藏着甲辰
- [[癸（天网之神）]]：癸是镶黑旗，藏着甲寅
- [[值符（八神）]]：值符所在的宫位，就是甲所在之处
- [[空亡]]：与六甲旬空有关

《神奇之门》第2章第1节
