# 日干

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **含义** | 代表求测者（预测当事人） |
| **地位** | 奇门预测的第一用神 |
| **来源** | 用事当天的天干（日柱天干） |
| **出处** | 《神奇之门》第3章、第4章 |

## 核心含义

**日干代表求测者本人**，是奇门预测中最核心的用神，没有之一。

> 无论测什么事，首先要看的就是日干所落的宫位。日干是整个盘中最直接代表"你是谁"的参数。

## 日干在奇门中的纲领性作用

### 1. 定主体
日干所在的宫位，代表**求测者本人的状态**：
- 日干落宫旺相：本人状态好、能力强
- 日干落宫衰弱：本人状态差、能量不足
- 日干入墓：本人困住、有力使不上
- 日干空亡：人不在状态、心不在焉

### 2. 定吉凶
日干与其他用神的生克关系决定吉凶：
- **日干生他神**：你需要主动付出
- **他神生日干**：别人来帮你（吉）
- **日干克他神**：你主动掌握局势
- **他神克日干**：别人压制你（凶）
- **比和**：双方地位相当

### 3. 定方位
日干所在的宫位方向，代表**求测者所处的方位**。

## 日干与其他用神的关系

| 用神 | 与日干的关系 | 用途 |
|------|------------|------|
| [[时干]] | 日干为人、时干为事 | 判断人和事的关系 |
| [[年命]] | 日干为当前，[[年命]]为根基 | 结合判断终身大事 |
| [[值符（八神）]] | 日干+值符=有贵人 | 判断是否有贵人 |
| [[乙奇]] | 日干生/克乙奇 | 男测婚姻看与乙的关系 |
| [[庚（阻隔之神）]] | 庚克日干 | 男测被情敌压制 |
| [[六合]] | 日干+六合 | 有合作、婚姻缘分 |

## 常见用法

```
测事业：日干宫 + 开门宫 → 看事业运
测婚姻：日干宫 + 六合宫 → 看婚姻运
测财运：日干宫 + 戊宫（资本） → 看财运
测疾病：日干宫 + 天芮星宫 → 看病和身体
测行人：日干宫 + 时干宫 → 看是否归
```

## 与其他概念的关系

- [[时干]]：日干为"人"，时干为"事"
- [[年命]]：日干看当前，年命看根基
- [[空亡]]：日干落空亡时，求测者状态不佳
- [[马星]]：日干临马星，本人主动行动
