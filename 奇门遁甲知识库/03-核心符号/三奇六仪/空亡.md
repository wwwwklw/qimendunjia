# 空亡

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **含义** | 十天干配十二地支，每旬有两个地支无天干可配——这两个地支即为空亡 |
| **别名** | 旬空 |
| **作用** | 空亡之宫无力、不落实 |
| **出处** | 《神奇之门》第2章 |

## 核心含义

空亡是奇门预测中判断**事物有无落实**、**宫位信息是否有效**的关键参数。

> 天干只有十个，地支有十二个。每旬十天干配十个地支后，剩下两个地支没有天干——这两位就是**旬空**（空亡）。

## 六甲旬空表

| 旬 | 旬首 | 空亡的地支 | 对应的宫位 |
|------|------|-----------|-----------|
| **甲子旬** | 甲子（戊） | **戌、亥** | 乾六宫 |
| **甲戌旬** | 甲戌（己） | **申、酉** | 兑七宫 |
| **甲申旬** | 甲申（庚） | **午、未** | 离九宫 + 坤二宫 |
| **甲午旬** | 甲午（辛） | **辰、巳** | 巽四宫 |
| **甲辰旬** | 甲辰（壬） | **寅、卯** | 艮八宫 + 震三宫 |
| **甲寅旬** | 甲寅（癸） | **子、丑** | 坎一宫 + 艮八宫 |

### 记忆口诀

```
甲子旬中戌亥空    甲戌旬中申酉空
甲申旬中午未空    甲午旬中辰巳空
甲辰旬中寅卯空    甲寅旬中子丑空
```

## 空亡的判断方法

第一步：根据预测当天的日柱，确定属于哪一旬
第二步：查表确定空亡的地支
第三步：找出空亡地支对应的宫位

> **判断例句**："今天是甲子日，甲子旬中戌亥空——乾六宫为空亡。"

## 空亡在预测中的应用

### 1. 空亡之宫无力
- 任何用神落入空亡之宫，**该信息不实、力量不足**
- [[日干]]落空亡：求测者状态不佳、不认真
- [[时干]]落空亡：事情不落实、信息不准确
- 财星（戊）落空亡：财不实在

### 2. 空亡的信息特点

| 空亡状态 | 含义 |
|----------|------|
| **全部空亡** | 信息完全无效 |
| **半空**（只有一个地支空） | 信息部分有效 |
| **空亡+吉神** | 吉事不实、要打折扣 |
| **空亡+凶神** | 凶事暂时不显、但暗伏 |
| **空亡+[[马星]]** | 想动动不了 |

### 3. 填空冲实为应期

这是奇门预测中最重要的应期法则之一：

| 应期类型 | 条件 | 说明 |
|----------|------|------|
| **填空** | 当地支走到空亡之支 | 信息由虚变实 |
| **冲实** | 当地支走到空亡之支的对冲位 | 以冲的方式填实 |

**举例**：
- 乾六宫（戌亥空） → 到了戌月/戌日（填空）或辰月/辰日（冲实），事情落实
- 兑七宫（申酉空） → 到了申酉月日（填空）或寅卯月日（冲实）

### 4. 远空近空

- **远空**：空亡之宫在地支上离当前较远——事情有待时日
- **近空**：空亡之宫在地支上临近——事情很快就有结果

## 开悟之门补充

### 六甲旬空的完整配法

《开悟之门》对六甲旬空的由来有详细说明。十天干与十二地支相配，总是阳干配阳支，阴干配阴支，按顺时针方向运动。天干之首甲依次与地支之首及轮空地支配对：

**第一轮：甲子旬**
- 甲子、乙丑、丙寅、丁卯、戊辰、己巳、庚午、辛未、壬申、癸酉
- 剩下地支戌、亥逢空 → **甲子旬中戌亥空**

**第二轮：甲戌旬**
- 甲（配第一轮轮空的戌）、乙亥、丙子、丁丑、戊寅、己卯、庚辰、辛巳、壬午、癸未
- 剩下地支申、酉逢空 → **甲戌旬中申酉空**

**第三轮：甲申旬**
- 甲（配第二轮轮空的申）、乙酉、丙戌、丁亥、戊子、己丑、庚寅、辛卯、壬辰、癸巳
- 剩下地支午、未逢空 → **甲申旬中午未空**

**第四轮：甲午旬**
- 甲（配第三轮轮空的午）、乙未、丙申、丁酉、戊戌、己亥、庚子、辛丑、壬寅、癸卯
- 剩下地支辰、巳逢空 → **甲午旬中辰巳空**

**第五轮：甲辰旬**
- 甲（配第四轮轮空的辰）、乙巳、丙午、丁未、戊申、己酉、庚戌、辛亥、壬子、癸丑
- 剩下地支寅、卯逢空 → **甲辰旬中寅卯空**

**第六轮：甲寅旬**
- 甲（配第五轮轮空的寅）、乙卯、丙辰、丁巳、戊午、己未、庚申、辛酉、壬戌、癸亥
- 这就是**六甲**（甲子、甲戌、甲申、甲午、甲辰、甲寅），剩下地支子、丑逢空 → **甲寅旬中子丑空**

至此，十天干与十二地支共配六次，形成**六十花甲子**。每一甲都剩下两个地支轮空，故叫六甲旬空。

> **背景**：据南京大学古天文系教授卢央考证，以六十甲子纳入奇门九宫格来记时的模型，原来是一种很古老的历法。奇门遁甲不过是利用了古历法而已，绝非荒诞不经，而是有天文学依据的。

### 空亡的四分类法

张大春先生将空亡进一步细分为四种类型，远超传统的"真空/假空"二分法：

| 类型 | 含义 | 判断要点 |
|------|------|----------|
| **真空** | 事物确实不存在，真正意义上的时空盲区 | 信息完全无效 |
| **动空** | 曾经发生过某事，现在事件已结束 | 反映过去的事 |
| **半空** | 某人尝试谋划某事，做过努力但未造成实质性影响 | 有始无成，有花无果 |
| **假空** | 随着时间发展，以后某时该盲点会被填实 | 等待时机即可 |

《奇门遁甲中元经》中有《四空赋》专门探讨四种空亡的差别。

### 四空的不同处理办法

- **假空**：填实时把该宫内原有信息直接读出。要把握当下状态信息，必须关注该宫的**先天信息**。如艮宫假空，在填实前需关注艮宫的先天宫位——震宫的意象，震宫内的符号组合就显示出艮宫填实前的情况。

- **真空**：**逢虚看孤**，着力解读对宫信息，对宫可以显示出造成空亡的原因及空亡状态下的相关信息。

- **半空**：表层象是虚象（类似海市蜃楼）。半空宫位对宫的象意也能解释造成半空的部分原因。

- **动空**：反映了过去发生的事。观测现在状态要看**翻宫后落宫**的状态。如巽宫属动空，巽宫本身反映过去，巽宫地盘干转为天盘干所落之宫的意象就反映出当下及将来情况。

### 日时旺相与空亡

如果日时旺相，值符值使俱空，这种空**不是真正意义上的空亡**。日时旺相代表行动人有能力从事某项事业，但此时大环境不允许。只要**守正待机**，时机一到，此事必能成功。

### 空亡与法术奇门

在法术奇门中，空亡是术者可以大显身手的好机会，运用得当可以转危为安，化险为夷。

### 时支空亡而非干空亡

在时家奇门预测体系中，空亡的划分以旬为单位，时是旬的基础。围绕时干展开的奇门布局体系所产生的空亡概念只能是**时支空亡**，与干空亡无关。
