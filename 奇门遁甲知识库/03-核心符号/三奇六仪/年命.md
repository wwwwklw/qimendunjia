# 年命

## 定义与基本属性

| 属性 | 内容 |
|------|------|
| **含义** | 求测者出生年的年干落宫 |
| **地位** | 终身预测和大事判断的核心用神 |
| **来源** | 出生年份的天干（如甲子年则年命为甲） |
| **出处** | 《神奇之门》第3章第6节、第4章 |

## 核心含义

**年命**是预测者出生年份对应的天干所在的宫位。它比[[日干]]更"底层"，代表一个人的**根基、命运底色**。

> [[日干]]看当前状态，年命看人生大势。大事判断必须看年命。

## 年命的判断层次

### 日干 vs 年命

| | **[[日干]]** | **年命** |
|------|-----------|------|
| **时效** | 当前、短期 | 长久、终身 |
| **层次** | 表面状态 | 深层根基 |
| **适用** | 日常小事 | 大事、终身 |
| **作用** | 看当下行为 | 看命中注定 |

### 使用原则
- **小事看日干**：日常琐事只需看日干落宫
- **大事看年命**：婚嫁、事业转折、投资、疾病等大事，必须结合年命
- **日干与年命同宫或相生**：当前状态与命理根基一致，顺遂
- **日干与年命相冲克**：当前行为与命理根基冲突，有波折

## 年命与他人六亲关系的判断

年命不光能判断自己，还能判断**他人与你的六亲关系**：

| 关系 | 以年命为基础的比较 |
|------|-------------------|
| **配偶** | 配偶年命与求测者年命落宫的生克 |
| **父母** | 年命生父母宫 = 孝顺 |
| **子女** | 年命被子女宫所生 = 子女有孝 |
| **朋友** | 双方年命比和 = 志同道合 |
| **合作伙伴** | 年命相生 = 合作顺利 |

## 常见用法

```
测婚姻：双方年命落宫是否相生相合
测事业：年命旺 + 开门吉 = 命中有官运
测财运：年命生戊（资本）= 命带财运
测疾病：年命入墓或受克 = 病重难愈
测终身：年命旺相+吉神吉星 = 命好
```

## 年命的旺衰判断

- **年命旺相**：命硬、根基好、运势强
- **年命入墓**：命中被困、施展不开
- **年命落空亡**：命途坎坷
- **年命被克**：命中多磨难

## 与其他概念的关系

- [[日干]]：日干看当前，年命看人生底色
- [[时干]]：时干看此事，年命看一生
- [[空亡]]：年命落空亡，一生多有波折
- [[甲（遁甲）]]：甲年生人，年命为甲——藏在戊中，看戊落宫
