# 八门概述

八门是奇门**人盘**上的八个门，代表人事吉凶和行动方向。

- 吉门被迫吉事不成，凶门被迫则凶灾尤甚

## 金函玉镜补充

> 出自《金函玉镜奇门遁甲秘笈全书》卷二十二——八门吉凶诗诀。

### 八门吉凶诗诀（金函玉镜版）

**休门**：一一气盈室，富贵子孙田土吉，祭祀修营入宅基，赴官迁徙事周悉，产招逃绝入兴隆，北旺冬时数六一，南北婚姻有远亲，送来六畜增官秩。

**生门**：八八气盈星，凶煞皆降尊土精，因待女财人寄物，从兹致富子孙兴，三年定有贵儿产，出入外州泉货盈，嫁娶种萌并造作，消灾发福有奇灵。

**伤门**：气短数三三，寅卯旺方音角间，渔猎捕征搜索债，更宜赌博追亡还，官司口舌重丧至，六畜邅瘟火盗艰，夫妇血光灾眼症，三旬产厄祸力残，刑名死以兼疯疾，蛇虎伤人居不安。

**杜门**：四四星凶恶，木性时方寅卯泊，闭堤绝水事封培，追邪伐盗并勾捉，出亡逃难断欲宜，隐伏邀遮俱可托，去佞远藏埋闭藏，克绝阴私能久约，用动似防盗贼侵，官刑财散瘟疫伤，蛇伤雷打疥疮浓，焚廪人亡家退落。

**景门**：九九紫气盈，巳午旺南寅戌结，遣使上书能解危，求谋修造访寻谒，葬埋嫁娶吉中斗，给赏吏人如手捉，献策求名墀陛亲，举科文士藻文洁。

**死门**：二二凶星逆，戊己坤艮方位即，穿猎渔网刑戳宜，送丧吊死葬埋益，修营妨长及平房，忤逆重丧亡产憾，所求不利不宜行，动见败亡官落职。

**惊门**：无化气为逆，旺在庚辛申酉地，罗网张疑立狱讼，攻门刑击一齐到，逃亡掩捕得功能，市贾营修皆可忌，致讼虚惊疾疫兴，败囚军贼牛羊毙。

**开门**：六六气营奇，谒贵求谋利有为，立宅扦修官职进，外来财帛马牛肥，蜂蜜窖冶横财发，富盛子孙名利齐，金上庚辛秋月旺，奴田畜产贾商宜。

## 上元经补充

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]八门九门章

### 门盘与天干的关系

门与干皆为人盘，门为静、干为动，一阴一阳，备人事之全。门为人行之路、为通道。干和门构成了一对阴阳矛盾统一体——天干是动态的、物质性的、占据主导地位的；门是通道，必须由"门"而入。

### 八门命名的易学根据

八门的命名实质上根据**甲木周游不同宫位产生的强弱状态**而来：

| 门 | 宫位 | 甲木状态 | 命名缘由 |
|----|------|----------|----------|
| [[开门]] | 乾宫亥 | 长生 | 甲木运行轨迹起始，申被亥穿庚金受伤 |
| [[休门]] | 坎宫子 | 沐浴 | 甲木沐浴、庚金死地，修养生息 |
| [[生门]] | 艮宫寅 | 临官（禄） | 甲木禄地、庚金墓，生机最旺 |
| [[伤门]] | 震宫卯 | 帝旺（刃） | 阳刃所在，过犹不及，双面性 |
| [[杜门]] | 巽宫辰巳 | 衰/病 | 甲木由盛转衰，庚金长生 |
| [[景门]] | 离宫午 | 死 | 内中生气耗尽，外呈繁茂绚烂之景 |
| [[死门]] | 坤宫未申 | 墓/庚禄 | 甲木墓地+庚金禄地，至凶 |
| [[惊门]] | 兑宫酉 | 胎 | 弱甲逢旺庚，惊恐心态 |

### 门的旺衰辨析（核心观点）

张大春先生驳斥了三种错误观点：

1. **门依据月令衰旺**：错误——春季生门土旺衰破，难道春天不能赚钱吗？
2. **门有长生禄旺入墓**：错误——开门金墓于丑，临艮宫受宫生其实反吉；开门临巽宫虽"长生"但门宫相迫反凶。不能把天干周游十二宫的原理硬套在门盘系统
3. **门衰旺与九星相关**：错误——星是天象，作用在示象而非直接与门宫构成生克。天体对地球的影响主要体现在太阳与月亮

### 门迫——唯一需要注意的门之凶象

门立于地上，与地的关系最紧密。"地气不可犯，犯之必有灾殃"。门克宫即为**门迫**：
- 吉门克宫（门迫）→ 吉性大减
- 凶门克宫 → 凶顽之性更强
- 宫克门 → 影响远小于门迫，实际预测中可忽略不计

> 上元经的结论：门与月令、星辰没有直接关系，也无生旺墓之别。唯一需要注意的就是**门迫**。

## 奇门宝鉴补充

> 出处：《御定奇门遁甲·奇门宝鉴》（金志文点校）卷一·释义四十四则 — 参见 [[御定奇门遁甲奇门宝鉴（金志文点校）]]

### 释门（八门命名之五行理据）

五行之理，畏克喜生。甲木位于东，庚金位于西，东西相对，木之畏金也甚矣，故用奇以制庚。庚制而退休于坎，以就水之生，故正北为[[休门]]。休于北而对于南，有向明而治之象。景者大也，故正南为[[景门]]。水能滋木，金能生水。生则金气泄，泄则木气舒。故金水之间，西北为[[开门]]。有开必有阖，故对开为[[杜门]]，杜者，言阳气欲尽也。木既休于水，则水气畅，故水木之间，东北为[[生门]]。有生必有死，故对生为[[死门]]，死者，言落其实而取其材也。庚欲犯甲，甲既休于北，而对之者惟有卯木，不无受伤之患，此伤而彼亦惊，故正西为[[惊门]]，正东为[[伤门]]。

### 释迫（门迫定义）

迫者，门制其宫也。

| 八门 | 迫宫 |
|------|------|
| 开门 | 临三、四宫 |
| 休门 | 临九宫 |
| 生门 | 临一宫 |
| 景门 | 临六、七宫 |
| 伤门、杜门 | 临二、八宫 |
| 死门 | 临一宫 |
| 惊门 | 临三、四宫 |

凡此为迫，不利诸事。《经》云：「吉门被迫，吉事成凶。凶门被迫，凶灾愈甚。」值此惟宜安静，不宜举动。

### 释阴阳刑德开阖

阴刑阳德，阳开阴阖也。

| 节气 | 德所在 | 刑所在 |
|------|--------|--------|
| 冬至 | 卯 | 酉 |
| 夏至 | 酉 | 卯 |
| 春分 | 午 | 子 |
| 秋分 | 子 | 午 |
| 立春 | 辰 | 戌 |
| 立秋 | 戌 | 辰 |
| 立夏 | 未 | 丑 |
| 立冬 | 丑 | 未 |

凡刑德所在，三气共之。刑德不及寅申巳亥者，以四孟为四生，故不及也。

**五阳星**：蓬、任、冲、辅、禽——凡五阳星加时为**开**
**四阴星**：英、芮、柱、心——凡四阴星加时为**阖**

将兵以开阖分主客，以刑德定坐击：**阖为主、开为客，坐阳德、击阴刑**。

### 释三甲（孟甲仲甲季甲）

三甲者，孟甲、仲甲、季甲也。

| 三甲 | 直符 | 入局 |
|------|------|------|
| 仲甲 | 甲子、甲午 | 上元入局 |
| 孟甲 | 甲寅、甲申 | 中元入局 |
| 季甲 | 甲辰、甲戌 | 下元入局 |

三元三甲直符之时，举事视刑德为动静，将兵视刑德为战守，皆以**在门**决之。在门者，刑德在直使之门也。

- 德在门之时，宜动宜战
- 刑在门之时，宜静宜守

凡德在门之时，阳星加之为**尽开**，阴星加之为**半开**；刑在门之时，阳星加之为**半阖**，阴星加之为**尽阖**。

> 《经》云：「能知三甲，一开一阖。不知三甲，六甲尽阖。」

### 统宗版特点

- 统宗卷二八门所主详列各门**宜作之事**与**不宜作之事**，较卷一歌诀更详
- 开门「不宜治政」、休门「不宜行刑断狱」、生门「不宜埋葬治丧」等禁忌为统宗独有
- 强调「**吉门有气益吉，无气减吉；凶门有气益凶，无气减凶**」——门气旺衰影响吉凶程度
