# 休门

## 基本属性

| 属性 | 内容 |
|------|------|
| **五行** | 水 |
| **阴阳** | 阴 |
| **对应宫位** | [[九宫|坎一宫]]（正北方） |
| **对应九星** | [[天蓬星]] |
| **对应八神** | 玄武（通常） |
| **吉凶** | 吉门 |

| **月份** | 亥、子月（农历十、十一月） |

## 名称释义

> 休门居坎一宫，属水。休门意即"休养生息"，代表休息、休养、退休、休闲、安宁之意。古人认为冬季万物收藏，人亦当休整，故休门配坎宫。

休门是"三吉门"之一（开、休、生）。与开门代表"开创"不同，休门代表的是"休整""保养"的阶段。如同一个人劳累了需要休息，万物到了冬天需要休养生息。休门也是公门、官贵之门，古书中说休门"宜见贵人"。

在奇门中，休门虽然属水，但因为坎宫的特殊性（坎为险），休门也有"以休止险"的含意。

## 详细象意

### 人事象意

- **职业身份**：公务员（公门人）、退休人员、休闲业从业者、酒店服务人员、水运工作者、养生师
- **与人相关**：懂休息的人、善保养的人、公门中人（政府部门的一般工作人员，区别于开门的领导）、性情温和的人
- **事情状态**：休息、停顿、养精蓄锐、等待时机、休闲娱乐、退休养老

### 地理象意

- 休闲场所：公园、度假村、温泉、浴场
- 水边：江河湖海、水库、码头、港口
- 酒店、宾馆、招待所
- 退休人员活动场所、养老院
- 政府机关（非领导层）、事业单位
- 正北方位的地点

### 物品象意

- 船只、水上交通工具
- 休闲娱乐用品
- 养生保健品
- 名片、证件（公门人）
- 饮料、酒水

### 身体象意

- 肾脏、泌尿系统
- 生殖系统
- 耳朵（坎为耳）
- 血液、体液
- 腰部

### 性格象意

- 温和、谦逊
- 善于休整、不急不躁
- 随遇而安、知足常乐
- 善交际（公门人）
- 聪慧、机灵

### 其他象意

- **动物**：鱼、龟、猪（水生动物）
- **颜色**：黑色、蓝色
- **数字**：1（坎一宫之数）
- **天干**：癸（休门藏癸）

## 旺相休囚

| 状态 | 条件 |
|------|------|
| **旺** | 冬（亥子月） |
| **相** | 秋（申酉月，金生水） |
| **休** | 春（寅卯月，水生木，子气泄） |
| **囚** | 夏（巳午月，水克火，耗力） |
| **死** | 四季末（土月，土克水） |

> 休门旺于冬季亥子月，此时水气最盛。若休门逢天蓬星同宫，虽为吉门临凶星，但仍以休门之吉为主。

落宫状态判断：

- 落[[九宫|坎一宫]]（本宫）：最吉，得地利
- 落[[九宫|乾六宫]]、[[九宫|兑七宫]]（金宫）：得生，吉
- 落[[九宫|震三宫]]、[[九宫|巽四宫]]（木宫）：泄气，一般
- 落[[九宫|离九宫]]（火宫）：耗力，次凶
- 落[[九宫|坤二宫]]、[[九宫|艮八宫]]、[[九宫|中五宫]]（土宫）：受克，不吉

## 应用宜忌

### 宜

- ✅ **谒贵求官**：休门代表公门，宜拜访官员贵人
- ✅ **休养身心**：生病恢复期宜用休门
- ✅ **聚会宴请**：休闲娱乐之事
- ✅ **求财经营**：尤其与水相关行业
- ✅ **退休规划**：退休、交接、退隐之事
- ✅ **出行**：水路出行尤其宜

### 忌

- ❌ 休门逢死门、惊门，求谋不利
- ❌ 休门落坤宫、艮宫（土克水），吉中有阻
- ❌ 休门逢天蓬星（凶星），吉中藏凶
- ❌ 休门逢空亡，养而不休，事难尽意

## 与其他概念的关系

- 休门属水，对应 **[[天蓬星]]**。天蓬虽为凶星，但休门为吉门，以吉为主，吉中藏险
- 休门居 **[[九宫|坎一宫]]**，坎为水、为险、为中男，休门以休止险
- 休门与 **[[癸（天网之神）|癸]]** 天干同宫（休门藏癸）
- 休门与 **[[开门]]**、**[[生门]]** 合称"三吉门"
- 休门加 **[[乙奇]]** 利谒贵，加 **[[丙奇]]** 利求财，加 **[[丁奇]]** 利文书

## 实战要点

1. **测求财**：休门落宫旺相，求财顺利；与水相关行业尤其有利
2. **测工作**：休门代表公职，落宫吉则表示工作稳定轻松
3. **测婚姻**：休门有阴人之象，利女性
4. **测疾病**：休门代表肾脏、泌尿系统，需注意
5. **测出行**：休门宜水路出行，需注意休门所临之星神是否有凶象
