# 死门

| 属性 | 内容 |
|------|------|
| **五行** | 土 |
| **阴阳** | 阴 |
| **对应宫位** | [[九宫|坤二宫]]（西南方） |
| **对应九星** | [[天芮星]] |
| **对应八神** | 九地（通常） |
| **吉凶** | 凶门 |

## 名称释义

> 死门居坤二宫，属土。死门意为"死亡、结束、终结"，代表死亡、丧事、地皮、坟墓等凶事。古人认为坤为地、为母、为收藏，万物入土即为死，故死门配坤宫。

死门是奇门中最凶的门之一，代表一切的结束和死亡。但死门也有相对"吉"的用法——死门代表"地皮"，在建筑、房地产预测中死门代表土地；死门还代表猎户、屠户等与"杀生"相关的职业。

值得注意的是，死门的吉凶与落宫关系极大。死门落在二五八九宫反而相对吉（因为这些宫位属土或得土生），落在一三四宫则更凶。

## 详细象意

### 人事象意

- **职业身份**：屠户、猎人、殡葬从业人员、行刑之人、地皮开发商、农民（与土地相关）
- **与人相关**：死人、将死之人、重病患者、绝望的人
- **事情状态**：死亡、结束、失败、终结、不可挽回

### 地理象意

- 坟地、墓地、陵园
- 地皮、空地、荒地
- 屠宰场
- 刑场、法场
- 殡仪馆、太平间
- 西南方位的地点

### 物品象意

- 丧葬用品
- 凶器、刑具
- 地契、土地证
- 死尸

### 身体象意

- 脾胃（土主脾胃）
- 腹部（坤为腹）
- 肌肉
- 疤痕、肿瘤
- 代表死亡的病症（绝症）

### 性格象意

- 固执、死板
- 心灰意冷
- 低调、沉默
- 坚韧（死门也有死守之意）
- 实在、不浮夸
- 悲观

### 其他象意

- **动物**：牛（坤为牛）等大型牲口
- **颜色**：黑色、黄色
- **数字**：2（坤二宫之数）
- **天干**：己（死门藏己）

| 状态 | 条件 |
|------|------|
| **旺** | 未申月（坤宫当令）、四季末（土月） |
| **相** | 夏（巳午月，火生土） |
| **休** | 秋（申酉月，土生金，子气泄） |
| **囚** | 冬（亥子月，土克水，耗力） |
| **死** | 春（寅卯月，木克土） |

> 死门旺于未申月（坤宫当令）和四季末土月。死门旺则土地之事可成。若死门逢天芮星同宫，则疾病死亡之象极强。

## 与其他概念的关系

- 死门属土，对应 **[[天芮星]]**。天芮为病星、凶星，代表疾病。死门逢天芮，疾病死亡之象最显
- 死门居 **[[九宫|坤二宫]]**，坤为地、为母、为众、为腹。死门得坤宫之气
- 死门与 **[[六仪（戊己庚辛壬癸）|己]]** 天干同宫（死门藏己）
- 死门与 **[[生门]]** 为对宫（坤二 vs 艮八），一生一死，互为对比
- 死门在建筑术中代表"鬼门线"相关概念
- 死门加 **[[乙奇]]**：地皮之事可成；加 **[[丙奇]]**：土地有纠纷；加 **[[丁奇]]**：可解部分凶性

## 实战要点

1. **测生死**：死门加日干或年命，病重不治
2. **测疾病**：死门加天芮星，病情危重
3. **测房产**：死门代表地皮，落宫旺相则土地有利
4. **测行人**：死门代表此人凶多吉少
5. **测事业**：死门代表事业已死、不可挽回
6. **死门与其他门的禁忌**：死加伤、死加杜、死加景、死加惊均为大凶，需特别警惕
