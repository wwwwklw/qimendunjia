# 天乙

## 定义与基本属性

**天乙**是奇门遁甲中**值符**下临地盘之星，是预测中的重要用神。

- **别名**：值符下临星
- **位置**：天盘值符所落之宫的地盘星
- **性质**：重要用神
- **象意核心**：天乙是值符的"脚下之星"，值符压在天乙上方，天乙是承受方、被动方

> **简单理解**：值符是最高统帅，天乙是统帅脚下踩着的那颗星——你可以理解为被统帅管着的人或事。

## 详细象意

### 天乙与值符的关系

| 对比 | 值符 | 天乙 |
|:---|:---|:---|
| **位置** | 天盘值符 | 值符下临地盘之星 |
| **性质** | 主动方、上级、原告 | 被动方、下级、被告 |
| **代表人物** | 债权人、原告、贷款方、管理者 | 债务人、被告、借款方、被管理者 |
| **判断** | 值符生天乙——主动方有利 | 天乙生值符——被动方有利 |

### 在预测中的具体应用

#### 1. 债务纠纷
| 角色 | 用神 | 说明 |
|:---|:---|:---|
| **欠债人** | 天乙 | 债务人是被动方 |
| **债主** | 值符 | 债权人是主动方 |
| **判断** | 生克关系 | 值符克天乙——能要回钱 |

#### 2. 诉讼官司
| 角色 | 用神 | 说明 |
|:---|:---|:---|
| **被告** | 天乙 | 被告是被动方 |
| **原告** | 值符 | 原告是主动方 |
| **判断** | 生克关系 | 天乙克值符——被告可能胜诉 |

#### 3. 借贷关系
| 角色 | 用神 | 说明 |
|:---|:---|:---|
| **贷款人（借款方）** | 天乙 | 求人借钱是被动方 |
| **放贷人（出资方）** | 值符 | 决定借钱是主动方 |
| **判断** | 生克关系 | 值符生天乙——对方愿意借钱 |

### 胜败判断规则

| 生克关系 | 胜负判断 |
|:---|:---|
| **值符生天乙** | 主动方愿意让步，被动方有利 |
| **天乙生值符** | 被动方愿意配合，主动方有利 |
| **值符克天乙** | 主动方压制被动方，主动方胜 |
| **天乙克值符** | 被动方反制主动方，被动方胜 |
| **比和** | 双方势均力敌，可能和解 |

## 应用宜忌

### 宜
- **诉讼**：明确原告被告定位，看值符和天乙的生克
- **讨债**：值符为债权方、天乙为债务方
- **谈判**：判明谁是主动方谁是被动方

### 忌
- 混淆值符和天乙的方向
- 忽视其他用神的辅助判断
- 只用天乙不用其他用神

### 注意事项
1. 天乙的判断要结合具体场景——不同事项主客判断不同
2. 天乙和值符之间还要看有没有其他星门神的影响
3. 天乙也可以理解为"被值符管理管辖的对象"
4. 与[[主客论]]呼应：值符为主、天乙为客
