# 三才断卦法

## 定义与基本属性

**三才断卦法**是奇门遁甲古籍中记载的三种断卦方法之一，以"天盘（九星）→人盘（八门）→地盘（九宫）"的顺序分层判断，各有侧重。

- **别名**：古籍三法
- **核心**：天、人、地三才各有偏重场景，不可混用
- **吉凶规则**：门宫相生俱吉，相克俱凶

> **简单理解**：不同的事要看不同的盘——出行看星和门谁克谁，搬家看九宫好不好。一层一层来，不能乱看。

---

## 三才分层

### 第一层：天盘——首重九星

**出行趋避时，星克门为吉、门克星为凶。**

| 要素 | 说明 |
|:---|:---|
| **适用场景** | 出行、趋避（趋吉避凶） |
| **判断依据** | 天盘九星与人盘八门的生克关系 |
| **吉凶** | 星克门——吉（天时控制人事） |
|  | 门克星——凶（人事违背天时） |

> 出行时，天时比人事更重要。星的意志高于人的意志，所以要顺从天时。

### 第二层：人盘——首重八门

**凡出行趋避者首重八门。**

| 要素 | 说明 |
|:---|:---|
| **适用场景** | 日常出行、人事活动 |
| **判断依据** | 八门的吉凶属性（开休生为吉、伤死惊杜景为凶/中平） |
| **核心** | 门是人的行动通道，门开了路才通 |

> 八门是人的行动指南，出门先看门好不好。好门就走得顺，凶门就绕道走。

### 第三层：地盘——首重九宫

**造葬迁移首重九宫。**

| 要素 | 说明 |
|:---|:---|
| **适用场景** | 造葬（修建坟墓）、迁移（搬家、动土） |
| **判断依据** | 九宫八卦的方位和属性 |
| **核心** | 地盘不动，选址最重要 |

> 造葬和搬家都是"动土"的事，土不动则安，动土就要看地盘的方位好不好。

---

## 三才判断规则总结

### 门宫关系判断

| 关系 | 吉凶 | 说明 |
|:---|:---|:---|
| **门生宫** | 吉 | 门（人事）主动生宫（地利）——人事助地利 |
| **宫生门** | 吉 | 宫（地利）生门（人事）——地利养人事 |
| **门克宫** | 凶 | 门克宫即门迫——人事违逆地利 |
| **宫克门** | 凶 | 宫克门——地利阻碍人事 |
| **比和** | 吉 | 门宫五行相同——人事地利和谐 |

> 核心口诀：**门宫相生俱吉，相克俱凶。**

### 星门关系判断

| 关系 | 吉凶 | 说明 |
|:---|:---|:---|
| **星生门** | 吉 | 天时生人事 |
| **门生星** | 吉 | 人事合天时 |
| **星克门** | 出行吉 | 天时压倒人事——顺天 |
| **门克星** | 出行凶 | 人事违背天时——逆天 |

## 与其他断卦法的关系

| 断卦法 | 侧重 | 来源 |
|:---|:---|:---|
| **三才断卦法** | 分层——哪个场景看哪层 | 古籍 |
| **横竖远近高低看** | 多维度——同时看多个角度 | 现代总结 |
| **用神断卦法** | 专项——看特定用神 | 古籍 + 现代 |

三才断卦法是"纵向分层"的思路，横竖远近高低看是"多维度交叉"的思路。两者结合使用效果最好。

## 应用宜忌

### 宜
- **出行**：先看星门关系，再看门本身吉凶
- **搬家迁坟**：先看地盘九宫方位，再结合门
- **日常活动**：先看八门是否吉利
- **遇门克宫**：考虑更换方向或时间

### 忌
- 造葬时只看门不看宫——大忌
- 出行时只看宫不看星——次忌
- 把三种场景的规则混用——三才各有专攻

### 注意事项
1. 三才断卦法各有专属场景：出行看星、日常看门、动土看宫
2. 现代预测中三种方法需要灵活结合
3. 门宫相生相克是最基本、最常用的判断规则
4. 与[[断卦思路（横竖远近高低看）]]互为补充
