# 奇门遁甲概述

## 定义与基本属性

**奇门遁甲**是中国古代最高层次的预测学之一，全称为"奇门遁甲"。它与[[太乙与六壬|太乙神数]]、[[太乙与六壬|大六壬]]并称为"三式"，是古人最为推崇的三大预测术。

- **"奇"**：指乙、丙、丁三奇。乙为日奇、丙为月奇、丁为星奇。
- **"门"**：指休、生、伤、杜、景、死、惊、开八门。
- **"遁甲"**："遁"是隐藏的意思，"甲"是十天干之首。甲木为十干之首，最为尊贵，在排盘时藏而不露，隐遁于六仪（戊、己、庚、辛、壬、癸）之下。

奇门遁甲源于古代军事上的**排兵布阵**，具体而言源于**九宫八卦阵**。古代将领用它来择时择方、运筹帷幄，因此被称为：

- **"帝王之学"**：只有帝王和高级将领才能学习掌握。
- **"中国方术之王"**：因其体系庞大、逻辑严密、准确率高，被视为方术中的王者。

## 详细象意

通俗理解：奇门遁甲像一面多层次的镜子，把时间（天干地支）、空间（九宫方位）、人事（八门）、天象（九星）、神秘力量（八神）同时映入其中，让你看到一个立体的"时空人事全息图"。

## 分类体系

奇门遁甲从不同角度可以分成多种类型：

| 分类角度 | 类型 | 说明 |
|---------|------|------|
| **时间单位** | 年家奇门 | 以年为单位起局，用于国家大事 |
| | 月家奇门 | 以月为单位起局 |
| | 日家奇门 | 以日为单位起局 |
| | **时家奇门** | 以时辰为单位起局（最常用，本书重点） |
| **排盘方式** | 活盘（排宫法） | 用活盘转动来模拟天地人神四层运转 |
| | 飞盘（飞宫法） | 九星八门按顺序飞布九宫 |
| **应用方向** | 数理奇门 | 用于预测、选择吉时吉方 |
| | 法术奇门 | 用于符咒、阵法等（难度高、传承少） |

## 四层结构

奇门盘面由四层叠加构成，从上到下依次是：

1. **神盘（八诈盘）**：最顶层，八神运转，代表神秘力量的影响
2. **人盘（门盘）**：八门运转，代表人事吉凶
3. **天盘**：九星运转，代表天时
4. **地盘**：九宫八卦固定，代表地理方位

这四层叠加起来，构成了"天、地、人、神"四维一体的预测模型。

## 格局数量

时家奇门分为阴遁和阳遁两大类：

- **阳遁九局** + **阴遁九局** = 共 **18 局**
- 每局60个时辰，18 × 60 = 1080 个时辰局
- 飞盘法定局数：9 × 60 × 18 = **9720 种格局**
- 排宫法定局数：**8640 种格局**

> 这八千多种格局构成了奇门遁甲的庞大信息库，每种格局都有其特定的解读。

### 遁甲神机赋（卷二十一）

《金函玉镜》卷二十一收录的《遁甲神机赋》，是一篇高度概括奇门遁甲核心法则的韵文：

> 「两仪主使三才攸，分步咒摄乎鬼神，存局通乎妙旨前，修删简灵文，裁整诸经要理原。」

**格局要诀**：
- 甲加丙兮龙回首，丙加甲兮鸟跌穴——回首则喜悦易遂，跌穴则显灼易成
- 乙遇辛而龙逃走（身残毁），辛遇乙而虎猖狂（财虚耗）
- 癸见丁兮螣蛇跃蹻，丁见癸兮朱雀投江

**遁格要诀**：
- 生临丙戊天遁用兵，开乙临己地遁安坟，休丁遇太阴人遁安营

**神煞要诀**：
- 阳遁直符后一为九天，后二为九地，前三为六合，前二为太阴
- 阴遁直符前一为九天，前二为九地，前五为六合，前七为太阴
- 太阴潜形而隐遁，六合遁身而谋议，九天之上扬威武，九地之下匿兵马

> 「天地备兮难量，神机妙兮莫测，学者欲临事，有谋存心，斯赋无惑。」

### 九遁歌（卷二十一）

金函玉镜卷二十一收录的九遁歌诀，概括了九遁的组合条件：

```
奇门九遁少人知，秘在仙经甚奥微，
丙合生门居丁上，便为天遁月光辉。
乙合开门临六己，地遁日精能蔽之。
丁共休门合太阴，人遁星精藉照临。
丙合生门合九天，神灵所蔽有威权。
丁合休门临九地，伏兵鬼遁能潜避。
乙合休门临坎水，即是蛟龙得云雨。
辛仪合生临艮出，猛虎神威净八蛮，
虎狂位遇丁景制，叱咤生风上将坛。
龙逃亦藉丁和景，一跃雷门震羽轮。
```

> 此九遁歌简洁概括了[[天遁]]、[[地遁]]、[[人遁]]、[[神遁]]、[[鬼遁]]、[[龙遁]]、[[虎遁]]、[[风遁]]的组合，各遁详细解读见对应笔记。

### 易数总断（卷二十二）

金函玉镜卷二十二收录的《易数总断》是奇门遁甲的概论性长篇韵文，涵盖排盘、格局、遁法等核心内容，可看作金函玉镜版的"烟波钓叟歌"。

**核心摘录**：
> 「玉册天元诚至理，分丁役甲洛书数；阴阳变化互逆顺，参伍错综并九坵；遁曰三才天地人，龙虎风云及鬼神；生土生中附神位，八卦推行列八门；一中造化露天机，上可凌虚升步飞；中可测推谨趋避，藏神合朔卜时期」

**格局摘要**：
- 冲辅禽任心最吉，三奇加互发祯祥
- 三奇得使诚堪用，六甲遇之非小补
- 乙归犬马丙鼠猴，六丁玉女骑龙虎
- 三奇入墓总非宜，五不遇时尤可嗔
- 逆顺直符统七神，地宜伏匿天扬兵
- 急从神兮缓从门，天三门兮地四户

## 应用宜忌

- **宜**：重大决策参考、择吉时吉方、运势分析、事件预测
- **忌**：不要事事依赖、不要忽略人为努力、初学勿用于害人之事

## 上元经补充

> **source:** [[奇门遁甲上元经（张大春）]] 释名章

### 奇/门/遁甲命名的真正含义

《奇门遁甲上元经》对"奇门遁甲"命名依据有**独到而核心的见解**，纠正了市面上流行的误解：

#### 奇 = 三奇，非六仪

"奇"实为**乙、丙、丁三个具有特殊能量的天干**，并非三奇六仪的总括性称呼。

- 奇门中大部分吉格的构成都依赖于三奇
- 风水择吉和法术奇门中乙、丙、丁也能发挥关键作用
- 关键在于：三奇对甲帅的天敌庚金具有**直接克制作用**——乙合制庚金，丙、丁为庚的官杀

#### 门 = 值使门，非诸门概称

"门"为**统领各门的值使门**，而非八门或九门的总称。

- 相较于其他门，值使门可以更好地把握事件运行的整体情况
- 值使门落宫为隐藏甲帅的当值六仪在某一具体时辰内所临之地
- 其旺衰吉凶是代表甲木执行既定方针政策的具体表现

#### 遁甲 = 值符

"遁甲"**专指隐藏六甲大将的当值天盘六仪**，即值符星落宫所在地。

- 遁甲好比是将帅甲木的司令部
- 遇大事首观搭配值符星的当值六仪是奇门预测的钥匙

#### 三者以甲为核心

> 无论是三奇，还是符使，它们有一个共同的核心——**甲**。奇门布局星罗棋布，错综复杂，但中心原则都围绕着甲木展开。观测隐藏的甲木的运动轨迹，就能把控奇门遁甲术数体系的关键脉络。

上元经明确指出：三奇、值符、值使在整个奇门遁甲预测体系中处于**枢纽地位**，明晰具体用神与它们三者之间的生克关系对事物情状的把握起着关键作用。

### 命名独取人盘的深意

奇门遁甲的命名中**隐去指称天时的九星、表征地利的九宫**，而单单留下代表人类活动的奇仪与诸门（三者皆属人盘范畴）。

深层奥义：强调**人的主观能动性**——"天时不如地利，地利不如人和"。

- 提醒后学始终保持进德修业、砥砺前行的奋斗姿态
- 用心努力征服自然界的各种艰难险阻

### 神盘何以不被命名

神盘系统在普通的奇门预测中**并不能从根本上决定事物的吉凶**，仅是代表着宇宙中的一种神秘场能，类似于现代科学中所说的暗物质。可在法术奇门中发挥一定效用，而在普通的数理奇门预测中不起决定性作用。
