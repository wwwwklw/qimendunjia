# 主客论

## 定义与基本属性

**主客论**是奇门遁甲应用中判断"我是主动方还是被动方"的核心理论。在任何预测和应用场景中，首先要分清谁是主、谁是客，这直接决定了你应该采取主动还是被动的策略。

通俗理解：主客论就是教你"该进攻还是防守"。主方是防守的一方，客方是进攻的一方。主客判断对了，奇门的吉凶建议才有意义。

## 主客判断的四条原则

| 原则 | 主 | 客 |
|------|-----|-----|
| **动静** | 静为主 | 动为客 |
| **先后** | 后动为主 | 先动为客 |
| **主动被动** | 被动为主 | 主动为客 |
| **天地盘** | 地盘为主 | 天盘为客 |

> 简单记：**动的是客、静的是主**。谁先动谁是客，谁后动谁是主。

## 时干的主客属性

十天干分为五阳时和五阴时：

### 五阳时——利客（宜主动出击）

**甲、乙、丙、丁、戊**

阳时气场偏动、偏阳，有利于主动一方。在阳时做事宜主动出击、先发制人。

### 五阴时——利主（宜以静制动）

**己、庚、辛、壬、癸**

阴时气场偏静、偏阴，有利于被动一方。在阴时做事宜等待时机、后发制人。

## 星宫生克的主客判断

通过天盘星和地盘宫的生克关系来判断利主还是利客：

| 生克关系 | 利主/利客 | 说明 |
|---------|-----------|------|
| **天盘星生地盘宫** | 利主 | 天星主动生宫，气泄于客，对主方有利 |
| **地盘宫生天盘星** | 利客 | 地盘生天星，主方之力助客，对客方有利 |
| **天盘星克地盘宫** | 利客 | 星克宫，客方克制主方，客有利 |
| **地盘宫克天盘星** | 利主 | 宫克星，主方克制客方，主有利 |

> 简单记：**谁被生/被克则利谁**——生助给谁就利谁，克制压制谁就利对方。

## 应用宜忌

- **宜**：每次分析前先分主客，这是奇门应用的第一步
- **宜**：利主时守住阵地、静待时机，不要贸然出击
- **宜**：利客时果断行动、先发制人，不要犹豫不决
- **忌**：利主时强出头、利客时拖延等待，大忌
- **忌**：把主客判断绝对化，需要结合具体格局综合分析

## 上元经补充

> **source:** [[奇门遁甲上元经（张大春）]] 主客功用章

### 主客法与日时用神法的关系

上元经指出主客法与日时用神法**并不冲突**，而是侧重点不同：

- **日时用神法**（张志春体系）：近景顺序扫描模式，注重天盘动态的、阳性信息，像讲故事一样娓娓道来
- **主客动静法**（古籍重视）：远景综观扫描模式，只重全局而忽视细节
- 两个体系**各有所长、各有所短**，需充分把握两者才能对奇门整体面貌有更深了解

### "动者为客、静者为主"须从全局着眼

上元经直接指出"动者为客，静者为主；先动为客，后动为主"的疏漏：

> 古代某支军队要攻占一个城池，在进攻方到达之前，守军先动了起来，加固城池，派遣守军出城设伏——敌未动而我先动。此时是否还能说"先动为客、后动为主"？

**关键**：主客动静必须从**宏观角度、事件全局**着眼，善用整体的、系统的观点看问题。

### 主客二分中的"有我"与"无我"

上元经将主客划分进一步细化为两种情况：

#### 无我之占
占测与己无关的事件，不需要观察日干，直接以**符使宫**或用神宫主客对比占断。

#### 有我之占（两种下位类型）

| 类型 | 特征 | 判断依据 |
|------|------|---------|
| **我为主客之一** | 求测人是主客矛盾中的一方 | 综合研判日干宫、符使宫、具体用神宫三个主客之间的生克制化 |
| **主客皆他人，我利益牵涉其中** | 不直接涉入矛盾但利益相关 | 日干宫与中宫的主客对比 |

### 多层次主客对待

奇门的星、神、门、干均成天地对峙、阴阳对立之象：

- 不仅天地两干形成主客矛盾
- **天地二星、天地二神、天地二门**也能形成相应的主客矛盾对立
- 一宫内的主客关系呈现符号的多元化对立（飞盘体系）
- 不同宫内的主客关系也要分辨（如符使主客二分）
- **天盘值符落宫与地盘值符落宫**也是一对主客矛盾

### 利主利客的精当辨析

上元经纠正了两个重要误解：

#### 伏吟利主、返吟利客的偏颇

- 伏吟局中癸加癸力量最弱，动性自然不强；但**双庚相合**力量强大，想静也难
- 马星、五合、三合可引动伏吟局，部分宫位合成动能
- 要善于观察伏吟中的**动性因素**，返吟中的静态因素
- 太极精义：阴中有阳，阳中含阴，阴阳环抱而成生生大道

#### 五阳时利客、五阴时利主的谬误

> 上元经明确指出："五阳时利客，五阴时利主"为**大谬**。

正确判断利主利客的依据：
- 必须看时干所落宫位**天地干组合所形成的大象**
- 天地全阳格局（戊加丙、丁加乙等）→ 利客
- 由阴转阳格局（壬加戊等）→ 利客
- 壬丁之合、戊癸之合等五合 → 利客
- 大象全阴、由阳入阴、明冲暗冲及部分五合 → 利主

### 飞盘与转盘的主客差异（飞盘体系）

飞盘奇门的天地盘同为跳涧九宫，天地之间的主客对应关系与转盘圆转排列方式不同。详见 [[飞盘系统精要]]。

## 奇门遁甲统宗补充

> 出处：《奇门遁甲统宗》卷之三·兵事分主客、主客雌雄 — 参见 [[奇门遁甲统宗（诸葛亮）]]

### 兵事分主客

统宗卷三以宫、星、门之生克论主客：

> 宫为主，星门为客。

| 生克关系 | 利主/利客 |
|---------|-----------|
| 客生星门（宫生星门）| 利客 |
| 星门生宫 | 利主 |
| 宫克星门 | 利主 |
| 星门克宫 | 利客 |
| 两相比和 | 主客势均 |
| 一克一生 | 主客互伤 |

### 主客雌雄（卷三·演卦论主客）

统宗卷三以演卦世应论主客雌雄：

> 世为客，应为主。

| 卦象关系 | 利主/利客 |
|---------|-----------|
| 世克应、应生世 | 利为客，宜先动 |
| 应克世、世生应 | 利为主，宜后动 |
| 世空 | 客兵不进 |
| 应空 | 主兵不前 |
| 世衰应旺 | 客弱主强 |
| 世旺应衰 | 客勇主怯 |
| 世应比和 | 战亦不分胜负 |
| 世应俱受月日之伤克 | 多败 |

### 统宗版特点

- 统宗卷三「兵事分主客」以**宫星门生克**论主客，简明扼要
- 「主客雌雄」则以**演卦世应**论主客，融入六爻纳甲体系
- 世应空亡论主客兵进退：世空客兵不进、应空主兵不前
- 世应比和则不分胜负，世应俱受伤克则多败
- 统宗版是奇门与六爻结合的典型文本
