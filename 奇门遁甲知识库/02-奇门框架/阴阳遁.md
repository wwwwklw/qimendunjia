# 阴阳遁

## 定义与基本属性

**阴阳遁**是时家奇门排盘的总纲。奇门遁甲的 18 种地盘格局（18 局）分为两大类：阳遁九局和阴遁九局。

阴阳遁的划分依据是**节气**，具体以冬至和夏至为分界：

- **冬至一阳生** → 用**阳遁**，顺排
- **夏至一阴生** → 用**阴遁**，逆排

通俗理解：冬至后阳气渐长，万物复苏，为"顺势而为"，所以顺排；夏至后阴气渐长，万物收敛，气势由盛转衰，所以逆排。这是"道法自然"在奇门中的体现。

## 详细象意

| 特征 | 阳遁 | 阴遁 |
|------|------|------|
| 起始节气 | 冬至 | 夏至 |
| 覆盖时段 | 冬至→芒种（上半年） | 夏至→大雪（下半年） |
| 局数 | 1~9 局 | 1~9 局 |
| 运转方向 | 顺排（1→2→...→9） | 逆排（9→8→...→1） |
| 三奇六仪排布 | 戊己庚辛壬癸丁丙乙 顺排 | 乙丙丁癸壬辛庚己戊 逆排 |
| 八神排布 | 顺时针：值符→腾蛇→太阴→六合→白虎→玄武→九地→九天 | 逆时针：值符→腾蛇→太阴→六合→白虎→玄武→九地→九天 |
| 性质 | 主动、向外、生长 | 主静、向内、收敛 |

## 四阳宫与四阴宫

八卦宫位也分阴阳：

- **四阳宫**（用阳遁）：坎一宫、艮八宫、震三宫、巽四宫
- **四阴宫**（用阴遁）：离九宫、坤二宫、兑七宫、乾六宫

> 注意：中五宫为太极，不属阴阳，但寄于坤二宫，实践中随坤宫。

## 三奇六仪的排列

### 阳遁顺排

戊 → 己 → 庚 → 辛 → 壬 → 癸 → 丁 → 丙 → 乙

从起局宫位（如阳遁一局从坎一宫）开始，按 1→2→3→...→9 的顺序依次布入。

### 阴遁逆排

乙 → 丙 → 丁 → 癸 → 壬 → 辛 → 庚 → 己 → 戊

从起局宫位（如阴遁一局从离九宫）开始，按 9→8→7→...→1 的顺序依次布入。

## 八神的排列

八神在阳遁和阴遁中排列方向不同：

- **阳遁**：值符、腾蛇、太阴、六合、白虎、玄武、九地、九天（顺时针转）
- **阴遁**：值符、腾蛇、太阴、六合、白虎、玄武、九地、九天（逆时针转）

即八神的**名称顺序不变**，但阳遁顺时针、阴遁逆时针转动。

## 应用宜忌

- **阳遁期间**（冬春）：宜主动出击、开拓进取、向外发展
- **阴遁期间**（夏秋）：宜以静制动、巩固防守、向内修行
- 错用阴阳遁方向会导致整个奇门盘面错乱，起局时必须首先判断

## 奇门遁甲统宗补充

> 出处：《奇门遁甲统宗》卷之一·阴阳二遁顺逆起例，卷之二·阳遁、阴遁 — 参见 [[奇门遁甲统宗（诸葛亮）]]

### 阴阳二遁顺逆起例歌诀

《奇门遁甲统宗》卷一载阴阳二遁顺逆起例歌诀，与通行版[[节气与定局]]所载歌诀基本一致：

**阳遁起例**：

> 冬至惊蜇一七四，小寒二八五相随。
> 大寒春分三九六，芒种六三九是仪。
> 谷雨小满五二八，立春八五二相宜。
> 清明立夏四一七，雨水九六三为奇。

**阴遁起例**：

> 夏至白露九三六，小暑八二五之间。
> 大暑秋分七一四，立秋二五八循环。
> 霜降小雪五八二，大雪四七一相关。
> 处暑排来一四七，立冬寒露六九三。

### 统宗版特点

- 歌诀分阳遁、阴遁两段，结构清晰
- 内容与[[烟波钓叟歌]]、金函玉镜版歌诀完全一致
- 是奇门遁甲定局的基础口诀，依节气确定上中下三元用局

### 阳遁布仪飞奇之法（卷二）

统宗卷二详述阳遁九局顺布六仪、逆布三奇之法：

> 阳遁九局，顺布六仪逆布三奇。

以阳遁一局为例：

| 六仪 | 所在宫 |
|------|--------|
| 甲子戊 | 坎一宫 |
| 甲戌己 | 坤二宫 |
| 甲申庚 | 震三宫 |
| 甲午辛 | 巽四宫 |
| 甲辰壬 | 中五宫 |
| 甲寅癸 | 乾六宫 |

三奇逆布：

| 三奇 | 所在宫 |
|------|--------|
| 丁 | 兑七宫 |
| 丙 | 艮八宫 |
| 乙 | 离九宫 |

> 因乙丙丁次于六七八九，故曰逆。

> 二局则甲子戊起坤二宫，三局则甲子戊起兑三宫，余仿此。

### 阴遁布仪飞奇之法（卷二）

统宗卷二详述阴遁九局逆布六仪、顺布三奇之法：

> 阴遁九局逆布六仪顺布三奇。

以阴遁九局为例：

| 六仪 | 所在宫 |
|------|--------|
| 甲子戊 | 离九宫 |
| 甲戌己 | 艮八宫 |
| 甲申庚 | 兑七宫 |
| 甲午辛 | 乾六宫 |
| 甲辰壬 | 中五宫 |
| 甲寅癸 | 巽四宫 |

三奇顺布：

| 三奇 | 所在宫 |
|------|--------|
| 丁 | 震三宫 |
| 丙 | 坤二宫 |
| 乙 | 坎一宫 |

> 八局则甲子戊起艮八宫，七局则甲子戊起兑七宫，余仿此。

### 卷二统宗版特点

- 阳遁顺布六仪（戊→己→庚→辛→壬→癸顺九宫）、逆布三奇（丁→丙→乙逆九宫）
- 阴遁逆布六仪、顺布三奇，与阳遁对称
- 各局以甲子戊所起之宫为局数（一局起坎一、二局起坤二…九局起离九）
- 统宗以「因乙丙丁次于六七八九，故曰逆」解释「逆布三奇」之理
