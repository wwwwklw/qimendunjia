# 六阴洞微真经

## 定义与基本属性

《**六阴洞微真经**》全称《秘藏通玄变化六阴洞微遁甲真经》，是[[法术奇门与数理奇门|法术奇门]]的代表性著作，托名宋初宰相赵普辑录。

| 属性 | 内容 |
|------|------|
| **全称** | 《秘藏通玄变化六阴洞微遁甲真经》 |
| **简称** | 《六阴洞微真经》 |
| **托名作者** | 赵普（宋初宰相） |
| **性质** | 法术奇门的代表性经典 |
| **位置** | 与《烟波钓叟歌》对立——《烟》为数理奇门经典，《六》为法术奇门经典 |

通俗理解：如果说《烟波钓叟歌》是奇门遁甲的"使用说明书"（教你排盘），那《六阴洞微真经》就是奇门遁甲的"咒语法本"（教你通过符咒作法改变现实）。

## 详细象意

### 书名含义

- "秘藏"：秘而不宣的珍贵内容
- "通玄变化"：通过玄妙的方法实现变化
- "六阴洞微"：六阴指阴遁六局？或六种阴性的深层奥秘？文献记载不详
- "遁甲真经"：真正涉及"遁甲"（隐遁、保护）的经典

### 托名赵普的问题

- 赵普是宋初著名宰相，"半部论语治天下"的典故就出自他
- 将法术著作托名于历史名人，是古代术数书籍的常见做法，用以增加权威性
- 实际作者不可考，成书时间可能在宋以后

## 应用宜忌

- **宜**：了解其存在和历史地位，作为法术奇门的知识背景
- **忌**：初学不应深究——法术奇门需要特定传承，自学难有成效
- **忌**：不要将此书中的法术内容与数理奇门的预测方法混淆
