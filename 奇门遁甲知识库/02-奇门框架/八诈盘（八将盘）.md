# 八诈盘（八将盘）

## 定义与基本属性

**八诈盘**是奇门遁甲[[地盘天盘人盘神盘|神盘]]的别称，又称八诈门、八将盘、针盘、顶盘。它是奇门遁甲四层盘面中的最顶层，代表八种神秘力量对人事的影响。

| 别名 | 含义 |
|------|------|
| **八诈门** | 八种带有"诈"（变化、隐蔽）性质的力量之门 |
| **八诈盘** | 以转盘形式呈现 |
| **八将盘** | 八位"神将"守卫各宫 |
| **针盘** | 可能与指针（天池）结合使用 |
| **顶盘** | 居于四层盘面的最顶端 |

通俗理解：如果把奇门盘面看成一辆车的仪表盘——地盘是导航地图、天盘是天时卫星信号、人盘是驾驶操作、八诈盘就像是"运气指数"和"意外干扰提示"，反映的是人力之外的、不可控的神秘因素。

## 详细象意

### 与六爻六神/六兽的类比

八诈盘上的八神与六爻预测中的六神/六兽有类似之处：

| 奇门八神 | 六爻六兽 | 对应关系 |
|---------|---------|---------|
| 值符 | 青龙 | 首领、正面主导力量 |
| 腾蛇 | 腾蛇 | 虚惊、怪异（一致） |
| 太阴 | — | 荫庇、密谋 |
| 六合 | 六合 | 和合、婚姻（一致） |
| 白虎 | 白虎 | 凶杀、病伤（一致） |
| 玄武 | 玄武 | 盗贼、暧昧（一致） |
| 九地 | — | 迟缓、稳固 |
| 九天 | — | 高远、张扬 |

六爻中还有**勾陈**和**朱雀**，在奇门八神中隐藏于白虎和玄武之下（见[[勾陈]]和[[朱雀（六兽）]]）。

## 应用宜忌

- **宜**：八神在奇门判断中起辅助作用——先看九星八门，再看八神加成
- **忌**：夸大八神的作用——八神是"加成"因素，不是决定性因素
- **忌**：将八神与六爻六神完全等同——虽有对应，但具体象意和使用规则不尽相同

## 与其他概念的关系
