# 用神

## 定义与基本属性

**用神**是预测时用来代表所测事物的符号。在奇门遁甲中，用神多达 **64 个**，是最复杂也是最核心的概念之一。

通俗理解：用神就是"你要看什么"。比如你想看自己的运势，日干就是你的用神；你想看财运，生门和戊就是你的用神。找到正确的用神，才能在奇门盘上找到对应信息。

## 详细象意

### 用神的三重特性

#### 1. 多义性

**同一个符号在不同场景下有不同含义。**

例如"开门"：
- 测工作 → 代表工作、单位
- 测出行 → 代表道路通畅
- 测官司 → 代表法官、判决

> 用神不是一个"固定标签"，而是要根据问题灵活解读。

#### 2. 多重性

**同一事物可以从多个角度观察，有多个用神。**

例如测婚姻：
- 日干 → 求测人自己
- 时干 → 婚姻这件事
- 乙奇 → 女方（或男方的妻子）
- 庚 → 男方（或女方的丈夫）
- 六合 → 婚姻媒介、关系质量
- 休门 → 家庭生活

> 多个用神相互印证，信息更全面。

#### 3. 纲领性

有主次之分，以下为三大纲领用神：

| 纲领 | 用神 | 作用 | 类似 |
|------|------|------|------|
| 第一纲领 | **日干** | 求测人、问事者本人 | 六爻中的"世爻" |
| 第二纲领 | **时干** | 所测的事情 | 六爻中的"应爻" |
| 第三纲领 | **年命** | 求测人的出生年干 | 补充个人信息 |

### 测不同事物取不同用神

| 被测事物 | 主要用神 | 辅助用神 |
|---------|---------|---------|
| 天时（天气） | 九星为主 | 天蓬（雨）、天英（晴）、天辅（风） |
| 人事 | 八门为主 | 日干时干 |
| 地理 | 九宫为主 | 地盘八卦 |
| 六亲 | 年月日时为主 | 十干对应 |
| 财运 | 生门、戊 | 日干时干 |
| 疾病 | 天芮星 | 时干、日干 |
| 考试 | 天辅星、景门 | 年命、日干 |

## 应用宜忌

- **宜**：先确定日干（自己）和时干（事情），再根据具体问题选辅助用神
- **宜**：多用神交叉验证，综合分析
- **忌**：用神太多抓不住重点，初学从日干时干入手最为稳妥
- **忌**：生搬硬套用神含义，要结合具体语境灵活理解

## 上元经补充

> **source:** [[奇门遁甲上元经（张大春）]] 用神章

### 动态用神体系

上元经提出区别于传统机械用神观的**动态用神体系**，核心包括历时（依时变化）和共时（宫位关联）两个维度。

### 全局用神与具体用神——沈氏用神相神理论

上元经借鉴沈孝瞻《子平真诠》的用神、相神概念，将其引入奇门体系：

#### 全局用神：甲木

奇门遁甲中隐藏在六仪下的六甲就是**全局的核心**：
- 甲是统帅、领袖、掌舵人
- 星、神、门、干、宫诸多元素都是为甲服务的
- **甲 = 全局用神**，保护甲的 = 相神

#### 三奇与符使为相神

- 乙合制庚，丙丁为庚之官杀 → 三者对庚有杀伤力或控制力 → **三奇为甲木的相神**
- 值符=元帅甲木头顶当值星辰，值使=甲木经过的门 → **星门也是相神**
- "奇"字的关键就在于它对庚这个七杀的制约

#### 全局用神与具体用神的关系

| 用神类型 | 作用范围 | 类比 |
|---------|---------|------|
| **全局用神（甲）** | 号令全局，统摄九宫 | 全省经济实力 |
| **具体用神** | 决单事吉凶祸福 | 某城市发展水平 |

**关键原则**：
- 具体用神为吉 ≠ 全局用神为吉（如并购成功但前景黯淡）
- 全局用神为吉 + 具体用神不佳 → **抓大放小**，认清矛盾主要方面
- 长远仍决之于符使（全局用神）

### 类占用神的四步选取法（飞盘体系核心方法论）

上元经针对奇门用神"星、门、神、干林立"的复杂性，提出**四步选取法**：

#### 第一步：观时之次序（历时维度）

星和神往往为**事件之初**，门随其后：
- 以高血压为例：天芮=病灶和病源（肝部），门与干=患者能直接感受到的病症（头晕脑涨、心脑血管问题）
- **时序不能搞乱**：星→神→门→干，各有先后

#### 第二步：通神之所辖（表征范围）

不同类型用神的应用范围有差别：

| 用神系统 | 表征范围 |
|---------|---------|
| 九星（天象） | 天时、天气、星辰 |
| 八门（人盘） | 与人关系最密切的事项 |
| 八神（神盘） | 意识、梦境等精神层面 |
| 天干（人盘） | 具体人事活动 |

#### 第三步：晓神之主次（宫位关联/共时维度）

同一类占符号内部有**主次差别**：
- 疾病：天芮为主（病源），天冲为辅（症状表象，如血压快速升高）
- 警察：白虎/勾陈=办案状态（神盘），伤门=具体侦查活动（人盘），两者构成**形与神的主客矛盾**
- **庚代表警察的条件**：①庚落宫与犯罪分子冲克 ②庚落宫与符使宫重合或三合/五合/六合——庚必须为甲所用

不同类占用神之间也有**表义相似性与差异性**，需仔细辨析。

#### 第四步：察力之聚散（能量关联）

类占用神与日时、符使、太岁、年命之间的关联：
- 通过三合、五合、六合构成能量统一体 → 成功概率大
- 类占用神落宫格局不佳，但符使全局用神大力支持 → 仍可成功
- 不能一叶障目，必须从**全局性、整体性**高度看问题

### 用神观与天人合一

上元经深入论证了用神选取背后的**天人合一整体观**：

- 二十二个干支文字通过**取类比象**表征宇宙万事万物
- 这是中华民族特有的思维方式，与西方物我二分的实验性研究方法根本不同
- 要提高干支取象水平，需深入理解《说文解字》的易学编排原理
- "戊"字为例：《说文》"中宫也。象六甲五龙相拘绞也"——"六甲"即六十甲子周游九宫，"五龙"即五日六十时辰，"相拘绞"是遁甲历气运流行

### 全局用神的易学地位

> 判定吉凶祸福根本上还是要从甲木身上着手，俗语所说"擒贼先擒王"就是这个道理。

抓住全局用神，从形而上的高度把握奇门核心，避免在枝节上纠缠不休。
