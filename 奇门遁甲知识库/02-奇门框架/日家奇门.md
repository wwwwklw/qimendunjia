# 日家奇门

## 定义与基本属性

**日家奇门**是奇门遁甲的一个独立分支体系，以"日"为单位起局（三日一局），与时家奇门以"时辰"为单位起局不同。日家奇门使用一套**不同的九星**系统，八门排布规则也异于时家奇门。

> **核心特点**：一卦管三日，三日换一门，五日换一奇。九星排列不同于时家奇门，使用太乙、摄提、轩辕、招摇、天符、青龙、咸池、太阴、天乙九星。

## 日家奇门九星

日家奇门使用的九星与时家奇门完全不同：

| 序号 | 日家九星 | 吉凶 | 说明 |

| 1 | **太乙** | **吉** | 求财万事通 |
| 2 | **摄提** | **凶** | 开打闹重重 |
| 3 | **轩辕** | **中平** | 出入多惊恐 |
| 4 | **招摇** | **凶** | 动处血光红 |
| 5 | **天符** | **中平** | 正宜打猎吉 |
| 6 | **青龙** | **吉** | 动处干财丰 |
| 7 | **咸池** | **凶** | 多怕官事起 |

> **日家九星歌诀**：「太乙求财万事通，摄提开打闹重重，轩辕出入多惊恐，招摇动处血光红，天符正宜打猎吉，青龙动处干财丰，咸池多怕官事起，太阴妇女结婚秾，天乙门中必遇贵，此法千金莫与穷。」

日家九星按六甲旬首起于艮宫（冬至后顺行，夏至后逆行）：

| 旬首 | 太乙起宫 |
|------|---------|
| **甲子** | 艮八宫 |
| **甲戌** | 离九宫 |
| **甲申** | 坎一宫 |
| **甲午** | 坤二宫 |
| **甲辰** | 震三宫 |

> 九星依次为：太乙→摄提→轩辕→招摇→天符→青龙→咸池→太阴→天乙，顺行排列。

### 卷廿二日家九星补录

卷二十二另有一段独立的日家九星歌诀：

> 「更有太乙九星位，冬至艮宫夏坤地，命起甲子顺逆行，本日住宫太乙立。」

## 排局规则

### 基本规则

- **一卦管三日**：每卦（宫）管三天，三日一换
- **三日换一门**：每三日休门换一宫
- **五日换一奇**：每五日三奇位置变换

### 日家奇门八门排布

#### 冬至阳遁四气（冬至、小寒、大寒、立春）

> 「甲戊壬子居坎，丁辛乙卯坤设，戊庚甲马震宫来，癸酉丁辛巽在，庚丙鼠逢干位，己癸兔走西街，丙壬午马艮游台，乙己鸡飞离界。」

以休门落宫起局：

| 干支日 | 休门落宫 |
|--------|---------|
| 甲子、乙丑、丙寅、戊子、己丑、庚寅、壬子、癸丑、甲寅 | **坎一宫** |
| 丁卯、戊辰、己巳、辛卯、壬辰、癸巳、乙卯、丙辰、丁巳 | **坤二宫** |
| 庚午、辛未、壬申、甲午、乙未、丙申、戊午、己未、庚申 | **震三宫** |
| 癸酉、甲戌、乙亥、丁酉、戊戌、己亥、辛酉、壬戌、癸亥 | **巽四宫** |
| 丙子、丁丑、戊寅、庚子、辛丑、壬寅 | **乾六宫** |
| 己卯、庚辰、辛巳、癸卯、甲辰、乙巳 | **艮八宫** |
| 壬午、癸未、甲申、丙午、丁未、戊申 | **兑七宫** |
| 乙酉、丙戌、丁亥、己酉、庚戌、辛亥 | **离九宫** |

> 八门排列顺序（从休门起）：休→生→伤→杜→景→死→惊→开（阳遁顺排）

#### 夏至阴遁四气（夏至、小暑、大暑、立秋）

> 「甲戊壬子离位，丁辛乙卯北居，戊庚甲马在坤期，癸酉丁辛震在，庚丙鼠归巽上，己癸兔走干室，丙壬午马兑宫吉，乙己鸡飞艮地。」

休门从离宫起，依九宫逆排。

### 大八门逐日出行诀

日家奇门有完整的逐节气八门出路歌诀，每四个节气一组：

#### 冬至四气（冬至→立春）
甲戊壬子居坎，丁辛乙卯坤设，戊庚甲马震宫来，癸酉丁辛巽在，庚丙鼠逢干位，己癸兔走西街，丙壬午马艮游台，乙己鸡飞离界。

#### 雨水四气（雨水→清明）  
甲戊壬子居兑，丁辛乙卯八宫，戊庚甲马在离中，癸酉丁辛坎用，庚丙鼠逢行坤位，己癸兔走东宫，丙壬午马巽为风，乙己鸡飞干应。

#### 谷雨四气（谷雨→芒种）
甲戊壬子居巽，丁辛乙卯干方，戊庚甲马兑宫藏，癸酉丁辛艮向，庚丙子行离位，己癸兔走北方，丙壬午马在坤冈，乙己酉归震上。

#### 夏至四气（夏至→立秋）
甲戊壬子离位，丁辛乙卯北居，戊庚甲马在坤期，癸酉丁辛震在，庚丙鼠归巽上，己癸兔走干室，丙壬午马兑宫吉，乙己鸡飞艮地。

#### 处暑四气（处暑→寒露）
甲戊壬子居震，丁辛乙卯东南，戊庚甲马却行干，癸酉丁辛兑换，庚丙鼠逢艮位，己癸兔走离轩，丙壬午马坎宫眠，乙己离飞坤转。

#### 霜降四气（霜降→大雪）
甲戊壬子干位，丁辛乙卯酉宫，戊庚甲马艮相逢，癸酉丁辛离应，庚丙鼠行坎位，己癸兔坤宫，丙壬午马震宫兴，乙己鸡逢巽用。

### 每日出行八门捷诀

> 「甲干丙艮戊子风，震庚壬坎起休门，乙卯东南丁卯火，兑辛干癸卯东中，离丙兑戊西北，庚午东南壬午，坤乙酉丁山坤母，辛酉南行癸酉东，此是休门真居处，每宫三日顺排轮。」

## 与时家奇门的区别

| 比对项 | 时家奇门（常用） | 日家奇门 |
|--------|----------------|---------|
| **起局单位** | 时辰（2小时） | 日（一卦管三日） |
| **九星** | 天蓬~天英 | 太乙、摄提、轩辕等 |
| **排盘方式** | 活盘/飞盘 | 固定八卦三日一换 |
| **常用场景** | 日常预测、择时 | 出行择日 |
| **复杂度** | 高 | 较低 |
| **三奇排布** | 五日一换 | 五日一换（同） |

## 应用宜忌

- **宜**：日家奇门主要用于出行选择、日常择吉
- **宜**：向三吉门行大吉
- **忌**：日家奇门的九星与时家奇门九星完全不可混淆
- **忌**："辰戌丑未皆为杀"——这些日子需谨慎
