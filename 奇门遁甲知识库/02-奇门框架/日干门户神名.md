# 日干门户神名

## 定义与基本属性

**日干门户神名**是奇门遁甲中十干所对应的门户神名系统，每一日干皆有特定的天神名号与门户象征。

> 出处：《御定奇门遁甲·奇门宝鉴》（金志文点校）卷一·释义四十四则 — 参见 [[御定奇门遁甲奇门宝鉴（金志文点校）]]

## 十干门户神名的应用

### 时加十干吉凶

| 时干 | 神名 | 出行所主 |
|------|------|----------|
| 六甲 | 天辅、青龙 | 阳开利客，阴开利主，阖则固守，开则扬兵 |
| 六乙 | 天德、蓬星 | 施恩布德、赏赉士卒、出兵大利 |
| 六丙 | 明堂、天威 | 振武扬威、出战聚众、安营立寨 |
| 六丁 | 太阴、玉女 | 阴谋暗计、偷营劫寨、伏匿逃亡 |
| 六戊 | 天武、天门 | 出师遣将、战必胜攻必克 |
| 六己 | 地户、六合 | 阴谋密计、设伏偷营、修理城郭 |
| 六庚 | 天狱、天伐 | 决狱固守、出兵不利 |
| 六辛 | 天庭、天尉 | 杜塞潜形、决罪囚（不宜出军） |
| 六壬 | 天牢、天廪 | 理刑狱、决罪囚（行兵利伏藏） |
| 六癸 | 天藏、华盖 | 断刑狱、积贮军粮、逃亡躲匿 |

### 出军运筹中的应用

出军运筹之法需依十干门户神名：
- 出**天门**（戊）
- 入**地户**（己）
- 过**太阴**（丁）
- 居**青龙**（甲）

详见 [[出军运筹法]]。

### 六甲安营中的应用

六甲安营法以十干门户神名分布营寨：
- 大将居**青龙**（甲）
- 旗鼓居**蓬星**（乙）
- 士卒居**明堂**（丙）
- 伏兵居**太阴**（丁）
- 奇兵居**天门**（戊）
- 裨将居**地户**（己）
- 斩断居**天狱**（庚）
- 治事居**天庭**（辛）
- 芻粮居**天廪**（壬）
- 兵器居**天藏**（癸）

详见 [[六甲安营法]]。

## 奇门遁甲统宗补充

> 出处：《奇门遁甲统宗》卷之二·日奇起青龙法 — 参见 [[奇门遁甲统宗（诸葛亮）]]

### 日奇起青龙法

统宗卷二载日奇起青龙法，以日支定青龙起位，顺排十二神：

| 日支 | 青龙起位 |
|------|---------|
| 子午日 | 申 |
| 丑未日 | 戌 |
| 寅申日 | 子 |
| 卯酉日 | 寅 |
| 辰戌日 | 辰 |
| 巳亥日 | 午 |

### 统宗版特点

- 统宗版日奇起青龙法以日支定青龙起位，与黄黑道十二神系统同源
- 十二神顺排：青龙、明堂、天刑、朱雀、金匮、天德、白虎、玉堂、天牢、玄武、司命、勾陈
- 其中**青龙、明堂、金匮、天德、玉堂、司命**为六黄道吉神
- **天刑、朱雀、白虎、天牢、玄武、勾陈**为六黑道凶神
- 此法与奇门遁甲十干门户神名系统互补，皆用于择吉出行
