# 勾陈

## 定义与基本属性

**勾陈**是隐藏在[[八诈盘（八将盘）|八神]]中白虎之下的神煞。在六爻预测的六兽体系中，勾陈是独立的六兽之一（六爻六兽：青龙、朱雀、勾陈、腾蛇、白虎、玄武）。但在奇门遁甲的八神体系中，八神只有八个位置，勾陈没有独立的席位，而是隐藏于白虎之下。

| 属性 | 内容 |
|------|------|
| **五行** | 中央土 |
| **隐藏于** | 白虎之下 |
| **与白虎的区别** | 白虎禀西方金，勾陈禀中央土 |
| **在六爻中** | 独立六兽之一 |

通俗理解：八个车位停了十辆车，勾陈和朱雀只能藏在别的车位里。勾陈藏在白虎下面——白虎主凶杀，勾陈主田土牢狱，两者都偏负面，但一个猛一个慢。

## 详细象意

### 勾陈的主要象意

- **田土**：土地、房产、农田、地基、建筑
- **牢狱**：官司、牢狱、纠纷、牵连
- **牵连**：被拖累、被纠缠、脱不了身
- **迟滞**：拖延、缓慢、不顺、停滞

### 勾陈与白虎的五行差异

| 对比 | 勾陈 | 白虎 |
|------|------|------|
| **五行** | 中央土 | 西方金 |
| **性质** | 迟缓、厚重 | 迅猛、锐利 |
| **伤害方式** | 慢慢磨、拖、缠 | 突然、猛烈、一击致命 |
| **象** | 慢性病、长年官司 | 急病、血光之灾 |

两者虽然在奇门中合在一起，但解读时应根据具体象意区分：是"突然打击"（白虎）还是"长期拖累"（勾陈），性质完全不同。

## 应用宜忌

- **宜**：在断牢狱、田土纠纷、慢性问题等事项时，注意勾陈的象意
- **忌**：机械套用勾陈象意——勾陈在奇门中不独立出现，只在特定层面的解读中参考
