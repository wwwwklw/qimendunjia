# 用神断卦法详解

## 定义与基本属性

**用神断卦法**是奇门遁甲中最主要、应用最广泛的断卦方法。其核心是根据所测事物的性质，在奇门盘面中确定对应的"[[用神]]"（代表所测事物的符号），然后看该用神落宫的状态，以及与其他用神之间的生合刑克关系，综合判断吉凶。

通俗理解：奇门盘面就像一个大舞台，上面有几十个"演员"（各宫的各种符号）。用神断卦法就是：找到你要看的那个"主角"（用神），看他在哪一宫、状态好不好、和谁互动，从而判断事情走向。

## 详细象意

### 用神断卦的步骤

1. **确定用神**：根据所测之事，确定用神是什么
   - 测婚姻 → 看乙（妻）和庚（夫）
   - 测工作 → 看开门和日干、年命
   - 测财运 → 看生门和戊（资本）
   - 测疾病 → 看天芮星和病神
2. **看用神落宫**：用神落在哪一宫？该宫的门、星、神、奇仪如何？
3. **看旺衰**：用神在当令还是失令？（参考节气与[[五行]]旺相休囚死）
4. **看生克**：用神与其他用神之间有生克吗？谁生谁？谁克谁？
5. **看格局**：用神落宫的组合是吉格还是凶格？

### 与六爻世应用神法的对照

| 方面 | 奇门用神法 | 六爻用神法 |
|------|----------|---------|
| 用神定义 | 根据不同事物用不同符号 | 根据六亲确定用神 |
| 核心依据 | 落宫状态 + 生克关系 | 爻位旺衰 + 六亲生克 |
| 信息量 | 大（同时看门星神奇仪） | 相对集中 |
| 层次 | 立体（天-地-人-神四层） | 平面（一卦六爻） |

两者都遵循"以用神为主"的断卦逻辑，但奇门的信息维度更多。

### 用神落宫状态的关键要素

用神落在一个宫里，同时受以下因素影响：
- **八门**：吉门还是凶门？
- **九星**：旺相还是休囚？
- **八神**：吉神还是凶神加持？
- **奇仪**：临三奇还是六仪？有无"甲"？
- **格局**：吉格（如青龙返首）还是凶格？

## 应用宜忌

- **宜**：先确定用神再看盘，不能先看盘再找用神
- **宜**：用神可能不止一个（如测婚姻同时看乙和庚），要看多个用神之间的关系
- **忌**：用神选错——同一件事可能涉及多个用神，选错了判断全盘皆错
- **忌**：只盯着用神不看整体——用神虽吉，但如果被其他因素克制，吉也不吉
