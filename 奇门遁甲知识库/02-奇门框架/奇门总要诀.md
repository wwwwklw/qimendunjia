# 奇门总要诀

## 定义与基本属性

**《奇门总要诀》**是金函玉镜奇门遁甲秘笈全书卷一收录的一首重要歌诀，位于《烟波钓叟歌》之后。与《烟波钓叟歌》不同，《奇门总要诀》侧重于奇门遁甲**实战要点的总括性归纳**，篇幅更短，信息更集中，更适合作为快速查阅的口诀。

## 全文

> 阴阳顺逆妙难穷，二至还归一九宫，若能了达阴阳理，天地都来一掌中，
> [[三才]]变化作三元，八卦分为八遁门，星符每逐十干转，值使常从天乙奔，
> 六仪六甲本同名，三奇即是乙丙丁，三奇若合开休生，便是吉门利出行，
> 万事从之无不利，能知玄妙得其灵，值符前三六合位，前二太阴须切记，
> 值符后一为九天，后二之宫为九地，地可伏匿天扬兵，六合太阴可藏避，
> 急从神兮缓从门，三五反复天道利，以上虽得三奇妙，不如更得三奇使，
> 得使犹来未为精，五不遇时损光明，用事须忧时尅日，反伏吟格犹不吉，
> 掩捕逃亡须格时，占稽行人信岂失，斗中三奇游六仪，[[天乙会合]]主阴私，
> 讨捕须用时下尅，行人信息遇三奇，三奇上见游六仪，六仪更见[[五阳干与五阴干|五阳时]]，
> 兼向八门寻吉位，万事开三万事宜，五阳在前五阴后，主客须知有盛衰，
> 阴后五干君须记，六仪加着更无利，六仪忽然加[[震三宫|三宫]]，便为[[六仪击刑|击刑]]先须记，
> 六仪击刑三奇墓，此时举动可惮惧，太白入荧贼即来，[[荧入太白]]贼即去，
> 丙为悖兮庚为格，格则不通悖乱逆，庚加日干为伏干，日干加庚飞干格，
> 庚加值符天乙伏，值符加庚天乙飞，加己为刑遁上格，加癸路中见疑格，
> 加壬之时为小格，又嫌年月日时逢，当此之时最不吉，举动行师皆不宜，
> 丙为甲兮鸟跌穴，甲加丙兮龙返首，辛加乙兮虎猖狂，乙加辛兮龙逃走，
> 丁加癸兮鸟投江，癸加丁兮蛇夭矫，符加丙兮为相佐，时加六丁为守星，
> 丙合戊开为天遁，地遁乙合入开宫，休承丁乙太阴人，天网四张时加癸，
> 蓬加英上为返吟，伏吟之时蓬加蓬，吉宿逢之事不吉，凶宿逢之事愈凶，
> 天辅冲任禽心吉，天蓬天芮英柱凶，阴宿天心英柱芮，阳星冲辅禽任蓬，
> 天网四张无走路，阴阳顺逆妙无穷，节气推移时候定，二至还乡一九宫，
> 三元超接游六仪，八卦周流遍九宫，若能了达阴阳理，天地消详一掌中。

## 逐句解释

### 开篇：总述纲领（句1-3）
| 原文 | 解释 |
|------|------|
| 阴阳顺逆妙难穷，二至还归一九宫 | 与[[烟波钓叟歌]]首句相同，点明阴阳遁以冬至（一宫坎）和夏至（九宫离）为分界 |
| 三才变化作三元，八卦分为八遁门 | 天地人三才对应上中下三元；八卦配八门，为八遁之门户 |

### 核心符号（句4-7）
| 原文 | 解释 |
|------|------|
| 星符每逐十干转，值使常从天乙奔 | 天盘九星随[[时干]]转动，八门值使从天乙（值符）所在宫位起算 |
| 六仪六甲本同名 | [[六仪（戊己庚辛壬癸）]]是六甲的替身，名异而质同 |
| 三奇若合开休生 | [[三奇（乙丙丁）]]配合[[三吉门|开休生]]三吉门，方为真正的吉门 |

### 八神方位（句8-9）
| 原文 | 解释 |
|------|------|
| 值符前三六合位，前二太阴须切记 | 值符前三位为[[六合]]，前二位为太阴 |
| 值符后一为九天，后二之宫为九地 | 值符后一位为九天，后二位为[[九地]] |
| 地可伏匿天扬兵，六合太阴可藏避 | 九地宜埋伏，九天宜扬兵，六合和太阴宜藏身躲避 |

### 用神判断（句10-15）
| 原文 | 解释 |
|------|------|
| 急从神兮缓从门 | 紧急看八神方位，从容看八门方位（同[[烟波钓叟歌]]） |
| 虽得三奇妙，不如更得三奇使 | 有三奇还不够，还要看[[三奇得使]] |
| 得使犹来未为精，五不遇时损光明 | 三奇得使还不够完美，遇到[[五不遇时]]就全盘皆损 |
| 掩捕逃亡须格时 | 拘捕、追逃看格（阻隔之神[[庚（阻隔之神）|庚]]），时干克日干为吉 |
| 斗中三奇游六仪，天乙会合主阴私 | [[玉女守门]]等格局主阴私和合之事 |

### 主客论（句16-17）
| 原文 | 解释 |
|------|------|
| 五阳在前五阴后 | [[五阳干与五阴干|五阳时]]（甲至戊）利客，五阴时（己至癸）利主 |
| 主客须知有盛衰 | 根据时令阴阳判断主客的盛衰 |

### 击刑与墓（句18-20）

|------|------|
| 六仪忽然加三宫，便为击刑先须记 | 六仪击刑——六仪加到相刑的地支宫位。三宫指震三宫（卯），寅巳申相刑等 |
| 六仪击刑三奇墓 | [[六仪击刑]]和[[三奇入墓]]都是大凶，不可轻易行动 |

### 九星吉凶（句34）
| 原文 | 解释 |
|------|------|
| 天辅冲任禽心吉 | 天辅、天冲、天任、天禽、天心为吉星 |
| 天蓬天芮英柱凶 | 天蓬、天芮、天英、天柱为凶星 |
| 阴宿天心英柱芮 | 阴星：天心、天英、天柱、天芮 |
| 阳星冲辅禽任蓬 | 阳星：天冲、天辅、天禽、天任、天蓬 |

> **注**：此九星阴阳分类法与《烟波钓叟歌》有差异——《烟波钓叟歌》中天心为阴星（此处列为阴星一致），但天蓬在《烟波钓叟歌》中为阳星，此处亦列阳星（一致）。具体差异需结合上下文。

## 与烟波钓叟歌的对比

| 对比维度 | [[烟波钓叟歌]] | 奇门总要诀 |
|----------|:---:|:---:|
| **篇幅** | 60句，1800余字 | 36句，约600字 |
| **风格** | 详细叙述，循序渐进 | 高度凝练，直奔要点 |
| **首尾** | 以历史开篇，以劝学结尾 | 以原理开篇（与烟波钓叟歌首句完全相同），以原理结尾 |
| **九星分类** | 蓬任冲辅禽—阳星；英芮柱心—阴宿 | 冲辅禽任蓬—阳星；心英柱芮—阴宿 |
| **独特内容** | 详述神龙负图、风后演文、太公张良等历史 | **无历史叙述**，从三才八卦直接进入实操 |
| **共同内容** | 三奇六仪、值符值使、吉凶格、九遁、反伏吟、天网四张等 | 内容高度重叠，但表述更简洁 |
| **独特表述** | "宫制其门不为迫，门制其宫是迫雄"（末尾） | **无此句**（此句仅在烟波钓叟歌末尾出现） |
| **独特表述** | "天目为客地耳主"（末尾） | **无此句** |
| **别称差异** | 庚加癸为"大格" | 庚加癸称"疑格"，庚加壬称"小格" |

### 核心差异总结

1. **定位不同**：烟波钓叟歌是"教科书"，总要诀是"速查手册"
2. **历史叙事**：烟波钓叟歌包含黄帝战蚩尤、风后演文等历史背景，总要诀完全省略
3. **格名称谓**：总要诀中庚加癸称"疑格"而非"大格"，此为独特称谓
4. **九星阴阳**：星名排列顺序不同，但分类结果一致
5. **首尾呼应**：总要诀以"二至还乡一九宫"开篇和结尾，形成闭环结构

## 应用宜忌

- **宜**：已掌握《烟波钓叟歌》后，用《奇门总要诀》快速回忆要点
- **宜**：起局后对照总要诀逐句校验，确保没有遗漏关键格局
- **忌**：单靠总要诀学奇门——它省略了基本概念的解释，初学者仍需从烟波钓叟歌入手

- [[烟波钓叟歌]]：总要诀的"母本"，内容高度重叠但表述更简洁
