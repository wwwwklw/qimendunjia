# 大事看星

## 定义与基本属性

**"大事看星"**是奇门遁甲应用中的一条重要原则：凡是遇到重大的事情，除了选择吉时吉方、分清主客之外，还必须**看九星的吉凶状态**。

通俗理解：小事看门，大事看星。日常琐事参考八门的吉凶就够了；但遇到人生大事（投资、婚嫁、搬迁、事业转型等），九星的吉凶状态是决定性因素。

## 详细象意

### 为什么大事要看星？

九星代表**天时**、**天命**。大事受天时影响大，小事更多受人事（八门）影响。

- 九星是天体运行力量的象征，代表不可抗拒的大趋势
- 八门是人事进退的象征，代表人可以选择和改变的方向
- 神助（八神）是辅助力量，锦上添花或雪上加霜

> **力量层级**：九星（天时） > 八门（人事） > 八神（神助）

### 大事的判断标准

什么算"大事"？
- 投资数额较大的事情
- 人生重要的转折点（婚嫁、换工作、搬家）
- 涉及多人利益的决策
- 时间跨度较长的事情
- 结果影响深远的事情

### 九星吉凶速查

| 九星 | 吉凶 | 五行 | 大事宜忌 |
|------|------|------|---------|
| **天蓬** | 凶 | 水 | 宜守不宜攻，忌投资扩张 |
| **天芮** | 凶 | 土 | 忌开业开店，宜解决老问题 |
| **天冲** | 次吉 | 木 | 宜速战速决，忌拖泥带水 |
| **天辅** | 大吉 | 木 | 宜学习考试、谋划策划 |
| **天禽** | 大吉 | 土 | 百事皆宜，居中调和 |
| **天心** | 大吉 | 金 | 宜医疗、求医、开创新局 |
| **天柱** | 凶 | 金 | 忌合作合伙，宜独立行动 |
| **天任** | 吉 | 土 | 宜置产置业、固本培元 |
| **天英** | 次凶 | 火 | 忌口舌之争，宜文书宣传 |

### 判断要点

1. **看值符星**：当前时辰值班的九星是什么，代表当前天时大势
2. **看旺衰**：星旺则有力量（吉上加吉或凶上加凶），星衰则力量不足（吉减凶也减）
3. **看落宫**：星落哪个宫，与地盘宫的生克关系
4. **看格局**：星与三奇六仪的组合格局

## 应用示例

**示例：想开一家新公司（大事）**
1. 先查当前时辰值符星是什么
2. 如果值符是天辅星（大吉星）旺相 → 天时有利，可以推进
3. 如果值符是天蓬星（凶星）旺相 → 天时不利，建议暂缓
4. 同时还要结合开门（事业）、生门（财运）、戊（资本）等用神综合判断

## 应用宜忌

- **宜**：大事决策前必看九星状态
- **宜**：九星吉 + 八门吉 + 时方吉 = 三吉具备，最佳时机
- **忌**：大事只看八门不看九星，本末倒置
- **忌**：小事也动不动看九星，浪费精力
