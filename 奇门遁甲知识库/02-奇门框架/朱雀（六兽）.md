# 朱雀（六兽）

## 定义与基本属性

**朱雀**是隐藏在[[八诈盘（八将盘）|八神]]中玄武之下的神煞。在六爻预测的六兽体系中，朱雀是独立的六兽之一（六爻六兽：青龙、朱雀、勾陈、腾蛇、白虎、玄武）。但在奇门遁甲的八神体系中，八神只有八个位置，朱雀没有独立的席位，而是隐藏于玄武之下。

| 属性 | 内容 |
|------|------|
| **五行** | 南方火 |
| **隐藏于** | 玄武之下 |
| **与玄武的区别** | 玄武禀北方水，朱雀禀南方火 |
| **在六爻中** | 独立六兽之一 |

通俗理解：八个车位停了十辆车，朱雀和勾陈只能藏在别的车位里。朱雀藏在玄武下面——两者都与"消息""隐秘"有关，但朱雀偏"明"（文书、口舌），玄武偏"暗"（盗贼、暧昧）。

## 详细象意

### 朱雀的主要象意

- **口舌**：争吵、口舌是非、言语纷争、辩论
- **文书**：文件、书信、合同、证书、通知
- **消息**：信息、新闻、传言、音讯
- **火灾**：火患、火灾（南方火之象）

### 朱雀与玄武的差异

| 对比 | 朱雀 | 玄武 |
|------|------|------|
| **五行** | 南方火 | 北方水 |
| **方向** | 明处 | 暗中 |
| **信息** | 公开消息、文书 | 暗箱操作、私密信息 |
| **伤害方式** | 口舌、言论 | 偷盗、诬陷 |
| **象** | 笔战、官司（打嘴仗） | 盗窃、暧昧（暗中来） |

两者虽然在奇门中合在一起（朱雀藏于玄武之下），但解读时应注意：如果涉及文书、口舌、公开争论，侧重朱雀的象意；如果涉及盗窃、暗昧、骗局，侧重玄武的象意。

### 六爻中的朱雀

在六爻预测中，朱雀是独立六兽之一，其象意与奇门中基本一致。口诀："朱雀南方火，主口舌文书"。

## 应用宜忌

- **宜**：在断官司、文书纠纷、消息真假等事项时，参考朱雀的象意
- **忌**：机械套用——朱雀在奇门中不独立出现，仅作辅助参考
