# 值符与值使

## 定义与基本属性

**值符**和**值使**是奇门排盘中最关键的两个概念，它们是驱动天盘和人盘转动的"发动机"。

- **值符**：天盘上**值班的九星**，由旬首（十干支为一旬，每旬的第一位叫旬首）所在宫位对应的天星确定
- **值使**：人盘上**值班的八门**，由旬首所在宫位对应的门确定

通俗理解：值符是"天盘的值班长"，值使是"人盘的值班长"。每个时辰该轮到九星中的哪颗星、八门中的哪个门值班，由当前时辰的旬首来决定。

## 详细象意

### 值符的确定步骤

1. 找出当前时辰的**旬首**（甲子、甲戌、甲申、甲午、甲辰、甲寅中的哪一个）
2. 看旬首落在地盘的哪个宫位
3. 该宫位所在地盘对应的**原属九星**即为值符
4. 例如：旬首甲子落坎一宫，坎一宫的天星是天蓬星，则**天蓬星为值符**

### 值使的确定步骤

1. 同样找出旬首
2. 看旬首落在地盘的哪个宫位
3. 该宫位所在地盘对应的**原属八门**即为值使
4. 例如：旬首甲子落坎一宫，坎一宫的门是休门，则**休门为值使**

## 运转规则

### 值符随时干

天盘值符转到**时干**所在的宫位，其余八星按顺序依次转动。

> 例如：时干是戊，戊在兑七宫，值符天蓬星就转到兑七宫，其他八星依次排布。

### 值使随时宫

人盘值使从**旬首地支所在宫**开始数，数到**时干地支所在宫**停下。

> 阳遁顺数、阴遁逆数。例如：旬首为甲子（子），时支为午，阳遁从子所在宫顺数到午所在宫，值使就停在该宫。

### 小值符追大值符

- **大值符**：天盘上的值符星
- **小值符**：神盘上的值符神

小值符永远追着大值符跑。大值符（天盘值符星）转到哪个宫，小值符（神盘值符）就跟到哪个宫，其余七神按阳顺阴逆排布。

## 上元经补充

> **source:** [[奇门遁甲上元经（张大春）]] 符使大义章

### 值符的核心枢纽地位

上元经明确指出：飞盘奇门与转盘奇门起局后**唯一相同的宫位就是值符落宫**，值符落宫是解局的**关键机要**。

值符的本质：
- "值"=当值，"符"=道家符箓 → 引申为具有印信凭证作用的兵符、官符
- 值符 = 当值的、正在发挥作用的符
- 符发挥效力有**时间范围限制**，离了限定时间范围就会失效

#### 太岁与值符的关系

- 太岁高于值符：太岁主经年之岁运，值符统一时之吉凶
- 一年内大事必须观测太岁落宫（飞盘体系：由于适宜短时事件，太岁作用更突出）
- 但太岁和值符不是同一体系——用神旺相且得符使之主的情况下，即使受太岁之制，也不妨成事
- 类比：《西游记》中太岁=玉皇大帝，值符=大唐皇帝李世民

#### 值符落宫衰旺判断真机

一宫之内星、门、干、神聚集，判断衰旺的**两条核心标准**：

1. **宫位本身的旺相休囚**——由月令决定（平台好坏）
2. **宫内天干落地宫十二支的生旺墓绝十二态变化**——求测人自身能量的旺相程度

星、门、干、宫中：
- 门的旺衰入墓之说**站不住脚**，唯一需考量的是**门迫**
- 神盘衰旺变化法则同于宫内天干，但神盘不起决定性作用

#### 解局时的观测顺序（大事首观星）

值符落宫中符号交织，解局顺序为：

1. **观星**：星主天时，其势先行，观星可知大势
2. **观帅甲衰旺**：帅甲之助首看当值之星，次观格局吉凶衰旺
3. **再察门宫之助**：干与门之间的主客动静关系

#### 庚与值符的复杂关系

上元经针对庚在值符宫中的不同角色做了**分门别类**的处理：

| 庚的角色 | 判断方法 |
|---------|---------|
| 甲申庚旬首 | 庚具有卫护功能，与其他五仪同；伏吟天显时格可化凶为吉 |
| 地盘庚 | 值符+庚格局，一般表阻碍困难；壬加庚例外（太白擒蛇） |
| 庚为太岁 | 需转看天盘庚落宫组合，与值符比和为助，冲战则为世仇 |
| 庚为暗干 | 防暗中侵害，察庚落宫与值符宫生克比和 |
| 庚不在值符宫 | 观测庚落宫与值符宫关系——庚只有发动起来（对宫+马星/合动）才会对值符做出攻击姿态 |

### 值使的性质与作用

- 值使奉旨办事，是直接的经办人
- 值符=董事长定大事，值使=总经理负责落实
- **事情最终办得怎么样，还是要看值使来说话**
- 值使无旺相死墓之别，唯观门吉凶及与迫制与否

#### 值使逢空的三种情况

| 状态 | 判定 |
|------|------|
| 事情已发动，值使逢空 | 好事坏事都成空 |
| 事情未发动，值使逢空 | 事情还未真正执行，会填实或冲实 |
| 执行阶段已结束 | 标示过去曾发动，当下已无行动 |

特别注意：死门、伤门、惊门作为值使门即使不遇门迫，多数情况也是负能量。

### 符使关系的六种深化分类

上元经发前人所未发，提出符使关系的**精细分类**：

#### 1. 符使同宫
符使同宫形成合力，但**不一定是吉利的**——需根据宫内格局符号具体判断。天芮+死门+双庚伏吟落艮宫则大凶。

#### 2. 值符生值使（三种子类型）

| 类型 | 含义 |
|------|------|
| **实生** | 值符宫搭配良好，扎实有力，生助大（如唐太宗全力支持李靖） |
| **虚生** | 值符宫格局不吉，休囚无力，有心无力（如汉献帝支持大臣） |
| **形生** | 坎→震（子卯刑）、震→离（卯午刑）：表面相生，下涌争斗暗流 |

#### 3. 值使生值符
亦分实生、虚生、形生三种，论断方法同。

#### 4. 符使比劫
转盘：震巽、乾兑可构成实比与虚比；飞盘：增加中宫与艮/坤的实比与虚比（飞盘体系）。

#### 5. 值符克值使（三种子类型）

| 类型 | 含义 |
|------|------|
| **实制** | 四正对冲，制而无情（如坎符离使），激烈矛盾 |
| **正制** | 宫位五行上克下，符合自然法则（如乾符克震使） |
| **合制** | 制中逢合，三合/六合/五合引动，化制为合（如离符乾使+三合火局） |

#### 6. 值使克值符
亦分实制、正制、合制。**只有实制才是真正不利**（君臣对立以下克上），正制与合制下格局搭配得当，符使可正常发挥效能（例：诸葛亮《出师表》使克符，蜀国治理效能仍好）。

### 天地值符的主客对待

天盘值符与地盘值符落宫是否构成主客矛盾？答案是**肯定的**：
- 天道速，地道迟：天盘值符每时辰变化，地盘值符十时辰一变
- 两者形成一对阴阳主客矛盾
- 建议多个时辰连续手工起局，观察天盘值符与地盘值符落宫对比的气息变化

## 奇门法窍补充

> **source:** [[奇门法窍（锡孟父子）]]
>
> 本节出自《奇门法窍》卷二「论直符用法」「论直使」「论直符直使」三章，系（飞盘体系）飞盘奇门对值符值使的独特论述。

### 论直符用法（飞盘体系·八神辅运）

> 《奇门法窍·论直符用法》论八神随直符辅运八宫之法

**原文要旨**：

> 「赋云：『直符前三六合位，太阴之神在前二，后一宫兮为九天，后二之神为九地。』此六合、太阴、九天、九地皆随直符而辅运八宫也。」

**中五前后之辨**：
- 如分九宫，则直符到中宫将何以分前后乎？
- **从气论**：中五以乾为前，以巽为后
- **从寄位论**：中五寄坤二，以兑为前，以离为后

**辅运八神之要**：
- 「前后」二字，有导引、随从直符之义
- 所重者在六合、太阴、九天、九地
- 螣蛇、白虎、元武虽凶，而不论及

**九天九地不可击之辨**：
- 汤谓云：「九天不可击，九地不可击」
- 若直符居离九（行阴遁以一为前），九天在坎一，击天蓬是击九天，与「不可击」相违
- **正解**：直符居离九
  - 阴遁宜屯兵于九天之坤二
  - 阳遁宜屯于九天之巽四
  - 故天蓬之可击，三胜宫第一胜也

**结论**：九遁五假皆从此中立论，不可妄生别议。

### 论直符直使（飞盘体系·符使体用）

> 《奇门法窍·论直符直使》论符使之体用关系

**核心定义**：

> 「符使者，星门用时之异名也。直者，代甲而直其用也。」

- **甲以星门为体**，以奇仪为用
- 奇仪无定位，而星门有定宫
- **符者**，即凭执之符也
- **门为使者**，即甲巡行之使也
- 凡甲在之宫，即以直宫之星为符，以宫之门为使

**奇仪与六十干之体用**：

| 概念 | 性质 | 定位 |
|------|------|------|
| 奇仪 | 九干之体 | 于卦无定位，于局有定宫（静） |
| 六十干 | 奇仪之用 | 随甲遁于宫（动） |
| 六甲 | 遁于六仪 | 不能遁于奇 |
| 六仪 | 随元遁六仪 | 奇合六仪之干，亦随甲遁于宫 |

**符使运转歌诀**：

> 「直符常以加时干，直使顺逆随宫去。」

- 每甲用时，以当直之星符临本局之定位
- 以当直之门使巡奇仪本元之遁宫
- 譬之用人：其体在此，而以符监之；其用在彼，而以使察之
- 「善藏其用以儆无虞也」

## 奇门宝鉴补充

> 出处：《御定奇门遁甲·奇门宝鉴》（金志文点校）卷一·释义四十四则 — 参见 [[御定奇门遁甲奇门宝鉴（金志文）]]

### 释直符直使

符使者，星门用时之异名也。直者，代甲而直其用也。

- 甲以星门为**体**，以奇仪为**用**
- 故奇仪无定位，而星门有定宫也
- **星为符**者，即凭执之符也
- **门为使**者，即甲巡行之使也
- 凡甲在之宫，即以直宫之星为符，以直宫之门为使也

#### 奇仪与六十干之体用关系

干本十也，以配十二支，而变为六十干，故甲称六而仪亦称六也。

- **奇仪**者，九干之体（于卦无定位，于局有定宫→静）
- **六十干**者，奇仪之用（随元遁于宫→动）

故奇仪随元，分职九宫，而六十干亦随元，分遁九宫也。

#### 六甲分遁九宫之理

六甲干首分遁九宫，盖以身先之，而分其势以弱之也。若六甲不率五十四干分遁，则各干各聚于一宫，非所谓「强木弱金」也。

| 六甲 | 同宫之干 | 五行关系 |
|------|---------|----------|
| 甲子、癸酉 | 遁戊 | 金盗土以生水 |
| 甲戌、癸未 | 遁己 | 土相比以求益 |
| 甲申、癸巳 | 遁庚 | 火制金以防合 |
| 甲午、癸卯 | 遁辛 | 木助火以制金 |
| 甲辰、癸丑 | 遁壬 | 土相连以败水 |
| 甲寅、癸亥 | 遁癸 | 水相合以益木 |

#### 符使运转歌诀

> 「直符常以加时干，直使逆顺时支去。」

每甲用时：
- 以当直之星符，临奇仪本局之定位
- 以当直之门使，巡奇仪本元之遁宫

> 譬之用人：其人体在是，而以符监之；其人用在彼，而以使察之。善藏其用，以儆戒无虞也。

> 出处：《御定奇门遁甲·奇门宝鉴》卷一·释义四十四则 — 参见 [[御定奇门遁甲奇门宝鉴（金志文）]]
