# 三元九运

## 定义与基本属性

**三元九运**是古代堪舆（风水）学中使用的时间单位体系，主要运用于玄空飞星风水中。它将时间划分为上、中、下三元，每元三运，共九运，每运二十年，合计180年为一个完整周期。

| 层次 | 名称 | 时长 | 总数 |
|------|------|------|------|
| 上元 | 一运、二运、三运 | 各20年 × 3 = 60年 | |
| 中元 | 四运、五运、六运 | 各20年 × 3 = 60年 | |
| 下元 | 七运、八运、九运 | 各20年 × 3 = 60年 | |
| **合计** | | **共180年** | 一个完整周期 |

通俗理解：三元九运就像风水学中的"大历法"——短的用年月日时，长的用三元九运来看大趋势。你现在住的房子属于哪一运建造，就带有那一运的气场特征。

## 详细象意

### 三元九运与奇门的关系

《神奇之门》提及三元九运主要是进行对比说明：
- 奇门遁甲和玄空风水都有"三元"概念，但含义不同
- 奇门三元：一个节气分上中下三元，每元五天（60个时辰）
- 风水三元：时间被分为上中下三元，每元60年
- 两者的"三元"不能混淆

### 当前所处运程（简要）

三元九运仍在风水实践中使用。根据传统推算：
- 当前（2004年—2023年）处于下元八运（八白土运）
- 下一运（2024年—2043年）为下元九运（九紫火运）

> 注：具体起止年份在不同流派中有细微差异，以上为常见说法。

## 应用宜忌

- **宜**：了解三元九运属于风水学范畴，与奇门遁甲的"三元"（节气三元）概念不同
- **忌**：将风水三元九运与奇门遁甲的三元混淆使用
- **忌**：初学不过多深入风水内容，分清学科边界

## 与其他概念的关系

- [[节气与定局]]：奇门的"三元"指节气上中下三元，与此不同
- [[置闰派与无闰派]]：三元确定的争论是奇门派别之争的来源之一
