# 节气与定局

## 定义与基本属性

**节气定局**是根据二十四节气来确定奇门遁甲用哪一局（阳遁几局还是阴遁几局）的方法。这是起局的第一步，也是最基础的一步。

通俗理解：节气定局就像"看天时开菜单"。古人发现太阳运行到不同位置（节气），天地气场就不同，对应不同的"局"。冬至后阳气升用阳遁，夏至后阴气升用阴遁。

## 上中下三元与符头

每个节气分为上、中、下三元，每元五天（60个时辰）：

- **符头**：每元开头第一天的天干。符头有甲、己之分
  - 甲子、甲午为**上元**符头
  - 己卯、己酉为**中元**符头
  - 甲辰、甲戌为**下元**符头

> 定局口诀：看天干找符头，符头定上中下元，再看节气表找局数。

## 应用宜忌

- **宜**：背熟阳遁歌和阴遁歌，是定局的捷径
- **宜**：先判断节气 → 再确定干支符头 → 最后定局数
- **忌**：阴阳遁搞反——冬至后用阴遁、夏至后用阳遁是大错
- **忌**：上中下三元搞混，局数错一档后续全盘错

## 开悟之门补充

### 二十四节气与八宫的错位原理

#### 地支月份节气的非等分性

按农历地支月份，各宫对应的节气数并非等分：

- **四正宫**（1/3/7/9）：各对应两个节气
- **四维宫**（2/4/6/8）：各对应四个节气

但在**奇门九宫格**内，将二十四节气按八宫**等分**，每宫平均对应三个节气。这一等分导致十二地支月份所对应的节气出现了**错位现象**：

| 地支月 | 原对应节气 | 错位后的宫位分布 |
|:---|:---|:---|
| 子月（十一月） | 大雪、冬至 | 跨在六宫与一宫之间 |
| 丑月（十二月） | 小寒、大寒 | 在一宫（原本在八宫） |
| 寅月（正月） | 立春、雨水 | 仍在八宫 |
| 卯月（二月） | 惊蛰、春分 | 跨在八宫与三宫之间 |
| 辰月（三月） | 清明、谷雨 | 在三宫（原本在四宫） |
| 巳月（四月） | 立夏、小满 | 仍在四宫 |
| 午月（五月） | 芒种、夏至 | 跨在四宫与九宫之间 |
| 未月（六月） | 小暑、大暑 | 在九宫（原本在二宫） |
| 申月（七月） | 立秋、处暑 | 仍在二宫 |
| 酉月（八月） | 白露、秋分 | 跨在二宫和七宫之间 |
| 戌月（九月） | 寒露、霜降 | 在七宫（原本在六宫） |
| 亥月（十月） | 立冬、小雪 | 仍在六宫 |

> 结论：子/午/卯/酉分别跨在两个宫之间，辰/戌/丑/未分别提前了一个宫位，寅/申/巳/亥仍在原来宫位。

### 二十四节气与阳历日期的对应规律

奇门遁甲干支历以**节气为准**，不按阴历初一为一个月的开始。二十四节气与阳历的对应比较稳定：

> **上半年来六二十一，下半年来八二十三，每月两气日期定，最多相差一二天。**

- 如冬至节：一般不是12月21日就是22日
- 如夏至节：一般不是6月21日就是6月22日

**实践建议**：在判断应期时，应以**节气为主、以阳历时间为主**，这样准确率才比较高。不要按阴历来断应期，否则误差较大。

### 与通行版歌诀的差异对比

| 节气 | 通行版歌诀 | 金函玉镜版 | 差异说明 |
|------|-----------|-----------|----------|
| 大寒 | "大寒春分三九六" | "春分大寒三九六" | 顺序互换，内容相同 |
| 立春 | "立春八五二相宜" | "立春八五二相随" | 一字之差 |
| 芒种 | 位于歌诀首段 | 位于歌诀末句 | 排列位置不同 |
| 大雪 | "大雪四七一相关" | 通行版"大雪四七一相关" | 完全相同 |

> 总体来看，金函玉镜版的用局表与通行版内容**完全一致**，仅在歌诀用字和排列顺序上略有差异，不影响起局结果。
