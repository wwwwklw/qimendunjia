# 日上起时法

## 定义与基本属性

**日上起时法**是根据日天干推算时辰天干的方法，又称**五鼠遁元**或**五子遁元**。在奇门遁甲起局中，需要知道当前时辰的完整干支（天干 + 地支），而时辰地支是固定的（23-1点为子时、1-3点为丑时……），关键在于时辰天干——这就是日上起时法要做的事。

通俗理解：时辰地支就像钟表上的刻度——每小时固定一个位置。但时辰天干每天不一样，需要用当天的日天干来"算"出来。日上起时法就是告诉你这个换算公式。

## 五鼠遁元歌诀

> **甲己还加甲**，—— 日干为甲或己时，子时为甲子
> **乙庚丙作初**。—— 日干为乙或庚时，子时为丙子
> **丙辛从戊起**，—— 日干为丙或辛时，子时为戊子
> **丁壬庚子居**。—— 日干为丁或壬时，子时为庚子
> **戊癸何方发**，—— 日干为戊或癸时，子时为何天干？
> **壬子是真途**。—— 日干为戊或癸时，子时为壬子

## 使用方法

### 第一步：确定日天干

查万年历或通过日干支计算，得到当天的日天干。

> 例如：今天日干支是"乙丑"，日天干是"乙"。

### 第二步：根据日天干确定子时天干

用五鼠遁元歌诀：日干乙 → "乙庚丙作初" → 子时为丙子。

### 第三步：推算任意时辰的天干

从子时开始，按天干顺序推算：

| 时辰 | 地支 | 天干（日干为乙庚时） |
|------|------|---------------------|
| 子时 | 子 | 丙（丙作初） |
| 丑时 | 丑 | 丁 |
| 寅时 | 寅 | 戊 |
| 卯时 | 卯 | 己 |
| 辰时 | 辰 | 庚 |
| 巳时 | 巳 | 辛 |
| 午时 | 午 | 壬 |
| 未时 | 未 | 癸 |
| 申时 | 申 | 甲 |
| 酉时 | 酉 | 乙 |
| 戌时 | 戌 | 丙 |
| 亥时 | 亥 | 丁 |

### 快捷推算公式

时辰天干 = (日天干序号 × 2 + 时辰地支序号) mod 10

- 日天干序号：甲=1, 乙=2, ..., 癸=10
- 时辰地支序号：子=1, 丑=2, ..., 亥=12

## 与五不遇时的关系

**五不遇时**是奇门中一种特殊的不吉利时辰：

- 时辰天干克日天干，称为"五不遇时"
- 日上起时法算出的时辰天干，如果恰好克制日天干，这个时辰就不吉利
- 五不遇时一般不宜用事（谋事不成、有损耗）

> 例如：日干乙（木），时辰为辛（金）。金克木 → 五不遇时，不利。

五不遇时的判断也需要先用五鼠遁元算出时辰天干，所以日上起时法是判断五不遇时的前提。

## 应用宜忌

- **宜**：背熟五鼠遁元歌诀，这是基本功
- **宜**：查表对照练习，直到能心算
- **忌**：忽略日上起时，直接用时辰地支排盘——缺少天干信息无法完整起局
- **忌**：甲/己日、乙/庚日等两两分组记混

## 记忆技巧

五对日干，每对共用一个歌诀：

- **甲己** → 甲子起（记住：首尾呼应，甲开始甲结束）
- **乙庚** → 丙子起（记住：乙第二、丙第三，挨着的）
- **丙辛** → 戊子起（记住：丙第三、戊第五，跳两个）
- **丁壬** → 庚子起（记住：丁第四、庚第七，对称感）
- **戊癸** → 壬子起（记住：戊第五、壬第九，接近收尾）
