# 真人步斗法

## 金函玉镜补充

### 定义与基本属性

**真人步斗法**是奇门遁甲法术层面的重要实操方法，通过在地上画北斗七星、配合禹步和咒语，达到与神人感通的目的。《步斗经》曰："夫步斗可以通神"，是古代术士修炼的核心功法之一。

通俗理解：步斗法就像是古代的"通灵仪式"——按照北斗七星的图案在地上走步，配合咒语，连续一百天后就能与"神人"感通。在行军、办事需要"法术加持"时使用。

### 步斗法实操步骤

#### 一、设坛
当于**夜半居星光下**，立场画地作北斗七星，中间相去三尺。

#### 二、布星
天蓬从天罡起，随次第布之。

#### 三、站位
居魁前逆布讫，正立天英。

#### 四、行步
先举**左足**，并诵咒，以次依轻步之，左右更遍履之。如法为之百日，则与神人自然感通。

### 禹步法

凡作事皆宜禹步。初移一步，不在九步之数，然相因仍之一步七尺，合二丈一尺。

**禹步步伐**：
1. 正立右足以，前左足在后
2. 次复前左足
3. 次复前右足
4. 以左足从右足并 (一禹步)
5. 次复前右足、次前左足并 (二禹步)
6. 次复前左、次前右、次左右足并 (三禹步)

### 未济禹迹图

以九宫表示步伐顺序（右、左表示足）：
- 芮（坤）右 → 柱（兑）左 → 心（干）右
- 英（离）左 → 禽（中）左 → 蓬（坎）左
- 辅（巽）右 → 冲（震）左 → 任（艮）右

### 既济禹迹图

- 芮（坤）右 → 柱（兑）左 → 心（干）右
- 英（离）左 → 禽（中）左 → 蓬（坎）左
- 辅（巽）右 → 冲（震）左 → 任（艮）右

**禹步口诀**：巽双离只坤单步，震亦依依兑是双，艮上独行坎只立，天门归去望天罡。

### 步斗咒（移动每步持咒一句）

「白气混沌灌我形，禹步相催合登明，天回地转步七星，蹑罡履斗齐九灵，亚指伏妖众邪惊，众灾消灭我长生，我得长生，我得长生。」

至心虔诚步星斗，毕即念后咒：
「六甲九章，天圆地方，四时五行，青赤白黄，太乙为师，日月为光，禹步治道，蚩尤避兵，青龙扶毂，白虎伏行，荧惑前引，辟不祥兆，北斗诛伐，去降凶殃，五神从我，周游八方，当我者死，嫉我者亡，左社右稷，寇盗伏藏，行者有喜，留者有福，万神护我，永除盗贼，急急如律令。」

### 真人闭六戊法

凡出军止宿，要安营下寨，避祸逃难，当用此法，鬼神不敢近也。其法于旬中六戊土，用刀从鬼门起左行画地一周，次取中央土一斗，分置六戊之上，六处各置一升余，乃置刀于原取土处，入中央咒曰：

「泰山之阳，恒山之阴，盗贼不起，虎狼不侵，城郭不完，闭以金关，千凶万恶，莫之敢干，急急如律令勅。」

六戊定位：
- 甲子旬戊辰
- 甲戌旬戊寅
- 甲申旬戊在子
- 甲午旬戊在戌
- 甲辰旬戊在申
- 甲寅旬戊在午

### 与众知的区别

- 现代奇门教材完全不涉及真人步斗法、禹步法、闭六戊法
- 禹步与九星配合在实战中有完整体系（未济禹迹/既济禹迹），非简单步法
- 步斗咒、六戊咒是法术奇门独有的修行功法
