# 正授超神接气

## 定义与基本属性

**正授、超神、接气**是符头与节气之间时间关系的三种情况。因为奇门的用局规则既依赖节气又依赖符头，而节气（天文时刻）和符头（干支历法）并不完全同步，就会出现这三种情况。

通俗理解：符头像是"值班的班长"，节气像是"日历上的交班时间"。大多数时候班长提前到（超神），少数时候班长迟到（接气），偶尔刚好准时到（正授）。

## 三种情况

### 一、正授

**交节当天正好是上元符头**。符头与节气完全同步，八字契合。

这是最理想的情况，符头和节气在同一天，不需要做任何调整。直接按该节气的上元局起局即可。

> 例如：冬至当天正好是甲子日（冬至的上元符头），这就是正授。

### 二、超神

**符头在节气之前到来**（多数情况）。符头先出来了，节气还没到。

> 这是最常见的情况。因为干支历法的节奏和天文节气的节奏有细微差异，大多数情况下符头会稍微超前。

处理方法：
- 超神时，按符头定局（不等节气到来）
- **超神九天及以上需置闰**：如果符头比节气早到了9天或以上，累计偏差过大，需要在芒种或大雪进行置闰（多过一轮上中下三元）

> 详见 [[超神接气置闰]] 中关于置闰的详细说明。

### 三、接气

**节气在符头之前到来**（置闰之后出现）。节气先到了，符头还没出来。

> 接气一般发生在置闰之后——因为置闰把历法往后"拖"了，所以接下来符头就比节气晚了。

处理方法：
- 接气时，等符头到来再换局
- 要补满完整的15天（上中下三元），不能缺斤少两

## 三者的关系

```
符头在节气前（大多数）  →  超神
                                 ↓
                        超神累积 ≥ 9天
                                 ↓
                       在芒种/大雪置闰
                                 ↓
符头在节气后  →  接气（直到重新对齐）
                                 ↓
符头与节气同步（少数）→  正授
```

> 这三者形成一种动态平衡关系——超神久了就置闰，置闰后就接气，接气后逐渐恢复正授，然后再慢慢出现超神……循环往复。

## 关键规则

| 规则 | 说明 |
|------|------|
| 符头在节气前 | 超神，按符头定局 |
| 符头在节气后 | 接气，等符头到来 |
| 超神 ≥ 9天 | 必须置闰 |
| 置闰时机 | 芒种（阳遁末）或大雪（阴遁末） |

## 九天为界的含义

为什么超神达到九天就必须置闰？

- 一元 = 5天
- 中元 + 下元 = 10天
- 如果超神9天，意味着上元几乎已经和下元重叠了——符头已经"跑"了将近两元
- 此时再不管，符头和节气的对应关系就会完全乱套

## 与其他概念的关系

- [[三元与符头]]：三元和符头是理解正授超神接气的前提
- [[节气与定局]]：定局时需要考虑正授超神接气的情况

## 应用宜忌

- **宜**：初学先用拆补法，暂时不深究正授超神接气（参见 [[超神接气置闰]]）
- **宜**：理解"大多数情况是超神"这个基本规律
- **忌**：遇到超神不管、遇到接气不等，导致局数用错
- **忌**：在非芒种/大雪的节气擅自置闰

## 奇门遁甲统宗补充

> 出处：《奇门遁甲统宗》卷之一·论超接之法、又诀 — 参见 [[奇门遁甲统宗（诸葛亮）]]

### 超接定义

《奇门遁甲统宗》卷一对超神接气有明确界定：

- **超**：超过也。神者，日辰也
- **接**：承接也。气者，诸节也
- **超神**：节气未至而日辰先到，则复节气为主而超越用未来之节气，此之谓超
- **接气**：节气已至而日辰未到，则伏辰为主而待日辰至方接承节。盖其气来则日辰未至，而奇星常用于前节，此之谓接也

### 正授之辨

凡节气或遇甲子、甲午、己卯、己酉，是甲己谓之**正授**。若不遇，则以官历挨究共过几日星当用第几局，名曰**超接**。其造化，每至冬至多是超，至十二三日又日闰也。

### 超接取效之速

> 丙午年四月十三日壬申交立夏节，然四月初五日甲子日。甲己是四仲日，已在立夏前九日矣，则合超，超先于甲子日，下用立夏上局奇，己巳后用甲局，此乃**先得奇后得节**，凡作用取效为速。

详细的置闰操作方法见 [[超神接气置闰]]。
