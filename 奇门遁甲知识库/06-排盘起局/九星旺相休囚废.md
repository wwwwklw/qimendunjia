# 九星旺相休囚废

## 定义与基本属性

**九星旺相休囚废**是九星的旺衰状态判断体系。与五行"旺相休囚死"不同，九星的旺衰有一个**重要区别**：九星**旺于"我生之月"**，而非五行中的"生我之月为旺"。这是一个容易搞混的关键点，必须特别注意。

通俗理解：五星（木火土金水）的旺衰规律是"谁生我，我旺"——就像孩子被父母照顾，最舒服；九星的旺衰规律是"我生谁，我旺"——就像父母照顾孩子，要付出精力。九星是"付出者"，旺于付出的月份。

## 九星旺衰的五个状态

九星旺衰有五种状态，与五行不同的是，五行叫"旺相休囚**死**"，九星叫"旺相休囚**废**"（九星没有"死"）。

| 状态 | 含义 | 九星与月令关系 |
|------|------|--------------|
| **旺** | 最强状态 | 我生之月（九星生月令） |
| **相** | 较强状态 | 同类之月（九星与月令同类） |
| **休** | 衰弱状态 | 生我之月（月令生九星） |
| **囚** | 被克状态 | 克我之月（月令克九星） |
| **废** | 最弱状态 | 我克之月（九星克月令） |

## 五行的旺相休囚死（对比参考）

作为对比，五行（木火土金水）的旺衰规律如下：

| 状态 | 含义 | 五行与月令关系 |
|------|------|--------------|
| **旺** | 最强 | 生我之月（月令生五行） |
| **相** | 较强 | 我生之月（五行生月令） |
| **休** | 衰弱 | 克我之月（月令克五行） |
| **囚** | 被克 | 我克之月（五行克月令） |
| **死** | 最弱 | 同类之月（与月令相同） |

> **核心区别**：九星旺于"我生之月"，而五行旺于"生我之月"。正好相反！

## 九星旺衰举例

### 天蓬星（属水）的旺衰

| 状态 | 对应月份 | 关系 |
|------|---------|------|
| **旺** | 亥、子月（冬） | 天蓬属水，水生木，亥子月为水→我生之月？不对。亥子月是水月，我生之月应该是木月（寅卯）。**纠正**：天蓬属水，水能生木，所以旺于**寅卯月（春/木月）** |
| **相** | 亥、子月（冬） | 天蓬属水，亥子为水月，同类之月 |
| **休** | 申、酉月（秋） | 申酉为金月，金生水，生我之月 → 休 |
| **囚** | 辰戌丑未月（四季） | 辰戌丑未为土月，土克水，克我之月 → 囚 |
| **废** | 巳、午月（夏） | 巳午为火月，水克火，我克之月 → 废 |

> 简单记忆：天蓬星属水，旺于春（我生之月=水生木=春=寅卯月），相于冬（同类之月=水=亥子月），休于秋（生我之月=金生水=申酉月），囚于四季（克我之月=土克水=辰戌丑未月），废于夏（我克之月=水克火=巳午月）。

### 天心星（属金）的旺衰

| 状态 | 对应月份 | 关系 |
|------|---------|------|
| **旺** | 亥、子月 | 天心属金，金生水（亥子=水月），我生之月 |
| **相** | 申、酉月 | 同类之月（金） |
| **休** | 辰戌丑未月 | 土生金，生我之月 |
| **囚** | 巳、午月 | 火克金，克我之月 |
| **废** | 寅、卯月 | 金克木，我克之月 |

## 天芮星的特殊规律

**天芮星属土**，有一个非常重要的特殊规律：

| 落宫 | 含义 |
|------|------|
| 落**震三宫、巽四宫**（木宫） | **不药而愈**——木克土，芮星受制，疾病容易好转 |
| 落**乾六宫、兑七宫**（金宫） | **旺不能治**——土生金（我生之月/宫），芮星旺，疾病难治 |

> 天芮星的"旺"状态（土生金=芮星旺）对应乾兑宫，此时疾病处于最强势——**旺不能治**。而"囚"状态（木克土=芮星被制）反而对病人有利——**不药而愈**。

## 记忆要点

> **九星旺于我生之月，相于同类之月。**
> **休于生我之月，囚于克我之月，废于我克之月。**

关键区别：
- 五行：**生我之月为旺**（被动、接受）
- 九星：**我生之月为旺**（主动、付出）

## 与其他概念的关系

- [[九星概述|九星]]：九星旺衰是对九星力量的判断
- [[五行]]：九星旺衰体系的底层逻辑基于五行生克
- [[天芮星]]：天芮星的旺衰有特殊规律，疾病预测中特别重要
- [[起局步骤]]：起局后需判断九星旺衰，辅助分析力量对比
- 八门旺衰：八门也有旺衰判断，但规律与九星不同

## 应用宜忌

- **宜**：牢记"九星旺于我生之月"这个最核心的规律
- **宜**：天芮星的特殊规律在疾病预测中要特别注意
- **忌**：把九星的旺衰与五行的旺衰混为一谈——核心规律正好相反
- **忌**：忘记天芮星的"旺不能治"和"囚则不药而愈"的特殊规律

## 开悟之门补充

### 九星旺相休囚废的落宫与月令之争

《开悟之门》提出了一个关键问题：九星旺相休囚废，是以**落宫**为主判断，还是以**月令**为主判断？

> **结论**：根据易理和实践经验，九星的旺相休囚废，**应以落宫为主来判断，兼看月令**。

**落宫判断法**（天蓬星属水为例）：

| 落宫关系 | 落何宫 | 状态 | 含义 |
|---------|-------|------|------|
| 与我同行 | 坎1宫（水） | **相** | 同五行，落入兄弟宫，只能帮扶，次旺 |
| 我生之宫 | 震3/巽4宫（木） | **旺** | 入子孙宫，作用最大发挥，最旺 |
| 我克之宫 | 离9宫（火） | **休** | 入妻财宫，火当令，水发挥不了作用 |
| 生我之宫 | 乾6/兑7宫（金） | **废** | 入父母宫，金当令能生水，不需天蓬再起作用 |
| 克我之宫 | 艮8/坤2宫（土） | **囚** | 入官鬼宫，土当令克水，天蓬被囚禁 |

**口诀**（《[[烟波钓叟歌]]》）：

> **"与我同行即为相，我生之月诚为旺，废于父母休于财，囚于鬼兮真不妄。"**

九星是在天上运行的星体，本身无所谓旺相休囚，它们不受地球上四季气候的影响。九星的旺相休囚废，主要是针对它们对地球的作用和影响而言，也就是它们飞临某宫对该宫的作用和影响。

> **关键**：九星旺衰以**落宫为主、月令为辅**，这是《开悟之门》的核心观点。
