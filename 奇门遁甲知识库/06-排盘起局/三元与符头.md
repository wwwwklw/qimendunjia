# 三元与符头

## 定义与基本属性

**三元与符头**是奇门遁甲定局的基础概念。一个节气15天，分为上、中、下三元；每元的符头（即该元第一天的天干地支）决定了该元用哪一局。

通俗理解：三元就像把一个月分成了上半月、中半月、下半月三段。符头就是每段第一天的"身份证号"——看它的天干地支，就知道该用哪个局。符头是定局的"钥匙"。

## 基本数量关系

| 时间单位 | 关系 | 说明 |
|---------|------|------|
| 一元 | = 5天 = 60个时辰 | 恰好演完一盘（一局） |
| 一节气 | = 15天 = 上中下三元 | 一个节气内演三局 |
| 一节气 | = 3元 × 5天/元 | 上元、中元、下元各5天 |

> **一元 = 5天 = 60时辰 = 演完一局**：这是奇门排盘的基本时间单位。因为九宫格每个宫位可以代表一个时辰，60个时辰刚好转动一圈。

## 上中下三元的划分

每个节气分为三段：

- **上元**：节气的前5天
- **中元**：节气的中间5天
- **下元**：节气的后5天

> 例如：冬至节气共15天，前5天用冬至的"上元"局，中间5天用"中元"局，后5天用"下元"局。

## 符头的确定

**符头**是每元第一天（即第1天、第6天、第11天）的干支。符头的**天干必为甲或己**。

### 上元符头

天干为**甲或己**，地支为**子午卯酉**（四仲）：

| 符头 | 天干 | 地支 | 地支类型 |
|------|------|------|---------|
| 甲子 | 甲 | 子 | 四仲 |
| 甲午 | 甲 | 午 | 四仲 |
| 己卯 | 己 | 卯 | 四仲 |
| 己酉 | 己 | 酉 | 四仲 |

> **四仲**：子、午、卯、酉，四正之位，位于每个季节的正中间。

### 中元符头

天干为**甲或己**，地支为**寅申巳亥**（四孟）：

| 符头 | 天干 | 地支 | 地支类型 |
|------|------|------|---------|
| 甲寅 | 甲 | 寅 | 四孟 |
| 甲申 | 甲 | 申 | 四孟 |
| 己巳 | 己 | 巳 | 四孟 |
| 己亥 | 己 | 亥 | 四孟 |

> **四孟**：寅、申、巳、亥，四隅之位，位于每个季节的开始。

### 下元符头

天干为**甲或己**，地支为**辰戌丑未**（四季/四库）：

| 符头 | 天干 | 地支 | 地支类型 |
|------|------|------|---------|
| 甲辰 | 甲 | 辰 | 四季 |
| 甲戌 | 甲 | 戌 | 四季 |
| 己丑 | 己 | 丑 | 四季 |
| 己未 | 己 | 未 | 四季 |

> **四季**（四库）：辰、戌、丑、未，每个季节的最后一个月，有"库藏"之意。

## 符头记忆口诀

> **子午卯酉上元是，寅申巳亥中元求。**
> **辰戌丑未下元定，甲己为头莫弄丢。**

解释：
- 符头天干一定是甲或己
- 地支是子午卯酉（四仲）→ 上元
- 地支是寅申巳亥（四孟）→ 中元
- 地支是辰戌丑未（四季）→ 下元

## 定局步骤

1. 查看万年历，确定当天的干支
2. 判断该天属于哪个节气的第几天
3. 找到该元的符头
4. 根据符头确定上/中/下元
5. 对照节气定局表，查出对应局数

> 详见 [[节气与定局]]。

## 与其他概念的关系

- [[节气与定局]]：三元是节气内部的时间划分，符头是定局的关键

## 应用宜忌

- **宜**：背熟"子午卯酉上元、寅申巳亥中元、辰戌丑未下元"的口诀
- **宜**：定局时先看天干是否为甲或己（找符头），再看地支是哪一类（定上中下）
- **忌**：地支分类搞混，把四仲当作中元、四孟当作上元
- **忌**：忘记天干必须是甲或己才能做符头

## 奇门宝鉴补充

> 出处：《御定奇门遁甲·奇门宝鉴》（金志文点校）卷一·释义四十四则 — 参见 [[御定奇门遁甲奇门宝鉴（金志文点校）]]

### 释符头

符头者，符合上元首甲也。**甲子、己卯、甲午、己酉**四日为符头。

#### 三元之制

- 一气统三元，分上、中、下三局
- 一局统五日，五日统六十时为一纪
- 一纪即一元纪法

以十干加十二支，首尾相合为一终。干以甲为首、癸为尾，支以子为首、亥为尾。故自甲子至癸亥，合得六十为一纪，谓之一元。

#### 奇门选时之法

时非日不能得元，元非气不能得局。是以一气统三元为三局，分上、中、下以别之。再别为周，再周为变，故终十二纪，而天地之气一变。

#### 符头之理

- 岁得日纪六
- 凡日纪一，统时纪十二，而变一气，是以谓之六气也
- 凡变气一，统节气四，故四分日纪，以起各气之元
- 用**甲己二干**，遁起甲子时为元首

#### 上中下元之分

| 元 | 符头地支 | 加临 |
|----|---------|------|
| 上元 | 子午卯酉（四仲） | 二干加仲为上 |
| 中元 | 寅申巳亥（四孟） | 加孟为中 |
| 下元 | 辰戌丑未（四季） | 加季为下 |

以仲之甲己四日，遁起甲子时，合时纪上元之首，故曰「符头」也。

> 出处：《御定奇门遁甲·奇门宝鉴》卷一·释义四十四则 — 参见 [[御定奇门遁甲奇门宝鉴（金志文点校）]]

## 奇门遁甲统宗补充

> 出处：《奇门遁甲统宗》卷之二·三元、八节、定三元法、八卦定图 — 参见 [[奇门遁甲统宗（诸葛亮）]]

### 三元

一年二十四气，一气三候，一候五日。故五日为一元，一气之中凡统三元，而下六十花甲之时一周。

### 八节起局

统宗载八节甲子起宫之法：

| 八节 | 甲子起宫 | 所辖节气 |
|------|---------|---------|
| 冬至 | 坎一 | 小寒、大寒 |
| 立春 | 艮八 | 雨水、惊蜇 |
| 春分 | 震三 | 清明、谷雨 |
| 立夏 | 巽四 | 小满、芒种 |
| 夏至 | 离九 | 小暑、大暑 |
| 立秋 | 坤二 | 处暑、白露 |
| 秋分 | 兑七 | 寒露、霜降 |
| 立冬 | 乾六 | 小雪、大雪 |

### 八卦定图

统宗列八卦定图（后天八卦配八节）：

```
坤（立秋、处暑、白露） 兑（秋分、寒露、霜降） 乾（立冬、小雪、大雪）
离（夏至、小暑、大暑）   中                     坎（冬至、小寒、大寒）
巽（立夏、小满、芒种） 震（春分、清明、谷雨） 艮（立春、雨水、惊蜇）
```

### 定三元法

三元者，上中下三局也。一元五日，以**甲己二干为一元之首**，谓之符头。

符头所临之支定三元：

| 符头地支 | 三元 |
|---------|------|
| 子午卯酉 | 上元 |
| 寅申巳亥 | 中元 |
| 辰戌丑未 | 下元 |

> 甲子日起上元，越五日己巳起中元，又起五日甲戌起下元。三局已游，又起五日，己卯复起上元，则又是一气矣。

### 统宗版特点

- 统宗版明确「**一气三候，一候五日**」的时间结构，是三元划分的天文依据
- 八节起局表以八节甲子所起之宫为主线，将二十四节气按八节分组
- 八卦定图直观展示后天八卦与八节的对应关系，是统宗独有的图示
- 定三元法以甲己为符头、以地支四类定三元，与神奇之门、奇门宝鉴体系一致
