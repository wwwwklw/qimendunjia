# 旬首

## 定义与基本属性

**旬首**是每旬（十天）的第一天。六十甲子分为六旬，每旬十天，每旬的第一天都以六甲之一为旬首。旬首是奇门起局中确定**值符**和**值使**的关键。

通俗理解：六十甲子就像60个人的大部队，每10个人分成一个小队（一旬）。每个小队的排头兵就是"旬首"——他永远是"甲×"（甲子、甲戌、甲申、甲午、甲辰、甲寅）。起局时先找到时辰属于哪个小队，再看排头兵在哪里，就能确定值符和值使。

## 六旬划分

六十甲子分为以下六旬，每旬十天：

| 六旬 | 旬首 | 包含的干支（从旬首起十位） |
|------|------|------------------------|
| **甲子旬** | 甲子 | 甲子、乙丑、丙寅、丁卯、戊辰、己巳、庚午、辛未、壬申、癸酉 |
| **甲戌旬** | 甲戌 | 甲戌、乙亥、丙子、丁丑、戊寅、己卯、庚辰、辛巳、壬午、癸未 |
| **甲申旬** | 甲申 | 甲申、乙酉、丙戌、丁亥、戊子、己丑、庚寅、辛卯、壬辰、癸巳 |
| **甲午旬** | 甲午 | 甲午、乙未、丙申、丁酉、戊戌、己亥、庚子、辛丑、壬寅、癸卯 |
| **甲辰旬** | 甲辰 | 甲辰、乙巳、丙午、丁未、戊申、己酉、庚戌、辛亥、壬子、癸丑 |
| **甲寅旬** | 甲寅 | 甲寅、乙卯、丙辰、丁巳、戊午、己未、庚申、辛酉、壬戌、癸亥 |

## 旬首与起局的关系

旬首是确定**值符**和**值使**的钥匙。

### 确定旬首的步骤

1. 获取起局时辰的干支（例如：丙寅时）
2. 从时柱的干支反推，找到该旬的第一天（旬首）
3. 例如：丙寅时 → 往回推：丙寅、乙丑、甲子 → 旬首为**甲子**
4. 找到地盘上旬首落于哪一宫

### 旬首与值符、值使

- **值符** = 地盘旬首所在宫位的九星
- **值使** = 地盘旬首所在宫位的八门

> 例如：如果旬首甲子落在地盘坎一宫，坎一宫的天蓬星就是值符，坎一宫的休门就是值使。

> 详见 [[值符与值使]]。

## 旬空（空亡）

每旬十天干配十二地支，每旬必然有两个地支配不上天干，这两个地支所在的宫位就是**旬空**（空亡）。

| 旬 | 空亡（缺的地支） | 空亡宫位 |
|----|---------------|---------|
| 甲子旬 | 戌、亥 | 乾六宫 |
| 甲戌旬 | 申、酉 | 兑七宫 |
| 甲申旬 | 午、未 | 离九宫、坤二宫 |
| 甲午旬 | 辰、巳 | 巽四宫 |
| 甲辰旬 | 寅、卯 | 震三宫 |
| 甲寅旬 | 子、丑 | 坎一宫、艮八宫 |

> 空亡在实际预测中非常重要，空亡之宫的信息往往不实或暂未发生。

## 六甲遁藏

奇门遁甲中，六甲（甲子、甲戌、甲申、甲午、甲辰、甲寅）不直接出现，而是隐藏在三奇六仪之中（以值符为代表），这就是"遁甲"的含义。

| 六甲 | 隐藏位置 | 九星代表 |
|------|---------|---------|
| 甲子 | 隐藏于戊 | 天蓬星 |
| 甲戌 | 隐藏于己 | 天芮星 |
| 甲申 | 隐藏于庚 | 天冲星 |
| 甲午 | 隐藏于辛 | 天辅星 |
| 甲辰 | 隐藏于壬 | 天禽星 |
| 甲寅 | 隐藏于癸 | 天心星 |

## 应用宜忌

- **宜**：熟练找出任意干支属于哪一旬、旬首是什么
- **宜**：记住六旬与空亡的对应关系
- **忌**：旬首找错 → 值符值使全错 → 整个盘面错误
- **忌**：忽略空亡——空亡之宫的信息在实际预测中需要特别注意
