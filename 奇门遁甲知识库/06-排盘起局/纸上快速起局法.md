# 纸上快速起局法

## 定义与基本属性

**纸上快速起局法**是在纸面上画一个井字格（九宫格），然后按照固定的步骤填入三奇六仪、九星、八门、八神，完成奇门遁甲排盘的方法。它与[[掌上起局法]]互为补充，是实际预测中最常用的起局方式。

通俗理解：掌上起局在大拇指上掐算，适合心算；纸上起局在纸上画格子填符号，适合初学和详细分析。两者原理相同，只是工具不同——一个是"手指"，一个是"纸张"。

## 基本步骤

### 第一步：画九宫格

在纸上画一个井字格，标出九宫固定位置：

```
| 四 | 九 | 二 |
| 三 | 五 | 七 |
| 八 | 一 | 六 |
```

> 注意：井字格的方位与地图方位不同——**上南下北、左东右西**（传统方位观念）。

### 第二步：确定地盘（定局）

根据当天的节气、三元、符头，确定用阳遁几局还是阴遁几局。

例如：冬至后阳遁一局。

### 第三步：布三奇六仪（排地盘）

**关键口诀**：**用几局就在几宫填"甲子戊"**。

- 从甲子戊所在的宫位开始，阳遁顺排、阴遁逆排
- 顺序：戊、己、庚、辛、壬、癸、丁、丙、乙（阳遁时）

> 例如：阳遁一局，"甲子戊"在坎一宫，然后己在坤二宫、庚在震三宫……以此顺排。

### 第四步：定值符值使

根据起局时辰的干支，找到所在旬的旬首，确定：
- **值符**：旬首所在宫位的九星
- **值使**：旬首所在宫位的八门

### 第五步：转天盘（九星带三奇六仪）

值符随时干走：
- 看时干落在哪个宫位
- 把值符（连同它所在宫的三奇六仪）转到时干所在的宫位
- 其余各星依次顺转（阳遁）或逆转（阴遁）

### 第六步：转人盘（八门）

值使随时支走：
- 从旬首的宫位出发（阳顺阴逆），数到时支所在的宫位
- 把值使转到该宫位
- 其余各门依次排布

### 第七步：排神盘（八神）

值符所在宫位排"值符"，然后阳遁顺时针、阴遁逆时针依次排八神。

## 井字格示意

```
┌─────┬─────┬─────┐
│ 巽4 │ 离9 │ 坤2 │
│     │     │     │
├─────┼─────┼─────┤
│ 震3 │ 中5 │ 兑7 │
│     │     │     │
├─────┼─────┼─────┤
│ 艮8 │ 坎1 │ 乾6 │
│     │     │     │
└─────┴─────┴─────┘
```

> 每个格子里填入：三奇六仪（地盘）、天盘星（含天盘奇仪）、八门、八神。四层信息一目了然。

## 与掌上起局法的对比

| 对比维度 | 纸上起局法 | [[掌上起局法]] |
|---------|-----------|-------------|
| 工具 | 纸和笔 | 手掌（大拇指掐算） |
| 信息量 | 可详细记录四层盘面 | 适合心算速断 |
| 适合场景 | 详细分析、学习、教学 | 快速判断、外出使用 |
| 优势 | 直观全面、不易遗漏 | 方便快捷、不需纸笔 |
| 劣势 | 需要纸笔 | 容易遗漏、不适合复杂分析 |

> 两种方法互为补充。建议初学先练纸上起局，熟练后过渡到掌上起局。

## 快速起局的口诀提示

1. **定点**：几局就在几宫填甲子戊
2. **定星**：找旬首，定值符值使
3. **转星**：值符随时干走
4. **转门**：值使随时支走
5. **排神**：值符在值符宫，顺/逆转

## 应用宜忌

- **宜**：初学时每次都在纸上画井字格，一步步来，不偷懒
- **宜**：九宫位置要写固定，不移位，养成好习惯
- **忌**：方向搞反——上为南（离九宫）、下为北（坎一宫）
- **忌**：跳步操作——每一步都必须正确，才能进入下一步
- **忌**：阴阳遁的顺逆转搞混
