# 十二长生

## 定义与基本属性

**十二长生**又称"寄生十二宫"，是描述事物从生到灭、再从灭到生的十二个循环阶段，用于衡量[[十天干]]在十二地支不同位置的旺衰状态。十二长生是五行之气在不同时空节点上的强弱表现。

十二长生为：

> **长生 → 沐浴 → 冠带 → 临官 → 帝旺 → 衰 → 病 → 死 → 墓 → 绝 → 胎 → 养**

这十二个阶段构成一个完整的生命周期循环，生、旺、墓、绝为其关键节点。

## 五行寄生规则

每种五行（即十天干）有自己的"长生"起始位：

| 五行 | 长生所在 | 说明 |
|------|----------|------|
| 木 | 亥 | 木长生在亥 |
| 火 | 寅 | 火长生在寅 |
| 金 | 巳 | 金长生在巳 |
| 水、土 | 申 | 水土长生在申 |

> **阳干顺行、阴干逆行**

- 阳干（甲、丙、戊、庚、壬）从长生开始，顺地支顺序推进十二宫
- 阴干（乙、丁、己、辛、癸）从长生开始，逆地支顺序推进十二宫

例如甲木（阳干）长生在亥 → 沐浴在子 → 冠带在丑 → ……（顺排）
乙木（阴干）长生在午 → 沐浴在巳 → 冠带在辰 → ……（逆排）

参见[[五阳干与五阴干]]。

## 各状态含义详解

| 状态 | 含义 | 象意 |
|------|------|------|
| **长生** | 初生，如婴儿初生 | 新生、开始、希望、萌芽 |
| **沐浴** | 洗浴去垢，又称"败地" | 桃花、情欲、变化不定、脆弱 |
| **冠带** | 成人加冠，逐渐成熟 | 成长、学习、初具规模 |
| **临官** | 入仕做官，又名"禄" | 事业有成、地位、担当 |
| **帝旺** | 权势鼎盛，旺极 | 高峰、极致、但旺极必衰 |
| **衰** | 盛极而衰，开始走下坡 | 递减、下滑、力不从心 |
| **病** | 衰而生病 | 问题、困难、衰退明显 |
| **死** | 气尽而亡 | 终结、死亡、无气无力 |
| **墓** | 入墓入库，又曰"库" | 收藏、入库、入狱、潜藏 |
| **绝** | 气已绝，又名"受气" | 绝灭、零基础、重新开始前 |
| **胎** | 受胎，重新孕育 | 孕育、准备、暗藏新机 |
| **养** | 养胎成形 | 养育、准备、蓄势待发 |

## 在奇门预测中的应用

十二长生是判断**旺衰**最重要的依据之一。

- 用神落宫处长生、临官、帝旺 → 旺相，吉利
- 用神落宫处死、墓、绝 → 衰弱，不利
- 用神入墓（墓库）→ 被困、被限制、无力施展，但墓也可能为"入库收藏"之意，需具体判断
- 帝旺虽旺，但"旺极必衰"，此时不宜再求进取

在具体预测中：

- 测疾病：病星临死、墓、绝 → 病情危重
- 测财运：财星临帝旺 → 财运正旺
- 测考试：用神临临官 → 状态正好
- 测行人：用神入墓 → 被困某处不能归

参见[[六十甲子]]中的干支组合旺衰判断。

## 开悟之门补充

### 墓与库的区分（关键）

在十二状态中，"墓"又叫"库"。既叫墓，又叫库，二者有本质区别：

| 判断维度 | 入库 | 入墓 |
|:---|:---|:---|
| **月令状态** | 天干旺相 | 天干休囚 |
| **含义** | 暂时放进仓库，发挥不了作用 | 如同死亡被埋进坟墓 |
| **可恢复性** | 库门打开（库被冲），仍可发挥作用 | 彻底完蛋 |

> **核心**：一定要根据月令区分用神是**入墓**还是**入库**，绝不能将二者混为一谈。这是断卦出错的主要原因之一。

### 开悟之门补充：墓库区分实例详解——辰月金旺不为墓

《神奇之门》实例九测外甥女失踪堪称墓库区分的**经典反面教材**。用神时干辛落巽4宫（辰墓），初学者未细分墓、库区别，一见辛在辰宫加上临死门、天柱凶星，就断"凶多吉少，恐怕已被害致死"——结果断错。

实际分析有三个层次纠正了误判：
1. **月令决定库/墓**：求测时为辰月，辰月**土生金**，时干辛金旺相，故辛入辰为**入库**而非入墓，入库表示暂时被收藏，并未死亡。
2. **冲解墓库**：格局虽为甲午辛+甲子戊（困龙被伤），但**子午相冲**，冲则不为入墓。
3. **月令分日用神**：4月5日清明节刚入辰月，头九天**乙木当令**（辰月乙木9天、癸水3天、戊土18天），木旺金囚土死，天柱星、死门均凶不起来。

最终此女孩被安全找回。**关键教训**：用神临死门在行人走失中反而容易找回（凶门反断）。

出处：《开悟之门》

### 古籍经验口诀

> **"老怕帝旺，少怕衰，中年就怕死、绝、胎。"**

| 年龄段 | 怕的状态 | 原因 |
|:---|:---|:---|
| **老** | 帝旺 | 违背衰老的自然规律 |
| **少** | 衰/病 | 青春年少不应衰病 |
| **中年** | 死/墓/绝/胎/养 | 正该兴旺发达时处于低潮 |

### 开悟之门补充：老怕帝旺实例详解

《神奇之门》实例二女测父病充分验证了"老怕帝旺"的规律：年干丙为用神（其父1921年生，1996年已75岁），落离9宫为**帝旺之地**。按常规，帝旺为旺相吉象，但因为求测人已年迈（75岁），老年人处于帝旺状态违背了"老则衰"的自然规律——如同枯木逢春反为妖。又逢**空亡**（帝旺+空亡=虚旺），故断不吉。最终其父果然在当年酉月去世。

> **核心**：十二长生的吉凶必须结合**年龄**判断。帝旺对年轻人是吉，对老年人反而是凶——违背事物发展规律的状态即为凶兆。

出处：《开悟之门》

### 天干与地支十二长生的重要区别

> **特别提醒**：天干与地支在十二长生中有重大区别，极易混淆！

| 区别 | 天干 | 地支 |
|------|------|------|
| 土的位置 | 戊己土**与火同宫** | 土**与水同宫**（水土同宫） |
| 阴阳路线 | 阳干顺行、阴干逆行 | 不分阴阳，均按顺时针（顺行） |
| 原理 | 火在天上，火能生土；午与未土相合 | 地中水与土在一起；子水与丑土相合 |

> 地支的十二长生：寅卯木均长生在亥、帝旺在卯、墓在未（即与天干甲木相同）；巳午火均长生在寅、帝旺在午、墓在戌（即与天干丙火相同）；辰戌丑未土均长生在申、帝旺在子、墓在辰（即与天干壬水相同）；申酉金均长生在巳、帝旺在酉、墓在丑（即与天干庚金相同）。

## 上元经补充

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]十干落宫章

### 天干十二态的适用范围

飞盘奇门多出来一个中宫，**中宫不同于他宫**，内含天干不像外八宫天干一样接受生旺墓十二态的制约。这一点飞盘研习者需特别注意。

### 十二态深层解读

**长生与临官**：公认最吉利的两个状态。从力量上说临官大于长生。但并非临官处处优于长生——如刚踏入社会的青年占测事业走向，临长生更好（代表成长空间大），而临官则代表已经到达高位。

**沐浴（败地）**：三重引申义——①桃花/情色（洗澡裸体）②败地（色字头上一把刀）③私欲（澡房隐私、不切实际的幻想）。临沐浴的人往往想入非非而没有实际行动。

**冠带**：双重性——①正面：穿衣戴冠，发奋求取功名；化妆/包装/文凭等助人事业的因素 ②负面：格局搭配不当→虚假包装、表里不一。

**帝旺（刃）**：能量最大值，但**物极必反**。帝旺又名"刃"（刀刃），极端旺相已隐含失败先兆。临帝旺需保持谦虚谨慎，切记高调外露。临官是事业高峰，帝旺是更高但暗藏凶险。

**衰**：由强转弱的起始点。区别于病——能量锐减但还没到疾病缠身的状态。如退下来的老总仍能发挥一定影响力。

**病**：身体有毛病、事业有问题。但只要格局符号搭配得当，一定能转危为安。

**死**：生命物理特征终止。引申为做事死板、状态消极，能量极其衰弱，阻力极大，难有转圜余地。

**墓**：奇门与四柱的关键区别！四柱中的刑冲可开库（墓转库），但在奇门系统中**一般的刑冲并不能化墓为库**，反而使能量进一步衰减。奇门中转墓为库需同时满足：①月令强有力支持 ②马星/三合/六合等引动。一旦出现刑冲等负能量信息，转墓为库即成泡影。

**绝**：前气与后气交接枢纽。引申为绝境、绝望、做事决绝不留后路。但如果宫内格局搭配得当，又可表**新旧事业转换时期**（刚辞职又开始组建新公司）。

**胎**：事物酝酿阶段。母子都十分脆弱，切记刑冲（遇刑冲往往流产/策划失败等）。特别提醒：根据占测经验，很多**老人起局时身体没有明显不适但局中已显示严重病患信息**——此时病患宫中天干多临于胎地。这表明病情已经出现，切不可盲目大意。

**养**：培养、养育、修养。临养地**适合积蓄力量，切记唐突盲动**。易静不易动，切记刑冲克害。

### 天干衰旺决定格局吉凶程度

天地两干所组成的格局有其先天吉凶，但在实际论断中必须根据落宫具体判断。以戊加庚为例：

| 落宫 | 戊状态 | 庚状态 | 吉凶 |
|------|--------|--------|------|
| 巽宫 | 帝旺（强旺） | 败地（被制） | 庚对戊的克害小得多 |
| 兑宫 | 死态 | 帝旺 | 戊受到的克害极大 |

**天干落宫十二态在具体论断中非常重要，要根据落宫的不同把握天干意象的差异。**
