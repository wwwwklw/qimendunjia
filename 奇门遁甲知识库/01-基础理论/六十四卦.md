# 六十四卦

## 定义与基本属性

**六十四卦**是由[[阴阳]]两个基本符号（阴爻 `- -` 和阳爻 `—`）通过六次组合而成，总共 2^6 = 64 种组合，即六十四卦。每卦由六爻组成，从下往上依次称为初爻、二爻、三爻、四爻、五爻、上爻。

六十四卦的生成逻辑：
1. **阴阳**（两仪）
2. → 两仪各加一爻 → **四象**（太阳、少阴、少阳、太阴）
3. → 四象各加一爻 → **八卦**（乾兑离震巽坎艮坤）
4. → 八卦两两重叠 → **六十四卦**

通俗理解：六十四卦是一套用六层"开关"（每爻开关一次，阴或阳）描述万物的符号系统，每一卦代表一种特定的变化状态。

## 详细象意

### 六十四卦与遗传密码的对应

现代科学发现的一个惊人巧合：六十四卦的结构与生物遗传密码（DNA密码子）完全对应。
- DNA有4种碱基（A、T、C、G），每3个组成一个密码子 → 4^3 = 64种密码子
- 易经有2种爻（阴、阳），每6个组成一卦 → 2^6 = 64卦
- 六十四卦与64个遗传密码子存在一一对应的映射关系

这一发现说明：阴阳符号系统与生命编码系统之间存在深刻的同构性，体现了[[宇宙全息论]]的基本思想。

### 六十四卦方图与圆图

北宋邵雍（邵康节）创制的两种排列方式：
- **方图**：按八卦次序纵横排列，形成8×8方阵，横看纵看斜看各有规律
- **圆图**：将方图转化为圆形排列，体现天地运行的圆周循环

方图和圆图是[[先天八卦]]方位与六十四卦结合的产物，是象数派的重要工具。

### 八宫六十四卦（京房排列）

汉代京房将六十四卦按八卦分为八宫，每宫八卦，按特定次序排列：

| 次序 | 名称 | 说明 |
|------|------|------|
| 一世 | 初爻变 | 最轻微的变化 |
| 二世 | 二爻再变 | 变化加深 |
| 三世 | 三爻再变 | 变化到中位 |
| 四世 | 四爻再变 | 变化继续深入 |
| 五世 | 五爻再变 | 变化达顶峰 |
| 游魂 | 四爻复归 | 变化的回归 |
| 归魂 | 三爻以下全归 | 变化回归本宫 |

加上每宫的本宫卦（八纯卦），共八宫六十四卦。这种排列方式是[[纳甲筮法]]和六爻预测的基础。

## 应用宜忌

- **宜**：熟悉八宫卦序，这是[[纳甲筮法]]、六爻预测的基础前提
- **宜**：了解卦名和基本卦义，但不必死记全部卦爻辞
- **忌**：初学者一上来就背全部六十四卦的卦爻辞，事倍功半

## 与其他概念的关系

- [[阴阳]]：六十四卦由阴爻和阳爻组成
- [[先天八卦]]：八卦是六十四卦的基本构件
- [[后天八卦]]：八卦的另一种排列体系
- [[连山易与归藏易]]：连山、归藏、周易三易都以六十四卦为基础
- [[纳甲筮法]]：京房在六十四卦基础上纳入天干地支
- [[十翼（易传）]]：十翼中《序卦》解释六十四卦的排列次序
- [[两派六宗]]：义理派和象数派都以六十四卦为研究对象
