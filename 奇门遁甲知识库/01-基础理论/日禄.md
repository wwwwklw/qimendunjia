# 日禄

## 定义与基本属性

**日禄**又称"十干禄"，是每个天干在地支中的"禄位"（临官之位）。"禄"即俸禄、福禄之意，天干得禄位为旺盛有气之象。

在择吉中，日干得禄的时辰或方位为吉利。

## 日禄口诀

金函玉镜所载：

> 甲禄在寅，乙禄在卯，丙戊在巳，丁己在午，
> 庚禄在申，辛禄在酉，壬禄在亥，癸禄在子。

## 十干禄位表

| 天干 | 禄位（地支） | 方位 | 说明 |
|------|------------|------|------|
| **甲** | 寅 | 东北 | 甲木禄在寅（寅为甲木临官之地） |
| **乙** | 卯 | 正东 | 乙木禄在卯（卯为乙木临官之地） |
| **丙** | 巳 | 东南 | 丙火禄在巳（巳为丙火临官之地） |
| **戊** | 巳 | 东南 | 戊土禄在巳（与丙同禄，因火生土） |
| **丁** | 午 | 正南 | 丁火禄在午（午为丁火临官之地） |
| **己** | 午 | 正南 | 己土禄在午（与丁同禄，因火生土） |
| **庚** | 申 | 西南 | 庚金禄在申（申为庚金临官之地） |
| **辛** | 酉 | 正西 | 辛金禄在酉（酉为辛金临官之地） |
| **壬** | 亥 | 西北 | 壬水禄在亥（亥为壬水临官之地） |
| **癸** | 子 | 正北 | 癸水禄在子（子为癸水临官之地） |

### 规律总结

| 规律 | 说明 |
|------|------|
| **阳干禄在寅申巳亥** | 甲（寅）、丙戊（巳）、庚（申）、壬（亥）——四隅方 |
| **阴干禄在子午卯酉** | 乙（卯）、丁己（午）、辛（酉）、癸（子）——四正方 |
| **丙戊同禄** | 丙和戊的禄位都在巳（火生土） |
| **丁己同禄** | 丁和己的禄位都在午（火生土） |

## 日禄的择吉应用

- 每日天干的禄位所在之方为吉方
- 禄位时辰（如甲日寅时）为吉时
- 出行向禄方吉利
- 谈判、签约宜在禄位时辰进行

## 与十二长生诀的关系

日禄即十二长生诀中的**"临官"**之位：

| 天干 | 临官（禄） | 帝旺 | 含义 |
|------|-----------|------|------|
| 甲 | 寅 | 卯 | 寅为临官（禄），卯为帝旺（更旺） |
| 乙 | 卯 | 寅 | 乙木阴阳逆行，卯为临官（禄） |
| 丙 | 巳 | 午 | 巳为禄，午为帝旺 |
| 丁 | 午 | 巳 | 丁火逆行，午为禄 |
| 庚 | 申 | 酉 | 申为禄，酉为帝旺 |
| 辛 | 酉 | 申 | 辛金逆行，酉为禄 |
| 壬 | 亥 | 子 | 亥为禄，子为帝旺 |
| 癸 | 子 | 亥 | 癸水逆行，子为禄 |

> **临官**是事物发展到"做官任职"的阶段——成熟而有力，所以为"禄"。

## 与其他概念的关系

- [[十二地支]]：禄位以地支表示
- [[十天干]]：禄位按天干推算
- [[贵人方]]：禄是"自身旺盛"的方位，贵人是"贵人相助"的方位，两者配合使用
- [[喜神方位]]：喜神是"喜气所在"，禄位是"力量所在"，皆为择吉参考
- [[日干]]：日禄以日干为本，看日干在地支中的旺衰状态

## 应用宜忌

- **宜**：重要事务安排在日禄时辰
- **宜**：出行、谈判向日禄方
- **忌**：禄位逢冲（如甲禄在寅，申日寅时即为冲禄）
- **忌**：过分依赖日禄而忽略奇门全盘格局
