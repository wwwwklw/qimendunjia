# 五行关系论

## 定义与基本属性

五行，指木、火、土、金、水五大类最基本的物质形态及其运动变化规律。五行不仅仅是五种物质，更是对宇宙间万事万物按其属性进行分类和归纳的五种"象"，是古代中国人的一种系统分类思维方式。

五行的核心关系包括：**相生**与**相克**。相生是促进、助长的关系，相克是抑制、制约的关系。两者相互配合，维持系统的动态平衡。

## 五行生克关系

### 五行相生（循环促进）

```
木 → 火 → 土 → 金 → 水 → 木（循环）
```

- **木生火**：木材可以燃烧生火
- **火生土**：火烧完后化为灰土
- **土生金**：金属矿物埋藏于土中
- **金生水**：金属熔化后变为液态（另说：金属器物可以取水）
- **水生木**：水可以滋养树木生长

### 五行相克（循环制约）

```
木 → 土 → 水 → 火 → 金 → 木（循环）
```

- **木克土**：树木扎根破土，吸收土壤养分
- **土克水**：土可以筑堤挡水（兵来将挡，水来土掩）
- **水克火**：水可以灭火
- **火克金**：火可以熔化金属
- **金克木**：金属刀具可以砍伐树木

### 生克关系总结

记住口诀：**"木火土金水，顺次相生，隔一相克"**

相邻的五行是相生关系，间隔一位的五行是相克关系。

## 六亲纳入五行

古人将人际关系归纳为五种基本关系，并与五行生克相对应，这就是"**六亲**"（实际是五种关系类型）：

| 关系名称 | 五行对应 | 含义 |
|---------|---------|------|
| **父母** | 生我者 | 生我者如父母，有庇护、养育之义 |
| **子孙** | 我生者 | 我生者如子孙，有产出、创造之义 |
| **官鬼** | 克我者 | 克我者如官鬼，有约束、克制之义 |
| **妻财** | 我克者 | 我克者如妻财，有掌控、支配之义 |
| **兄弟** | 同类者 | 同类者如兄弟，有帮扶、竞争之义 |

举例：以"木"为我，则——水生木 → 水为父母；木生火 → 火为子孙；金克木 → 金为官鬼；木克土 → 土为妻财；木见木 → 木为兄弟。

## 应用宜忌

- **宜**：利用生克关系分析事物之间的促进与制约关系，找到平衡点
- **忌**：片面强调生而忽略克，或强调克而忽略生；生克是一种自然调节机制，过犹不及

在奇门遁甲中，天干地支之间、门星神之间、日时与落宫之间，都离不开五行生克的分析。六亲关系在预测中用以判断人与事之间的利害关系。

## 旺相休囚/落宫状态

五行在一年四季中有不同的旺衰状态，即[[五行]]中的旺相休囚死。五行生克的力度取决于各自所处的旺衰状态：旺者克力大，衰者无力克人；旺者生力足，衰者自顾不暇难以生人。
