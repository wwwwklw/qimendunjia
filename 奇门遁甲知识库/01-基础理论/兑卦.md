# 兑卦（兑为泽）

## 定义与基本属性

兑卦，卦符为 ☱，口诀"**兑上缺**"——两阳爻在下，一阴爻在上断（缺口），像湖泊沼泽上方的缺口。兑卦代表泽、少女、口、喜悦、言说，是八卦中最具沟通表达、欢悦交流特性的卦象。

| 属性 | 内容 |
|------|------|
| **卦名** | 兑 |
| **卦符** | ☱ |
| **口诀** | 兑上缺 |
| **代表** | 泽 |
| **五行** | 金（阴金） |
| **阴阳** | 阴卦（一阴二阳，以少爻定阴阳） |
| **先天卦数** | 2 |
| **后天卦数** | 7 |
| **后天方位** | 西 |
| **对应宫位** | [[兑七宫]] |
| **家庭成员** | 少女（幼女） |
| **身体部位** | 口、舌、齿、喉、肺 |
| **季节** | 秋（申酉月，尤其酉月） |
| **时辰** | 酉时（17:00-19:00） |
| **地支** | 酉 |
| **对应奇门符号** | [[惊门]]、[[天柱星]] |

兑卦在[[后天八卦]]中位于西方，是"秋"之卦，代表秋季收获、万物成熟的阶段。"说言乎兑"——万物在兑卦时最为欢悦成熟。兑卦的核心精神是**喜悦、言说、交流、收获**。

在[[先天八卦]]中，兑卦居东南（左下），数序为2。"山泽通气"——兑（泽）与艮（山）相对，泽低山高，气息相通。先天兑卦代表地球上的洼地及有水的地方。

兑卦上爻为阴，按"上爻定少"的规律，为少女（幼女）。兑卦上爻空缺，像人的口——因此兑卦与口、舌、言语有密切的关系。"兑者说也"——"兑"字通"说"（悦），既表示言说，也表示喜悦。

## 详细象意

### 人事

- **身份**：少女、幼女、歌手、演讲者、翻译、律师、教师、销售、服务员、媒人、巫师
- **性格**：活泼、开朗、善言、爱笑、受欢迎、善于交际、有艺术气质
- **负面**：口舌惹祸、多嘴多舌、巧言令色、虚伪、说谎、搬弄是非
- **职业**：歌手、主持人、销售员、律师、翻译官、教师、心理咨询师、客服、公关、导游
- **行为**：说话、唱歌、辩论、说服、谈判、交流、翻译、欢笑、庆祝
- **关系**：对应幼女、少女、口齿伶俐之人、娱乐界人士

### 地理

- **自然**：泽、湖泊、沼泽、水塘、洼地、水井、湿地
- **方向**：西方
- **城市**：水城（如绍兴、苏州）、沿海城市、西方城市
- **建筑**：剧场、歌剧院、音乐厅、KTV、饭店、酒吧、茶馆、商场
- **地形**：沼泽地带、低洼水地、盐碱地、西方
- **场所**：餐厅、食堂、厨房、会议室、讲台、舞台、接待处
- **奇门方位**：西方（酉位），兑七宫

### 身体

- **主要**：口、嘴、唇（言语饮食之出口）
- **口腔**：舌（味觉、言谈）、齿（咀嚼）、喉（发声）
- **脏腑**：肺（金对应脏腑）、气管、呼吸道
- **其他**：唾液、痰、女性生殖系统（兑为缺为阴）
- **疾病**：口腔疾病、咽喉炎、牙齿病、肺部疾病、哮喘、咳嗽、语言障碍（口吃、失声）、性病

### 性格

- **正面**：活泼开朗、能说会道、善于沟通、乐观爱笑、有魅力、交际广、受欢迎
- **负面**：口舌招祸、巧言令色、搬弄是非、大嘴巴、虚伪、说谎、轻浮
- **心理特质**：外向型人格、善于表达、重视人际关系、情绪容易外露

### 物品

- 发声物品：乐器（笛子、箫、钢琴、吉他）、麦克风、音响、喇叭
- 餐具：刀、叉、勺、碗、盘子、杯子
- 金属物品：剪刀、小刀、金属制品、金银首饰
- 缺口物品：破碗、破镜、裂缝之物
- 娱乐用品：扑克、麻将、棋牌、玩具
- 化妆品：口红、化妆品、香水（兑为悦为美丽）

### 动物

- **主要**：羊（温顺可爱）
- **其他**：鸟（尤其是会鸣叫的鸟）、鹦鹉、金鱼、兔子
- **特点**：善于发声的动物、温顺可爱的动物、有缺口特征（如兔唇）的动物

### 天时

- 秋雨、细雨、阵雨、露水
- 秋季凉爽之天气
- 阴雨天气

### 颜色

- 白色（金之色）、银色、金色

## 应用宜忌

### 宜

- **预测言说交流之事**：兑为口舌，演讲、谈判、销售、教学之事宜
- **预测喜悦娱乐之事**：兑为悦，聚会、娱乐、喜庆之事宜
- **预测少女**：看兑宫状态，判断幼女的情况
- **预测口腔喉咙**：兑为口，口腔、咽喉疾病看兑宫
- **预测婚姻**：兑为少女为喜悦，婚姻之事宜看兑宫（配合其他符号）
- **预测艺术表演**：兑有表演性质，艺术演出之事宜
- **与惊门、天柱星同宫**：惊门为凶门，天柱星为凶星，需谨慎判断
- **落兑七宫（本宫）**：得地利，力量最强
- **秋季申酉月**：金当令，兑卦最旺

### 忌

- **预测保密之事**：兑为口舌开敞，不适合保密
- **预测口舌是非**：兑宫出现凶象时容易有口舌之争
- **落离九宫（火克金）**：受制，口舌受压制
- **落震三宫、巽四宫（金克木，门迫）**：为迫，言语伤人
- **夏季巳午月（火旺金死）**：兑卦力量极弱
- **谨防口舌惹祸**：兑的负面是巧言令色、搬弄是非，要慎言
- **惊门为凶门**：兑宫惊门主惊恐、口舌是非，本身带有凶性

## 旺相休囚/落宫状态

### 季节旺衰

兑卦五行属金，其旺衰随季节变化（与乾卦同理）：

| 季节 | 月份 | 状态 | 说明 |
|------|------|------|------|
| **秋** | 申、酉月 | **旺** | 金当令，兑卦最旺 |
| **四季土月** | 辰、戌、丑、未月 | **相** | 土生金，第二大旺 |
| **冬** | 亥、子月 | **休** | 金生水，泄气 |
| **春** | 寅、卯月 | **囚** | 金克木，但木旺反侮金 |
| **夏** | 巳、午月 | **死** | 火克金，最为衰弱 |

**口诀**：兑金旺于秋，相于四季土，休于冬，囚于春，死于夏。

### 落宫状态

兑卦（或作为用神）落入九宫各宫的状态：

| 落宫 | 宫位五行 | 关系 | 状态 |
|------|---------|------|------|
| **兑七宫** | 金 | 比和，本宫 | 最旺 |
| **乾六宫** | 金 | 比和 | 旺，同气相求 |
| **坤二宫** | 土 | 宫生卦（生入） | 相，得土生 |
| **艮八宫** | 土 | 宫生卦（生入） | 相，得土生 |
| **中五宫** | 土 | 宫生卦（生入） | 相 |
| **坎一宫** | 水 | 卦生宫（生出） | 休，泄气 |
| **震三宫** | 木 | 卦克宫（迫） | 囚 |
| **巽四宫** | 木 | 卦克宫（迫） | 囚 |
| **离九宫** | 火 | 宫克卦（制） | 死，严重受克 |

### 旺衰对预测的影响

- **旺相时**：口才出众、喜悦度满、少女健康、肺气充足、交际顺利
- **休囚时**：言语不利、喜悦不足、少女小病、肺部不适
- **死时**：失声无言、口舌大祸、肺病严重、牙齿大问题
- **入本宫（兑七宫）**：无论季节，力量加成

## 开悟之门补充

### 兑卦在奇门预测中的应用技巧

1. **预测口舌是非**：兑为口舌，惊门亦主口舌官司，矛盾纠纷看兑宫
2. **预测少女**：看兑宫或用神是否临兑卦之象
3. **预测口腔疾病**：看兑宫是否有凶门凶星凶格
4. **预测演讲谈判**：兑为言说，演讲谈判看兑宫
5. **预测娱乐**：兑为喜悦，娱乐聚会看兑宫
6. **预测婚姻**：兑为少女为喜悦，结合用神判断婚姻状态

### 兑与惊门的关系

惊门居兑七宫，属金。惊门为凶门，主惊恐、口舌、官司：
- 惊门旺相（秋季）时，口舌是非多，诉讼有利（对原告方）
- 惊门休囚时，惊恐减轻
- 惊门在预测口舌是非时是必看之门

惊门虽然有凶性，但在某些特定情况下反而是"工具"——例如需要通过法律诉讼解决问题时，惊门旺相反而是利好的信号。

### 兑与天柱星的关系

天柱星为九星之一，属金，居兑七宫。天柱星主口舌、毁折、顶梁柱：
- 天柱星旺相：善于言辞、有支柱性作用；但口舌是非也重
- 天柱星休囚：口舌之力减弱
- 天柱星在预测建筑、房屋时，代表顶梁柱、支撑结构

天柱星与惊门同落兑七宫，是奇门中口舌是非最重的宫位之一——这个宫位旺相时尤其要注意说话谨慎。

### 兑卦"喜悦"的双面性

兑卦代表喜悦，但喜悦也有两面：
1. **真正的喜悦**：发自内心的快乐、人际关系的和谐
2. **表面的喜悦**：强颜欢笑、虚伪的奉承、酒肉朋友的虚假情谊

在奇门预测中，兑象出现时要学会分辨：这是真正的喜悦，还是表面的欢愉？如果是后者，往往伴随口舌是非的隐患。

### 兑与"说言乎兑"

《说卦传》曰"说言乎兑"——万物在兑卦的西方、秋季时最为成熟欢悦，因此兑卦也有"言说"的含义。在奇门预测中：
- 何时有消息：看兑宫（言说即信息）
- 何时能谈成：看兑宫（言说即谈判）
- 何时有消息传回：兑宫旺相则信息通畅

### 兑卦"上缺"的现实含义

兑卦上爻为阴爻，卦形为"兑上缺"——上方开口。这个"缺"在家宅预测中是一个重要信号：兑宫出现凶象时，往往代表房屋的缺口破损（墙裂、漏水等），或者家中有口舌纷争（口如缺口，言语漏出）。

### 兑卦与西方的地理成因

《开悟之门》解释：雨水往往从西边来（中国位于北半球中纬度，受西风带控制），西边又有青海湖等沼泽地，故将代表沼泽的兑卦放在西方。
