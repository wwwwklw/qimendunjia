# 地支六合

## 定义与基本属性

**地支六合**是十二地支之间的两两相合关系：子丑合、寅亥合、卯戌合、辰酉合、巳申合、午未合。

张大春先生认为六合可以概括以地球为核心的整个天象运行体系，直接模拟了地表气化的运行模式，具有普适性与广泛的应用价值。

## 六合的宇宙观解读

| 合 | 解读 |
|----|------|
| 子丑合 | 水土合德——海水（咸）经土地净化后成为生命之源的淡水。水气肇始于子，合于丑土。 |
| 寅亥合 | 亥水生化艮宫木气，木为生命的象征 |
| 卯戌合 | 木脱胎化火，气逐级上升 |
| 辰酉合 | 象征大气层阴压之气 |
| 巳申合 | 生克并立 |
| 午未合 | 两阳比助 |

## 奇门中六合的两种应用形式

### 同宫之合（天干与所在宫藏支）

天干落入某宫，与宫中藏支形成六合：

| 组合 | 合 | 关系 | 含义 |
|------|-----|------|------|
| 甲子戊落艮宫 | 子丑合 | 丑土制合子水 | 戊的行动受制约，谋事被他人他事牵连 |
| 甲寅癸落乾宫 | 寅亥合 | 亥寅水木相生 | 干支两情相悦，谋事有人相助。但占病需防亥水合灭寅火→心脑血管问题 |
| 甲戌己落震宫 | 卯戌合 | 卯木克合戌土 | 己受宫位制合，行动受限 |
| 甲辰壬落兑宫 | 辰酉合 | 辰酉土金相生 | 干生宫，壬落桃花、兑宫情色→有**情爱意象** |
| 甲申庚落巽宫 | 巳申合 | 相合与击刑并存 | 一般不用于同宫论断，只在研讨巽坤二宫关系中考虑 |
| 甲午辛落坤宫 | 午未合 | 两阳比助 | 午未均从对方获得阳性力量，阳性最强的一对 |

### 两宫之合（宫位间的六合）

两宫之间由于地支形成六合而关联，但**有条件**：必须**双宫同动**。

动宫条件：宫内天干相冲、天盘干引动、宫位被冲或合而发动、临马星等。

**四维宫的六合归属**：四维宫每宫含两个地支（如艮宫含丑、寅），可与两个宫形成六合。此时需**取动宫**确定与谁成合。

## 暗合

地支除明显的六合外还存在三组**暗合**：寅丑、卯申、午亥。

暗合的特点：两干处于**阴阳互补**状态。
- 寅中的甲丙戊 + 丑中的己辛癸 = 甲己合、丙辛合、戊癸合
- 申中的庚 + 卯中的乙 = 乙庚合
- 午中的丁己 + 亥中的壬甲 = 丁壬合、甲己合

> 暗合**只能作用于同一宫中**，且需要宫位逢马星或被他宫冲动引动才能构成合力。
