# 二十八宿

## 概述

二十八宿是中国古代天文学中将黄道和天赤道附近的恒星分为二十八个星区的体系。每个星宿有其对应的度数、五行属性以及与地支十二宫的对应关系。在奇门遁甲中，二十八宿用于分野判断、地理定位以及星宿度推算。

---

## 星宿过度诀

> 氐二度过卯，尾三度过寅，斗四度过丑，女二度过子，
> 危十三度过亥，奎二度过戌，胃四度过酉，毕七度过申，
> 井九度过未，柳四度过午，张十五度过巳，轸十度过辰。

周天共计三百六十五度二十五分五十杪，分布十二宫，每宫计三十度四十二分七十九杪。

## 二十八宿总度数计算

周天总计：365度25分50杪

各星宿度数总和：
- 角12 + 亢9 + 氐16 + 房5 + 心6 + 尾18 + 箕9 = 75度（东宫苍龙七宿）
- 斗23 + 牛7 + 女15 + 虚9 + 危16 + 室17 + 璧9 = 96度（北宫玄武七宿）
- 奎17.5 + 娄12 + 胃14 + 昴7 + 毕16 + 觜0.5 + 参9 = 76度（西宫白虎七宿）
- 井30 + 鬼2.5 + 柳14 + 星7 + 张18 + 翼20.17 + 轸18.5 = 110.17度（南宫朱雀七宿）

## 应用要点

1. **分野判断**：根据二十八宿确定地理方位，用于奇门遁甲中的地脉、风水判断
2. **星宿度数**：用于精确推算太阳、月亮的位置，辅助奇门排盘中的节气定位
3. **军事运用**：不同星宿临不同宫位，判断战争胜负和行军方向
4. **吉凶判断**：某些星宿有特殊的吉凶属性，遇上奇门凶格则加倍不利
