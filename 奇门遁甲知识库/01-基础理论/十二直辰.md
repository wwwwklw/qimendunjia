# 十二直辰

## 定义与基本属性

**十二直辰**即建、除、满、平、定、执、破、危、成、收、开、闭十二神。以其逐月随斗所建，分直月内日辰，故曰「直辰」。

> 出处：《御定奇门遁甲·奇门宝鉴》（金志文点校）卷一·释义四十四则 — 参见 [[御定奇门遁甲奇门宝鉴（金志文点校）]]

## 直辰的推算法

### 推算方法

如正月立春后，斗柄建寅，即用寅为月建。本月内，凡寅日，即以建加之。

### 示例：正月建寅

以建加寅日，则十二直辰分布如下：

| 直辰 | 地支 |
|------|------|
| 建 | 寅 |
| 除 | 卯 |
| 满 | 辰 |
| 平 | 巳 |
| 定 | 午 |
| 执 | 未 |
| 破 | 申 |
| 危 | 酉 |
| 成 | 戌 |
| 收 | 亥 |
| 开 | 子 |
| 闭 | 丑 |

余月类推。

## 直辰的应用

### 地四户

十二直辰中的四位用于**地四户**：

| 直辰 | 用途 |
|------|------|
| 除 | 地户 |
| 危 | 地户 |
| 定 | 地户 |
| 开 | 地户 |

> 如立春后建寅之月，则除卯、危酉、定午、开子之类也。

凡急难时，不及择奇门，以月将月建加正时，视三将四建之下，为天门地户，乘其方去，百恶不能害。

详见 [[三门四户与太冲天马]]。
