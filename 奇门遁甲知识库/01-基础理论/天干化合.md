# 天干化合

## 定义与基本属性

**天干化合**又称"天干五合"，是[[十天干]]之间的一种特殊关系。十天干两两配对相合，共组成五组合化关系，每组合化后产生一种新的五行。

> **甲己合化土、乙庚合化金、丙辛合化水、丁壬合化木、戊癸合化火**

| 天干组合 | 合化五行 | 别称 |
|----------|----------|------|
| 甲 + 己 | 土 | 中正之合 |
| 乙 + 庚 | 金 | 仁义之合 |
| 丙 + 辛 | 水 | 威制之合 |
| 丁 + 壬 | 木 | 淫慝之合（苟合） |
| 戊 + 癸 | 火 | 无情之合 |

合化的原理来源于[[河图洛书]]中的天地生成数：天一地二天三地四天五地六天七天八天九地十中，一甲（1）配六己（6）合化土，二乙（2）配七庚（7）合化金，三丙（3）配八辛（8）合化水，四丁（4）配九壬（9）合化木，五戊（5）配十癸（10）合化火。

## 详细象意

### 各组合化在奇门中的意义

| 组合 | 奇门意义 |
|------|----------|
| **乙庚合** | 乙为阴木（女）、庚为阳金（男），代表**夫妻之合**、婚姻关系。测婚恋时逢乙庚合格外重要 |
| **丁壬合** | 丁为阴火、壬为阳水，代表**苟合**、私情、非正式的两性关系 |
| **丙辛合** | 丙为阳火、辛为阴金，威制之合，主**权威与顺从的结合** |
| **戊癸合** | 戊为阳土、癸为阴水，无情之合，主**无感情基础的结合**，合作需谨慎 |
| **甲己合** | 甲为阳木、己为阴土，中正之合，主**正直诚信的结合**，事业合作吉利 |

### 合的通用象意

- 合多主吉，但并非所有合都吉利（如丁壬苟合、戊癸无情合）
- 逢合主**和解**：争端中遇合，双方有和解可能
- 逢合主**了结**：事情发展遇合，即将结束或有结果
- 逢合主**平局**：竞争、比赛逢合，难分胜负，容易打平
- 逢合主**牵扯**：测行人逢合，可能被事情牵绊不能来

## 应用宜忌

- **宜**：
  - 测婚姻遇乙庚合 → 夫妻感情好、婚姻稳定
  - 测合作遇甲己合 → 合作方信实可靠
  - 测纠纷遇合 → 有望和解

- **忌**：
  - 测婚姻遇丁壬合 → 可能有第三者或私情
  - 测合作遇戊癸合 → 合作方可能无诚意
  - 测出行遇合 → 可能被事情牵绊，走不了

### 地支化合（金函玉镜独有）

金函玉镜同时记录了**地支六合化五行**的规则：

| 地支合 | 化五行 | 合称 | 说明 |
|--------|--------|------|------|
| 子丑合 | 化土 | — | 阳水（子）合阴土（丑），化成土 |
| 寅亥合 | 化木 | — | 阳木（寅）合阴水（亥），化成木 |
| 辰酉合 | 化金 | — | 阳土（辰）合阴金（酉），化成金 |
| 巳申合 | 化水 | — | 阴火（巳）合阳金（申），化成水 |
| 卯戌合 | 化火 | — | 阴木（卯）合阳土（戌），化成火 |
| **午未合** | **合而不化** | — | **阳火（午）合阴土（未），但合而不化五行** |

### "午未合而不化"详解

金函玉镜中关于地支化合最独特的说法是"**午未合而不化**"——这在通行版的化合理论中极为罕见。

| 说法 | 通行版 | 金函玉镜版 |
|------|--------|-----------|
| 子丑 | 子丑合土 | 子丑化土（一致） |
| 寅亥 | 寅亥合木 | 寅亥化木（一致） |
| 辰酉 | 辰酉合金 | 辰酉化金（一致） |
| 巳申 | 巳申合水 | 巳申化水（一致） |
| 卯戌 | 卯戌合火 | 卯戌化火（一致） |
| **午未** | **午未合土/火**（诸说不一） | **午未合而不化**（独有） |

#### "合而不化"的含义

"合而不化"指两方虽然相合，但不会化出新的五行，即：
- **合**：午和未仍然是合的关系（吸引力、配合度仍在）
- **不化**：不会产生第三种五行（土或火），保持各自原有属性

这对奇门判断的意义：
- 午未合不化土 → 测合作时午未日的合不能视为"化成新局"
- 保持原有火土属性 → 午火、未土各自独立判断
- 在五行生克制化中，午未合的力量较其他五组合弱

#### 其他五组合化的五行规律

| 合 | 化出 | 原因 |
|----|------|------|
| 子丑→土 | 水+土，得土之气 |
| 寅亥→木 | 木+水，得木之气 |
| 辰酉→金 | 土+金，得金之气 |
| 巳申→水 | 火+金，得水之气 |
| 卯戌→火 | 木+土，得火之气 |
| 午未→不化 | 火+土，二气相争，不能成局 |

> 午为阳火极盛之位，未为燥土含火，两者皆为火土相杂之气。火旺时土随火走，土旺时火被土晦，故不化新五行。

### 天干化合与地支化合的配合

金函玉镜将天干化合与地支化合并列，暗示使用时需**天干化合与地支化合配合判断**：

- 天干化合看**天时**（大环境趋势）
- 地支化合看**地利**（具体环境和条件）
- 两者俱合时，化合之力最强；仅一合时，力量减半

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]干支合会章

### 奇门五合的特殊性——宫位决定合的真实性

张大春先生指出，两干之间是否能够形成事实上的相合关系，**必须考虑两干所在宫位的五行生克**。这是奇门与四柱的根本区别。五合在奇门中可分为三种类型：

**类型一：合力**（天干相合，宫位印比）
> 天干和宫位力量相向而行，构成事实上最强的合力。

**类型二：虚合**（天干相合，宫位冲克）
> 天干和宫位力量正好相反，宫位力量大于天干，能抵消天干的合力。"水花镜月"——合意不足，有名无实。如三国中的刘关之情虽深，但因处于对立阵营（宫位相克），无法形成事实上的合力。

**类型三：暗合**（天干相合，宫位表面相克但有暗助）
> 两宫在地支上构成六合关系，或形成先后天关系 → **明克暗合**。克中有合的特殊情形。在夫妻关系预测中，一方霸道压制另一方，但家庭关系却可能一直稳固——不能仅从表面不平等关系断定存在问题。

### 六亲定位问题

奇门通过**年、月、日、时四干取六亲、十神**的办法不能轻易被废弃。仅从天干角度或宫位生克关系来划定六亲十神都具片面性——原因在于把四柱与六爻中的原则直接套用在奇门中，没有观测到奇门选取用神的特殊性。

### 甲己合的奇门特殊性

奇门预测体系中，五合中的甲己相合不会直接出现。因为甲为值符潜藏在六仪之中，如果值符为甲戌己，则己本身同时具有甲、己两干的性质，无法形成正常意义的五合。
