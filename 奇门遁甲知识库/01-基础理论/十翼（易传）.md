# 十翼（易传）

## 定义与基本属性

**十翼**，又称**易传**，是解释《[[奇门遁甲概述|周易]]》的十篇文章的总称，相传为孔子及其弟子所作。因为十篇文章像十只翅膀辅翼着《易经》的经义，所以称为"十翼"。

十翼后来被纳入[[奇门遁甲概述|周易]]，成为其不可分割的组成部分，使得《周易》从一部单纯的占筮之书升华为一部哲学经典。

**十翼的十篇文章**：

| 篇名 | 分类 | 说明 |
|------|------|------|
| 《彖传（上）》 | 彖传 | 解释六十四卦卦辞 |
| 《彖传（下）》 | 彖传 | 同上，按卦分上下 |
| 《象传（上）》 | 象传 | 解释卦象和爻象，分"大象"（释全卦）和"小象"（释爻辞） |
| 《象传（下）》 | 象传 | 同上 |
| 《系辞（上）》 | 系辞 | 通论易经大义，哲学价值最高 |
| 《系辞（下）》 | 系辞 | 同上 |
| 《文言》 | 文言 | 专释乾卦和坤卦 |
| 《说卦》 | 说卦 | 解释八卦的取象原理 |
| 《序卦》 | 序卦 | 解释六十四卦排列次序 |
| 《杂卦》 | 杂卦 | 以错杂方式解释各卦大义 |

通俗理解：如果说《易经》原文是一部"说明书"（告诉你卦爻辞是什么），那"十翼"就是"操作手册的解读本"（告诉你为什么这样写，背后有什么深意）。

## 详细象意

- **"推天道以明人事"**：这是十翼最核心的哲学贡献。易经本是占卜工具，十翼将其提升为"通过了解自然规律来指导人的行为"的哲学体系
- **[[太极阴阳论]]的源头**："易有太极，是生两仪，两仪生四象，四象生八卦"（出自《系辞》），成为后世太极阴阳思想的核心论述
- **人文精神的注入**：十翼强调人的道德修养和主观能动性，如"天行健，君子以自强不息""地势坤，君子以厚德载物"
- **象数思维的奠基**：《说卦》系统归纳了八卦的取象，《系辞》阐述了数与理的关系

## 应用宜忌

- **宜**：学习十翼是深入理解易经和[[六十四卦]]的必经之路，尤其《系辞》和《说卦》最为基础
- **宜**：将十翼中的哲学思想与奇门遁甲的具体技法结合理解，体会"道"（原理）与"术"（方法）的统一
- **忌**：只把它当哲学读而忘了它与占筮的关系——十翼本身就是解释占筮原理的

## 与其他概念的关系

- [[六十四卦]]：十翼的核心内容就是解释六十四卦
- [[连山易与归藏易]]：连山、归藏失传，十翼是专为周易而作
- [[两派六宗]]：十翼中的哲学论述是后世"义理派"的理论源头；象数论述是"象数派"的依据
- [[简易变易不易]]：十翼中"易"的多重含义（变易、不易、简易）在《系辞》中有系统表述
