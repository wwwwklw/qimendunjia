# 天将十二神

## 定义与基本属性

**天将**是奇门遁甲中十二位贵神的统称，应乎天干，故曰「天将」，即**日将**也。十二神分别为：贵人、螣蛇、朱雀、六合、勾陈、青龙、天空、白虎、太常、元武、太阴、天后。

> 出处：《御定奇门遁甲·奇门宝鉴》（金志文点校）卷一·释义四十四则 — 参见 [[御定奇门遁甲奇门宝鉴（金志文）]]

## 求将之诀

### 总诀

> 始子申，终未丑。避魁罡，不冲首。干取合，用惟九。阴阳分，顺逆数。

### 纳甲法诀

> 乾宫甲子申，支分干不分。此是纳甲法，支始乾，干终坤。日干寻纳合，支上贵人生。

## 纳甲之法

以乾、坤、艮、兑、坎、离、震、巽环列，以十干甲起于乾，一卦一干，纳至辛，止于巽。又纳壬于乾、癸于坤。

| 八卦 | 所纳干 |
|------|--------|
| 乾 | 甲、壬 |
| 坤 | 乙、癸 |
| 艮 | 丙 |
| 兑 | 丁 |
| 坎 | 戊 |
| 离 | 己 |
| 震 | 庚 |
| 巽 | 辛 |

## 阳贵与阴贵

### 阳贵入宫（冬至以后用）

阳贵支以**子**为首。去午（冲子）、辰（罡）、戌（魁）三支不用，余九支。

阳贵入宫定卦：
| 八卦 | 所纳 |
|------|------|
| 乾 | 壬亥、甲子 |
| 坤 | 乙癸丑 |
| 艮 | 丙寅 |
| 兑 | 丁卯 |
| 坎 | 戊己 |
| 离 | 己未 |
| 震 | 庚申 |
| 巽 | 辛酉 |

**阳贵起法**：以日干寻纳甲合神，本日贵人即起于合神所纳之支。
- 甲日：甲与己合，己纳于离，未纳于己，贵人起于**未**
- 戊日：戊与癸合，癸纳于坤，丑纳于乙癸，贵人起于**丑**
- 庚日：庚与乙合，乙纳于坤，贵人起于**丑**

### 阴贵入宫（夏至以后用）

阴贵支以**申**为首。去寅（冲申）、戌（魁）、辰（罡）三支不用，余九支。

阴贵入宫定卦：
| 八卦 | 所纳 |
|------|------|
| 乾 | 壬酉、甲申 |
| 坤 | 乙癸未 |
| 艮 | 丙午 |
| 兑 | 丁巳 |
| 坎 | 戊卯 |
| 离 | 己丑 |
| 震 | 庚子 |
| 巽 | 辛亥 |

**阴贵起法**：以日干寻纳甲合神，本日贵人即起于合神所纳之支。
- 甲日：甲与己合，己纳于离，丑纳于己，贵人起于**丑**
- 戊日：戊与癸合，癸纳于坤，未纳于乙癸，贵人起于**未**
- 庚日：庚与乙合，乙纳于坤，贵人起于**未**

## 阳贵阴贵立成歌

### 阳贵立成歌

> 甲羊戊庚牛，己鼠乙寻猴。丙鸡丁猪位，壬兔癸蛇游。辛从虎上起，阳贵顺相求。

### 阴贵立成歌

> 甲牛戊庚羊，乙鼠己寻猴。丙猪丁鸡位，壬蛇癸兔游。辛从马上起，阴贵逆推求。

## 阴贵阳贵的本义

### 为何以子、申为首

- 先天坤在子，后天坤在申，故用子、申为首
- 从坤不从乾：先阴后阳之理
- 阴极则阳生，阳生而万物生，故从坤则机生
- 阳极则阴生，阴生而万物息，故从乾则机息

> 「不观儿在母腹乎？母呼亦呼，母吸亦吸，寂然不动者，息机于阴也。及其生也，叻的一声，则机生于阳矣，故曰『天地再造，必先有阴而后有阳也』。」

### 何以独称纳甲

乾纳甲壬二干，阴阳贵人支首皆纳于乾之甲下。恐误纳于壬，故称纳甲以儆之。
