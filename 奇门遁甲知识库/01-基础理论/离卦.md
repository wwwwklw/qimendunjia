# 离卦（离为火）

## 定义与基本属性

离卦，卦符为 ☲，口诀"**离中虚**"——两阳爻在外，一阴爻在中，像火焰外焰明亮而中心空虚。离卦代表火、中女、明、丽、文采，是八卦中最具光明灿烂、美丽华彩特性的卦象。

| 属性 | 内容 |
|------|------|
| **卦名** | 离 |
| **卦符** | ☲ |
| **口诀** | 离中虚 |
| **代表** | 火 |
| **五行** | 火 |
| **阴阳** | 阴卦（一阴二阳，以少爻定阴阳） |
| **先天卦数** | 3 |
| **后天卦数** | 9 |
| **后天方位** | 南 |
| **对应宫位** | [[离九宫]] |
| **家庭成员** | 中女（次女） |
| **身体部位** | 目、心、小肠 |
| **季节** | 夏（巳午月，尤其午月） |
| **时辰** | 午时（11:00-13:00） |
| **地支** | 午 |
| **对应奇门符号** | [[景门]]、[[天英星]] |

离卦在[[后天八卦]]中位于南方，是"夏"之卦，代表夏季炎热、万物繁盛、光彩夺目的阶段。"相见乎离"——万物在离卦时最为明显可见。离卦的核心精神是**光明、美丽、附丽、文明**。

在[[先天八卦]]中，离卦居东（左），数序为3。"水火不相射"——离（火）与坎（水）相对，火炎上水润下，各安其位。先天离卦代表太阳从东边升起。

离卦中爻为阴，按"中爻定中"的规律，为中女（次女）。离为火，火必须附着在可燃物上才能燃烧，因此离卦有"附丽"（依附、附着）的含义——美丽的东西往往需要依附于某个载体。

## 详细象意

### 人事

- **身份**：中女、次女、艺术家、美容师、设计师、明星、模特、文人、记者、教师
- **性格**：热情、明朗、美丽、文雅、善于表达、有魅力、有艺术气质
- **负面**：虚荣、浮夸、好面子、情绪化、急躁、容易上火、华而不实
- **职业**：艺术家、设计师、美容美发、演艺人员、主持人、教师、作家、摄影师、摄像师
- **行为**：显示、表现、照亮、照亮、美容、化妆、写作、绘画、摄影
- **关系**：对应次女、中女、文艺界人士、美人

### 地理

- **自然**：火、太阳、电、光、热
- **方向**：南方
- **城市**：南方城市、热带地区、火炉城市
- **建筑**：火电厂、冶炼厂、灯光设施、灯塔、剧院、影院、画廊、美容院
- **地形**：干燥炎热之地、火山地带
- **场所**：厨房、炉灶间、舞台、展厅、化工厂、美容院
- **奇门方位**：南方（午位），离九宫

### 身体

- **主要**：目（眼睛，火照万物如目视万物）
- **脏腑**：心（主血脉、主神志）、小肠（与心相表里）
- **其他**：血液（属火）、舌（心开窍于舌）、面部
- **疾病**：眼疾（近视、老花、白内障）、心脏病、高血压、中风、发烧、上火、炎症、血液病

### 性格

- **正面**：热情开朗、聪明伶俐、文采斐然、有艺术修养、美丽大方、善于交际
- **负面**：虚荣浮躁、好面子、情绪化、急躁易怒、华而不实、容易上火
- **心理特质**：感性思维为主、重视外表形象、情绪波动大、追求美和认可

### 物品

- 火源：炉灶、打火机、火柴、蜡烛、火把
- 照明：灯、灯泡、灯笼、手电筒、屏幕、显示器
- 电器：电视、电脑、手机、相机、摄像机
- 文书：书籍、字画、文凭、证书、文件、信函
- 美术品：画作、雕塑、装饰品、艺术品
- 镜类：镜子、玻璃、晶体、透镜、眼镜
- 红色物品：红衣、红布、红纸、红花

### 动物

- **主要**：雉（野鸡/孔雀，羽毛美丽）
- **其他**：凤凰（神鸟，光明灿烂）、金鱼（色彩艳丽）、蝴蝶、萤火虫
- **特点**：毛色鲜艳美丽的动物、会发光的动物、鸟类

### 天时

- 烈日、酷暑、干旱、晴天
- 夏季炎热之天气
- 闪电、火山爆发

### 颜色

- 红色（火之色）、紫色、橙色、彩色、鲜艳的颜色

## 应用宜忌

### 宜

- **预测文化艺术之事**：离为文采和美丽，艺考、创作、展示之事宜
- **预测美容美发**：离主美丽，与外表相关的事宜宜
- **预测中女**：看离宫状态，判断次女的情况
- **预测眼睛**：离为目，眼疾之事看离宫
- **预测光明之事**：离为明，求曝光、上镜、出名之事宜
- **与景门、天英星同宫**：同气相应，但景门为小凶门
- **落离九宫（本宫）**：得地利，力量最强
- **夏季巳午月**：火当令，离卦最旺

### 忌

- **预测冷静沉稳之事**：离为火热，过于热情冲动之事不吉
- **预测心血管疾病**：离宫有凶象时要注意心脏问题
- **落坎一宫（水克火）**：受制严重，光明被遮蔽
- **落乾六宫、兑七宫（火克金，门迫）**：为迫，过于强势
- **冬季亥子月（水旺火死）**：离卦力量最弱
- **过于虚荣浮夸**：离的负面是华而不实

## 旺相休囚/落宫状态

### 季节旺衰

离卦五行属火，其旺衰随季节变化：

| 季节 | 月份 | 状态 | 说明 |
|------|------|------|------|
| **夏** | 巳、午月 | **旺** | 火当令，离卦最旺，光明灿烂 |
| **春** | 寅、卯月 | **相** | 木生火，第二大旺，得生扶 |
| **四季土月** | 辰、戌、丑、未月 | **休** | 火生土，泄气，力量减弱 |
| **秋** | 申、酉月 | **囚** | 火克金，但金旺反侮火，受制 |
| **冬** | 亥、子月 | **死** | 水克火，被当令之水所克，最弱 |

**口诀**：离火旺于夏，相于春，休于四季土，囚于秋，死于冬。

### 落宫状态

离卦（或作为用神）落入九宫各宫的状态：

| 落宫 | 宫位五行 | 关系 | 状态 |
|------|---------|------|------|
| **离九宫** | 火 | 比和，本宫 | 最旺，得地利 |
| **震三宫** | 木 | 宫生卦（生入） | 相，得木生 |
| **巽四宫** | 木 | 宫生卦（生入） | 相，得木生 |
| **坤二宫** | 土 | 卦生宫（生出） | 休，泄气 |
| **艮八宫** | 土 | 卦生宫（生出） | 休，泄气 |
| **中五宫** | 土 | 卦生宫（生出） | 休 |
| **兑七宫** | 金 | 卦克宫（迫） | 囚 |
| **乾六宫** | 金 | 卦克宫（迫） | 囚 |
| **坎一宫** | 水 | 宫克卦（制） | 死，严重受克 |

### 旺衰对预测的影响

- **旺相时**：美丽动人、才华横溢、眼睛明亮、事业光彩、中女有为、心脏健康
- **休囚时**：光彩不再、文采不济、中女平凡、眼睛疲劳
- **死时**：完全暗淡、心脏病严重、眼睛疾病、事业灰暗
- **入本宫（离九宫）**：无论季节，力量加成

## 与其他概念的关系

- [[先天八卦]]：先天离卦数序为3，居东，为太阳。"水火不相射"——离与坎相对
- [[后天八卦]]：后天离卦数序为9，居南，为夏。南方炎热，故将离火放在南边
- [[八卦概述]]：离为中女、为火、为明、为丽
- [[九宫]]：对应离九宫，地支为午，数序为9
- [[五行]]：属火，火性炎上光明，与离卦光明美丽之性相通
- [[阴阳]]：阴卦（一阴在中），以少爻定阴阳
- [[景门]]：离卦对应景门，景门属火，主文书、信息、道路、血光。景门为小凶门，但适合上书、通讯、旅游
- [[天英星]]：离卦对应天英星，天英星属火，为小凶星，主血光、火灾、声音。但天英星也有光明美丽的一面
- [[十二地支]]：离宫寄午支，午为火之正位
- [[十天干]]：离属火，对应丙火（阳火）之性，为太阳之火
- [[六十四卦]]：离为上卦、下卦形成离宫八卦（离为火、火山旅、火风鼎、火水未济、山水蒙、风水涣、天水讼、天火同人）
- [[时空统一论]]：离卦位于南方、夏季、午时（正午时分）
- [[河图洛书]]：洛书九宫中离九宫之数为9（戴九）
- [[三才]]：离在"人"的层面代表中女、美丽之人、文艺人士
- [[奇门遁甲概述]]：离宫是关键宫位，景门和天英星的本位

## 开悟之门补充

### 离卦在奇门预测中的应用技巧

1. **预测美丽美容**：看离宫状态，离主美丽外表
2. **预测中女**：看离宫或用神是否临离卦之象
3. **预测眼睛疾病**：看离宫是否有凶门凶星凶格
4. **预测考试文书**：离为文采，与巽宫天辅星配合判断学业
5. **预测火灾**：离为火，火灾之事主要看离宫
6. **预测心脏疾病**：离为心，心的疾病看离宫

### 离与景门的关系

景门居离九宫，属火。景门为小凶门，主文书、信息、道路、血光：
- 景门旺相（夏季）时，利于文化传播、旅游出行
- 景门休囚时，凶性减弱
- 景门与天英星同宫——火上加火，光明加倍但也容易过火

景门虽然是凶门，但有凶中有吉的特征——在预测文书、考试、信息传播时，景门反而是必要的"曝光之门"。

### 离与天英星的关系

天英星为九星之一，属火，居离九宫。天英星主血光、火灾、声音：
- 天英星旺相：光明美丽，适合演艺、音乐、艺术
- 天英星休囚：凶性显现，容易有血光、火灾之灾
- 天英星在预测性格时，代表热情外向、善于表现的人

### 离卦"中虚"的现实含义

离卦卦形为"离中虚"——外阳内阴，外实内虚。这个结构有两层含义：
1. **火的特征**：火焰外焰温度最高（阳），焰心温度反而较低（阴），与"外阳内阴"的卦象吻合
2. **人的认识**：美丽的外表下往往有虚空的内在——"虚"字提醒人们不要只看表面

在奇门预测中，离象出现时要注意：一个人可能外表光鲜但内心空虚，或者一件事表面看起来热烈但实际基础不牢。

### 离卦与"相见乎离"

《说卦传》曰"相见乎离"——万物在离卦的南方、夏季时最为茂盛可见。因此离卦也有"显现、面见、展示"的含义。预测何时能见到某人、某事物何时显现时，离宫的状态是一个重要参考。
