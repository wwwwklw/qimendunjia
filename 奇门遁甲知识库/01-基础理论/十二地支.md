# 十二地支

## 定义与基本属性

十二地支是：**子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥**。

"支"即树枝，地支是与天干相对的概念，表示地球自身的运行节律和对万物的影响，偏重于"地"的方面。地支在天干之下，展开为十二种时空状态。

十二地支配合十天干，构成了六十甲子纪时体系，同时地支也是空间方位的代名词。

| 午 | 火 | 阳 | 正旺之火，炎烈当空 |

**五行归属口诀**：亥子水、寅卯木、巳午火、申酉金、辰戌丑未土。

**阴阳归属**：子、寅、辰、午、申、戌为阳支（奇数位）；丑、卯、巳、未、酉、亥为阴支（偶数位）。

## 详细象意

- 地支象意（后续会有专门的扩展笔记，此处为基本象意）
- **子**：水、北方、冬季、鼠、智慧、隐秘、寒冷
- **丑**：湿土、东北偏北、晚冬、牛、稳重、缓慢、田宅
- **寅**：木、东北偏东、初春、虎、权威、勇敢、开始
- **卯**：木、东方、仲春、兔、柔顺、敏捷、门户
- **辰**：湿土、东南偏东、晚春、龙、变化、争斗、水库
- **巳**：火、东南偏南、初夏、蛇、灵活、变化、文书
- **午**：火、南方、仲夏、马、光明、热烈、礼仪
- **未**：燥土、西南偏南、晚夏、羊、温和、稳定、木库
- **申**：金、西南偏西、初秋、猴、好动、灵活、传递
- **酉**：金、西方、仲秋、鸡、精致、美妙、言说
- **戌**：燥土、西北偏西、晚秋、狗、忠诚、守护、火库
- **亥**：水、西北偏北、初冬、猪、懒散、享受、智慧

### 辰戌丑未四库

辰、戌、丑、未四个地支不仅是土，还各是四行（除土以外）的"库"：
- **辰**为水库（藏癸水）
- **戌**为火库（藏丁火）
- **丑**为金库（藏辛金）
- **未**为木库（藏乙木）

四库在奇门遁甲中非常重要，库为收藏、潜伏之地，入墓入库有吉凶不同的含义。

## 应用宜忌

- **宜**：熟悉地支的五行、时辰、方位和生肖对应，这是排盘和判断的基本功
- **忌**：混淆阴阳地支在不同盘局（阳遁/阴遁）中的顺逆作用

在奇门遁甲排盘中，地支主要用于确定时辰、旬首、空亡等关键信息。

## 旺相休囚/落宫状态

地支与季节的关系决定其旺衰：
- **春**：寅卯木旺、巳午火相、其余依此类推
- **夏**：巳午火旺
- **秋**：申酉金旺
- **冬**：亥子水旺
- **四季末**（每个季节的最后十八天）：辰戌丑未土旺

具体一个地支落在奇门九宫的哪个宫位，与该宫五行的关系决定其落宫状态。

## 与其他概念的关系

- [[十天干]]：天干为天、地支为地，干支配合构成六十甲子，是奇门排盘的根本
- [[五行]]：地支各有五行属性，是五行在时空中的具体化
- [[阴阳]]：地支分阴阳，奇数为阳支、偶数为阴支
- [[六十甲子]]：十天干与十二地支组合产生六十甲子
- [[九宫]]：十二地支分布在九宫之中（每宫一个或多个地支）
- [[后天八卦]]：后天八卦的方位与地支的方位有对应关系
- [[时空统一论]]：地支既是时间单位（月、时）又是空间方位，是时空统一的典型

## 开悟之门补充

### 十二地支与九宫固定对应（流年流月流日流时判断法）

十二地支与奇门九宫格的对应关系是**固定不变**的：

| 地支 | 对应宫位 | 备注 |
|------|---------|------|
| 子 | 坎1宫 | 正北 |
| 丑、寅 | 艮8宫 | 东北 |
| 卯 | 震3宫 | 正东 |
| 辰、巳 | 巽4宫 | 东南 |
| 午 | 离9宫 | 正南 |
| 未、申 | 坤2宫 | 西南 |
| 酉 | 兑7宫 | 正西 |
| 戌、亥 | 乾6宫 | 西北 |

这种固定对应体现了地支"属静、属阴"的特性。但地支同时又是永远按顺时针方向流动的，它象时间本身一样永远向前流动，不可能倒退。

**流年流月流日流时判断法**：

正因为地支与九宫的对应关系是固定的，所以奇门在判断流年、流月、流日、流时上，从地支角度就可以找到它相对应的宫位来判断：

- **流年判断**：如1996年（丙子年），子年对应坎宫 → 观察坎宫情况。若坎宫临[[惊门]]，主官司是非，则可断该人1996年有官司是非发生过。
- **流月判断**：如农历八月 → 看兑宫。
- **流日判断**：如辰日 → 看巽宫。
- **流时判断**：如戌时 → 看乾宫。

> **实战案例**：《神奇之门》实例——甲午旬中辰巳空，地支辰巳在巽4宫，临生门，生门填实之日冲开死门就可出院。查万年历近期5月19日丙辰日、20日丁巳日，故断5月19日或20日出院。

> **另一案例**：测外籍医生四柱——1967年丁未、1968年戊申，地支未申所在坤2宫临庚（为其丈夫），但逢空亡，填实之年自然就可能是结婚之年了。

### 地支的阴阳五行属性

十二地支按顺序奇偶分阴阳：

| 地支 | 五行 | 阴阳 | 说明 |
|------|------|------|------|
| 子 | 水 | 阳 | 子为1数（奇数），故为阳水 |
| 丑 | 土 | 阴 | 丑为2数（偶数），故为阴土 |
| 寅 | 木 | 阳 | 寅为3数（奇数），故为阳木 |
| 卯 | 木 | 阴 | 卯为4数（偶数），故为阴木 |
| 辰 | 土 | 阳 | 辰为5数（奇数），故为阳土 |
| 巳 | 火 | 阴 | 巳为6数（偶数），故为阴火 |
| 午 | 火 | 阳 | 午为7数（奇数），故为阳火 |
| 未 | 土 | 阴 | 未为8数（偶数），故为阴土 |
| 申 | 金 | 阳 | 申为9数（奇数），故为阳金 |
| 酉 | 金 | 阴 | 酉为10数（偶数），故为阴金 |
| 戌 | 土 | 阳 | 戌为11数（奇数），故为阳土 |
| 亥 | 水 | 阴 | 亥为12数（偶数），故为阴水 |

也可按农历月份奇偶来分，与上述结果一致。

### 地支相合

按照异性相吸的原则，十二地支形成**六对相合**的关系：

| 合 | 五行关系 | 合中关系 | 趋势 |
|----|---------|---------|------|
| 子丑合 | 阳水+阴土 | 丑土克子水 | 先合后分 |
| 寅亥合 | 阳木+阴水 | 亥水生寅木 | 越合越好 |
| 卯戌合 | 阴木+阳土 | 卯木克戌土 | 先合后分 |
| 辰酉合 | 阳土+阴金 | 辰土生酉金 | 越合越紧密 |
| 巳申合 | 阴火+阳金 | 巳火克申金 | 先合后分 |
| 午未合 | 阳火+阴土 | 午火生未土 | 越合越好 |

> 三个阳性也可以相合。申子辰合水局、寅午戌合火局；三个阴性亥卯未合木局、巳酉丑合金局。

金函玉镜在十二支神将地盘之前，已有[[十二神应验要说]]记录十二神临门时的应验。此处的十二支神将地盘是十二神体系的完整版，将地支、天将（神）、月将（将）三者合一。
