# 阴阳

## 定义与基本属性

阴阳是中国古代哲学中最基本、最核心的概念，是易学思维的基石。阴阳最初指的是日光的向背：向着太阳的一面为"阳"，背着太阳的一面为"阴"。后来抽象为哲学概念，用以概括宇宙万物中两种对立统一的属性。

阴和阳不是绝对的，而是相对的。同一事物在不同的参照系中，可能是阴也可能是阳。例如，白天的太阳是阳，晚上的月亮是阴；但在晚上，月为阳、暗夜为阴。

阴阳的基本规律是：**孤阴不生，独阳不长**。阴和阳必须相互配合、相互依存，才能生生不息。

## 基本分类

书中对阴阳的基本分类（也是最早的分类依据）：

| 阳 | 阴 |
|----|----|
| 天 | 地 |
| 日（太阳） | 月（月亮） |
| 男 | 女 |
| 动 | 静 |
| 刚 | 柔 |
| 奇数（一三五七九） | 偶数（二四六八十） |
| 上 | 下 |
| 外 | 内 |
| 热 | 冷 |
| 明 | 暗 |
| 昼 | 夜 |
| 春夏 | 秋冬 |
| 进 | 退 |
| 发散 | 收敛 |
| 上升 | 下降 |

## 详细象意

- **阳**：积极、主动、向外、向上、刚健、明亮、温暖、扩张、领导、开创
- **阴**：消极、被动、向内、向下、柔顺、黑暗、寒冷、收缩、跟随、守成

在实际应用中，要根据具体情况灵活运用阴阳分类，一个事物到底属阴还是属阳，要看参照物和讨论的语境。

## 应用宜忌

- **宜**：在任何判断中首先分清阴阳，这是奇门遁甲分析的第一步
- **忌**：僵化地理解阴阳，认为某事物永远是阳或永远是阴

在奇门遁甲中，阴阳无处不在：天干分阴阳（甲丙戊庚壬为阳，乙丁己辛癸为阴）、地支分阴阳（子寅辰午申戌为阳，丑卯巳未酉亥为阴）、八卦分阴阳（乾震坎艮为阳，坤巽离兑为阴）、八门分阴阳、九星分阴阳。掌握阴阳是学习和运用奇门遁甲的基本功。

## 旺相休囚/落宫状态

阴阳与旺衰状态密切相关：
- 阳生阴死：阳气生发的季节，万物生长（春夏季），为旺相之时
- 阴盛阳衰：阴气盛行的季节，万物收藏（秋冬季），为休囚之时
- 具体到宫位：阳遁和阴遁的区别，阳遁顺排、阴遁逆排，都源于阴阳的消长规律

## 与其他概念的关系

- [[太极阴阳论]]：太极生两仪，阴阳是由太极分化而来的，太极阴阳论是阴阳概念的总论
- [[五行]]：五行是阴阳的进一步分化，木火为阳（上升之气），金水为阴（下降之气），土居中调和
- [[十天干]]：十天干分阴阳五行，是阴阳在时间轴上的具体编码
- [[十二地支]]：十二地支分阴阳五行，是阴阳在时空中的具体体现
- [[先天八卦]]：先天八卦以阴阳爻的组合构成，乾为纯阳，坤为纯阴
- [[天人合一论]]：天地人各有阴阳，天地人三才统一也就是阴阳的和谐统一
- [[穷通分合论]]：泰卦与否卦的区别在于阴阳是否相交，穷通本质上取决于阴阳运行的状态
