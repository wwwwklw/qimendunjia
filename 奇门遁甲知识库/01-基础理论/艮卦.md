# 艮卦（艮为山）

## 定义与基本属性

艮卦，卦符为 ☶，口诀"**艮覆碗**"——一阳爻在上，两阴爻在下，像倒扣的碗。艮卦代表山、少男、止、阻、坚守，是八卦中最具稳固不动、坚毅忍耐特性的卦象。

|------|------|
| **卦名** | 艮 |
| **卦符** | ☶ |
| **口诀** | 艮覆碗 |
| **代表** | 山 |
| **五行** | 土（阳土） |
| **阴阳** | 阳卦（一阳二阴，以少爻定阴阳） |
| **先天卦数** | 7 |
| **后天卦数** | 8 |
| **后天方位** | 东北 |
| **对应宫位** | [[艮八宫]] |
| **家庭成员** | 少男（幼子） |
| **身体部位** | 手、鼻、背、骨节、脾胃 |

| **地支** | 丑、寅 |
| **对应奇门符号** | [[生门]]、[[天任星]] |

艮卦在[[后天八卦]]中位于东北方，是"冬春之交"的卦，代表冬季将尽、春天将至的转折阶段。"终万物始万物者莫盛乎艮"——万物在艮卦处终结旧循环、开启新循环。艮卦的核心精神是**止、阻、坚守、稳固**。

在[[先天八卦]]中，艮卦居西北（右上），数序为7。"山泽通气"——艮（山）与兑（泽）相对，山高泽低，气息相通。先天艮卦代表地球上突起的丘陵和高山。

艮卦上爻为阳，按"上爻定少"的规律，为少男（幼子）。"艮者止也"——山在高处停止不动，因此艮卦的最大特点是"止"：停止、阻止、静止。但这个"止"不是消极的停止，而是积蓄力量后重新出发的"止"。

## 详细象意

### 人事

- **身份**：少男、幼子、少年、学生、修行人、守门人、保安、隐居者
- **性格**：稳重、踏实、谨慎、有耐心、有定力、坚韧不拔、保守
- **负面**：固执、保守、封闭、顽固不化、不开窍、死板、迟钝
- **职业**：保安、门卫、仓库管理员、档案管理员、考古人员、地质人员、矿工、学生
- **行为**：停止、阻止、坚守、守候、隐居、储蓄、积蓄
- **关系**：对应幼子、少男、晚辈、少年儿童

### 地理

- **自然**：山、丘陵、高地、石头、岩石、泥土
- **方向**：东北方
- **城市**：山城、东北城市
- **建筑**：城墙、堡垒、仓库、监狱、寺庙、道观、钟楼、门楼
- **地形**：山地、丘陵地带、高原、石山、东北方
- **场所**：银行、保险柜、图书馆、档案馆、仓库、山地
- **奇门方位**：东北方（丑寅位），艮八宫

### 身体

- **主要**：手（止物之用）、手指
- **五官**：鼻（面之山，位于面部最高处）
- **骨骼**：背、脊柱、骨节、关节
- **脏腑**：脾胃（土属），但有厚土之性
- **其他**：左足（洛书八为左足）
- **疾病**：手伤、手指疾病、鼻病、背部疾病、关节疼痛、脾胃不适、肿块、结节、增生

### 性格

- **正面**：稳重大方、诚实可靠、有原则、坚守岗位、有定力、有耐心、谨慎细心
- **负面**：固执己见、顽固不化、死板教条、保守封闭、不善变通、反应迟钝
- **心理特质**：理性而固执、不易改变、重视原则和纪律、有储蓄倾向

### 物品

- 石制品：石头、玉石、石碑、石雕、石器
- 泥土制品：陶器、瓷器、瓦罐、砖块
- 家具：桌子、柜子、箱子、保险柜
- 门类：门、门槛、门闩、锁、钥匙
- 防御物：盾牌、墙壁、栅栏、堡垒
- 储存物：储蓄罐、存钱箱、钱袋、包裹
- 山形物品：假山、石山盆景

### 动物

- **主要**：狗（看家护院、忠于职守）
- **其他**：虎（百兽之王，山中霸主）、鼠（钻洞掘土）、熊、牛
- **特点**：有守卫性的动物、挖洞打洞的动物、山地动物、忠实的动物

### 天时

- 雾气、阴霾、多云
- 冬春之交的天气
- 干燥、不见天日之天气

### 颜色

- 黄色（土之色）、棕色、咖啡色、褐色

## 应用宜忌

### 宜

- **预测留守等待之事**：艮为止，坚守待时，等待之事宜
- **预测储蓄积财**：艮为蓄，储蓄理财、积攒家业之事宜
- **预测学生少年**：看艮宫状态，判断少男、学生的情况
- **预测防守保安之事**：艮为阻为止，安保防御之事宜
- **预测房地产**：艮为山为土，与坤宫配合判断房产
- **预测停止结束之事**：艮为止，结束某件事、停止某种行为吉
- **与生门、天任星同宫**：生门为三大吉门之一，天任星为吉星，组合大吉
- **落艮八宫（本宫）**：得地利，力量最强
- **四季土月（辰戌丑未）**：土当令，艮卦最旺

### 忌

- **预测前进进取之事**：艮为止，不利于开拓进取
- **预测流通之事**：艮为阻，物流、资金流动之事不吉
- **落震三宫、巽四宫（木克土）**：受制严重
- **落坎一宫（土克水，门迫）**：为迫，强行阻止
- **春季寅卯月（木旺土死）**：艮卦力量最弱
- **过于固执保守**：艮的负面是顽固封闭

## 旺相休囚/落宫状态

### 季节旺衰

艮卦五行属土，其旺衰随季节变化（与坤卦同理）：

| 季节 | 月份 | 状态 | 说明 |
|------|------|------|------|
| **四季土月** | 辰、戌、丑、未月 | **旺** | 土当令，艮卦最旺 |
| **夏** | 巳、午月 | **相** | 火生土，第二大旺 |
| **秋** | 申、酉月 | **休** | 土生金，泄气 |
| **冬** | 亥、子月 | **囚** | 土克水，但水旺反侮土 |
| **春** | 寅、卯月 | **死** | 木克土，最为衰弱 |

**口诀**：艮土旺于四季土月，相于夏，休于秋，囚于冬，死于春。

但需注意：艮宫地支含丑寅，丑为土（艮旺），寅为木（克土），故艮卦在冬春之交时力量最为复杂。

### 落宫状态

艮卦（或作为用神）落入九宫各宫的状态：

| 落宫 | 宫位五行 | 关系 | 状态 |
|------|---------|------|------|
| **艮八宫** | 土 | 比和，本宫 | 最旺 |
| **坤二宫** | 土 | 比和 | 旺，同气相求 |
| **中五宫** | 土 | 比和 | 旺 |
| **离九宫** | 火 | 宫生卦（生入） | 相，得火生 |
| **兑七宫** | 金 | 卦生宫（生出） | 休，泄气 |
| **乾六宫** | 金 | 卦生宫（生出） | 休，泄气 |
| **坎一宫** | 水 | 卦克宫（迫） | 囚 |
| **震三宫** | 木 | 宫克卦（制） | 死，严重受克 |
| **巽四宫** | 木 | 宫克卦（制） | 死，严重受克 |

### 旺衰对预测的影响

- **旺相时**：稳固如山、储蓄丰厚、少男健康、手巧鼻通、防守严密
- **休囚时**：站立不稳、储蓄不多、少男平庸、手鼻有疾
- **死时**：根基动摇、完全被困、脾胃重疾、骨节严重问题
- **入本宫（艮八宫）**：无论季节，力量加成

## 开悟之门补充

### 艮卦在奇门预测中的应用技巧

1. **预测财运**：生门居艮八宫，为利润之门，财运看生门状态
2. **预测少男**：看艮宫或用神是否临艮卦之象
3. **预测手鼻疾病**：看艮宫是否有凶门凶星凶格
4. **预测投资**：生门主利润，投资回报看艮宫
5. **预测房地产**：艮坤二宫皆属土，配合判断不动产
6. **预测学业**：艮为少男学生，看艮宫状态

### 艮与生门的关系

生门居艮八宫，属土。生门是三大吉门之一（开门、休门、生门），主生机、生长、利润：
- 生门旺相时：利润丰厚、事业生长、投资有回报
- 生门休囚时：生长力弱、利润微薄
- 生门为八门中唯一的"纯吉"——没有负面含义

在奇门预测中，生门旺相是最有力的吉门信号之一。尤其在预测求财时，生门的状态直接决定财运的好环。

### 艮与天任星的关系

天任星为九星之一，属土，居艮八宫。天任星为吉星，主责任、担当、稳重：
- 天任星旺相：有责任心、稳重可靠、能担大任
- 天任星休囚：责任心不足、承压能力弱
- 天任星在预测人品时，代表稳重诚实之人

天任星与生门同落艮八宫，是奇门中最具稳定性的吉宫。

### 艮卦"止"的哲学含义

艮卦的核心是"止"——停止、阻止。但"止"不是消极的停止，而是一种智慧：
1. **知止不殆**（《道德经》）：知道何时停止才不会陷入危险
2. **止于至善**（《大学》）：停在最好的状态
3. **以止为进**：暂时停止是为了积蓄力量更好地前进

在奇门预测中，艮象出现时，往往是提醒当事人"该停一停了"——不要盲目冲动，审时度势、积蓄力量才是上策。

### 艮与"终万物始万物"

《说卦传》曰"终万物始万物者莫盛乎艮"——艮卦位于东北，是冬天结束、春天开始的位置，万物在这里结束旧的循环、开启新的循环。因此艮卦也有"结束+开始"的双重含义：
- 预测何时结束：看艮宫
- 预测何时开始：也看艮宫（因为结束即新开始）

### 艮卦与东北方的地理成因

《开悟之门》解释：东北方为山西高原和太行山，故将代表山的艮卦放在东北方。中国地形东北多山，山势巍峨，艮卦的"止、阻"之意从地理上就可以找到依据——高山阻路，使人止步。
