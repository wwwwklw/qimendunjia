# 震卦（震为雷）

## 定义与基本属性

震卦，卦符为 ☳，口诀"**震仰盂**"——一阳爻在下，两阴爻在上，像仰面朝上的盆盂。震卦代表雷、长男、震动、行动，是八卦中最具爆发力、行动力的卦象。

| 属性 | 内容 |
|------|------|
| **卦名** | 震 |
| **卦符** | ☳ |
| **口诀** | 震仰盂 |
| **代表** | 雷 |
| **五行** | 木（阳木） |
| **阴阳** | 阳卦（一阳二阴，以少爻定阴阳） |
| **先天卦数** | 4 |
| **后天卦数** | 3 |
| **后天方位** | 东 |
| **对应宫位** | [[震三宫]] |
| **家庭成员** | 长男（长子） |
| **身体部位** | 足、肝、筋、声带 |
| **季节** | 春（寅卯月，尤其卯月） |
| **时辰** | 卯时（5:00-7:00） |
| **地支** | 卯 |
| **对应奇门符号** | [[伤门]]、[[天冲星]] |

震卦在[[后天八卦]]中位于东方，是"春"之卦，代表春季万物萌动、雷霆震动、生机勃发的阶段。"帝出乎震"——万物都从震卦开始发动。震卦的核心精神是**震动、行动、唤醒、启动**。

在[[先天八卦]]中，震卦居东北（左下），数序为4。"雷风相薄"——震（雷）与巽（风）相对，雷动风起，互相激荡。先天震卦代表地球上的气候变化（雷）。

震卦初爻为阳，象征一阳初生，如春天阳气从地下萌动上升，破土而出。因此震卦为长男——家中第一个男孩，代表家族的新生代力量。

## 详细象意

### 人事

- **身份**：长子、长男、大儿子、青年男子、继承人、开创者、先锋、出头人物
- **性格**：有活力、行动力强、果断、急躁、冲动、勇敢、有魄力、刚直
- **负面**：急躁冒进、鲁莽冲动、脾气暴躁、缺乏耐心、容易发怒、虎头蛇尾
- **职业**：运动员、战士、警察、消防员、司机、机械师、工程师、创业者、音乐人
- **行为**：行动、出发、冲击、震动、发出声音、宣告、打破沉默、激发
- **关系**：对应长子、大儿子、有冲劲的年轻人

### 地理

- **自然**：雷、电、地震、火山爆发
- **方向**：东方（日出之地）
- **城市**：东方城市、新兴城市、开发区
- **建筑**：电厂、广播电台、信号塔、钟楼、运动场馆
- **地形**：东方之地、有声响之地、震动频繁之地、竹林、树林
- **场所**：运动场、健身房、赛车场、机场、火车站、工厂车间
- **奇门方位**：东方（卯位），震三宫

### 身体

- **主要**：足、脚、下肢、腿
- **脏腑**：肝、胆（肝主怒，怒伤肝）
- **其他**：声带、喉咙（发声之处）、筋、神经系统
- **疾病**：肝病、肝胆不适、脚伤、腿部疾病、抽筋、中风、神经衰弱、声带疾病、惊悸

### 性格

- **正面**：有冲劲、敢作敢为、雷厉风行、有开创精神、朝气蓬勃、直爽坦诚
- **负面**：急躁冲动、暴躁易怒、缺乏耐心、虎头蛇尾、容易泄气、不够稳重
- **心理特质**：行动导向、反应迅速、情绪波动大、喜怒形于色

### 物品

- 发声物品：鼓、钟、铃、喇叭、音响、乐器（尤其打击乐）
- 交通工具：汽车、摩托车、火车（快速移动之物）
- 电子设备：电话、手机、电报机、信号发射器
- 武器：鞭炮、炸药、火箭
- 植物：竹子（节节高升）、乔木、大树、高耸的植物
- 其他：绿色物品、青色物品、运动器械、健身器材

### 动物

- **主要**：龙（龙行天际、雷霆万钧）
- **其他**：蛇（行动迅速）、骏马（奔跑快速）、鸣虫、飞鸟
- **特点**：善于发声的动物、行动迅速的动物、有爆发力的动物

### 天时

- 雷、电、暴风雨、地震、火山爆发
- 春季生机勃发的天气
- 突然变化的天气

### 颜色

- 青色（木之色）、绿色、碧色、蓝色

## 应用宜忌

### 宜

- **预测行动之事**：震为动，出行、起步、开工等主动行为之事宜
- **预测长子**：看震宫状态，判断长子、青年男子的情况
- **预测竞争**：震有竞争之意，运动比赛、商业竞争看震宫
- **预测声音相关**：音乐、演讲、广播、通讯等事宜宜
- **预测新生事物**：震为初生之力，新项目启动、创业之初宜
- **与伤门、天冲星同宫**：同气相应（需结合具体情况判断吉凶）
- **落震三宫（本宫）**：得地利，力量最强
- **春季寅卯月**：木当令，震卦最旺，最为有利

### 忌

- **预测平稳安静之事**：震过于动荡，求稳定之事不吉
- **预测脾胃之事**：木克土，与坤艮土宫之事冲突
- **落兑七宫、乾六宫（金克木）**：受制，行动受阻
- **落坤二宫、艮八宫（木克土，门迫）**：为迫，冲动过激
- **秋季申酉月（金旺木死）**：震卦力量最弱，不利行动
- **容易冲动误事**：震的负面是鲁莽急躁，要提醒自己冷静

## 旺相休囚/落宫状态

### 季节旺衰

震卦五行属木，其旺衰随季节变化：

| 季节 | 月份 | 状态 | 说明 |
|------|------|------|------|
| **春** | 寅、卯月 | **旺** | 木当令，震卦最旺，行动力最强 |
| **冬** | 亥、子月 | **相** | 水生木，第二大旺，得生扶 |
| **夏** | 巳、午月 | **休** | 木生火，泄气，力量减弱 |
| **四季土月** | 辰、戌、丑、未月 | **囚** | 木克土，但土旺反侮木，受制 |
| **秋** | 申、酉月 | **死** | 金克木，被当令之金所克，最弱 |

**口诀**：震木旺于春，相于冬，休于夏，囚于四季土，死于秋。

### 落宫状态

震卦（或作为用神）落入九宫各宫的状态：

| 落宫 | 宫位五行 | 关系 | 状态 |
|------|---------|------|------|
| **震三宫** | 木 | 比和，本宫 | 最旺，得地利 |
| **巽四宫** | 木 | 比和 | 旺，同气相求 |
| **坎一宫** | 水 | 宫生卦（生入） | 相，得水生扶 |
| **离九宫** | 火 | 卦生宫（生出） | 休，泄气 |
| **坤二宫** | 土 | 卦克宫（迫） | 囚，主动出击但对方厚重 |
| **艮八宫** | 土 | 卦克宫（迫） | 囚，同上 |
| **兑七宫** | 金 | 宫克卦（制） | 死，严重受克 |
| **乾六宫** | 金 | 宫克卦（制） | 死，严重受克 |
| **中五宫** | 土 | 卦克宫（迫） | 囚 |

### 旺衰对预测的影响

- **旺相时**：行动力强、事业有突破、长子有为、脚步有力、生机勃勃
- **休囚时**：行动受阻、有力使不出、长子平庸、脚部不适
- **死时**：完全被困、动弹不得、肝病严重、脚部重伤、事业停滞
- **入本宫（震三宫）**：无论季节，力量加成，地利之助

## 与其他概念的关系

- [[先天八卦]]：先天震卦数序为4，居东北。"雷风相薄"——震与巽相对，雷动风起
- [[后天八卦]]：后天震卦数序为3，居东，为春。"帝出乎震"——春雷震动，万物复苏
- [[八卦概述]]：震为长男、为雷、为动
- [[九宫]]：对应震三宫，地支为卯，数序为3
- [[五行]]：属木（阳木），木性生发条达，与震卦初生发动之性相通
- [[阴阳]]：阳卦（一阳爻在下，阳气初生），以少爻（阳爻）定阴阳
- [[伤门]]：震卦对应伤门，伤门属木，主伤害、竞争、捕盗。伤门虽为凶门，但在特定场合如捕猎、竞争反而有用
- [[天冲星]]：震卦对应天冲星，天冲星属木，主行动、冲动、冲突。是九星中最具行动力的星
- [[十二地支]]：震宫寄卯支，卯为木之正位
- [[十天干]]：震属木，对应甲木（阳木）之性，为栋梁之材
- [[六十四卦]]：震为上卦、下卦形成震宫八卦（震为雷、雷地豫、雷水解、雷风恒、地风升、水风井、泽风大过、泽雷随）
- [[时空统一论]]：震卦在时空模型中位于东方、春季、卯时（日出之时）
- [[河图洛书]]：洛书九宫中震三宫之数为3（左三）
- [[三才]]：震在"人"的层面主要代表长子、青年男子
- [[奇门遁甲概述]]：震宫是重要宫位，伤门和天冲星的本位

## 开悟之门补充

### 震卦在奇门预测中的应用技巧

1. **预测出行**：看震宫和伤门的状态，伤门也是出行之门
2. **预测长子**：看震宫或用神是否临震卦之象
3. **预测竞争**：震为动为竞争，伤门亦为竞争之门，结合判断
4. **预测脚部疾病**：看震宫是否有凶门凶星凶格
5. **预测声音/音乐**：震为声响，预测音乐、演讲、广播看震宫
6. **预测雷电相关**：震为雷，预测电子设备、电力系统看震宫

### 震为长男的含义

震卦初爻为阳，按"初爻定长、中爻定中、上爻定少"的规律，震为长男。在奇门预测家庭关系、子女前途时，震宫是判断长子状况的关键宫位。

### 震与伤门的关系

伤门居震三宫，属木。伤门为凶门之一，主伤害、竞争、捕猎：
- 伤门旺相（春季）时，有利于竞争、捕猎，但也容易发生伤害事件
- 伤门休囚时，凶性减轻
- 伤门与天冲星同宫，冲动加倍

在奇门预测中，伤门并不绝对为凶——适度的"伤"（如竞争中的优胜劣汰）反而是事物发展的必要推动力。

### 震卦"帝出乎震"的原始含义

《说卦传》曰"帝出乎震"——"帝"指北斗星所指的方向，也象征主宰万物的力量。震代表东方、春天，是万物开始生长、一切开始运动的方向和时间。因此震卦有"开始、启动、发动"之意。
