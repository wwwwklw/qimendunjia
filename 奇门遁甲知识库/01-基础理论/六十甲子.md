# 六十甲子

## 定义与基本属性

六十甲子，又称六十花甲子，是十天干与十二地支按固定顺序组合而成的六十个基本计时单元。

因为十天干和十二地支的最小公倍数是六十，所以甲子不是一百二十个，而是恰好六十个，形成一个完整的循环。六十甲子循环一周，古代称为"六十花甲"。

组合规律：天干与地支从各自的起始位置（甲和子）开始，一一配对——甲配子、乙配丑、丙配寅……癸配酉之后，天干又回到甲，地支继续往下走到戌、亥，甲再配戌、乙配亥……如此循环，直到甲再配子，整整六十对完成一个循环。

## 六甲

六十甲子中有六个以"甲"开头的组合，称为**六甲**：

| 六甲 | 起始 | 包含的干支 | 用途 |
|------|------|-----------|------|
| **甲子旬** | 甲子 | 甲子至癸酉（10个） | 第一旬 |
| **甲戌旬** | 甲戌 | 甲戌至癸未（10个） | 第二旬 |
| **甲申旬** | 甲申 | 甲申至癸巳（10个） | 第三旬 |
| **甲午旬** | 甲午 | 甲午至癸卯（10个） | 第四旬 |
| **甲辰旬** | 甲辰 | 甲辰至癸丑（10个） | 第五旬 |
| **甲寅旬** | 甲寅 | 甲寅至癸亥（10个） | 第六旬 |

六十甲子每十个一组分为六旬，每旬以甲开头，故称六甲。在奇门遁甲中，六甲隐于六仪之下，是"遁甲"的具体表现形式。

## 六甲旬空

每一旬中，由于天干有十个、地支有十二个，每旬总有两个地支配不到天干，这两个落空的地支就是"**旬空**"（也叫"空亡"）：

| 六甲旬 | 旬空地 |
|--------|--------|
| 甲子旬 | 戌、亥 |
| 甲戌旬 | 申、酉 |
| 甲申旬 | 午、未 |
| 甲午旬 | 辰、巳 |
| 甲辰旬 | 寅、卯 |
| 甲寅旬 | 子、丑 |

在奇门遁甲中，**旬空是一个极其重要的概念**。宫位为旬空之地时，往往表示事物不落实、空旷、虚假、未发生等含义，具体吉凶需要结合其他条件判断。

## 详细象意

- **六十甲子整体**：时间的循环编码，代表完整的周期
- **六甲**：首领、主帅、隐藏的力量（奇门遁甲的核心——把甲藏起来）
- **旬空**：空虚、未落实、虚假、等待、尚未发生、力有不逮
- **每一组干支**：各有其纳音五行和特殊含义

## 应用宜忌

- **宜**：熟练掌握六甲旬空的规定，这是奇门判断的基础
- **忌**：忽略旬空的影响，只看表面格局

奇门遁甲排盘时，首先要确定用时的旬首（属于哪一甲），从而确定值符和值使，以及空亡所在的宫位。这些都是判断中不可或缺的信息。

## 旺相休囚/落宫状态

六十甲子本身不直接参与旺相休囚，但每组的干支各有五行属性，根据其天干地支的五行在落宫中的旺衰状态进行判断。
