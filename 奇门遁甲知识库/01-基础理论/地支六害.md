# 地支六害

## 定义与基本属性

**地支六害**又称"地支相害"或"六穿"，是[[十二地支]]之间的一种暗中破坏、互相伤害的关系。与[[地支六冲|六冲]]的公开对抗不同，六害的特点是**暗中使坏、背后拆台、表面和谐实则有害**。

> **子未害、丑午害、寅巳害、卯辰害、申亥害、酉戌害**

| 相害组合 | 五行关系 | 特点 |
|----------|----------|------|
| 子 ↔ 未 | 水 ↔ 土（燥土） | 水被土害 |
| 丑 ↔ 午 | 湿土 ↔ 火 | 火被湿土所害 |
| 寅 ↔ 巳 | 木 ↔ 火 | 木被火泄害 |
| 卯 ↔ 辰 | 木 ↔ 湿土 | 木克土，互相伤害 |
| 申 ↔ 亥 | 金 ↔ 水 | 金生水被泄害 |
| 酉 ↔ 戌 | 金 ↔ 燥土 | 戌为火库害金 |

六害的推导方法有两种理解方式：

1. **以合推害**：每个地支与其所合地支的六冲方相害
   - 如：子丑合，丑未冲 → 子未害（子与丑的冲方未相害）
2. **同类相害**：五行同类的地支之间相害
   - 如：寅卯皆木，卯与辰害；巳午皆火，丑与午害

## 详细象意

### 六害的通象

- 暗中使坏、表面和气内心不满
- 口蜜腹剑、背后拆台
- 挑拨离间、搬弄是非
- 暗中破坏、内耗
- 关系表面的维持实则已有裂痕

### 各害的具体象意

| 相害 | 象意 |
|------|------|
| **子未害** | 水被燥土所害，主**家宅不宁、内部矛盾** |
| **丑午害** | 火被湿土所害，主**官非口舌、被人暗算** |
| **寅巳害** | 木被火焚，主**出力不讨好、被人利用** |
| **卯辰害** | 木土相害，主**邻里不和、亲戚矛盾** |
| **申亥害** | 金水相害（泄），主**交通意外、文书出错** |
| **酉戌害** | 金被火库所害，主**失窃、被骗、暗中损失** |

## 应用宜忌

- **忌**：
  - 测人际关系逢害 → 注意小人在背后搞鬼
  - 测合作逢害 → 合作方可能表面配合、暗中使绊
  - 测婚姻逢害 → 夫妻间有暗中的不满和隔阂
  - 测官运逢害 → 有小人暗算、仕途阻力

六害一般**无宜可言**，逢害皆需警惕暗中的不利因素。

## 与相关概念的关系

- [[地支六冲]]：冲是公开对抗，害是暗中破坏——明枪易躲、暗箭难防
- [[地支三合]]：三合可以化解六害
- [[地支相刑]]：刑与害都是不和谐关系，刑偏重于自我折磨与冲突、害偏重于被人暗算
- [[十二地支]]：六害的基础
- [[天干化合]]：遇合可在一定程度上缓解六害

## 上元经补充

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]刑害章

### 穿（害）大于刑冲

张大春先生明确指出：**穿（害）的力量大于冲与刑**。很多奇门书籍对穿的作用隐而不谈，这是不应该的。必须熟练掌握六穿在奇门遁甲中的具体应用。

奇门中穿的应用与四柱有所不同——单纯的六穿关系并不一定能够直接套用在奇门遁甲中，奇门体系自身的格局特征（如半合局、宫位旺衰等）可能会在一定程度上改变原有力量的对比。

详见 [[地支相刑]] 笔记中的"穿害六组（奇门专用）"部分。
