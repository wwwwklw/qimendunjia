# 孤虚法

## 定义与基本属性

**孤虚法**是古代兵法与择吉中的重要方位判断法，以六甲旬首确定孤方（吉方/背靠之方）和虚方（凶方/攻击之方）。核心原则是"背孤击虚"——背靠孤方、面向虚方行动。

> **通俗理解**：孤虚法就是告诉你在战斗中应该站在哪个方位、攻击哪个方位。孤方是你的"后盾"（背靠的安全方位），虚方是敌人的"弱点"（应该攻击的方位）。

## 经典论述

> 黄石公曰：「**背孤击虚，一女可敌十人**。」

## 六甲孤虚方位

| 六甲旬 | 旬首 | 孤方 | 虚方 |
|--------|------|------|------|
| **甲子旬** | 甲子 (子) | **戌、亥** | **辰、巳** |
| **甲戌旬** | 甲戌 (戌) | **申、酉** | **寅、卯** |
| **甲申旬** | 甲申 (申) | **午、未** | **子、丑** |
| **甲午旬** | 甲午 (午) | **辰、巳** | **戌、亥** |
| **甲辰旬** | 甲辰 (辰) | **寅、卯** | **申、酉** |
| **甲寅旬** | 甲寅 (寅) | **子、丑** | **午、未** |

> 规律：六甲旬中，旬首前二位为孤，旬首对冲及对冲前进一位为虚。

## 孤虚层级

根据使用场景确定用何种孤虚：

| 使用场景 | 孤虚层级 | 说明 |
|---------|---------|------|
| **个人/小规模** | 时孤 | 十人用时孤，**最验** |
| **中等规模** | 日孤 | 百人用日孤 |
| **较大规模** | 月孤 | 千人用月孤 |
| **国家/大战** | 年孤 | 万人用年孤 |

## 旺相休囚的倍数效应

孤虚的效果受旺衰影响：

| 状态 | 效果倍数 |
|------|---------|
| **旺气** | 十倍 |
| **相气** | 五倍 |
| **休气** | 如数（一倍） |
| **囚气** | 减少 |
| **死气** | 减半 |

## 六甲青龙安营法

卷二十载六甲青龙安营法，与孤虚法配套使用：

> 「六甲起青龙，大将居之；六乙为蓬星，旗鼓居之；六丙为明堂，次将居之；六丁为太阴，出入私路；六戊为天门，吉事出入；六己为地户，伏兵凶事；六庚为天狱，宜积粮草；六辛为天庭，判断斩决；六壬为天牢，宜积蓄；六癸为华盖/天藏，若欲脱难入太阴出天藏而去。」

**十干安营对应表**：

| 天干 | 神名 | 安营用途 |
|------|------|---------|
| **甲** | 青龙 | 大将居之 |
| **乙** | 蓬星 | 旗鼓居之 |
| **丙** | 明堂 | 士卒/次将居之 |
| **丁** | 太阴 | 伏兵、探听、脱身 |
| **戊** | 天门/军门 | 吉事出入 |
| **己** | 地户 | 伏兵、凶事、躲藏 |
| **庚** | 天狱 | 积粮草 |
| **辛** | 天庭 | 判断、斩决 |
| **壬** | 天牢 | 积蓄 |
| **癸** | 华盖/天藏 | 脱难之用 |

> 《三元经》曰：「夫将兵四出，统众安营，必取其法，以六甲为首，十时一易。」

## 博奕胜负

> 《金匮》云：「得与无，视孤虚。」

博奕（赌博）以正时六甲旬孤上坐者胜，虚上坐者负。又参以三奇吉门，斗罡以杓指他人必胜。

## 陟险危论

> 「九山河水际深大为绝过山朦胧为天牢过。绝人者为天罗。兼葭者众草也。潢者池也。险者疾也。」

若贼从利方上来，阳时令士卒坦前左肩引声大呼鸣钟鼓而先击之；阴时令士卒衔枚驻车立马桴鼓静以待之。

**突围三部法**：一部居月建上，一部居月德上，一部居生神上。大将居亭亭上引兵击之大胜。

## 应用宜忌

- **宜**：背孤击虚——自己坐在孤方、向虚方进攻
- **宜**：旺相时使用孤虚法，效果倍增
- **宜**：日常小事用时孤（最验）
- **忌**：死气时使用，效果大减
- **忌**：仅靠孤虚法不看奇门格局

### 论孤虚

**原文要旨**：

> 「凡主将用兵、安营、布阵，必须依孤虚之法，方能保全士卒，百战百胜，所向无敌。」

**孤虚之定义**：
- 孤者，乃**高峻独尊**之象
- 常可背而击其冲
- 其法即**旬中空亡**也

**分阳孤阴虚**：
- 务使阳孤而击阳虚
- 阴孤而击阴虚
- 则**一女可敌十夫**

**孤虚层级**（与金函玉镜互补）：

| 规模 | 用何孤 |
|------|--------|
| 万人 | 年孤 |
| 千人以上 | 月孤 |
| 百人以上 | 日孤 |
| 十人以上 | 时孤 |

**六花大阵**（奇门法窍独有详细论述）：

主将列阵方位：
- 主将坐于**中军**
- 裨将坐于**青龙**
- 旗鼓振于**天蓬**
- 士卒驻于**明堂**
- 伏兵潜于**太阴**
- 领兵出于**天门**
- 旋师入于**地户**
- 斩决出于**天狱**
- 囚罪系于**天牢**
- 粮草统于**天庭**
- 府库藏于**华盖**

**六甲旬青龙**：
- 甲子旬，子为青龙
- 甲戌旬，戌为青龙
- （余仿此，以旬首为青龙）

**阴阳孤虚对位**：
- 六甲旬**亥酉未巳卯丑**为**阴虚**
- 六甲旬**戌申午辰寅子**为**阳孤**
- 对而击其冲，分阳孤击阳虚，阴孤击阴虚
