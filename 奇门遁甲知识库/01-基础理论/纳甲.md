# 纳甲

## 定义与基本属性

**纳甲**是汉代易学家京房创立的八卦与天干相配的学说，将十天干分别纳入八卦之中，用于解释卦象与天象、时间的关系。"纳"即纳入，"甲"指天干（以甲为首代指十天干）。

纳甲是连接[[八卦概述|八卦]]与[[十天干]]的桥梁，在奇门遁甲中主要用于判断天干落入何宫以及入墓的宫位。

## 纳甲口诀

金函玉镜所载：

> 干纳甲壬、坤纳乙癸、坎戊、离己、震庚、
> 巽辛、艮丙、兑丁

## 八卦纳甲表

| 八卦 | 纳天干 | 说明 |
|------|--------|------|
| **乾卦** ☰ | 甲、壬 | 乾为天，纳甲壬（阳干之首与阳水之干） |
| **坤卦** ☷ | 乙、癸 | 坤为地，纳乙癸（阴干之首与阴水之干） |
| **坎卦** ☵ | 戊 | 坎为水，纳戊（阳土之干） |
| **离卦** ☲ | 己 | 离为火，纳己（阴土之干） |
| **震卦** ☳ | 庚 | 震为雷，纳庚（阳金之干） |
| **巽卦** ☴ | 辛 | 巽为风，纳辛（阴金之干） |
| **艮卦** ☶ | 丙 | 艮为山，纳丙（阳火之干） |
| **兑卦** ☱ | 丁 | 兑为泽，纳丁（阴火之干） |

### 纳甲规律

| 天干 | 所在八卦 | 宫位 |
|------|---------|------|
| 甲、壬 | 乾 | 乾六宫 |
| 乙、癸 | 坤 | 坤二宫 |
| 戊 | 坎 | 坎一宫 |
| 己 | 离 | 离九宫 |
| 庚 | 震 | 震三宫 |
| 辛 | 巽 | 巽四宫 |
| 丙 | 艮 | 艮八宫 |
| 丁 | 兑 | 兑七宫 |

> **注意**：天干在九宫中的分布不是均等的——乾、坤各纳二干，其他六卦各纳一干。在奇门遁甲中，甲遁于六仪之中，并不单独出现。

## 甲墓

金函玉镜原文：

> 甲墓于干，戊墓坎之类。

即：天干甲入墓在乾宫（甲属乾），戊入墓在坎宫（戊属坎）。同理可推：

| 天干 | 墓位（依纳甲所在宫） |
|------|-------------------|
| 甲、壬 | 乾六宫 |
| 乙、癸 | 坤二宫 |
| 戊 | 坎一宫 |
| 己 | 离九宫 |
| 庚 | 震三宫 |
| 辛 | 巽四宫 |
| 丙 | 艮八宫 |
| 丁 | 兑七宫 |

> **注意**：此处的"墓"是纳甲之墓，与三奇六仪入墓（如乙奇入坤二宫墓）的概念有所不同。纳甲墓偏重于天干本身所入之宫的"归宿"属性。

## 纳甲在奇门遁甲中的应用

1. **判断天干落宫**：通过纳甲可以快速判断某天干落在哪个八卦宫位
2. **入墓判断**：天干入其所纳之宫即为"入墓"（但需结合具体语境，奇门中三奇入墓另有专属规则）
3. **八卦与天干互推**：知道宫位可知对应天干，知道天干可知对应宫位

## 与其他概念的关系

- [[八卦概述|八卦]]：纳甲是八卦与天干的配对体系
- [[十天干]]：天干通过纳甲分别归属各卦
- [[三奇入墓]]：奇门中的入墓概念与此相关但规则不同
- [[九宫]]：纳甲中的每个卦对应一个宫位
- [[六十甲子]]：纳甲是理解干支与八卦关系的关键
- [[喜神方位]]：喜神方位的推算用到天干，与纳甲相关

## 应用宜忌

- **宜**：背熟八卦纳甲表，能快速判断天干对应的宫位
- **宜**：结合[[后天八卦]]的方位理解天干的空间属性
- **忌**：混淆纳甲之墓与三奇六仪入墓——两者规则不同，不可混用
