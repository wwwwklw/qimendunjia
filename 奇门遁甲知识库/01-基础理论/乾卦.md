# 干卦（乾为天）

## 定义与基本属性

乾卦，八卦之首，卦符为 ☰，口诀"**乾三连**"——三个阳爻并列，纯阳之卦。乾卦代表天、父、君、刚健、创造，是八卦中最具阳刚之气、主动进取精神的卦象。

| 属性 | 内容 |
|------|------|
| **卦名** | 乾 |
| **卦符** | ☰ |
| **口诀** | 乾三连 |
| **代表** | 天 |
| **五行** | 金（阳金） |
| **阴阳** | 阳卦（纯阳） |
| **先天卦数** | 1 |
| **后天卦数** | 6 |
| **后天方位** | 西北 |
| **对应宫位** | [[乾六宫]] |
| **家庭成员** | 父亲、老年男子 |
| **身体部位** | 头、首、骨、肺、大肠 |
| **季节** | 秋冬之交（戌亥月） |
| **时辰** | 戌时、亥时（19:00-23:00） |
| **地支** | 戌、亥 |
| **对应奇门符号** | [[开门]]、[[天心星]] |

乾卦在[[后天八卦]]中位于西北方，是"秋冬之交"的卦，代表秋收后收藏、积蓄力量的阶段。乾为天，天行健，君子以自强不息。乾卦的核心精神是**刚健、创造、领导、永动**。

在[[先天八卦]]中，乾卦居南（上），代表天在上，是宇宙的顶端，数序为1，为八卦之首。"天地定位"——乾在上为天，坤在下为地，确立了宇宙的基本框架。先天为体，后天为用，乾卦从先天之"上（南）"变为后天之"西北"，体现了从"体"到"用"的转化。

乾卦三爻皆阳，为纯阳之体，在[[阴阳]]体系中代表着阳的极致——最刚健、最主动、最积极的力量。

## 详细象意

### 人事

- **身份**：领导、官员、上级、老板、君王、总统、主席、CEO、决策者、权威人物、父亲、祖父、老年男人、长者、尊贵之人、掌权者
- **性格**：刚强、果断、勇敢、自信、有领导力、正直、有威严、好动、进取心强
- **负面**：过于刚硬、固执己见、独断专行、霸道、好斗、不近人情、刚愎自用
- **职业**：政府官员、公务员、军人、警察、法官、企业高管、创始人、体育健将
- **行为**：开创、领导、指挥、决策、颁布命令、行使权力、征服、竞争
- **关系**：对应父亲、丈夫（男性长辈一方）、上级领导、导师

### 地理

- **自然**：天、高空、苍穹、太空、高处
- **城市**：首都、省会、政府所在地、权力中心
- **建筑**：政府大楼、宫殿、官邸、高楼大厦、指挥中心、司令部、塔楼、钟楼
- **地形**：高原地带、高山之巅、西北方、干燥开阔之地
- **场所**：领导办公室、董事会会议室、高级俱乐部、老干部活动中心、体育馆
- **奇门方位**：西北方（戌亥位），乾六宫

### 身体

- **主要**：头、首（首领之象征）、大脑、额头、面部
- **骨骼系统**：骨骼、脊椎、关节（尤其是上肢关节）
- **脏腑**：肺、大肠（金对应脏腑）
- **其他**：皮肤（护卫全身如天覆万物）、右侧身体
- **疾病**：头痛、头晕、脑部疾病、高血压、肺部疾病、大肠疾病、骨折、皮肤病

### 性格

- **正面**：刚健不息、自强自立、光明磊落、胸怀宽广、意志坚定、勇于开拓、正直不阿、有担当
- **负面**：固执僵化、独断专行、好高骛远、过于强硬、不近人情、刚愎自用、缺乏柔和
- **心理特质**：理性思维强、逻辑清晰、目标明确、行动力强

### 物品

- 金属制品、贵重金属（金、银、铜、铁）
- 圆形物品：球体、圆盘、圆环、车轮、钟表
- 坚硬物品：钻石、玉石、宝石、金刚石
- 交通工具：汽车、飞机、火车（尤其是高级轿车）
- 武器：刀剑、枪炮、盾牌
- 权力象征：印章、权杖、王冠、徽章、奖杯、国徽
- 贵重物品：古玩、珠宝、钱币、黄金
- 科技产物：电脑、精密仪器、机器人、机械
- 冰、雪（乾为寒、为冰）

### 动物

- **其他**：狮子（百兽之王）、鹰（空中之王）、天鹅

## 应用宜忌

### 宜

- **求官求职**：乾卦代表权力、领导，预测仕途最宜
- **事业开创**：刚健进取之象，创业、开拓新项目大吉
- **竞争取胜**：乾为刚健有力，竞争性事务有力
- **求见领导**：乾为上级，求见领导、官员得时则顺
- **金属行业**：乾五行属金，与金属、金融、机械相关行业利
- **与开门、天心星同宫**：力量加倍，贵人相助、事业有成
- **落乾六宫（本宫）**：得地利，力量最强
- **秋季申酉月**：金当令，乾卦最旺，最为有利

### 忌

- **预测柔顺之事**：乾过于刚硬，求柔顺、调和之事不吉
- **与人争执**：乾刚健好斗，争吵容易升级
- **落离九宫（火克金）**：受制，权威受挑战
- **落震三宫、巽四宫（金克木，门迫）**：为迫，主动出击但劳而无功
- **女性预测婚姻**：乾代表男性，过于阳刚可能不利感情（需结合具体用神）
- **春季寅卯月（木旺金囚）**：乾卦力量大减，不利乾象之事
- **过于刚愎自用**：乾的负面特质是要注意的

## 旺相休囚/落宫状态

### 季节旺衰

乾卦五行属金，其旺衰随季节变化：

| 季节 | 月份 | 状态 | 说明 |
|------|------|------|------|
| **秋** | 申、酉月 | **旺** | 金当令，乾卦最旺，力量最强 |
| **四季土月** | 辰、戌、丑、未月 | **相** | 土生金，第二大旺，得生扶有力 |
| **冬** | 亥、子月 | **休** | 金生水，泄气，力量开始减弱 |
| **春** | 寅、卯月 | **囚** | 金克木，但木旺反侮金，严重被制 |
| **夏** | 巳、午月 | **死** | 火克金，被当令之火所克，最为衰弱 |

**口诀**：乾金旺于秋，相于四季土，休于冬，囚于春，死于夏。

### 落宫状态

乾卦（天盘或作为用神）落入九宫各宫的状态：

| 落宫 | 宫位五行 | 关系 | 状态 |
|------|---------|------|------|
| **乾六宫** | 金 | 比和，本宫 | 最旺，得地利，如鱼得水 |
| **兑七宫** | 金 | 比和 | 旺，同气相求 |
| **坤二宫** | 土 | 宫生卦（生入） | 旺相，得生扶 |
| **艮八宫** | 土 | 宫生卦（生入） | 旺相，得生扶 |
| **坎一宫** | 水 | 卦生宫（生出） | 休，泄气 |
| **震三宫** | 木 | 卦克宫（迫） | 囚，主动出击但劳而无功 |
| **巽四宫** | 木 | 卦克宫（迫） | 囚，同上 |
| **离九宫** | 火 | 宫克卦（制） | 死，受克最凶 |
| **中五宫** | 土 | 宫生卦 | 旺相，得生扶 |

### 旺衰对预测的影响

- **旺相时**：领导力强、权威重、贵人有助、事业顺利、身体健康（尤其头部）
- **休囚时**：权威受损、领导不信任、事业受阻、头疾易发、贵人无力
- **死时**：权威尽失、职位不保、身体严重问题（尤其头部、骨骼）
- **入本宫（乾六宫）**：无论季节，力量加成，得地利之助

## 与其他概念的关系

- [[先天八卦]]：先天乾卦数序为1，居南（上），为天，象征宇宙顶端。"天地定位"——乾与坤相对，确立宇宙格局
- [[后天八卦]]：后天乾卦数序为6，居西北，为秋冬之交。"天高西北，地陷东南"——西北多高山，为天之象征
- [[八卦概述]]：乾为八卦之首，纯阳之卦，代表天、父、君
- [[九宫]]：对应乾六宫，地支为戌亥，数序为6
- [[五行]]：属金（阳金），金性刚硬肃杀，与乾卦刚健之性相通
- [[阴阳]]：纯阳之卦，三爻皆阳，代表阳的极致
- [[开门]]：乾卦对应开门，开门属金，为吉门之首，主开创、事业、出行
- [[天心星]]：乾卦对应天心星，天心星属金，为吉星，主心计、策划、医疗
- [[十二地支]]：乾宫寄戌、亥二支，戌为火库、亥为水
- [[十天干]]：乾属金，对应庚金（阳金）之性，与辛金（阴金）相对应
- [[六十四卦]]：乾为上卦、下卦的组合形成乾宫八卦（乾为天、天风姤、天山遁、天地否、风地观、山地剥、火地晋、火天大有）
- [[时空统一论]]：乾卦在时空模型中位于西北方、秋冬之交、戌亥时，代表一天中夜幕降临的时段
- [[河图洛书]]：洛书九宫中乾六宫之数为6（右足）
- [[三才]]：乾为天，代表三才之"天"的层面
- [[太极阴阳论]]：乾为纯阳，与坤（纯阴）构成太极的两极
- [[奇门遁甲概述]]：乾宫是奇门九宫格的重要宫位之一，开门和天心星的本位

## 开悟之门补充

### 乾卦在奇门预测中的应用技巧

1. **预测仕途**：看开门落宫与乾宫的关系，若开门落乾宫则事业最旺
2. **预测领导**：看值符（最高领导）、太岁（当年有权之人）所在的宫位，若与乾宫相生则得上司器重
3. **预测父亲/男性长辈**：看乾宫或用神是否临乾卦之象
4. **预测头部疾病**：看乾宫是否有凶门凶星凶格
5. **预测金属/机械行业**：看乾兑二宫的状态

### 乾卦之"元亨利贞"

《周易》乾卦卦辞曰"**元、亨、利、贞**"四德：
- **元**：创始、开始、万物之始
- **亨**：亨通、顺利、发展壮大
- **利**：和谐有利、各得其宜
- **贞**：正固、坚守正道

这四德是乾卦精神的核心——开创、通达、和利、坚正，也是奇门中乾象取用的四个维度。

### 乾与开门的关系

开门属金，居乾六宫，为乾卦所对应之八门。开门是奇门三大吉门之首（开门、休门、生门），主开创、远行、谒贵、经商。开门落乾宫为本位，力量最足；开门落其他宫位，需结合门宫关系判断。

### 乾与天心星的关系

天心星为九星之一，属金，居乾六宫。天心星为吉星，主心计谋略、医药、策划。在预测中，天心星落乾宫力量最强，对于谋略策划、医疗治病最为有利。天心星与开门同落乾宫，是奇门中极为有利的组合。
