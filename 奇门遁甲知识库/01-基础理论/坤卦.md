# 坤卦（坤为地）

## 定义与基本属性

坤卦，八卦之末，卦符为 ☷，口诀"**坤六断**"——三个阴爻并列，纯阴之卦。坤卦代表地、母、柔顺、包容、承载，是八卦中最具阴柔之美、含藏养育精神的卦象。

| 属性 | 内容 |
|------|------|
| **卦名** | 坤 |
| **卦符** | ☷ |
| **口诀** | 坤六断 |
| **代表** | 地 |
| **五行** | 土（阴土） |
| **阴阳** | 阴卦（纯阴） |
| **先天卦数** | 8 |
| **后天卦数** | 2 |
| **后天方位** | 西南 |
| **对应宫位** | [[坤二宫]] |
| **家庭成员** | 母亲、老年女子 |
| **身体部位** | 腹、脾胃、肌肉、子宫 |
| **季节** | 夏秋之交（未申月） |
| **时辰** | 未时、申时（13:00-17:00） |
| **地支** | 未、申 |
| **对应奇门符号** | [[死门]]、[[天芮星]] |

坤卦在[[后天八卦]]中位于西南方，是"夏秋之交"的卦，代表万物从繁盛走向收藏的过渡阶段。坤为地，地势坤，君子以厚德载物。坤卦的核心精神是**柔顺、包容、承载、养育**。

在[[先天八卦]]中，坤卦居北（下），数序为8。"天地定位"——乾在上为天，坤在下为地，坤代表地球的下边，是承载万物的基底。先天为体，坤卦代表了宇宙的"容器"和"承载面"。

坤卦三爻皆阴，为纯阴之体，在[[阴阳]]体系中代表着阴的极致——最柔顺、最包容、最具养育性的力量。

## 详细象意

### 人事

- **身份**：母亲、祖母、老年妇女、皇后、女领导、妇女主任、群众、大众、平民百姓、下属、随从
- **性格**：柔顺、善良、包容、温和、厚道、勤劳、忍耐、节俭、贤惠
- **负面**：过于柔顺则懦弱无能、缺乏主见、优柔寡断、拖沓、迟钝
- **职业**：教师（尤其是幼儿教师）、护士、服务员、农妇、家庭主妇、后勤人员、财务人员、秘书
- **行为**：养育、照顾、服务、配合、服从、积蓄、收藏、等待
- **关系**：对应母亲、妻子（女性长辈一方）、群众关系、上下级关系中的下级

### 地理

- **自然**：大地、平原、田野、土地、盆地、平地
- **城市**：成都、四川盆地（"坤为地，西南盆地"）
- **建筑**：民居、平房、仓库、粮仓、地下室、停车场、农村房屋
- **地形**：平原、盆地、农田、牧场、低洼地、西南方
- **场所**：厨房、食堂、餐厅、后勤部、仓库、农场、畜牧场
- **奇门方位**：西南方（未申位），坤二宫

### 身体

- **主要**：腹部、肚子、胃肠（尤其是胃、脾）
- **消化系统**：脾胃、胰脏、整个消化吸收系统
- **生殖系统**：子宫、卵巢（女性生殖器官）
- **其他**：肌肉、软组织、肉、右肩
- **疾病**：脾胃病、消化不良、胃病、腹部疾病、妇科病、子宫肌瘤、肌肉萎缩、肥胖

### 性格

- **正面**：厚道老实、包容大度、柔顺温和、勤俭持家、耐心细致、踏实稳重、善良有爱
- **负面**：懦弱无能、逆来顺受、缺乏主见、保守落后、拖沓迟钝、随波逐流
- **心理特质**：感性思维强、善于包容、重视家庭、有养育之心

### 物品

- 土地产物：粮食、谷物、面粉、土豆、蔬菜、水果
- 容器类：锅、碗、盆、缸（坤为腹，肚大能容）
- 布匹类：衣服、布料、被褥、毛巾、棉麻制品
- 柔软物品：棉被、枕头、沙发、垫子
- 方形容器：箱子、柜子、车厢（坤为方）
- 负重物：担子、背包、货车、拖拉机
- 陶器、瓷器、土制品、泥塑
- 生活用品：炊具、餐具、清洁工具

### 动物

- **主要**：牛（坤为牛，任劳任怨、负重致远）
- **其他**：母马（牝马）、母畜、老牛
- **特点**：食草动物、负重动物、温和驯良的动物、多胎多产动物

### 天时

- 阴天、多云、雾气、阴霾
- 夏秋之交的气候
- 潮湿、闷热之天气

### 颜色

- 黄色（土之色）、黑色（阴之色）、咖啡色、土黄色

## 应用宜忌

### 宜

- **预测母亲/女性长辈**：坤为母，预测母亲、祖母之事最宜
- **预测房地产**：坤为大地，土产房产之事有利
- **预测怀孕生育**：坤为腹、为母，看坤宫和天芮星状态
- **预测收藏储蓄**：坤含藏之性，储蓄、积蓄之事宜
- **预测群众关系**：坤为大众，群众路线、民主事宜宜
- **预测农业种植**：坤为田地，农业、种植业之事宜
- **与死门、天芮星同宫**：死门和天芮星也属土，同气相应
- **落坤二宫（本宫）**：得地利，力量最强
- **四季土月（辰戌丑未）**：土当令，坤卦最旺

### 忌

- **预测主动进取之事**：坤过于柔顺被动，开创性事务不吉
- **落震三宫、巽四宫（木克土）**：受制严重，诸事不顺
- **落坎一宫（土克水，门迫）**：主动勉强，效果不佳
- **春季寅卯月（木旺土死）**：坤卦力量最弱，不利坤象之事
- **与凶门凶星凶格同宫**：死门、天芮星本为问题之星，坤宫临之要小心
- **过于保守**：坤的负面是拖沓保守，要注意平衡

## 旺相休囚/落宫状态

### 季节旺衰

坤卦五行属土，其旺衰随季节变化：

| 季节 | 月份 | 状态 | 说明 |
|------|------|------|------|
| **四季土月** | 辰、戌、丑、未月 | **旺** | 土当令，坤卦最旺，力量最强 |
| **夏** | 巳、午月 | **相** | 火生土，第二大旺，得生扶 |
| **秋** | 申、酉月 | **休** | 土生金，泄气，力量减弱 |
| **冬** | 亥、子月 | **囚** | 土克水，但水旺反侮土，受制 |
| **春** | 寅、卯月 | **死** | 木克土，被当令之木所克，最弱 |

**口诀**：坤土旺于四季土月，相于夏，休于秋，囚于冬，死于春。

坤卦（或作为用神）落入九宫各宫的状态：

| 落宫 | 宫位五行 | 关系 | 状态 |
|------|---------|------|------|
| **坤二宫** | 土 | 比和，本宫 | 最旺，得地利 |
| **艮八宫** | 土 | 比和 | 旺，同气相求 |
| **中五宫** | 土 | 比和 | 旺 |
| **离九宫** | 火 | 宫生卦（生入） | 相，得生扶 |
| **兑七宫** | 金 | 卦生宫（生出） | 休，泄气 |
| **乾六宫** | 金 | 卦生宫（生出） | 休，泄气 |
| **坎一宫** | 水 | 卦克宫（迫） | 囚，主动勉强 |
| **震三宫** | 木 | 宫克卦（制） | 死，受克严重 |
| **巽四宫** | 木 | 宫克卦（制） | 死，受克严重 |

### 旺衰对预测的影响

- **旺相时**：包容力强、群众基础好、田地丰收、母亲健康、房产增值
- **休囚时**：软弱无力、群众不拥护、田地歉收、母亲身体有恙
- **死时**：严重受制、完全被动、腹部疾病严重、女性生殖系统问题
- **入本宫（坤二宫）**：无论季节，力量加成，得地利

## 与其他概念的关系

- [[先天八卦]]：先天坤卦数序为8，居北（下），为地，象征宇宙基底。"天地定位"——坤与乾相对
- [[后天八卦]]：后天坤卦数序为2，居西南，为夏秋之交。"坤为地，西南是四川盆地"
- [[八卦概述]]：坤为八卦之末，纯阴之卦，代表地、母
- [[九宫]]：对应坤二宫，地支为未申，数序为2
- [[五行]]：属土（阴土），土性厚重包容，与坤卦柔顺承载之性相通
- [[阴阳]]：纯阴之卦，三爻皆阴，代表阴的极致
- [[死门]]：坤卦对应死门，死门属土，主死亡、结束、固定不变。死门并非都是凶，在特定场合（如出殡、狩猎）反而是吉
- [[天芮星]]：坤卦对应天芮星，天芮星属土，为病星，主疾病、问题、学生。预测疾病时必看天芮星
- [[十二地支]]：坤宫寄未、申二支，未为土之库，申为金之始
- [[十天干]]：坤属土，对应己土（阴土）之性
- [[六十四卦]]：坤为下卦、上卦的组合形成坤宫八卦（坤为地、地雷复、地泽临、地天泰、雷天大壮、泽天夬、水天需、水地比）
- [[时空统一论]]：坤卦在时空模型中位于西南方、夏秋之交、未申时
- [[河图洛书]]：洛书九宫中坤二宫之数为2（右肩）
- [[三才]]：坤为地，代表三才之"地"的层面
- [[奇门遁甲概述]]：坤宫是关键宫位，死门和天芮星的本位

## 开悟之门补充

### 坤卦在奇门预测中的应用技巧

1. **预测母亲/女性长辈**：看坤宫状态，结合用神判断
2. **预测怀孕生育**：坤为腹，天芮星主孕产，两者结合判断
3. **预测疾病**：看天芮星所落宫位和坤宫的关系，坤宫本身也主脾胃腹部的疾病
4. **预测房地产/土地**：看坤宫和生门（房产相关）的状态
5. **预测群众**：坤为大众，预测群众路线、民主事项看坤宫
6. **预测收藏**：坤含藏之性，预测储蓄、收藏、保存之事看坤宫

### 坤卦与死门的关系

死门居坤二宫，属土。死门是八门中的凶门之一，但并不绝对：
- 死门旺相（四季土月）时，反而有利于出殡、打猎、捕杀等事务
- 死门休囚时，其凶性减轻
- 预测疾病时死门旺相为凶，病情可能加重
- 预测失踪人时，若用神临死门且旺相，则大凶

吉门被迫吉不就，凶门被迫祸重重。死门落其他宫位时需看门宫关系来判断凶的程度。

### 坤卦与天芮星的关系

天芮星为九星之一，属土，居坤二宫。天芮星为病星：
- 预测疾病：必看天芮星，天芮星旺相则病情严重，休囚则病情较轻
- 预测学业：天芮星亦为学生、学子之象
- 天芮星落坤二宫为本位，力量最强

### 坤卦"厚德载物"的现实意义

坤卦的精神是"厚德载物"——以宽广深厚的品德承载万物。在奇门预测中，坤象取用体现了包容、等待、顺应的智慧：很多时候，不需要主动出击，顺势而为、厚积薄发反而是最好的策略。
