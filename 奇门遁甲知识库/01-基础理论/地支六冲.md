# 地支六冲

## 定义与基本属性

**地支六冲**是[[十二地支]]之间最强烈的对立关系。十二地支在方位上两两相对，形成六组对冲组合：

> **子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲**

冲的规律是：在十二地支的圆环排列中，**相隔六位（180度）的地支相对冲**。这正是"六冲"名称的由来——两者之间隔了六个位置。

| 对冲组合 | 方位 | 五行冲突 | 阴阳 |
|----------|------|----------|------|
| 子 ↔ 午 | 北 ↔ 南 | 水 ↔ 火 | 阳 ↔ 阳 |
| 丑 ↔ 未 | 东北 ↔ 西南 | 土 ↔ 土 | 阴 ↔ 阴 |
| 寅 ↔ 申 | 东北偏东 ↔ 西南偏西 | 木 ↔ 金 | 阳 ↔ 阳 |
| 卯 ↔ 酉 | 东 ↔ 西 | 木 ↔ 金 | 阴 ↔ 阴 |
| 辰 ↔ 戌 | 东南偏东 ↔ 西北偏西 | 土 ↔ 土 | 阳 ↔ 阳 |
| 巳 ↔ 亥 | 东南偏南 ↔ 西北偏北 | 火 ↔ 水 | 阴 ↔ 阴 |

六冲的本质是**方位对冲 + 五行相克**的叠加，与[[天干相冲]]在原理上一致。

## 详细象意

### 冲的三大特征

> **冲主动、冲主快、冲为应期**

**1. 冲主动**
- 安静之时逢冲 → 事情将发生变化
- 不动之时逢冲 → 将被迫动起来
- 测工作逢冲 → 可能要换工作、调动
- 测婚姻逢冲 → 感情不稳、有变动

**2. 冲主快**
- 冲代表突然、迅速
- 测事情逢冲 → 来得快去得快
- 测疾病逢冲 → 急性病、病情迅速变化

**3. 冲为应期**
- 这是冲在预测中最重要的作用之一
- 推断事情何时发生，常以"逢冲"为应期
- **关键规律：逢冲以合定应期**——意思是，冲表示事情将发生，但何时发生？等到与冲相关的地支相合之时，就是应期

例如：子午冲 → 应期可能在丑（子丑合）或未（午未合）

### 各冲的具体象意

| 对冲 | 象意 |
|------|------|
| 子午冲 | 水火相激，情绪波动大，明暗之争 |
| 丑未冲 | 土土相冲，固执之争，事情反复 |
| 寅申冲 | 金木交战，道路奔波，竞争激烈 |
| 卯酉冲 | 金木交锋，口舌是非，礼节之争 |
| 辰戌冲 | 土土相冲，库门打开，钱财进出 |
| 巳亥冲 | 水火相激，交通事故，口舌是非 |

## 应用宜忌

- **宜**：
  - 占行人逢冲 → 人快回来了
  - 占事逢冲 → 事情有转机
  - 病情迁延日久逢冲 → 病将有变化（可能好转也可能恶化，需看格局）

- **忌**：
  - 测婚姻逢冲 → 婚姻不稳、有分离之象
  - 测财运逢冲 → 财运被冲散
  - 测合作逢冲 → 合作关系不稳
  - 格局良好逢冲 → 好事被冲散
