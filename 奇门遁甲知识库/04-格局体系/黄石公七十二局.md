# 黄石公七十二局

## 金函玉镜补充

### 定义与基本属性

**黄石公奇门七十二局军中应验神符经**是《金函玉镜》全书中篇幅最大（318行）、最独特的内容。黄石公（张良之师）总结奇门72种格局，每种格局配有：格局名称、星门组合、用兵方向吉凶、应验事象、具体时辰局例。是奇门军事应用的最高指南。

通俗理解：这就像古代的"72张战术卡片"——每一种都有专属的名称、用什么星和门、往哪个方向打好打、出门会看到什么预兆（天空的鸟、地上的动物、遇到的人），照着做就能打赢。

## 第一至第九局（乙奇格局一）

### 第一局：飞龙在天
- **组合**：乙奇 + 天辅星 + 休门
- **断语**：主将军于京地千里，利西方进兵札营，出门有二鸟齐鸣成奇勋也。
- **局例**：阳五局甲己日乙亥时（无九地）、阳三局乙庚日丙戌时（无玄武）、阴三局丙辛日戊子时、阳三局甲己日乙亥时（有天辅）

### 第二局：青云得路
- **组合**：乙奇 + 九天 + 天柱星 + 生门
- **断语**：出军主大胜，得首级五千、地百里，出门有老妪到左逢，军行五百里有壶浆来迎，吉。

### 第三局：太阴当权
- **组合**：乙奇 + 天任星 + 太阴 + 伤门
- **断语**：主大将出阵得胜，有获辎重粮饷之类，军行坎宫五百里有云铺顶，西方有鼓乐，大利。

### 第四局：霹雳雷全
- **组合**：乙奇 + 值符 + 天英星 + 杜门
- **断语**：宜副将出军正将坐营，得地四十里、首级五千余，军行宜兑宫，出门有道人相逢，本营有旗折，大利。
- **局例**：阳六局丙辛日乙未时、阴二局乙庚日乙酉时

### 第五局：天马驰星
- **组合**：乙奇 + 六合 + 景门
- **断语**：军行宜出兑宫，巽宫坐营，有异人献策，得军马于京，出门有大风，行至五百里大雨骤至，吉。
- **局例**：阳八局戊癸日乙卯时（是否天柱星应查）、阴一局戊癸日乙卯时（同局又是天芮）

### 第六局：五虎临门
- **组合**：乙奇 + 白虎 + 天心星 + 死门
- **断语**：宜出巳宫坐午宫，正将大胜，得城池、首级、车马、财帛，军行出门有紫色云气，行三里有黄伞人至，大吉。

### 第七局：月出天衢
- **组合**：乙奇 + 太阴 + 天禽星 + 惊门
- **断语**：军行宜出干宫坐营巳宫，攻城利西北，夺地利东南，出门有三人飞马报献消息，有鹰鸣，大利。

### 第八局：太阴得令
- **组合**：乙奇 + 九天 + 天辅星 + 开门
- **断语**：至于天子御驾亲征，判伏四夷，剿除不法。
- **局例**：阴三局甲己日乙丑时

### 第九局：白虎扬威
- **组合**：乙奇 + 天柱星 + 值符 + 休门
- **断语**：宜劫人营寨、夺人粮草，大胜。宜坎宫出阵，巽宫坐营，主将有功，出门三里有青雀飞噪，途中遇火起，大吉。
- **局例**：阳三局丁壬日乙巳时、阴八局甲己日乙亥时

## 第十至第十八局（乙奇格局二）

### 第十局：天罗地网
- **组合**：乙奇 + 九天 + 天柱星 + 生门
- **断语**：主大将出阵，百战百胜，得地千里，城池十余，所向皆利，军行出门有大风起自西北，东北有雷声相应，大吉。

### 第十一局：六合同春
- **组合**：乙奇 + 太阴 + 天柱星 + 伤门
- **断语**：出军有损攻战，宜与敌人有弓弩埋伏东西，出军宜防之。出行有黄云四起，辰巳时利。

### 第十二局：玉女归关
- **组合**：乙奇 + 六合 + 天柱星 + 杜门
- **断语**：出战大将有伤，利东北不利西南，军行出门微雨，不利。

### 第十三局：喜鹊争巢
- **组合**：乙奇 + 九天 + 天禽星 + 景门
- **断语**：利用火攻，后劫人营寨，得人器械、戈盾之属，军行午时，出门未时，兵行三里遇兔，大吉。

### 第十四局：鸟占枯枝
- **组合**：乙奇 + 朱雀 + 天任星 + 死门
- **断语**：出军主士卒失散，所为不遂，战不成功，军行出门有雁南来，不吉。
- **局例**：阳九局乙庚日丙戌时

### 第十五局：黄龙升天
- **组合**：乙奇 + 螣蛇 + 天辅星 + 惊门
- **断语**：出阵宜水战不宜陆战，水战有东风天助，利火攻，军行五百里有紫色云气，大吉。

### 第十六局：天虎司衡
- **组合**：乙奇 + 太阴 + 天蓬星 + 开门
- **断语**：利守不利战，正将出师有忧，利出巽宫，丁甲焚符可免，途中遇云掩红日，不利。

### 第十七局：天枢直日
- **组合**：乙奇 + 九地 + 天芮星 + 休门
- **断语**：军行宜出干宫坐未宫，不利攻城，若在大寒节乃利攻城，军行出门有五彩人至，大利。
- **局例**：阳三局甲己日乙亥时

### 第十八局：白露横江
- **组合**：乙奇 + 太阴 + 天冲星 + 生门
- **断语**：宜出坎宫或午宫，用符水噀之，利水战，可得首级二千、舟车数百，出门有白鱼入舟，大利。

## 第十九至第二十四局（乙奇格局三）

### 第十九局：风散百花
- **组合**：乙奇 + 六合 + 天禽星 + 伤门
- **断语**：不宜出师，不利主将，有折兵覆舟之厄，军行出门遇西风，有恶风不利。行师若事急不得已，当出巽宫，先书六甲符，噀之、烧之，可免凶煞。
- **局例**：阴四局戊癸日乙卯时

### 第二十局：虎啸生风
- **组合**：乙奇 + 天辅星 + 白虎 + 杜门
- **断语**：军行宜出寅宫，坐巳宫，利陆战，五十里地有伏兵，宜退不宜进，出行有瞽至，主小利。

### 第二十一局：飞蛾扑火
- **组合**：乙奇 + 九天 + 天任星 + 景门
- **断语**：不利守城，宜死战守城，破敌则可，若敌人有反间之计，不可从。出门有白犬当路，不利。

### 第二十二局：尸横遍野
- **组合**：乙奇 + 朱雀 + 天柱星 + 死门
- **断语**：不宜出师，宜固守营垒，谨防西、北二方有贼劫营，午时不利。
- **局例**：阴六局乙庚日乙酉时

### 第二十三局：昆虫变化
- **组合**：乙奇 + 螣蛇 + 天任星 + 惊门
- **断语**：宜出西方，水陆俱利，城池可破，大将可擒，出门有白云当空，行三里天有微雨，大利。

### 第二十四局：天门大开
- **组合**：乙奇 + 天任星 + 入符 + 开门
- **断语**：大将出师，得城池、首级数百，军行宜从坎宫，主闻鼓乐声，有野人献策，大利。
- **局例**：阳八局甲己日乙丑时、阴四局戊癸日乙卯时、阴七局戊癸日乙卯时

## 第二十五至第三十二局（丙奇格局一）

### 第二十五局：五星聚会
- **组合**：丙奇 + 六合 + 天芮星 + 休门
- **断语**：大将出师，宜出坤宫坐巽宫，得首级五千，又得一名大将知天文地理，可招服之。大将出师宜遇大雪，春遇微雨，有狗二只从西方来。
- **局例**：阴九局丁壬日丙午时

### 第二十六局：如鱼失水
- **组合**：丙奇 + 朱雀 + 天任星 + 生门
- **断语**：大将出师不利，主士卒伤火矢，若事急不得已出坎宫，宜书丁甲符，水制之可反凶为吉，出门忌见人持彩物来，不利。

### 第二十七局：得出天罗
- **组合**：丙奇 + 值符 + 天心星 + 伤门
- **断语**：主大将出师，宜出南方坐正北，始不利，主午时大获辎重、器械，途中遇青衣道人持书献策来迎，大吉。

### 第二十八局：六甲逢时
- **组合**：丙奇 + 值符 + 天芮星 + 杜门（遇甲己日）
- **断语**：出军有小伤，士卒皆有忧色，午时出兵大利，得土地、破城，人心归顺，出门五百里有妇人衣色服自西方来，大利。
- **局例**：阳八局丙辛日丙申时

### 第二十九局：鱼龙变化
- **组合**：丙奇 + 勾陈 + 天蓬星 + 景门
- **断语**：主将出师单骑破，虏获首级数十，宜坤宫坐营，军行出门遇仙风、道人，大利。
- **局例**：阴八局丙辛日丁酉时

### 第三十局：宝镜重磨
- **组合**：丙奇 + 勾陈 + 天英星 + 死门（遇午未时）
- **断语**：大将出师，先败后胜，得地五千，宜震宫，出门遇黄衣道人骑白马至，大利。
- **局例**：阳二局丙辛日丙申时、阴二局丙辛日戊戌己亥皆同

### 第三十一局：蛇入龙穴
- **组合**：丙奇 + 太阴 + 惊门
- **断语**：主将出师可，宜入人城池，得金宝数万，军行出门西方有鼓声，白云射斗牛，黄云盖顶，大利。
- **局例**：阳九局戊癸日丙辰时（遗漏星名）

### 第三十二局：鸿鹄冲天
- **组合**：丙奇 + 九天 + 六合 + 开门
- **断语**：主大将出师大胜，可夺人土地，斩将搴旗，军中有员归营，出门东南有红霓射天市垣，大利。
- **局例**：阴五局丁壬日丙午时（遗漏天柱）

## 第三十三至第四十局（丙奇格局二）

### 第三十三局：天罡择地
- **组合**：丙奇 + 螣蛇 + 天芮星 + 休门（遇壬子日）
- **断语**：主将先凶后吉，有弓矢之伤，至吉时得胜，可得土地，杀人盈野。军行出门三里主阴雨微微，大吉。

### 第三十四局：蛇入燕巢
- **组合**：丙奇 + 九地 + 天任星 + 生门
- **断语**：不可出军，有敌人伏兵入营内，伤副将军。军行三里南方有鼓声，主军中有火起，不利。

### 第三十五局：天门大开
- **组合**：丙奇 + 六合 + 天辅星 + 伤门
- **断语**：大将从震宫出可缚敌首，有擒王斩将之功，军行有黑云气起震宫，大利。

### 第三十六局：枯木逢春
- **组合**：丙奇 + 六合 + 天任星 + 杜门（遇庚申日）
- **断语**：主战阵已破之，兵复整出阵，竟成大利。军行出门三里有鸦鸣之声飞过，大吉。

### 第三十七局：蛟龙失水
- **组合**：丙奇 + 九天 + 天芮星 + 景门
- **断语**：主出师不利，折车马、损辎重。若事急宜向兑宫出，以行水解之，亦可半吉。军行出门遇烈日，行至十里有鼓声，大利。

### 第三十八局：天冲临阵
- **组合**：丙奇 + 朱雀 + 天辅星 + 死门
- **断语**：出师宜从震方，主敌人退三舍，得首级四十，获大将一员，出门西北方有巨雷声，二十里大利。

### 第三十九局：风云聚会
- **组合**：丙奇 + 值符 + 天芮星 + 惊门（遇卯未日时）
- **断语**：主大将出师，弃西北出东南，伏兵能得首级五百。军行出门三里有野飞来，大吉。
- **局例**：阳二局甲己日丙寅时、阴二局乙庚日丙子时

### 第四十局：群羊搏虎
- **组合**：丙奇 + 九天 + 天芮星 + 开门
- **断语**：辰戌日时出兵大败，折马卒三千，不宜战。军行遇虎，不利。

## 第四十一至第四十八局（丙奇格局三）

### 第四十一局：天日照临
- **组合**：丙奇 + 太阴 + 天蓬星 + 休门
- **断语**：主兵得胜，吉。缚大将一员，得地三十里，城池可破，军行宜从艮宫出坐坤宫，行至五百里有大火，风雨十里有。
- **局例**：阳三局戊癸日丙辰时、阴八局日时皆同

### 第四十二局：秋风四起
- **组合**：丙奇 + 六合 + 天冲星 + 生门
- **断语**：军行宜在艮宫出，坐在兑宫，四路伏兵，破败敌五百骑，俱不利水战。军行出门有一人献舆图，大利。

### 第四十三局：河魁夺魄
- **组合**：丙奇 + 朱雀 + 伤门（遇午未日）
- **断语**：主将宜出离宫坐坎宫，利水战，可夺舟楫，军行出门西方有火起，大利。
- **局例**：阳一局乙庚日丙子时（是否天任星应查）

### 第四十四局：天乙司冲
- **组合**：丙奇 + 九地 + 天心星 + 杜门（遇申子辰日）
- **断语**：军行宜出兑宫坐震宫，可得地五千里。出门有黄衣人自南方来报献消息，大利。

### 第四十五局：火焚昆冈
- **组合**：丙奇 + 太阴 + 天英星 + 景门
- **断语**：主大将出师百里，破寨斩首级七百。出门遇黄衣人自西来，大吉。
- **局例**：阳二局癸酉日丙辰时、阴七局戊癸日丙辰时

### 第四十六局：天河转运
- **组合**：丙奇 + 六合 + 天柱星 + 死门（遇卯未日时）
- **断语**：出师不利，主伤士卒、车马。军行出师有微雨，行至三百里有群鸟向北飞来，不利。

### 第四十七局：太乙得令
- **组合**：丙奇 + 朱雀 + 天任星 + 惊门
- **断语**：军行出门可得城池，战必胜，攻必取。若在惊蛰时节，军行有黄云从巽宫起，大吉。

### 第四十八局：河魁直时
- **组合**：丙奇 + 勾陈 + 天心星 + 开门（在雨水前）
- **断语**：利水战，可缚敌首，军行遇西方有鼓声，大吉。
- **局例**：阳八局丙辛日丁酉时、阴四局日时同、阳一局丙辛日甲午时

## 第四十九至第五十六局（丁奇格局一）

### 第四十九局：雨霁沧江
- **组合**：丁奇 + 值符 + 天任星 + 休门（遇丑未日时）
- **断语**：出师宜出坎宫坐坤宫，可夺寨、焚粮、用计、伏兵，大胜。军行三里有微雨，遇西风，大利。
- **局例**：阳七局乙庚日丁丑时、阴四局戊癸日丁巳时

### 第五十局：胜光争位
- **组合**：丁奇 + 太阴 + 天禽星 + 生门
- **断语**：出师不利，主大将有灾，士卒多厄。军行宜退不宜进，军行出门遇有北风大起，不利。
- **局例**：阳八局丁壬日乙巳时

### 第五十一局：群鸡化凤
- **组合**：丁奇 + 六合 + 天任星 + 伤门
- **断语**：主将宜出震宫坐离宫，可破敌五百骑，遇午未时更胜。军行有老人携子来见，大吉。

### 第五十二局：火炼金丹
- **组合**：丁奇 + 朱雀 + 杜门（遇寅卯日时）
- **断语**：主大将出师，从兑上出先凶后吉，军行出门遇妇人抱子来见，群鸦飞噪，大吉。
- **局例**：阳四局甲己日丁卯时（遗漏星名）、阴四局同（天柱）、阳二局丙辛日戊子时

### 第五十三局：鼠落糠盘
- **组合**：丁奇 + 勾陈 + 天禽星 + 景门
- **断语**：出师先胜后不成，败大功。军出门见西方有火起，不利。

### 第五十四局：孤舟渡江
- **组合**：丁奇 + 九天 + 天英星 + 死门（遇子午日时）
- **断语**：主大将军宜出坎宫，东方坐营，可斩大将三员者，首级三千。天子出师，可服万国。出兵南方，有五色云见，大吉。

### 第五十五局：太岁坐营
- **组合**：丁奇 + 六合 + 天心星 + 惊门
- **断语**：宜出大兵八方皆利，可得土地数千里，敌兵尽扫。军出南方有大风起，白云见顶，大吉。

### 第五十六局：豹变南山
- **组合**：丁奇 + 值符 + 天辅星 + 开门
- **断语**：主将出师大胜，军出宜出巳宫坐营震宫，出门有二小儿从北方来见，大吉。

## 第五十七至第六十四局（丁奇格局二）

## 第六十五至第七十二局（丁奇格局三）

### 第六十五局：荧惑生光
- **组合**：丁奇 + 天蓬星 + 休门（遇辰戌日）
- **断语**：宜离宫出军，坎宫坐营，宜用火攻，可破敌都邑，俘获上将。出门见黄衣人自北来，不利。

### 第六十六局：罗猴过渡
- **组合**：丁奇 + 值符 + 天辅星 + 生门
- **断语**：宜出师坎宫坐营坤宫，主破敌三十骑，得地百里，斩首级三千。军出遇大风一阵，即不利。

### 第六十七局：月孛当空
- **组合**：丁奇 + 朱雀 + 天心星 + 伤门
- **断语**：出师不利，事急不得已，宜以鬼掌符水噀北方可出，得三里有跛足人自西北来报，大利。

### 第六十八局：月德临门
- **组合**：丁奇 + 勾陈 + 天禽星 + 杜门
- **断语**：宜副将出阵，可得地五千里，宜兑宫坐营，出军东方向西而行，有雷炮声应，大吉。

### 第六十九局：天德照门
- **组合**：丁奇 + 九天 + 天任星 + 景门
- **断语**：利副将出师，军行五里遇伏兵，不可战，行二里东方有火光，不利。

### 第七十局：阴阳不交
- **组合**：丁奇 + 九地 + 天柱星 + 死门
- **断语**：主大将出师，有弓矢之灾。军出不宜北方，利西南方，坐营艮宫，其灾可免。行三里有公差骑马，大吉。

### 第七十一局：三台得位
- **组合**：丁奇 + 九天 + 天任星 + 惊门
- **断语**：出师大胜，宜艮宫出军，坎宫坐营可，军千乘地千里，出门有微雨，大吉。

### 第七十二局：瑞生九鼎
- **组合**：丁奇 + 勾陈 + 天辅星 + 开门
- **断语**：主出师大胜，四方皆利，得城池、土地并得上将首级。军行三里有白云自西北方来，东方风起，皆大利。
