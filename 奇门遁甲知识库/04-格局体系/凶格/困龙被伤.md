# 困龙被伤

## 定义与基本属性

**困龙被伤**是奇门遁甲中天盘辛金加临地盘戊土所形成的凶格，即辛加戊（$\text{辛}_\text{天} + \text{戊}_\text{地}$）。

- **别名**：无
- **组合**：天盘辛 + 地盘戊
- **五行关系**：戊土生辛金（土生金，辛得土助）
- **吉凶**：凶
- **象意核心**：戊为青龙（甲子戊）、为大财，辛为天庭、为刑罚，龙被金伤，屈抑难伸

> **简单理解**：戊是大财也是青龙（贵人），辛是罪错和刑罚。龙被困住了还被打伤——想动动不了，动了就受伤。

## 详细象意

### 自然取象
- 戊为甲子戊、为青龙、为资本、为大山
- 辛为天庭、为刀兵、为刑罚
- 龙困山中，又被刀伤
- 龙本应飞腾于天，却被困被伤

### 人事取象
- **官司**：官非破财、因诉讼损失钱财
- **事业**：有志难伸、才华被压制
- **财务**：破财、投资失败、资金被套牢
- **出行**：受阻、困于途中
- **健康**：筋骨受伤、气血不通

### 特殊凶应
- 官司必然破财
- 屈抑守分尚可——保持低调还能勉强应付
- 妄动带来祸殃——一旦主动出击必然出事
- 进退两难的困境

## 应用宜忌

### 宜事
- **守成**：安分守己、维持现状
- **低调**：韬光养晦、不做出头鸟

### 忌事
- **妄动**：主动出击、贸然行事
- 诉讼打官司：必破财
- 投资理财：资金被困
- 远行：被困途中
- 升职加薪诉求：反遭打压

### 注意事项
1. 此格的核心建议是"忍"——忍一时风平浪静
2. 辛加戊的"戊"是大财，意味着事情与钱有关
3. 只要不动就不会受伤，一动就有凶应
4. 戊也是太岁，此格若在太岁所在宫位更凶
