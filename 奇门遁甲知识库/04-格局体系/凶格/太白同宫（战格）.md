# 太白同宫（战格）

## 定义与基本属性

**太白同宫**（又名战格）是奇门遁甲中天盘庚加临地盘庚所形成的极凶格，即庚加庚（$\text{庚}_\text{天} + \text{庚}_\text{地}$）。

- **别名**：战格、太白重刑
- **组合**：天盘庚 + 地盘庚
- **五行关系**：庚金 + 庚金（金金比和，但庚为极凶，双庚叠加）
- **吉凶**：大凶
- **象意核心**：庚为太白金星、为白虎、为敌人、为阻隔，两个庚相遇，敌人加敌人、障碍叠障碍

> **简单理解**：庚是最大的凶神，一个庚就够麻烦了，两个庚叠加在一起，凶上加凶。就像两把刀架在你脖子上。

## 详细象意

### 自然取象
- 庚为太白金星、为刀兵、为白虎
- 庚也是阻隔、障碍、关卡
- 两个庚叠加——刀兵相接、障碍重重
- 比和之义变为硬碰硬的正面冲突

### 人事取象
- **官司**：官灾横祸，诉讼必然严重
- **兄弟/朋友关系**：兄弟或同辈朋友相互冲撞
- **同行竞争**：同行之间的恶性竞争
- **事业**：处处受阻、关卡不断
- **出行**：路途多阻、交通事故

### 具体凶应

| 领域 | 象意 |
|:---|:---|
| **兄弟** | 兄弟反目、手足相残 |
| **朋友** | 朋友间因竞争而决裂 |
| **同辈** | 同事/同学/同业者之间的恶性对抗 |
| **官非** | 必发官非，双倍严重 |
| **健康** | 外伤、骨折、手术 |

## 应用宜忌

### 忌事
- **兄弟合作**：合伙创业、兄弟同舟——必反目
- 竞技对抗：两败俱伤
- 诉讼官司：双倍不利
- 与朋友争利：得不偿失
- 出行远门：路途险阻

### 宜事
- **独行**：独自行动反而安全
- **远离纷争**：不参与任何竞争对抗
- **居间调停**：作为中间人调和反而有利可图

### 注意事项
1. "战格"之名已说明本质——这就是一个战斗格局
2. 核心在"兄弟/同辈相冲"，所以团队合作的事尤其不宜
3. 两个庚叠加，若有丙奇（火克金）来制，可减其凶
4. 与[[白虎出力]]（辛加庚）不同：白虎出力是主客相残，战格是兄弟相残
5. 庚加庚占出行则路途必受阻

## 与其他概念的关系

- [[庚（阻隔之神）|庚]]：庚为十干中最凶之神，代表阻隔、白虎、敌人
- [[白虎出力]]：辛加庚，同为庚金相关的两强相争格局
- [[荧入太白]]：丙加庚（贼退格），与庚加庚同为庚涉及的凶格
- [[伏宫格]]：庚加值符，庚相关的另一凶格
- [[飞宫格]]：值符加庚，与伏宫格同为庚与值符的组合
- [[五行六亲|兄弟象]]相关的其他格局
