# 奇格

## 定义与基本属性

**奇格**是奇门遁甲中天盘庚金加临地盘三奇（乙丙丁）所形成的一组凶格统称。

- **别名**：庚加三奇格
- **组合**：天盘庚 + 地盘乙/丙/丁
- **五行关系**：庚金克乙木（庚加乙）、丙火克庚金（庚加丙）、丁火克庚金（庚加丁）
- **吉凶**：大凶
- **象意核心**：庚为白虎、为阻隔、为敌人，加临三奇意为"凶神占据吉位"，吉被凶压，大不利

> **简单理解**：三奇是三个好位置，但庚金这个凶神站到它们上面去，就等于"坏人占了指挥部"。

## 详细象意

奇格分为以下三种：

### 1. 庚加乙——合格（又称"合格格"）

| 属性 | 内容 |
|:---|:---|
| **组合** | $\text{庚}_\text{天} + \text{乙}_\text{地}$ |
| **五行** | 庚金合乙木（天地之合） |
| **名称** | 合格（又称） |
| **象意** | 天盘庚合地盘乙，虽为合但庚为主凶，是"不情愿的合" |
| **人事** | 合作不顺、被迫妥协、表面和合内心不合 |
| **出行** | 不宜出行，中途受阻 |
| **用兵** | 不宜用兵，进退两难 |

### 2. 庚加丙——白入荧（贼格）

| 属性 | 内容 |
|:---|:---|
| **组合** | $\text{庚}_\text{天} + \text{丙}_\text{地}$ |
| **五行** | 丙火克庚金 |
| **别名** | 白入荧、贼格 |
| **象意** | 天盘庚（白虎/贼）入地盘丙（荧惑/火），贼入明处 |
| **人事** | 贼人来犯、被暗算、财物损失 |
| **出行** | 出行大凶，必被盗贼所伤 |
| **用兵** | 主贼人入营，防盗 |

> 白入荧与[[荧入太白]]（丙加庚）互为反吟格局。白入荧是"贼来"，荧入太白是"贼走"。

### 3. 庚加丁——破格

| 属性 | 内容 |
|:---|:---|
| **组合** | $\text{庚}_\text{天} + \text{丁}_\text{地}$ |
| **五行** | 丁火克庚金 |
| **别名** | 破格 |
| **象意** | 天盘庚被地盘丁火所克，金被火熔 |
| **人事** | 破败、损坏、计划被破坏 |
| **出行** | 出行不利，半途而废 |
| **用兵** | 大凶，阵型被破 |

## 应用宜忌

### 忌事
- **出行用兵**：三种奇格均大凶，不可出行、不可用兵
- 谈判合作：易被压制、被迫妥协
- 签约文书：合同易出问题
- 拜访贵人：反遭冷遇

### 注意事项
1. 庚加乙（合格）相对庚加丙、庚加丁略轻——因为有合，未必全凶
2. 三种奇格虽五行有生有克，但庚为凶神，统论为凶
3. 若有值符、太岁所在宫来救应，凶力可减
4. 奇格重在出行用兵之事，日常小事可不论

## 与其他概念的关系

- [[荧入太白]]：丙加庚，与庚加丙（白入荧）互为反吟，同为凶格
- [[庚（阻隔之神）|庚]]：庚为十干中最凶之神，代表阻隔、白虎、敌人
- [[三奇（乙丙丁）|三奇]]：乙丙丁为三奇，被庚临则失其吉力
- [[白虎猖狂]]：辛加乙，同为凶神克三奇
- [[白虎出力]]：辛加庚，同为庚辛相关的凶格
- [[岁格、月格、日格、时格、飞干格]]：天盘庚加地盘年/月/日/时干，扩展的凶格体系

## 经典断语

> "庚加三奇为奇格：庚加乙为合格，庚加丙为白入荧，庚加丁为破格。出行用兵均大凶。"
