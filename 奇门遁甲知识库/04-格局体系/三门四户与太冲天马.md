# 三门四户与太冲天马

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]奇格异局章

## 适用范围

**仅适用于转盘奇门**。飞盘重地气需补天气（与太乙结合），转盘接天气需补人事（与六壬结合）。天三门与地四户如果与飞盘奇门结合是南辕北辙、画蛇添足。

## 天三门

### 方法

以月将加于占时之上，顺时旋转成天地二盘，观**卯、未、酉下临之支**即为天三门。

### 十二月将

| 月将 | 月份 | 地支 |
|------|------|------|
| 登明 | 正月 | 亥 |
| 河魁 | 二月 | 戌 |
| 从魁 | 三月 | 酉 |
| 传送 | 四月 | 申 |
| 小吉 | 五月 | 未 |
| 胜光 | 六月 | 午 |
| 天乙 | 七月 | 巳 |
| 天罡 | 八月 | 辰 |
| 太冲 | 九月 | 卯 |
| 功曹 | 十月 | 寅 |
| 大吉 | 十一月 | 丑 |
| 神后 | 十二月 | 子 |

### 用法

- 紧急情况下**逃生避难**
- 地理位置选择（如地产集团竞拍多个地块时作堪舆择方参考）
- 但天三门所在宫位内部格局不能大凶——如果本身格局大凶再受冲制则百无一用

> 占测时要首重奇门格局大象，不能完全用这些局外异格代替奇门本身格局的吉凶。

## 地四户

### 方法

将建除十二神加于占时之上，**除、定、危、开下临之支**即为地四户。

### 建除十二神

建、除、满、平、定、执、破、危、成、收、开、闭（依次顺序排列）

### 用法

- 利于**暗中行事**（偷袭、伏兵、设伏）
- 利于逃难求生
- 同样以所在宫位吉凶衰旺为前提条件

### 地私门辨析

张大春先生明确：地私门不能应用于时家奇门：
1. 以日支为基准点，与奇门的旬内占时法相左
2. 阴阳贵人起法异于奇门重阳抑阴法则
3. 六合、太阴、太常三神只完整出现于飞盘，转盘奇门中太常不出现

## 太冲天马

### 方法

以月将加占时成天地二盘，观**天盘卯下临之位**即为天马之方。

### 核心特征

- **速度**：天马显其速——速度越快与局象越相符，事半功倍
- **光明正大**：与三门四户的暗中行事不同
- 古代应用中：李靖建议唐太宗以少量精锐骑兵突击敌阵即太冲天马格局的应用

### 与三门四户的差异

| 特征 | 三门四户 | 太冲天马 |
|------|----------|----------|
| 方式 | 暗中策划 | 光明正大 |
| 速度 | 一般 | 强调快速 |
| 适用 | 趋避求生 | 强冲暴击 |

## 奇门宝鉴补充

> 出处：《御定奇门遁甲·奇门宝鉴》（金志文点校）卷一·释义四十四则 — 参见 [[御定奇门遁甲奇门宝鉴（金志文）]]

### 释三门四户

**三门**者，太冲、小吉、从魁也（即卯、未、酉三将）。
**四户**者，除、危、定、开也（即建除十二神中之四神）。

> 如立春后建寅之月，则除卯、危酉、定午、开子之类也。

凡急难时，不及择奇门，以月将月建加正时，视三将四建之下，为**天门地户**，乘其方去，百恶不能害。

### 释天马

天马者，**房日兔**也。房为天驷，故曰「天马」。兔隶卯，卯为太冲，每月太冲下天马方，又曰「太冲天马」。

凡急难时，不及择奇门，以月将加月建，视太冲所临之下，乘其方去，**凶恶不能侵，剑戟不足畏**。

### 释地私门

私门者，**太阴、六合、太常**所临之方也。

以月将加正时，用日干寻纳甲合神所纳之支，即以贵人泊其支，依阴阳顺逆，布定十二贵神，然后视三神所临之下，即**地私门方**。

> 私门之方，更得奇门凑合，百事乘之大吉。

### 三门四户命名之本义（重要纠正）

奇门宝鉴指出三氏诸书之非：

> 三氏诸书，以地户用时建法。月建四孟，则四仲时为地户；月建四仲，则四季时为地户；月建四季，则四孟时为地户。是**用地户时**，而非**用地户方**矣。

考小吉，未将也；太冲，卯将也；从魁，酉将也。

- **未**为二气交会之门
- **卯酉**为日月出入之门

故以三神为天门也。

- **太常**象于未
- **六合**象于卯
- **太阴**象于酉

故以三神为私门也。

> 三氏谓「小吉为门」，其义莫解。曷若去小吉，以成**参天两地**更妙。
