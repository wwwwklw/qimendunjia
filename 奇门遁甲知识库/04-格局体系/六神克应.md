# 六神克应

> 出处：《奇门遁甲统宗》卷之三·六神克应 — 参见 [[奇门遁甲统宗（诸葛亮）]]

## 定义

**六神克应**是《奇门遁甲统宗》卷三所载的**六神（六兽）所主之物类克应**，详列青龙、[[朱雀（六兽）|朱雀]]、[[勾陈]]、[[腾蛇]]、[[白虎]]、[[玄武]]六神所对应的人物、事物、物类。

## 六神克应详表

### 青龙

> 青龙：为儒者，为僧道，为医生，为书客，为轿伞，为鱼，为树枝。为喜庆事，如嫁娶、祝寿、馈送等事。

| 类别 | 所主 |
|------|------|
| 人物 | 儒者、僧道、医生、书客 |
| 物品 | 轿伞、鱼、树枝 |
| 事件 | 喜庆事：嫁娶、祝寿、馈送 |

### 朱雀

> 朱雀：为文人，为书吏，为口舌，为呼唤，为牙，行人，为哭啼，为哭语，为飞禽，为红色物。

| 类别 | 所主 |
|------|------|
| 人物 | 文人、书吏、行人 |
| 物品 | 牙、飞禽、红色物 |
| 事件 | 口舌、呼唤、哭啼、哭语 |

### 勾陈

> 勾陈：为流作，为农人，为跎子，为牵牛羊人，为捕捉，为公门人，为拗子，为提携，为跌，为阻塞，为争斗。

| 类别 | 所主 |
|------|------|
| 人物 | 流作、农人、跎子、牵牛羊人、公门人、拗子 |
| 物品 | 提携之物 |
| 事件 | 捕捉、跌、阻塞、争斗 |

### 腾蛇

> 腾蛇：为妇女，为卖婆，为朵花，为骂詈，为奇闻，异见为蛇，为绳索，为污秽，为臭气。

| 类别 | 所主 |
|------|------|
| 人物 | 妇女、卖婆 |
| 物品 | 朵花、蛇、绳索 |
| 事件 | 骂詈、奇闻异见 |
| 性质 | 污秽、臭气 |

### 白虎

> 白虎：为恶人，为屠人，为病人，为孝服人，为血光，为斗打，为行丧，为石匠，为刀斧，为鼓声，为庖人。

| 类别 | 所主 |
|------|------|
| 人物 | 恶人、屠人、病人、孝服人、石匠、庖人 |
| 物品 | 刀斧、鼓声 |
| 事件 | 血光、斗打、行丧 |

### 玄武

> 玄武：为卖鱼人，为卖盐人，为乞丐，为\*，为贼人，为酒醉人，为雨伞，为木匠，为水夫，为鱼盐油酒。

| 类别 | 所主 |
|------|------|
| 人物 | 卖鱼人、卖盐人、乞丐、贼人、酒醉人、木匠、水夫 |
| 物品 | 雨伞、鱼盐油酒 |
| 性质 | 阴私、\*邪 |

## 应用法则

### 推占克应

六神克应主要用于：

1. **行人推占**：见六神临何宫，推占来者身份
2. **物类推占**：见六神所主之物以断物之形类
3. **事件推占**：见六神所主之事以断吉凶

### 与八卦克应、地支克应并用

六神克应常与[[八卦概述|八卦克应]]、[[十二地支|地支克应]]并用，综合推占来人、物、事之吉凶。
