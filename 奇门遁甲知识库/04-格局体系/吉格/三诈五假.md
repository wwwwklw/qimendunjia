# 三诈五假

## 定义与基本属性

**三诈五假**是奇门遁甲中两组特殊格局的合称，由吉门、三奇、八神组合而成，共八种格局。

- **三诈**：三吉门+三奇+特定八神
- **五假**：五事（天/地/人/神/鬼）+ 特定组合
- **吉凶**：吉（但各有专用范围）

---

## 三诈

三诈是指三吉门加三奇再加特定八神形成的格局。

| 名称 | 组合 | 条件 | 象意 |
|:---:|:---|:---|:---|
| **真诈** | 三吉门 + 三奇 + 太阴 | 开/休/生门 + 乙/丙/丁 + 太阴 | 暗中行事、秘密行事大吉 |
| **休诈** | 三吉门 + 三奇 + 六合 | 开/休/生门 + 乙/丙/丁 + 六合 | 合作、合伙、交易大吉 |
| **重诈** | 三吉门 + 三奇 + 九地 | 开/休/生门 + 乙/丙/丁 + 九地 | 埋伏、隐藏、储蓄大吉 |

### 三诈详释

| 格局 | 宜事 | 忌事 | 象意 |
|:---|:---|:---|:---|
| **真诈** | 秘密行动、暗中行事、夜间活动 | 不宜公开透明之事 | 太阴为阴私，得奇门相助，暗中得利 |
| **休诈** | 合作、合伙、婚姻、交易 | 不宜独自行事 | 六合为合作之神，得奇门相助，合作顺利 |
| **重诈** | 隐藏、埋伏、储蓄、积蓄 | 不宜主动出击 | 九地为地下、隐藏，得奇门相助，深藏不露 |

## 五假

五假是以"天、地、人、神、鬼"五事命名的五种特殊格局。

### 天假
- **组合**：景门 + 三奇
- **条件**：天盘景门 + 天盘三奇之一
- **吉凶**：吉
- **象意**：借天之力，宜求官、谒贵、上书
- **宜事**：谒见贵人、求取功名、上书献策

### 地假
- **组合**：杜门 + 丁/己/癸
- **条件**：天盘杜门 + 地盘丁/己/癸之一
- **吉凶**：吉（专用）
- **象意**：借地之力，宜潜伏、藏身、侦探
- **宜事**：隐匿藏形、暗中侦查、潜逃

### 人假
- **组合**：惊门 + 六合 + 壬
- **条件**：惊门 + 六合 + 地盘壬
- **吉凶**：中平
- **象意**：借人之力，宜捕盗、抓人
- **宜事**：抓捕逃犯、围捕

### 神假（神遁之变）
- **组合**：伤门 + 值符/九天
- **条件**：伤门 + 值符或九天
- **吉凶**：中平
- **象意**：借神之力，宜祈求、祭祀
- **宜事**：祭祀、祈祷

### 鬼假（鬼遁之变）
- **组合**：死门 + 九地 + 丁/己
- **条件**：死门 + 九地 + 地盘丁或己
- **吉凶**：中平（专用）
- **象意**：借鬼之力，宜丧葬、超度
- **宜事**：安葬、超度亡灵

## 应用宜忌

### 三诈应用要点
1. **真诈**偏暗事，利阴不利阳
2. **休诈**偏合作，利合不利分
3. **重诈**偏深藏，利藏不利露

### 五假应用要点
1. 五假各有所主，不可混用
2. 五假之力不如三诈强
3. 需严格确认条件是否齐备

### 注意事项
1. 三诈五假都需要特定条件，缺一不可
2. 日常生活中三诈比五假更实用
3. 五假多数为古代特定用途（如军事、祭祀）

## 与其他概念的关系

- [[天遁]]、[[地遁]]、[[人遁]]：九遁中的前三遁，与三诈五假同为门奇神组合
- [[鬼遁]]：丁+杜门+九地，与鬼假有相似之处
- [[神遁]]：丙+生门+九天，与神假有相似之处
- [[三奇得使]]：三奇加值使门
