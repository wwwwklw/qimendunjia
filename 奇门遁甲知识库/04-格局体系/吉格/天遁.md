# 天遁

## 定义与基本属性

**天遁**是奇门遁甲"九遁"格局之一，也是最高级别的遁格。

- **别名**：无
- **组合**：天盘丙奇 + 生门 + 地盘丁奇
- **条件**：$\text{丙}_\text{天} + \text{生门} + \text{丁}_\text{地}$
- **吉凶**：大吉
- **象意核心**：丙为天威（月奇），生门为生发之门，丁为星奇，三者会合为天遁，得天时、地利、人和之全

## 详细象意

### 自然取象
- 丙为太阳，在天照耀
- 生门为生长之门，为地之利
- 丁为星奇，为人之文
- 日月星三光汇聚，天地人三才合一

### 人事取象
- **战争**：攻无不克、战无不胜
- **事业**：万事亨通、飞黄腾达
- **求财**：财运亨通
- **出行**：出行大利

### 吉应
- 行军打仗大吉
- 百事皆吉
- 谋为必成
- 所求皆得

## 应用宜忌

### 宜事
- 战争征伐（古代军事第一吉格）
- 开业远行
- 求财求官
- 一切主动出击之事

### 忌事
- 无特别禁忌
- 但需确认丙、生门、丁三者齐备

### 注意事项
1. 天遁是古代军事上的第一吉格
2. 必须同时满足三个条件：天盘丙+生门+地盘丁
3. 在其他任何一位缺失的情况下不构成天遁

## 与其他概念的关系

- [[地遁]]：乙+开门+己，遁格之二
- [[人遁]]：丁+休门+太阴，遁格之三
- [[风遁]]、[[云遁]]、[[龙遁]]、[[虎遁]]、[[神遁]]、[[鬼遁]]：九遁的其余六格
- [[十千克应概述]]：丙加丁为星奇朱雀，吉格
- [[八门克应概述]]：生门加丁奇

## 经典断语

> "天遁，丙+生门+丁，得天时地利人和，百事大吉，攻无不克。"

## 金函玉镜补充

> **source:** [[金函玉镜奇门遁甲秘笈全书（诸葛亮 刘伯温）]] 卷二十一

### 天遁歌诀

> 「丙合生门居丁上，便为天遁月光辉。」

### 卷二十一详述

金函玉镜卷二十一对天遁的详细描述：

- **条件**：丙奇临生门下合六丁，不犯奇墓刑
- **得蔽**：得月精所蔽，内应其心，外主其身，其气上升于首，其质下注于心，名曰「元珠」。能听而修者升天
- **玉女咒**：「丁卯玉女，护我佑我，毋令有鬼伤我，视我者瞽，恶我者反受其殃。」咒毕行之，慎勿反顾
- **宜事**：称王侯之权、利朝君王、谢苍穹、礼求福、利征战使敌自伏、上书献策、求官进职、修身隐迹、剪恶除凶、市贾出行、婚姻入宅——百事俱吉

## 金函玉镜补充

> 出自《金函玉镜奇门遁甲秘笈全书》卷六——九遁变化法。

### 金函玉镜版天遁判断条件

金函玉镜中天遁的条件较《神奇之门》更宽泛：

1. **主条件**：开、休、生三吉门，天盘丙奇下临地盘丁奇者，皆名「天遁」，得月精所蔽。
2. **扩展条件**：或临地盘九地、太阴者，亦名「天遁」，可以遁迹、隐形。

### 八神配合详论

金函玉镜对各神煞在天遁中的影响有详细说明：

| 乘神 | 影响 |
|------|------|
| 天月二德、天恩、天赦、日禄、喜神 | 百事皆吉 |
| 朱雀 | 利文书、奏对 |
| 螣蛇 | 主疑惑 |
| 青龙 | 主财喜 |
| 白虎、玄武 | 主灾病、损失，细事不得全吉 |

> **特殊用法**：若练祭丁甲，呼风唤雨等项之事，用「天遁」可以全吉也。

### 与《神奇之门》差异

- 《神奇之门》要求「丙+生门+丁」三者特定组合
- 金函玉镜更宽：三吉门（开休生）任一+丙加丁即可，且九地/太阴亦可替代丁奇
