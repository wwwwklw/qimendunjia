# 吉格概述

吉格是奇门遁甲中**吉利格局**的总称，由吉门、吉星、吉神与[[三奇（乙丙丁）]]相配合而成。

## 核心原理

- **吉门+吉星+吉神+三奇** → 吉格
- 星、门、宫、奇、仪之间五行相生或比和为吉
- [[甲（遁甲）]]、[[乙奇]]、[[丙奇]]、[[丁奇]]、[[甲（遁甲）|甲子戊]]（[[五阳干与五阴干|五阳干]]）组合多为吉

## 金函玉镜补充

以下内容引自《金函玉镜奇门遁甲秘笈全书》卷十五"奇门吉格"。

### 九遁详解（金函玉镜版）

金函玉镜中的九遁定义更为详细，补录如下：

**天遁**：生门主兴隆合丙奇，临下六丁是也。此遁得月华之蔽，宜行兵、献策、摄王侯之权，百事生旺。又开门与六丙亦相同，宜祭祷天神。

**地遁**：开门合乙奇，临地下六己是也。此遁得日精之蔽，宜安葬、埋伏、出门、造屋一切皆吉。

**人遁**：休门与丁奇合太阴是也。此遁得星精之蔽，宜密探贼营、隐迹、谒贵、招贤、求亲、经营、求将、对敌、盟誓，吉；百事和集。

**风遁**：乙奇合休开生临巽是也。又开丙临巽亦是。此时风从西北方来名"天罡风"，宜顺风击贼。如风从东来名"雷门风"，若贼在东方不可与战。风从东南来名"大门风"，如贼在南方亦不可与战。《参筹秘书》云："开门乙奇临巽，宜祭风伯，以火攻敌，战立旌旗，以候风应"。

**云遁**：乙合开休生，下临六辛宫，又乙开临坤亦是。此遁得云精之蔽，宜藏形、蔽敌。如云气形如妇人双手双足，主三日有喜，所临处吉。白云在外中有黑云，主有伏兵。形似雁行，主有大将出。形如坐狗，有奇兵埋伏。云震片片被风吹断，主大败。一云：乙休临坤，利求雨泽、建立云寨、制造军器，以候云应。

**龙遁**：休门与乙奇临六甲于坎宫，临六癸亦是。此时宜祭龙神、祈求雨泽、掩捕贼人、密计渡河、把守水口、设机伏谋、攻敌、量水面，得龙神助，又利水战。

**虎遁**：乙奇合辛临休门于艮宫，又辛仪合生临艮亦是。此时宜招安亡命及战用急攻，必胜；设伏游击、藏兵、计渡要害、道路量险，用谋得虎之威。一云：生门丙奇合辛亦是，利守御，宜建立山寨，以候虎应。

**神遁**：丙奇合九天临生门，此时宜祷神灵、行神术、画地布筹、禹罡、造坛、驱邪、命将、呼风召雨，及制伏鬼神魔魅，得天灵百神之蔽。一云："修塑神像，神灵佑之"。

**鬼遁**：生门与丁奇、九地临艮。又乙奇合九地于杜门亦是。此时宜哨探贼机、偷营刦寨、设伪为伏虚、驱神役鬼，得鬼神之蔽。一云："开门、丁奇合九地亦是鬼遁"。

### 三奇上吉门格

乙丙丁三奇到，而无开休生三吉门者，不可用。如三吉门到，而无三奇者，尚可用。经云："背生向死，百战百胜"。总之三奇吉星、吉门俱到，又不犯格，方为万全。若事在危急，诸星不到，须用背生向死之法。如敌势尚缓，候三奇吉门俱到，斯为全美矣。

## 上元经补充

### 九遁的性质和适用范围

九遁（天遁、地遁、人遁、风遁、云遁、龙遁、虎遁、神遁、鬼遁）在法术奇门中有广泛使用，但在**普通奇门象数理预测中只能作为一种辅助参考**，干支的阴阳五行克应起主导作用。

#### 各遁构成条件

| 遁名 | 核心组合 | 关键条件 |
|------|----------|----------|
| **天遁** | 丙加丁 | 门盘为生门或开门（月星相会，夜空中只有月+星，故为天遁） |
| **地遁** | 乙加己 | 开门（日奇在黑夜不能发挥作用，开门等于打开地下通道） |
| **人遁** | 丁+地盘三奇与戊壬 | 太阴+休门 |
| **神遁** | 丙+地盘三奇与辛 | 生门+九天 |
| **鬼遁** | 丁+地盘三奇与戊 | 九天/九地+杜门（杜门阻隔阴阳） |
| **风遁** | 乙落巽宫 | 三吉门（仅用于法术奇门） |
| **云遁** | 乙加辛 | 三吉门（法术奇门中可借助风势击杀敌人） |
| **龙遁** | 乙+坎宫癸 | 适于水战伏兵 |
| **虎遁** | 乙加辛落艮宫 | 辛金在艮宫被穿，庚金在艮宫入墓击刑，白虎无力 |

#### 关键辨析

- **鬼遁只能用杜门**：有的书中说开门也能成鬼遁是错误的。杜门起阴阳阻隔作用，开门无此效能
- **天假不包含乙奇**：日奇在奇门天象体系中不直接显现，借上天之气的天假只含丙丁二奇
- **地假不临杜门**：杜门五行克土，与地假借助地气的原理不符
- **虎遁无法在兑宫**：部分书籍把庚落兑宫看成虎遁是荒谬的，兑宫白虎当旺

### 三诈：法术奇门格局

三奇乙丙丁合开生休三吉门构成三诈必要条件：
- 临**太阴**为真诈（主天时）
- 临**九地**为重诈（主地利）
- 临**六合**为休诈（主人事——如占测人员逃遁方向可看六合落宫）

### 三奇得使的真义

张大春先生指出《烟波钓叟歌》"三奇得使"的解释真义：
> "乙逢犬马丙鼠猴，六丁玉女骑龙虎"中的乙加己/乙加辛、丙加戊/丙加庚、丁加壬/丁加癸并不都是一般意义上的吉格。

《烟波钓叟歌》杂糅了法术奇门元素，三奇得使是**法术奇门中对三奇和值使门的应用**——普通的数理奇门预测只需根据阴阳五行克应规则判断即可。
