# 地遁

## 定义与基本属性

**地遁**是奇门遁甲"九遁"格局之一。

- **别名**：无
- **组合**：天盘乙奇 + 开门 + 地盘己仪
- **条件**：$\text{乙}_\text{天} + \text{开门} + \text{己}_\text{地}$
- **吉凶**：吉
- **象意核心**：乙为日奇（柔木），开门为通达之门，己为地户，日光照地户而开，地下通达之象

## 详细象意

### 自然取象
- 乙为日奇，为草木、为柔顺
- 开门为金，为开通、为开创
- 己为地户，为阴土、为隐藏
- 日光照开地户，地中之物可出

### 人事取象
- **藏身**：宜藏形匿迹
- **出行**：宜出地下通道
- **安葬**：宜安营扎寨、埋葬
- **建设**：宜修筑工事

### 吉应
- 藏兵埋伏大吉
- 修筑工事吉
- 安葬吉
- 求财亦有得

## 应用宜忌

### 宜事
- 埋伏、隐藏
- 修筑、建设
- 安葬、丧事
- 储蓄、埋财

### 忌事
- 不宜公开行动
- 不宜主动出击

### 注意事项
1. 地遁偏重"地下"之事，与天遁的"光明"不同
2. 必须同时满足三个条件：天盘乙+开门+地盘己
3. 是专门宜"暗事"的吉格

## 与其他概念的关系

- [[天遁]]：丙+生门+丁，最高级别的遁格
- [[人遁]]：丁+休门+太阴
- [[风遁]]、[[云遁]]、[[龙遁]]、[[虎遁]]、[[神遁]]、[[鬼遁]]：九遁的其余格
- [[十千克应概述]]：乙加己为日奇入墓，但在遁格中得开门化解

## 经典断语

> "地遁，乙+开门+己，宜藏形匿迹、修筑安葬。"

## 金函玉镜补充

> **source:** [[金函玉镜奇门遁甲秘笈全书（诸葛亮 刘伯温）]] 卷二十一

### 地遁歌诀

> 「乙合开门临六己，地遁日精能蔽之。」

### 卷二十一详述

金函玉镜对地遁的详细描述：
- **条件**：乙奇与开门合，下临六己，得日精所蔽
- **得蔽**：其气黄，内应其脾，外应其形，又名「黄婆金公」。能修之者南宫列仙
- **宜事**：藏兵伏将、扎寨安营、建府造仓、筑垣墙、安坟开圹、修道求仙、逃亡绝迹、出阵攻城——全师捷胜，百事大吉

> 「盖己为地户，得日精盖之。」

## 出处

《神奇之门》第2章；《金函玉镜奇门遁甲秘笈全书》卷二十一

## 金函玉镜补充

> 出自《金函玉镜奇门遁甲秘笈全书》卷六——九遁变化法。

### 金函玉镜版地遁判断条件

金函玉镜中地遁的条件与《神奇之门》有显著差异：

1. **主条件**：开、休、生三吉门合乙奇，下临地盘六合、九地、太阴宫者，皆名「地遁」，得日精所蔽，可以设伏、谋为，百事皆吉。

> **注**：《神奇之门》要求「乙+开门+己」，金函玉镜不要求己仪，而是要求八神（六合/九地/太阴），且三吉门任一即可。

### 八神配合详论

| 乘神 | 影响 |
|------|------|
| 朱雀 | 惟宜设词、行诈、间谍 |
| 螣蛇 | 宜蛊惑 |
| 太常 | 有酒食、宴会 |
| 青龙 | 有财喜 |
| 白虎 | 主斗胜 |
| 玄武 | 主窃盗、探私 |
| 勾陈 | 主淹滞 |

> 得地遁之吉者，宜修筑、起造、埋葬、藏匿之事，俱皆大吉。
