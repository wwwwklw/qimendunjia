# 地假

## 定义与基本属性

**地假**是奇门遁甲"五假"之一，由杜门、特定天干、九地或太阴或六合组合而成的专用吉格。

- **别名**：无
- **组合**：**杜门** + **丁/己/癸**（三者之一）+ **九地**或**太阴**或**六合**
- **条件**：杜门在宫 + 天盘丁或己或癸 + 神盘九地/太阴/六合，三者同宫
- **吉凶**：吉（专用）
- **象意核心**：杜门为隐匿闭塞之门，加丁/己/癸皆有隐蔽之性，配以阴隐之神（九地/太阴/六合），借地之力潜伏藏身

> **简单理解**：地假就是"借地的力量躲起来"。杜门是关门躲藏的门，配上隐蔽的天干和神，就能藏得严严实实，适合躲灾、潜伏、暗中侦查。

## 详细象意

### 组合本质
| 要素 | 含义 |
|:---|:---|
| **杜门** | 隐匿、闭塞、关门 |
| **丁/己/癸** | 丁为星光隐蔽、己为地下、癸为天网暗水 |
| **九地/太阴/六合** | 地下、阴私、结合——均为隐蔽之神 |

### 三种变化

| 天干 | 八神 | 象意侧重 |
|:---|:---|:---|
| **丁** + 九地 | 星火藏于地下 | 暗中侦查、潜藏 |
| **己** + 太阴 | 阴土配阴神 | 逃亡躲灾 |
| **癸** + 六合 | 天网暗藏 | 谋探私事 |

### 人事取象
- **潜藏**：躲藏起来不被发现
- **埋伏**：暗中埋伏等待
- **逃亡**：躲避灾祸、逃亡避祸
- **侦探**：暗中调查、侦查私事
- **隐身**：暂时隐退、避免曝光

## 应用宜忌

### 宜事
- **潜藏埋伏**：躲藏不被发现
- **逃亡躲灾**：避祸躲灾
- **谋探私事**：暗中调查
- **侦查取证**：秘密调查
- **隐身避世**：暂时退出公众视野

### 忌事
- 公开活动（与地假相反）
- 高调宣扬
- 主动出击（潜伏阶段不宜妄动）

### 注意事项
1. 地假是"借地之力"，宜向下、宜隐藏
2. 各组合有细微差别：丁重谋探、己重逃亡、癸重暗事
3. 五假之力不如三诈强，但有专用场景
4. 现代社会适合：暗中调查取证、暂时避风头
