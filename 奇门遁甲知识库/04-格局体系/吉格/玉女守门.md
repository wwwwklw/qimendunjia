# 玉女守门

## 定义与基本属性

**玉女守门**是奇门遁甲中的吉格之一，指值使门落宫遇到地盘丁奇。

- **别名**：无
- **组合**：值使门所在宫位 + 地盘丁奇
- **吉凶**：吉
- **象意核心**：丁为玉女，为阴火之星，守于值使门，如美女守门，凡事有喜

## 详细象意

### 自然取象
- 丁为玉女，属阴火，为文明之象
- 值使门为执行之门
- 玉女守门，门内有喜

### 人事取象
- **婚姻**：喜事临门、婚嫁大利
- **宴会**：适合请客、聚会
- **文书**：书信往来、消息传达
- **女性**：与女性相关之事皆宜

### 吉应
- 婚嫁喜庆
- 宴请宾客大吉
- 文书信息有利
- 女性谋事顺利

## 应用宜忌

### 宜事
- 结婚嫁娶
- 宴请宾客
- 书信往来
- 与女性相关的事务

### 忌事
- 无特别禁忌
- 不宜征战杀伐

### 注意事项
1. 玉女守门以"喜"为核心，偏向喜事
2. 与其他吉格叠加更佳
3. 要注意值使门的吉凶属性（值使为凶门则减力）

## 与其他概念的关系

- [[三奇得使]]：天盘三奇加地盘值使门，玉女守门是值使门临地盘丁奇
- [[天遁]]：丙+生门+丁，遁格之一种
- [[奇仪相合]]：丁壬合，丁相关的相合关系

## 经典断语

> "玉女守门，值使门落宫遇地盘丁奇，宜婚嫁宴会。"

## 上元经补充

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]奇格异局章

### 玉女守门的纠正

张大春先生对传统理解作出重要纠正：

> 丁为玉女，门为值使门，**只有在丁为天盘干、门为天盘值使门的情况下**才成立。

部分书籍说玉女守门是天盘值使门与地盘丁奇的组合，这不符易理。原因是奇门中所有门、星、干、神都有明暗两套对应符号系统：
- 天盘值使门处于**阳性动态**，与天盘丁形成阳性对应关系
- 地盘丁处于**阴性状态**，分属不同层面

> 法术奇门中天盘值使门与天盘丁之间可形成很大合力；数理奇门中遇丁加癸等凶格，玉女守门也不能转凶为吉。

> 出处：《奇门遁甲上元经》 — 参见 [[奇门遁甲上元经（张大春）]]奇格异局章

## 奇门宝鉴补充

> 出处：《御定奇门遁甲·奇门宝鉴》（金志文点校）卷一·释义四十四则 — 参见 [[御定奇门遁甲奇门宝鉴（金志文点校）]]

### 释玉女守门

玉女守门者，**地盘六丁守直使之门**也。

| 直符 | 时辰 |
|------|------|
| 甲子直符 | 庚午时 |
| 甲戌直符 | 己卯时 |
| 甲申直符 | 戊子时 |
| 甲午直符 | 丁酉时 |
| 甲辰直符 | 丙午时 |
| 甲寅直符 | 乙卯时 |

值此六时，则直使遁于丁奇所在之宫。丁为玉女，故曰「玉女守门」也。

#### 所宜之事

- **利谋秘密、阴私之事**：乘玉女守门方出，人不能见
- **入宜营建、宴会、和乐之事**
- 若遇三奇吉门直使，又得太阴临合地户，**宜福食远行，出入皆吉**

### 玉女守门与三奇游六仪之辨

三氏诸书（李氏、林氏、陶氏）皆以「玉女守门」为「三奇游六仪」，奇门宝鉴指出此说之非：

> 玉女守门者，守其宫以待直使之门，来加所遁之时也。玉女，丁奇也。既以丁奇所在之宫，守直使来加为守门，只可谓之「八门游丁奇」，奚可谓之「三奇游六仪」乎？

考《钓叟歌》：「号为三奇游六仪，又有玉女守门时」——歌义明分游仪、守门为二格，三氏混而为一，于歌义大相戾矣。
