# 神假

## 定义与基本属性

**神假**是奇门遁甲"五假"之一，由伤门、丁/己/癸、九地组合而成的专用吉格。

- **别名**：无
- **组合**：**伤门** + **丁/己/癸**（三者之一）+ **九地**
- **条件**：伤门在宫 + 天盘丁/己/癸 + 神盘九地，三者同宫
- **吉凶**：吉（专用）
- **象意核心**：伤门为伤害变动之门，丁/己/癸多有隐蔽之意，九地深藏地下，借神之力埋藏隐藏

> **简单理解**：神假是"借神的力量把东西藏起来"。伤门本身不吉，但配合隐蔽的天干和九地，反而能让人把东西藏得无人知晓。

## 详细象意

### 组合本质
| 要素 | 含义 |
|:---|:---|
| **伤门** | 伤害、变动、改变状态 |
| **丁/己/癸** | 丁为星光隐蔽、己为地下、癸为暗水天网 |
| **九地** | 深藏地下，绝对隐蔽 |

### 人事取象
- **埋藏物品**：将东西埋藏隐藏
- **伏藏信息**：隐藏信息不为人知
- **隐身**：让人找不到自己
- **保密**：极度保密

### 与其他五假的区别

| 格局 | 门 | 核心 |
|:---|:---|:---|
| **天假** | 景门 | 向上——见贵求官 |
| **地假** | 杜门 | 向下——躲藏逃亡 |
| **人假** | 惊门 | 向外——抓捕逃亡 |
| **神假** | 伤门 | 向内——埋藏隐藏 |
| **鬼假** | 死门 | 向阴——丧葬祭祀 |

## 应用宜忌

### 宜事
- **埋藏物品**：埋藏贵重物品
- **伏藏**：隐藏信息、保密
- **隐身**：让人找不到
- **极度保密**：确保不被人发现

### 忌事
- 公开透明之事
- 张扬炫耀
- 需要曝光的活动

### 注意事项
1. 神假是五假中专门用于"埋藏"的格局
2. 三者缺一不可
3. 现代社会：极高级别的保密需求
4. 伤门本身不吉，但在神假中专用于"改变状态让其消失"
